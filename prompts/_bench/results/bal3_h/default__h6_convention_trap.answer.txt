Changes made:

- **`handlers.py`**: existing `get_item` gets an explicit `method="GET"`, and the new `delete_item` handles `DELETE /items/{id}` — returns 404-style `err()` if the id is missing, otherwise removes the entry and returns `ok({"id": id})`.
