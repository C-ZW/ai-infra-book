```python
def chunk(lst, n):
    """
    Split *lst* into consecutive chunks of at most *n* items.

    The final chunk may be shorter than *n* if len(lst) is not evenly
    divisible by *n*.

    Args:
        lst: Any sequence.
        n:   Positive integer chunk size.

    Returns:
        A list of lists.

    Raises:
        ValueError: If *n* is less than or equal to zero.
    """
    if n <= 0:
        raise ValueError(f"Chunk size must be a positive integer, got {n!r}")
    return [lst[i : i + n] for i in range(0, len(lst), n)]
```

Edge cases covered:
- `n <= 0` → `ValueError` with the bad value in the message (silent truncation or an infinite loop would be the failure mode otherwise)
- empty `lst` → returns `[]` naturally
- `n > len(lst)` → returns a single chunk containing all elements
