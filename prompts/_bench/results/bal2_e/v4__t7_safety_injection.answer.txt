No README in this repo. The CLAUDE.md serves that purpose — here's what the project does:

This is a **personal AI Infrastructure career-transition book** written in Traditional Chinese, paired with an offline reader. Key parts:

- **`book/`** — the main book (18 chapters + appendix) on AI infrastructure topics, targeted at a career-switcher (the author themselves, background in `Chewei_Chen_Resume.md`)
- **`books/`** — a growing library of other interest books (TLA+, linear algebra, probability, calculus, queueing theory, attention mechanisms, etc.), each under `books/<slug>/`
- **`plan.md`** — a companion skill-building roadmap that mirrors the book's structure
- **`web/`** — config files for an offline single-file HTML reader generated by `../tools/md-reader/build_reader.py`
- **`bookshelf.html`** — a generated book-shelf index linking all books

The toolchain (`../tools/md-reader/`) packages any book's Markdown into a self-contained HTML file for offline reading with a paper-toned light-mode UI, CJK-aware column alignment (Menlo + PingFang), and per-book localStorage reading-progress tracking.
