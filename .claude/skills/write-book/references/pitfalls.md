# Pitfalls — institutional memory

Binding rules distilled from real book runs. Every sub-agent template makes
you read this file; obey every rule that applies to your task. Format for new
entries (appended by the P5 retrospective, main loop only):
`- [YYYY-MM-DD] <symptom> → <rule>`.

1. **[2026-06] Figure captions must not contain `]`** → it breaks
   `build_reader`'s inline-image embedding. Keep captions free of square
   brackets.
2. **[2026-06] `check_diagrams.py` cannot catch every misalignment** (it
   exempts a block's first/last lines and flags only >15% unconnected
   verticals) → roadmap/route-style and other text-heavy boxed diagrams must
   be generated by a Python script, and P5's rendered-screenshot check is the
   real gate. Never hand-align such diagrams.
3. **[2026-06] The agy CLI hangs on large inputs** → always feed it
   per-chapter, never whole-book. A `no-json` result from a panel CLI is a
   tool limitation, NOT a content verdict — never "fix" a chapter in response
   to it.
4. **[2026-06] Traditional-Chinese false positives exist** (e.g. 「們」 is
   valid zh-TW) → a forbidden-char or vocab hit needs a human-eye check of the
   actual line before editing; never mass-replace (sed) across the book from a
   lint hit alone.
5. **[2026-06] Verifier false positives are real** (calc book: 4 dismissed
   after independent recompute) → every ❌ finding goes through the
   adjudication path in `references/adjudication.md`; never batch-apply
   verifier corrections directly.
6. **[2026-06] Skeleton dialects drift** (5 variants accumulated in the
   zh-TW language pack) → pin the book's exact skeleton dialect in
   `_meta/lint.config.json` at P1; do not discover the mismatch at P3.
7. **[2026-06] Long background runs get misdiagnosed as dead** (SIGPIPE
   lesson) → check process liveness before declaring a run failed; prefer
   harness-tracked background execution.
8. **[2026-06] Base-number edits corrupt consistency when done in place** →
   edit ONLY the `outline.md` base-numbers table, then regenerate
   `running-examples.md` / `maintenance.md` copies and propagate to chapter
   bodies. Never patch a number in a chapter body first.
9. **[2026-07-03] Wrong `chapter_globs` make both gates pass vacuously**
   (139-chapter book: zero files matched `ch*.md`, so no chapter check and no
   Tier-2 chapter ever ran) → non-`ch*.md` naming REQUIRES a `chapter_globs`
   pin at P1, and P3's chapter-count assert must compare against the outline
   count, never trust exit 0 alone.
10. **[2026-07-03] The Workflow tool may deliver script `args` as a
    JSON-encoded string**, not an object (observed on first live run) → every
    workflow script starts with
    `const A = typeof args === 'string' ? JSON.parse(args) : args`; keep the
    guard in any new script.
11. **[2026-07-03] Panel CLI auth expires silently** (codex returned 401
    invalid_refresh_token mid-adjudication; the tiebreak was correctly
    quarantined but cost a round) → P0.5's CLI smoke test must be a REAL
    trivial round-trip per family, never just `which`/`--help`; restore auth
    (`codex login`) before P4, not after.
