# 如何不騙自己

### 科學方法的直覺與驚嘆

> 科學不是「證明真理」，而是有系統地把自己被騙的機會壓到最低。這本書想讓你親身感受這一刀——並且能用自己的話，對另一個工程師講清楚：**一個結論要滿足什麼，才值得相信；以及我此刻最可能在哪裡騙了自己。**

---

這是一本寫給資深後端工程師的科學方法「欣賞」書。純興趣、和工作無關、純紙上推演——它不教你跑統計軟體、不發 lab 作業、不要你背公式。它只想帶你看懂一件反直覺的事：**科學不是一張流程圖，而是人類花了三百年、一個漏洞一個漏洞地去堵自己騙自己的管道。**

你已經在工作裡碰過這套方法的影子，只是沒這樣叫它：凌晨三點你去拉兩個時段的錯誤率對照、而不是相信「我覺得是 DB」，這就是科學的起手式（系統性比較）；你做 A/B test 讓流量隨機分流、而不是靠感覺決定版本好壞，這就是對照實驗的核心（堵分組偏差）；你的 alert 一響你第一個想的是「是 false alarm 嗎」，這就是貝氏直覺（先驗低時一次觀察不夠更新信念）；你 root cause 的時候從一堆症狀跳到「最能解釋這組線索的根因」，這不是歸納也不是演繹，是溯因推理。這本書把這些直覺接上一個更大的故事。

## 這本書怎麼讀

- **純紙上推演**：每章的「紙上推演」是口頭自答、手算示範、論證拆解，不是要你開終端機。所有數值示範一律手算、攤在你眼前的 ` ```text ` 表裡。
- **直覺與精確意義優先**：每個概念先回答「它在堵哪一種自欺」（可證偽在防「怎樣都對的假理論」、隨機化在防「分組偏差」、雙盲在防「期望污染量測」、預註冊在防「事後挑分析」），再給形狀。
- **講得出來才算過**：每章末的「自我檢核」是口頭題——對齊「能不能把這個概念講給另一個工程師聽」的深度。優先問「為什麼」而不是「是什麼」。
- **陷阱視角是賣點**：科學方法正是「想當然耳」最常出錯的領域。每章的「直覺的陷阱」段，專門守住那些被講爛了的庸俗版（「科學方法就是五步驟」「p<0.05 代表 95% 為真」「觀察就是客觀事實」「塞麥爾維斯讓死亡率從 18% 降到 1%——那是用單月峰值對全年、誇大效果的假誠實」），還你正確版。

## 一條脊椎貫穿全書

整本書反覆回到**同一個 1847 年的洗手案**（史實底座釘在 `_meta/landscape-2026-06.md`）：

```text
脊椎：塞麥爾維斯洗手案的核心數字（每章從不同透鏡重看它）

  地點／背景  維也納總醫院，1846–1848
  兩病房對照  第一病房（醫師）98.4‰ vs 第二病房（助產士）36.2‰，1840–46 平均
              （同院同期、單雙日交替分配——最乾淨的對照）
  介入前後    1846 全年約 11.4% → 1848 全年約 1.27%（全年 vs 全年、百分比版）
  轉捩點      1847 年 5 月：Kolletschka 之死 → 溯因出「屍體微粒」假說 → 氯化石灰洗手
  缺口        無細菌理論（Pasteur/Koch 1860s–80s 才立）；有效果、無機制
  結局        被否決二十年；1865 年死於敗血症；細菌說後平反
```

這個案子之所以是完美脊椎——它**整套科學方法的每一環都齊全又都出問題**：扎實的觀察與對照、漂亮的介入效果、卻機制未明、又被社群否決、主角憂鬱而終、最後平反。它同時是「科學如何防自欺」的正面教材與「科學社群如何失靈」的反面教材。Hempel 1966 年在《Philosophy of Natural Science》裡選它當科學方法課本範例，絕非偶然。

每深入一章，就用一個方法論透鏡重看它一次——ch18 把十個核心透鏡攤成一張對帳表，那就是全書的終點口試。

## 書架上的鄰居

這本書只用自己的聲音講「科學的本質是有系統的防自欺」；需要更深的計算或心理機制時，它會把你指向書架上相鄰的書：

- 機率計算太在意——貝氏定理怎麼算、p 值的統計機制、相關係數 ρ、大數法則 →《馴服隨機》（本書只給科哲意義，「怎麼算」一律指向它）
- 「我如何理性地下注、更新信念」的規範面——信念更新作為理性義務、期望效用、賭注大小 →《決策與理性》
- 個人為什麼對證據偏心——確認偏誤、動機性推理的心理機制、為何 p-hacking 的人不自知 →《思考的陷阱》（本書講「科學制度怎麼防」，它講「大腦為什麼這樣」）
- 數學的真 vs 科學的真——形式系統裡「真」如何溢出「可證」的那道分裂 →《這句話無法被證明》（ch10 做這個對照，把形式系統那一邊指向它）

本書屬「推理六書」家族，上述姊妹書一律以主題方式指向（不寫死章號），因為彼此都是活的書。

## 給誰讀

唯一讀者：一位六年以上的資深後端工程師。你不需要任何科學哲學背景——你需要的直覺，A/B test、canary 放量、post-mortem、相關性故障、alert fatigue，早就教會你了。本書把這些工程經驗當成錨點，從「你去拉兩個時段的指標對照」起步，一路帶你看到「一整套科學制度如何用三百年累積的防線，系統性地把人被自己騙的機會壓到最低——以及它到底在哪裡也會失靈」。

準備好了，就從 **ch01 — 兩個病房的死亡率：科學的起手式** 開始。你的直覺，正準備在接下來每一章被它自己騙一次——這正是值得你慢慢讀的地方。
