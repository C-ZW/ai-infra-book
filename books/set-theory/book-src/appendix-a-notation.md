# 附錄 A — 記號與讀法速查表

這本書的驗收靠**口述**——你能不能把一個證明講給另一個工程師聽、讓他自己重走一遍，才算真的懂。所以每個記號你都得**唸得出來**：看到 ℵ₀ 嘴巴要能吐出「aleph-null」，看到 ↔ 要能說「一一對應於」，而不是只在腦袋裡認得那個形狀。這張表就是給口述用的——把全書十九章**真正出現過**的記號收齊，每個給「怎麼唸（中文口語）」「白話意義」「在哪一章第一次或主要登場」。沒在書裡出現的記號（例如 ℂ、序型同構符號）不收，免得你背了用不到的東西。

記號一律純文字（閱讀器不渲染 LaTeX）；唸法欄裡的英文是你跟人講的時候真的會說出口的那個音。按用途分五組：基礎集合與映射、基數、序數、公理與宇宙、邏輯與計算。

## 基礎集合與映射

| 記號 | 怎麼唸 | 白話意義 | 出現章 |
|---|---|---|---|
| `∈` | 「屬於」 | x ∈ A：x 是集合 A 的元素 | ch01 |
| `∉` | 「不屬於」 | x ∉ A：x 不是 A 的元素（羅素悖論 x∉x 的關鍵） | ch04 |
| `⊆` | 「包含於／是…的子集」 | A ⊆ B：A 的每個元素都在 B 裡（可相等） | ch04 |
| `⊊` | 「真包含於／是…的真子集」 | A ⊊ B：A 是 B 的子集但不等於 B；本書招牌教訓——真子集**不**蘊含基數較小（ℕ ⊊ ℚ 但 \|ℕ\|=\|ℚ\|） | ch03 |
| `∪` | 「聯集／並」 | A ∪ B：A 和 B 合起來的所有元素 | ch02 |
| `∩` | 「交集」 | A ∩ B：同時屬於 A 和 B 的元素 | ch08 |
| `∅` | 「空集」 | 沒有任何元素的集合；累積階層的第 0 層 V₀=∅（見 ch11） | ch06 |
| `ℕ` | 「自然數集」 | {1,2,3,…}（點名/基數脈絡）；序數脈絡（ch08 起）改 von Neumann 約定 ω={0,1,2,…}，**同一個集合、\|ℕ\|=ℵ₀ 不變**，只是起點標法不同 | ch01 |
| `ℤ` | 「整數集」 | {…,−2,−1,0,1,2,…}；用交錯列舉證 ℤ↔ℕ、可數 | ch02 |
| `ℚ` | 「有理數集」 | 所有分數 p/q；稠密卻仍可數（對角蛇行列舉，見 ch03） | ch01 |
| `ℝ` | 「實數集」 | 整條數線；對角論證證它**不可數**（見 ch04） | ch01 |
| `f: A → B` | 「f 從 A 映到 B」 | 一個從 A 到 B 的函數（映射）；`→` 唸「映到」 | ch01 |
| `↦` | 「映到／對應到」 | 指明函數對單一元素怎麼送，如 x ↦ x/(1−\|x\|)（給出對應規則本身） | ch04 |
| `↔` | 「一一對應於」 | A ↔ B：A 與 B 之間**存在一一對應**（bijection），即本書「一樣多」的唯一判準 | ch01 |
| `↪` | 「單射嵌入到」 | A ↪ B：存在從 A 到 B 的單射（injection，不重地塞進去）；對應 \|A\|≤\|B\| | ch06 |
| `A × B` | 「A 叉乘 B／笛卡兒積」 | 所有有序對 (a,b) 的集合；ℕ×ℕ↔ℕ 是「可數×可數=可數」的核心 | ch02 |
| `π(a,b)` | 「pi of a, b／配對函數」 | 康托爾配對函數 π(a,b)=(a+b)(a+b+1)/2+b，把一對 (a,b) 編成單一自然數（工程橋：composite key 編碼） | ch02 |
| `{x ∈ A : P(x)}` | 「所有 A 裡滿足 P 的 x 所成的集合」 | 集合建構式（分離公理的安全版：只在既有集合 A 裡篩，見 ch11）；冒號唸「使得」 | ch04 |
| `{0,1}^ℕ` | 「ℕ 到 0、1 的無窮二元序列集」 | 所有 ℕ→{0,1} 的無窮 0/1 序列；對角論證的標準靶子，與 𝒫(ℕ)、(0,1) 等勢 | ch04 |
| `~` | 「等價於」 | 等價關係，如 x ~ y ⟺ x−y∈ℚ（Vitali 集分等價類，見 ch13） | ch13 |
| `Aᶜ` | 「A 的補集」 | 不在 A 裡的那些元素（ch13 球面分解用到） | ch13 |

## 基數（多少）

| 記號 | 怎麼唸 | 白話意義 | 出現章 |
|---|---|---|---|
| `\|A\|` | 「A 的基數／A 的大小」 | 集合 A 的「多少」；\|A\|=\|B\| 定義為 A↔B（見 ch01） | ch01 |
| `\|A\| = \|B\|` | 「A 的基數等於 B 的基數」 | 兩集合一樣多（存在一一對應） | ch01 |
| `\|A\| ≤ \|B\|` | 「A 的基數小於等於 B 的基數」 | A 能單射進 B（A 不比 B 大）；兩向 ≤ 夾出相等＝CSB（見 ch07） | ch07 |
| `\|A\| < \|B\|` | 「A 的基數嚴格小於 B 的基數」 | A 嚴格比 B 小（冪集定理：\|A\|<\|𝒫(A)\|，見 ch06） | ch06 |
| `ℵ₀` | 「aleph-null／aleph-naught／阿列夫零」 | 第一個無限基數＝\|ℕ\|＝「可數無限」的大小 | ch02 |
| `𝔠` | 「continuum／fraktur c／連續統」 | 連續統的基數＝\|ℝ\|＝\|(0,1)\|＝\|{0,1}^ℕ\|＝\|𝒫(ℕ)\|＝2^ℵ₀，全書同一個基數的五種寫法；𝔠>ℵ₀ | ch04 |
| `ℵ₁` | 「aleph-one／阿列夫一」 | ℵ₀ **之後緊接的下一個**基數（中間沒有別的基數，良序保證） | ch06 |
| `2^ℵ₀` | 「2 的 aleph-null 次方」 | 𝒫(ℕ) 的基數＝𝔠；「取冪才會跳上一階」的具體運算 | ch04 |
| `2^𝔠` | 「2 的 continuum 次方」 | 𝒫(ℝ) 的基數，比 𝔠 更高一階（階梯往上沒有頂） | ch06 |
| `ℶ₀` | 「beth-zero／beth 零」 | Beth 階梯第 0 階＝ℵ₀ | ch06 |
| `ℶ₁` | 「beth-one／beth 一」 | Beth 階梯第 1 階＝2^ℵ₀＝𝔠；CH 等價於 ℵ₁=ℶ₁ | ch06 |
| `ℶ₂` | 「beth-two／beth 二」 | Beth 階梯第 2 階＝2^ℶ₁＝2^𝔠；階梯往上 ℶ₃、ℶ₄… 依此類推、沒有頂 | ch06 |
| `𝒫(A)` | 「power set of A／A 的冪集」 | A 的所有子集所成的集合；恆有 \|A\|<\|𝒫(A)\| | ch04 |
| `CH：2^ℵ₀ = ℵ₁` | 「連續統假設」 | ℵ₀ 與 𝔠 之間**沒有別的基數**；獨立於 ZFC（見 ch15） | ch14 |
| `GCH：2^ℵ_α = ℵ_{α+1}` | 「廣義連續統假設」 | 對每個序數 α，取一次冪集恰好跳到下一級 ℵ_{α+1}；CH 是 α=0 那一格 | ch14 |

## 序數（第幾／順序型）

| 記號 | 怎麼唸 | 白話意義 | 出現章 |
|---|---|---|---|
| `ω` | 「omega／小寫 omega」 | 第一個無限序數＝所有自然數**之後**那個位置；序型 vs 基數的分水嶺 | ch02 |
| `ω+1` | 「omega 加 1」 | ω 後面再接一個位置，是比 ω 大的新序數（基數仍 ℵ₀） | ch08 |
| `ω·2` | 「omega 乘 2」 | ω+ω，兩段 ℕ 順序接起來；注意 1+ω=ω ≠ ω+1（序數加法不可交換） | ch08 |
| `ω²` | 「omega 平方」 | ω 段 ω 接起來（ω 個 ω）；可數序型，但比任何 ω·k 都大 | ch08 |
| `ω^ω` | 「omega 的 omega 次方」 | 序數取冪，仍是可數序型，往上爬得很快 | ch08 |
| `ε₀` | 「epsilon-nought」 | ω 的次方塔疊到極限：ε₀=ω^ε₀，α↦ω^α 的第一個不動點；仍可數，通向不完備（見 ch16） | ch08 |

## 公理與宇宙

| 記號 | 怎麼唸 | 白話意義 | 出現章 |
|---|---|---|---|
| `R = {x : x ∉ x}` | 「所有不屬於自己的集合所成的『集合』」 | 羅素悖論的禍首：問 R∈R？兩邊都矛盾——天真概括公理的自爆 | ch10 |
| `V₀` | 「V-zero／第 0 層」 | 累積階層的底層＝∅ | ch11 |
| `V_{α+1} = 𝒫(V_α)` | 「V α 加 1 等於 V α 的冪集」 | 累積階層的後繼層：上一層取冪集往上長一級（用超限遞迴定義，見 ch09） | ch11 |
| `V_λ` | 「V lambda／極限層」 | 極限序數 λ 處取下面所有層的聯集；宇宙一層層蓋上去，沒有「所有集合的集合」 | ch11 |
| `ZFC` | 「Z-F-C」 | Zermelo–Fraenkel 集合論加選擇公理；本書用的地基 | ch11 |
| `AC` | 「A-C／選擇公理」 | 任意一族非空集合有選擇函數；等價良序定理、佐恩引理；非構造性 | ch12 |
| `Con(ZFC)` | 「Con of ZFC／ZFC 是一致的」 | 「ZFC 不會推出矛盾」這個斷言；獨立性陳述 Con(ZFC) ⇒ Con(ZFC+CH) 用它 | ch15 |
| `L` | 「宇宙 L／可構成宇宙」 | 哥德爾的構造宇宙（只承認被明確定義出來的集合的最小宇宙）；在 L 裡 CH（甚至 GCH）成立 ⇒ ZFC 不能反駁 CH | ch15 |

## 邏輯與計算

| 記號 | 怎麼唸 | 白話意義 | 出現章 |
|---|---|---|---|
| `∀` | 「對所有／任意」 | 全稱量詞（對每一個…都成立） | ch06 |
| `∃` | 「存在」 | 存在量詞（至少有一個…）；構造性 vs 純存在的爭論核心（見 ch17） | ch09 |
| `¬` | 「非／否定」 | 邏輯否定，如 ¬CH（CH 不成立）；獨立性＝既證不出 CH 也證不出 ¬CH | ch12 |
| `⇒` | 「則／蘊含」 | 若…則…（推得出來）；本書推導骨架的主力 | ch01 |
| `⟺` | 「若且唯若／當且僅當」 | 雙向蘊含，兩邊等值；如「A 可數 ⟺ A↔ℕ 或有限」 | ch01 |
| `=` 與 `≠` | 「等於」與「不等於」 | 相等與相異；對角線「每位翻轉」造出與第 n 列第 n 位**≠**的新對象，是全書脊椎 | ch01 |
| `⌈φ⌉` | 「φ 的編碼／φ 的哥德爾數」 | 哥德爾編碼：把公式 φ（或證明）編成一個自然數，於是「可證」變成算術命題 | ch16 |

> **一句話收尾**：這張表裡你最該唸到滾瓜爛熟的是 `↔`（一一對應於）、`ℵ₀`（aleph-null）、`𝔠`（continuum）、`𝒫`（power set）、`2^ℵ₀`、`CH`——它們是六條驗收主線的關鍵字。其餘記號是配角，講證明時順手就會用到。如果某個記號你唸不出來，那通常代表那個概念你還沒真懂——回去翻它的「出現章」。
