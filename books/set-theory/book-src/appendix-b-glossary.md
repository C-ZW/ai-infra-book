# 附錄 B — 術語表

這份術語表只收錄本書正文真的 introduce 過的術語——每一條都在某一章第一次出現時給過「中文（English）」、或被當成核心概念正式定義過。定義一律寫成一句白話、可以直接口頭講出來的話，刻意對齊該章正文的講法；若你對某條的精確邊界有疑慮，以該章正文為準。每條最後標出**首次出現章**（跨章用 chNN）。

排序按主題分成六組，順序大致跟著全書的敘事弧走：先把「一樣多」與「比較大小」講清楚，再分出可數與不可數，把無限做成數（基數與序數），補上地基（公理與選擇），走到判定的邊界（獨立與不完備），最後落回哲學與計算。同一組內大致按概念出場與相依順序排（先給判準、再給工具）。

讀法（怎麼唸出來）以附錄 A 為準；這裡只給意義。

## 一、比較大小：把「一樣多」操作化

| English | 中文 | 一句話定義（白話、可口頭講） | 首次出現章 |
|---|---|---|---|
| bijection | 一一對應（亦譯雙射） | 一個同時不重（單射）又不漏（滿射）的對應；兩個集合之間配得出一個一一對應，就叫「一樣多」，寫成 `A ↔ B`。 | ch01 |
| injection | 單射 | 一個「不重」的對應：A 裡不同的元素配到 B 裡不同的位置、彼此不撞（工程版：hash 無碰撞）；存在 A 到 B 的單射就寫 `\|A\| ≤ \|B\|`。 | ch01 |
| surjection | 滿射 | 一個「不漏」的對應：B 裡每個元素都有人配到它（工程版：hash 填滿值域）；單射加滿射就是一一對應。 | ch01 |
| cardinality／cardinal | 基數（亦稱勢） | 一個集合「有多少個元素」這個量，寫 `\|A\|`；兩個集合基數相等的意思就是它們之間配得出一一對應，跟數字無關。 | ch01 |
| Cantor–Schröder–Bernstein (CSB) | 康托爾–施洛德–伯恩斯坦定理 | 只要 A 塞得進 B、B 也塞得進 A（兩個方向各一個單射），就保證 `\|A\|=\|B\|`——省去硬造一一對應的力氣。 | ch07 |

## 二、可數與不可數：第一道裂縫

| English | 中文 | 一句話定義（白話、可口頭講） | 首次出現章 |
|---|---|---|---|
| countable | 可數 | 一個集合若有限、或能跟 ℕ 一一對應，就叫可數；白話就是「能排成一列 a₁,a₂,a₃,…、每個元素都在某個有限位置出現」，等於替它寫得出 generator。 | ch02 |
| enumerable | 可列舉 | 一個集合的元素能被一個 generator「最終每一個都 yield 得出來」（任一元素 yield 夠多次總會輪到）；本書證明它和「能跟 ℕ 一一對應」「可數」是同一件事的三種說法。 | ch02 |
| aleph-null (ℵ₀) | 阿列夫零 | ℕ 的基數，第一個無限基數；一個無限集合若可數，它就有 ℵ₀ 個元素。 | ch02 |
| Hilbert's Hotel | 希爾伯特旅館 | 一家有可數無限多間房（房號 ℕ）的旅館，用來示範可數無限的怪算術：客滿了還能用「room n → n+1」塞進一人、用「room n → 2n」塞進可數無限多人，房間始終夠用。 | ch02 |
| Cantor pairing function | 康托爾配對函數 | 一條把一對自然數 (a,b) 直接算成單一編號的閉式公式，證明 ℕ×ℕ 可數（工程橋：把複合鍵壓成一個 ID）。 | ch03 |
| countable choice | 可數選擇公理 | 選擇公理的「低劑量版」：只在可數多個非空集合裡各挑一個；本書證「可數個可數集的聯集可數」時用到，誠實標一下但不展開（完整版見 ch12）。 | ch03 |
| uncountable | 不可數 | 一個無限集合大到無法排成一列、跟 ℕ 配不起來，就叫不可數；ℝ 是第一個被證明不可數的集合，比 ℵ₀ 嚴格大。 | ch04 |
| Cantor's diagonal argument | 對角論證 | 把假設的「完整列表」寫成一張表、取對角線、每一位都翻轉，造出一個不在表中的新對象，逼出矛盾——證明 ℝ（與 {0,1}^ℕ）不可數。全書脊椎的那條對角線，ch04 定版。 | ch04 |
| continuum (𝔠) | 連續統 | ℝ 的基數，寫 𝔠；它等於 `\|(0,1)\| = \|{0,1}^ℕ\| = \|𝒫(ℕ)\| = 2^ℵ₀`，是第一個比 ℵ₀ 大的無限。 | ch04 |
| halting problem | 停機問題 | 沒有一個程式能判定任意程式會不會停；它的證明跟對角論證是同一招（造一個專門跟判定器作對的對角程式），是不可數性的計算版。 | ch04 |
| order type | 序型 | 一個良序集「排出來的形狀」；基數只看個數，序型看的是排列的順序型態，同一個 ℵ₀ 可以有很多不同序型。 | ch08 |

## 三、把無限做成數：基數、序數、冪集

| English | 中文 | 一句話定義（白話、可口頭講） | 首次出現章 |
|---|---|---|---|
| power set | 冪集 | 一個集合 A 的所有子集構成的集合，寫 `𝒫(A)`；有限時 n 個元素有 2ⁿ 個子集，無限時照樣是 `2^\|A\|`。 | ch06 |
| Cantor's theorem | 康托爾定理（冪集定理） | 對任意集合 A，冪集嚴格更大：`\|A\| < \|𝒫(A)\|`；推論是無限的階梯一級級往上、沒有最大的無限。它是對角論證的一般化（本書硬證之一）。 | ch06 |
| Cantor's paradox | 康托爾悖論 | 「所有集合的集合」不存在：若有全集 U，它的冪集基數會比它還大，矛盾；康托爾約 1899 年就自己撞上了這面牆（比羅素早兩年）。 | ch06 |
| ordinal | 序數 | 用來標記良序集「序型」的數，回答「第幾、什麼順序」；0,1,2,…→ ω → ω+1 → ω·2 → ω² → … 一路往上。 | ch08 |
| successor ordinal | 後繼序數 | 有「直接前驅」的序數，例如 ω+1（它前面緊鄰 ω）；可以由前一個序數加一得到。 | ch08 |
| limit ordinal | 極限序數 | 沒有直接前驅的序數，例如 ω、ω·2、ω²；它是前面所有序數累積到的極限位置，找不到「ω−1」這種東西。 | ch08 |
| well-order | 良序 | 一個全序，而且每個非空子集都有最小元；等價於「沒有無限下降鏈」，這正是能做歸納、保證遞迴會停的關鍵性質。 | ch08 |
| transfinite induction | 超限歸納 | 把數學歸納法推到所有序數的版本：要證性質對每個序數都成立，得多走一個「極限步驟」（所有比 λ 小的都成立 ⇒ λ 也成立）。 | ch09 |
| transfinite recursion | 超限遞迴 | 超限歸納的「建造版」：用同樣的基底／後繼／極限三步去**定義**東西（ℵ 階層、V 階層、序數算術全靠它）。 | ch09 |
| well-ordering theorem | 良序定理 | 每個集合都能被良序排列——包括 ℝ；這個驚人結論等價於選擇公理。 | ch09 |

## 四、地基的危機：悖論、公理、選擇

| English | 中文 | 一句話定義（白話、可口頭講） | 首次出現章 |
|---|---|---|---|
| comprehension | 概括公理 | 天真集合論的信條：任何性質 P 都決定一個集合 `{x : P(x)}`，不限定 x 從哪來；正是這條太慷慨的規則種下了羅素悖論。 | ch10 |
| Russell's paradox | 羅素悖論 | 取 R = {x : x ∉ x}，問「R 屬於 R 嗎」——答是也矛盾、答否也矛盾，兩邊都死；它證明天真概括公理本身有毒，逼出了公理化。 | ch10 |
| Burali-Forti paradox | 布拉利-福爾蒂悖論 | 「所有序數的集合」會有一個比自己每個成員都大的序數、於是比自己還大，矛盾；跟羅素、康托爾悖論同屬「太大的集合」家族。 | ch10 |
| ZFC | ZFC 公理系統 | 現代集合論的標準地基（策梅洛–弗蘭克爾公理加選擇公理）；策略是「只能從已有集合、用受限的方式造新集合」，把悖論擋在門外。 | ch10 |
| separation (axiom) | 分離公理 | 只准在一個**已有的**集合 A 裡篩：`{x ∈ A : P(x)}`；這一刀讓羅素的 R 從「合法但矛盾」變成「根本造不出來」。 | ch11 |
| Axiom of Infinity | 無限公理 | 保證存在一個無限集合（含 ∅、且對每個 x 都含 x∪{x}）；沒有它，ℕ 都生不出來，整套無限免談。 | ch11 |
| cumulative hierarchy (V) | 累積階層 | ZFC 的宇宙觀：集合一層層蓋上去，V₀=∅、V_{α+1}=𝒫(V_α)、極限層取聯集；每個集合都在某一層出生，「所有集合的集合」永遠在下一層之外。 | ch11 |
| axiom of choice (AC) | 選擇公理 | 給一族非空集合，保證存在一個選擇函數從每個集合各挑一個；有限時顯然，無限且沒有挑選規則時才需要當公理。 | ch12 |
| choice function | 選擇函數 | 從一族非空集合的每個集合裡各挑出一個元素的那個函數；AC 保證它**存在**，卻不告訴你**怎麼挑**。 | ch12 |
| Zorn's lemma | 佐恩引理 | 一個偏序集若每條鏈都有上界，就必有極大元；它和選擇公理、良序定理是同一個東西的三張臉（代數裡天天用）。 | ch12 |
| non-constructive | 非構造性的 | 一個只證明「東西存在」、卻給不出怎麼造它的論證；AC 是最典型的例子，這正是它最折磨工程師的地方。 | ch12 |
| Banach–Tarski paradox | 巴拿赫–塔斯基悖論 | 用 AC 可以把一顆實心球分成有限多塊、只靠旋轉平移重組成兩顆一樣大的球；它真正揭露的不是球的事，是「不是每個點集都能有體積」。 | ch13 |
| non-measurable set | 不可測集 | 一個沒辦法被賦予一致「體積／長度」的點集；巴拿赫–塔斯基那幾塊就是這種東西，倍增的是點的安排、不是體積。 | ch13 |
| measure theory | 測度論 | 研究怎麼給集合一致地賦予「長度／面積／體積／機率」的理論；它必須把測度限制在 σ-代數上，正是為了躲開不可測集（系統建立指向《馴服隨機》）。 | ch13 |
| Vitali set | 維塔利集 | 用 AC 從 [0,1] 上「相差有理數」的每個等價類各挑一個代表元造出的集合，可被證明不可測；是巴拿赫–塔斯基的可懂前身。 | ch13 |

## 五、判定的邊界：獨立與不完備

| English | 中文 | 一句話定義（白話、可口頭講） | 首次出現章 |
|---|---|---|---|
| continuum hypothesis (CH) | 連續統假設 | 問 `2^ℵ₀ = ℵ₁` 嗎——ℵ₀ 和 𝔠 之間到底有沒有夾著別的無限；康托爾終生想證、希爾伯特列為 20 世紀第一問題。 | ch14 |
| undefined behavior | 未定義行為 | 工程師的話：規格沒約束到的地方，怎麼填都不算違規；本書用它類比「ZFC 對 CH 兩邊都沒規定」這件事。 | ch15 |
| independence | 獨立性 | 一個命題 P 獨立於某公理系統，意思是系統既證不出 P、也證不出 ¬P；不是「還沒證出來」，是「被證明了證不出來」，兩個方向都相容。 | ch15 |
| consistent / consistency | 一致／一致性 | 一個公理系統推不出矛盾（不會同時證出 P 和 ¬P）；不一致的系統會爆炸——能推出任何命題，整套變廢墟。 | ch15 |
| relative consistency | 相對一致性 | 「如果原系統一致，那麼加上某命題後也一致」這種形式的陳述（Con(ZFC) ⇒ Con(ZFC+P)）；獨立性就是靠兩個方向的相對一致性夾出來的。 | ch15 |
| constructible universe (L) | 可構成宇宙 | 哥德爾造的最小宇宙，只承認「被明確定義出來」的集合；在 L 裡 CH（甚至 GCH）成立，所以 ZFC 反駁不了 CH。 | ch15 |
| forcing | 力迫法 | 科恩的技術：從一個模型「外加」新的實數撐大連續統、造出 ¬CH 成立的模型，所以 ZFC 也證不出 CH（本書只給直覺 taste）。 | ch15 |
| large cardinal axioms | 大基數公理 | 假設存在大到 ZFC 推不出其存在的基數的新公理；多數活躍集合論家指望靠這類新公理替 CH 定真假（本書只 taste 不展開）。 | ch15 |
| Gödel numbering | 哥德爾編碼 | 把每個公式、每段證明都編成一個自然數，於是「某命題可證」變成一個關於數的算術命題（工程橋：把程式碼當資料、序列化）。 | ch16 |
| incompleteness theorem | 不完備定理 | 哥德爾 1931：任何夠強、一致、可遞迴公理化的系統，都存在一個為真卻證不出的命題 G（第一定理），且這種系統證不出自己一致（第二定理）。 | ch16 |
| diagonal lemma | 對角化引理（不動點引理） | 保證任何「只有一個自由變數的算術公式」都能機械地造出一個自我指涉的句子；哥德爾用它讓 G 說出「我不可證」。 | ch16 |

## 六、哲學與計算：無限到底存不存在

| English | 中文 | 一句話定義（白話、可口頭講） | 首次出現章 |
|---|---|---|---|
| potential infinity | 潛無限 | 亞里斯多德版的無限：「永遠可以再加一」的可能性，但「全部加完」這個完成狀態不存在。 | ch17 |
| actual infinity | 實無限 | 康托爾版的無限：一個已經完成、可以整個拿起來當對象的東西（ℵ₀ 就是一個數）；本書前十六章全建立在它上面。 | ch17 |
| logicism | 邏輯主義 | 弗雷格、羅素那一派：要把整個數學化約到邏輯；被羅素悖論重創。 | ch17 |
| formalism | 形式主義 | 希爾伯特那一派：別管無限對象真不真實，數學是只要一致就合法的符號遊戲；被哥德爾第二定理重創。 | ch17 |
| intuitionism／constructivism | 直覺主義／構造主義 | 布勞威爾那一派：只承認能被造出來的對象、拒絕把排中律用在無限上、拒絕非構造的存在證明；跟「只承認可計算的東西」最親，最對工程師胃口。 | ch17 |
