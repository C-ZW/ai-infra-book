# ch06 — 冪集定理：沒有最大的無限

> **本章解決什麼問題**：ch04 證明了 ℝ 不可數，於是有「比 ℵ₀ 大的無限」；ch05 把那個新基數 𝔠 摸熟了，但也留下一個沒回答的問題——𝔠 是不是最大的無限？這章給出一個比「不是」更狠的答案：**沒有最大的無限，永遠沒有**。任何集合 A，它的冪集 𝒫(A) 一定嚴格更大，所以無限的階梯一級一級往上，沒有頂。證明它的招數，正是 ch04 對角線的一般化——同一條對角線第二次出場。順帶還會炸掉一個你可能寫過的天真念頭：「把所有東西收進一個集合」。那次爆炸，是 ch10 羅素悖論的引線。

## 從你已知的出發

你寫過 feature flag。三個布林開關，配置空間是 2³ = 8 種組態；十個開關，2¹⁰ = 1024 種。每多一個開關，可能的世界翻一倍——「設定項」加一，「設定的所有可能組合」乘二。這是你對冪集最樸素的直覺，雖然你大概沒這樣叫它：n 個元素的集合，它的子集有 2ⁿ 個。

換成位元你更熟。給一個固定的元素清單，任何一個子集就是一張勾選表：每個元素勾或不勾，一個 bit。三個元素的子集對應一個 3-bit 的 bitmask，剛好 2³ = 8 個 mask，從 `000` 到 `111`。**子集 ↔ 特徵位元串**，這個對應不重不漏，你在做權限旗標、做集合的點陣表示時用過無數次。

有限世界裡，「組合永遠比項多一個數量級」是廢話——2ⁿ 把 n 甩在後面，這你不需要人教。本章要問的是同一句話搬到無限會怎樣：ℕ 有 ℵ₀ 個元素，那 ℕ 的所有子集——𝒫(ℕ)——有幾個？「2^ℵ₀ 個」，但 2^ℵ₀ 是什麼？它還是 ℵ₀ 嗎（無限乘二還是自己，ch02 教過）？還是像有限情形那樣，真的更大？

答案是後者，而且不只 ℕ：**任何**集合，冪集都嚴格更大。這就是康托爾定理。它把你那個「組態比開關多」的有限直覺，提升成一條關於無限的鐵律——而且證明只用一頁紙，用的還是你 ch04 已經見過的那一招。

## 子集就是一張勾選表

先把「子集」這個東西換成工程師看得順眼的形狀，後面的證明才不會卡在直覺上。

給定集合 A。A 的一個子集 S，本質上是對 A 的每個元素回答一個是非題：「你在 S 裡嗎？」在＝1，不在＝0。把這些答案排成一串，就得到一個函數 χ_S: A → {0,1}（χ 念 chi，叫**特徵函數** characteristic function）：

```text
χ_S(x) = 1   ← 若 x ∈ S
χ_S(x) = 0   ← 若 x ∉ S
```

反過來，任何一個 A → {0,1} 的函數，都唯一決定一個子集（把回答 1 的元素收集起來）。所以：

```text
A 的子集  ↔  函數 A → {0,1}  ↔  {0,1}^A 的元素
  𝒫(A)              特徵函數          「A 上的 0/1 序列」
```

這個對應不重（不同子集給不同的勾選表）也不漏（每張勾選表都對應一個子集），是一個如假包換的一一對應。於是**𝒫(A) 和 {0,1}^A 是同一個東西的兩種寫法**，往後我會在兩者之間自由切換——談「子集」時想 𝒫(A)，談「對角線翻位元」時想 {0,1}^A。

具體摸一下 A = {a, b, c}。把 a, b, c 固定成第 1、2、3 位，每個子集就是一個 3 位元的勾選表：

```text
子集        特徵位元 (a b c)
∅           0 0 0
{a}         1 0 0
{b}         0 1 0
{c}         0 0 1
{a,b}       1 1 0
{a,c}       1 0 1
{b,c}       0 1 1
{a,b,c}     1 1 1
```

八個子集，八個 3-bit 串，從 000 數到 111。**2³ = 8 > 3**：冪集有 8 個元素，原集合只有 3 個。在有限世界，冪集更大是因為 2ⁿ > n，國中就會。問題是無限——ℕ 的勾選表是無限長的 0/1 串，那種「串」有多少個？「無限長的二元序列集」這個詞你在 ch04 已經盯著它看過一整章了。{0,1}^ℕ 不可數。所以 𝒫(ℕ) 不可數，比 ℕ 嚴格大。

但這只是 ℕ 一個特例。康托爾定理要的是**所有** A。

## 有限版熱身：手造一個對角集

在搬去無限之前，先在 {a, b, c} 上把證明的核心動作走一遍——這一步是本章最划算的十分鐘，看懂這裡，無限版只是把同一件事再做一次。

我們想證明的是：**不可能有一個 f: {a,b,c} → 𝒫({a,b,c}) 是滿射**。注意目標——不是說沒有一一對應（那靠數數 3 ≠ 8 就行），而是說連「漏不掉」的滿射都造不出來。為什麼挑滿射開刀，等一下無限版會看到原因：在無限世界你不能數數，但「漏掉一個」這件事可以直接抓現行。

隨便給一個 f 試試。比方說：

```text
f(a) = {a, b}
f(b) = {b, c}
f(c) = {a, b, c}
```

f 把三個元素各送到一個子集。我們要當場抓出一個「沒被任何人送到」的子集，證明 f 不滿。怎麼抓？用對角線。

逐個元素問一個是非題：「**x 在不在它自己被送到的那個子集 f(x) 裡？**」把不在的收集起來，組成一個新子集 D：

```text
D = { x ∈ {a,b,c} : x ∉ f(x) }

a 在 f(a)={a,b} 裡嗎？ 在  → a 不收進 D
b 在 f(b)={b,c} 裡嗎？ 在  → b 不收進 D
c 在 f(c)={a,b,c} 裡嗎？在  → c 不收進 D

D = ∅
```

現在問關鍵問題：**D 是誰的像？有沒有某個 x 使 f(x) = D？** 來查。f 的像只有 {a,b}、{b,c}、{a,b,c} 三個，D = ∅ 不在裡面——抓到了，D 沒被送到，f 不滿。

但「不在那三個裡面」是這個特定 f 的巧合嗎？不是。D 的造法保證它和**每一個** f(x) 都至少差一個元素，這才是對角線的威力。看 D 和 f(a) 的關係：a 進不進 D，是看「a 在不在 f(a)」決定的——而且是**反著來**：a 在 f(a) 裡，所以 a 故意不進 D。於是在「a 這一格」上，D 和 f(a) 必然相反：

```text
            a 這一格      所以
f(a)={a,b}  含 a          D 在 a 格 = 不含（故意反過來）
                          ⇒ D ≠ f(a)（至少 a 格不同）
```

同理 D 和 f(b) 在 b 格相反、和 f(c) 在 c 格相反。**D 被設計成跟每個 f(x) 都至少在「第 x 格」上唱反調**，所以 D 不可能等於任何一個 f(x)。f 不滿——而這跟我們挑的 f 是哪個無關，任何 f 都會被自己的對角線出賣。

把它擺成一張表，你會認出 ch04 的老朋友。把三個子集寫成特徵位元，對角線就是「x 格看 f(x)」那條斜線：

```text
        a   b   c
f(a)   [1]  1   0       ← f(a)={a,b} 的位元，方括號是對角格
f(b)    0  [1]  1       ← f(b)={b,c}
f(c)    1   1  [1]      ← f(c)={a,b,c}
       ───────────
對角線   1   1   1       ← 每個 f(x) 在第 x 格的值
翻 轉    0   0   0       ← D 的位元：每格取反
                         ⇒ D = ∅，與每一列至少差對角那格
```

這就是 ch04 的對角線表，**換了皮**。ch04 的列是「假設能排成一列的二元序列」，這裡的列是「f 把每個元素送到的子集（寫成特徵位元）」；ch04 沿對角線翻位元造出不在表中的序列，這裡沿對角線翻位元造出不是任何人像的子集 D。同一招。差別只在 ch04 預設列表由 ℕ 編號、這裡列表由集合 A 的元素編號——而這個差別，正是讓它能套到**任意** A 的關鍵。

## 康托爾定理：把同一招搬到無限

現在把熱身的動作原封不動搬到任意集合 A，一個字都不用改。這是本書硬證的少數定理之一，要寫到你能合上書自己重講——所以慢走，每一步都對得上剛才的有限版。

**康托爾定理**：對任何集合 A，|A| < |𝒫(A)|。

「<」要拆成兩半說清楚：(1) |A| ≤ |𝒫(A)|，A 不比冪集大；(2) |A| ≠ |𝒫(A)|，兩者不一樣大。合起來才是嚴格小於。第一半幾乎免費，第二半是全部的戲。

**第一半（≤，容易）**：造一個 A → 𝒫(A) 的單射就行。把每個元素 x 送到它自己的單元素集 {x}：

```text
g(x) = {x}     ← x ↦ {x}
```

不同的 x 給不同的 {x}（x ≠ y ⇒ {x} ≠ {y}），單射成立，所以 |A| ≤ |𝒫(A)|。直覺上：A 的元素都能塞進冪集裡（每個元素認領自己的單元素子集），所以冪集至少裝得下 A。

**第二半（≠，核心）**：要證沒有 A 到 𝒫(A) 的一一對應。其實證更強的——**沒有 A 到 𝒫(A) 的滿射**（連漏不掉都做不到，更別說又不重又不漏）。

用反證法。假設存在滿射 f: A → 𝒫(A)。仿照熱身，造對角集：

```text
D = { x ∈ A : x ∉ f(x) }
```

白話：D 收集所有「不屬於自己被送到的那個子集」的元素。逐個元素問「x 在不在 f(x) 裡」，不在的收進 D。這個 D 是 A 的一個子集，所以 D ∈ 𝒫(A)——它是 f 的值域裡的合法目標。

既然假設 f 滿射，D 一定是某個元素的像。設 D = f(d)，d ∈ A。現在問那個會殺死一切的問題：

**d 在不在 D 裡？**

兩條路，兩條都死。

```text
情況一：假設 d ∈ D
  D 的定義是「x ∉ f(x) 的那些 x」
  d ∈ D ⇒ d 滿足入會條件 ⇒ d ∉ f(d)
  但 f(d) = D，所以 d ∉ D
  ⇒ 從 d ∈ D 推出 d ∉ D，矛盾

情況二：假設 d ∉ D
  d ∉ D ⇒ d 不滿足入會條件 ⇒ 「d ∉ f(d)」為假 ⇒ d ∈ f(d)
  但 f(d) = D，所以 d ∈ D
  ⇒ 從 d ∉ D 推出 d ∈ D，矛盾
```

d 在 D 裡也矛盾，不在也矛盾，無處可逃。矛盾的源頭只有一個——我們假設了「D = f(d) 存在」，也就是假設了 f 滿射。所以 f 不可能滿射。

任何 f 都不滿 ⇒ 沒有 A 到 𝒫(A) 的滿射 ⇒ 更沒有一一對應 ⇒ |A| ≠ |𝒫(A)|。配上第一半的 ≤，得到 |A| < |𝒫(A)|。證畢。

停一下，認領這個證明的本質。**D 跟每個 f(x) 都至少在「第 x 格」上不同**——x 進不進 D，是拿「x 在不在 f(x)」反過來定的，所以 D 在第 x 格永遠和 f(x) 唱反調。這跟有限版一字不差，跟 ch04「s* 與第 n 列在第 n 位不同」也一字不差。三個地方，同一條對角線。差別只在這裡的「位」是用 A 的元素索引的，不是用 ℕ——這一鬆綁，證明就對**任意大小**的 A 都成立，包括那些大到沒有名字的無限。

這就是我認為本章最了不起的地方，值得你停下來品一分鐘：ch04 費一整章證明 ℝ 不可數，這裡用同一招、同一頁紙，一口氣證明了**每一個**集合都被自己的冪集超過。對角線不是只能對付 ℝ 的特技，它是一台通用的「造一個跟清單上每一項都不同的東西」的機器——你給它一份清單，它就吐回一個不在清單上的對象。記住這台機器的形狀，ch10 羅素悖論、ch16 哥德爾與停機問題，都是它換皮再來。

## 沒有最大的無限：階梯往上沒有頂

康托爾定理對「任何」A 都成立，這個「任何」是它真正的炸藥。把它接連套用，無限的階梯就一級一級長出來，而且永遠停不下來。

從 ℕ 起步。|ℕ| = ℵ₀。套康托爾定理：

```text
|ℕ|        = ℵ₀
|𝒫(ℕ)|     = 2^ℵ₀         > ℵ₀        ← 康托爾定理
|𝒫(𝒫(ℕ))|  = 2^(2^ℵ₀)     > 2^ℵ₀      ← 再套一次
|𝒫(𝒫(𝒫(ℕ)))| = 2^(2^(2^ℵ₀)) > 2^(2^ℵ₀)  ← 再套
   ⋮                          ⋮
```

每取一次冪集，基數嚴格變大。這條鏈沒有終點：給我任何一個無限基數 κ，2^κ 一定更大。所以——

**沒有最大的無限。** 不是「我們還沒找到最大的」，是「最大的不存在」，而且這是被**證明**的，跟 ch04「ℝ 不可數」同一個等級的硬結論。任何你指著說「這個最大了吧」的無限，它的冪集當場打臉。

這件事該讓你起一點雞皮疙瘩。ch04 第一次撕開「不只一種無限」時，你可能還想：好吧，那就兩種，ℵ₀ 和 𝔠，可數和不可數，二分天下。錯了。是無窮多種，排成一座沒有頂的塔，𝔠 = 2^ℵ₀ 只是第一級往上踏的那一階，它上面還有 2^𝔠、2^(2^𝔠)……無限本身有無限多個尺寸。

### 用 ℶ（beth）給這座塔編號

這座塔的台階有個專用記號，希伯來字母 ℶ（beth，唸 bet），這樣寫：

```text
ℶ₀ = ℵ₀          ← 第 0 階，可數
ℶ₁ = 2^ℶ₀ = 2^ℵ₀ = 𝔠    ← 第 1 階，連續統
ℶ₂ = 2^ℶ₁ = 2^𝔠         ← 第 2 階
ℶ₃ = 2^ℶ₂              ← 第 3 階
  ⋮
ℶ_{n+1} = 2^(ℶ_n)        ← 每階是前一階取冪集
```

ℶ 階梯就是「從 ℵ₀ 開始反覆取冪集」得到的基數序列，康托爾定理保證它嚴格遞增、永不到頂。一個歷史小修正，免得你跟坊間說法一起記錯：**ℶ 這個記號不是康托爾發明的**。康托爾只引入了 ℵ（aleph，用來標記良序集的基數，1895 年定型）；ℶ 符號是邏輯學家皮爾士（C. S. Peirce）1900 年 12 月寫信給康托爾時提議的，康托爾自己沒給冪集基數另立專名（2026-06，依 landscape）。很多書把 ℶ 一股腦歸給康托爾，是錯的。

ℵ 和 ℶ 是兩套不同的編號系統，這個區別在 ch14 連續統假設會變成主角，這裡先埋下：ℵ 數的是「第幾個基數」（ℵ₀ 之後是 ℵ₁、ℵ₂……，一個緊接一個沒有縫，靠良序保證，見 ch08/ch09），ℶ 數的是「取了幾次冪集」。兩套系統在 ℶ₀ = ℵ₀ 對齊起點，但之後就不一定對得上——**ℶ₁ = ℵ₁ 嗎？** 也就是「2^ℵ₀ 是不是 ℵ₀ 之後緊接的下一級」，這正是連續統假設（CH 等價於 ℶ₁ = ℵ₁），而它的答案會在 ch14/ch15 揭曉成一個你想不到的形狀。這裡不展開，只要你看清楚：取冪集這把梯子（ℶ）和基數排隊這把尺（ℵ）是兩回事，別混。

## 2^ℵ₀ = 𝔠：把幾個基數收成同一個

順手把 ch04、ch05 欠的帳收一下。前面散落著好幾個「不可數的 𝔠」，現在用特徵函數這把鑰匙把它們串成一個基數。

```text
𝒫(ℕ)      ↔  {0,1}^ℕ       ← 子集就是特徵序列（本章開頭那把鑰匙）
{0,1}^ℕ   ↔  (0,1) 區間    ← 二元序列就是 (0,1) 的二進位展開
(0,1)     ↔  ℝ            ← tan 類的明確一一對應（ch05）
```

於是 |𝒫(ℕ)| = |{0,1}^ℕ| = |(0,1)| = |ℝ| = 𝔠，而 |𝒫(ℕ)| 又寫成 2^ℵ₀（冪集就是「對 ℵ₀ 個元素各做一次二選一」）。串起來：

```text
2^ℵ₀ = |𝒫(ℕ)| = |{0,1}^ℕ| = |(0,1)| = |ℝ| = 𝔠
```

全書同一個基數，五種長相，從今以後我會混著用，看到哪個寫法都該認得是 𝔠。

但這串等號有一個地方我得對你誠實，不能含糊過去。中間那條 {0,1}^ℕ ↔ (0,1) 並**不是**乾乾淨淨的一一對應——卡在二進位的雙重表示：0.0111… = 0.1000…（就像十進位的 0.4999… = 0.5000…，ch04 你已經被這個咬過一次）。同一個實數有兩串二元展開，所以「序列 ↦ 它代表的實數」會把某些不同序列送到同一個數，不是單射。

這個瑕疵不致命，但要老實補。補法是 ch07 的 CSB 定理（Cantor–Schröder–Bernstein）：只要造得出**兩個方向各一個單射**——{0,1}^ℕ ↪ (0,1) 和 (0,1) ↪ {0,1}^ℕ——CSB 就保證它們等勢，不必硬湊出一個完美的一一對應。雙重表示只影響可數多個點，兩向單射都繞得開，所以 |{0,1}^ℕ| = |(0,1)| 是真的相等，2^ℵ₀ = 𝔠 站得住。完整的 CSB 機制留給 ch07，這裡你只要知道：**「幾乎是一一對應」的那點髒，有一台叫 CSB 的洗衣機專門處理**，本章先打欠條。

## 沒有「所有集合的集合」：第一聲警鐘

康托爾定理還有一個你身為工程師會立刻在意的後果——它證明了**一個你可能想寫的東西根本不能存在**。

假設有一個「萬有集」U，裝下所有集合（`Set<Everything>`，所有東西收進一個容器）。那 U 的每個子集本身也是集合，也該在 U 裡，於是 𝒫(U) 的每個元素都是 U 的元素：

```text
𝒫(U) ⊆ U          ← U 的子集也是集合，所以也在 U 裡
⇒ |𝒫(U)| ≤ |U|    ← 子集關係給出基數的 ≤

但康托爾定理說：
  |U| < |𝒫(U)|     ← 對任何集合都成立，U 也不例外

兩個合起來：
  |𝒫(U)| ≤ |U| < |𝒫(U)|
  ⇒ |𝒫(U)| < |𝒫(U)|   ← 一個基數嚴格小於自己
  ⇒ 矛盾
```

一個基數嚴格小於它自己，這是純粹的爆炸。結論：**萬有集 U 不存在**。「所有集合的集合」是一個會讓系統當機的概念，不是某個你還沒造出來的物件，是一個被證明造不出來的物件。

這叫**康托爾悖論**（Cantor's paradox）。康托爾自己約 1899 年就意識到了，在寫給戴德金的信裡用定理的形式講出來——「所有集合的集合」其冪集基數會更大，所以沒有最大的基數，也就沒有裝下一切的集合（2026-06，依 landscape）。注意這個時間點：**1899 年，比羅素 1901 年發現他那個悖論早了兩年**。康托爾不是被羅素的悖論嚇到，他自己先撞上了同一面牆——「天真地把太多東西收成一個集合會爆炸」。

你應該已經聞到 ch10 的火藥味了。看 U 不存在的證明的核心——對角集 D = {x ∈ A : x ∉ f(x)}。把它和「所有不屬於自己的集合的集合」R = {x : x ∉ x} 並排，你會發現它們是**同一個構造**：當 f 是恆等對應（把每個集合 x 送到它自己），「x ∉ f(x)」就退化成「x ∉ x」，D 就變成 R。康托爾悖論的對角集和羅素悖論的 R 是同一條對角線在兩個地方的化身。這個同構 ch10 會正面拆開，這裡先點名：你以為冪集定理是在講「無限的尺寸」，其實它順手在天真集合論的地基底下埋了一顆雷。引線已經點著，ch10 引爆。

## 直覺的陷阱

冪集定理的結論太乾淨（「沒有最大的無限」），乾淨到容易被收進一句口號，然後你就講不出它憑什麼成立、也躲不開下面這幾條溝。

| 錯誤直覺 | 它把你帶去哪條溝 | 自我察覺的徵兆 |
|---|---|---|
| 「2^ℵ₀ 不過是 ℵ₀ 乘很多次，乘來乘去還是 ℵ₀」 | 把取冪當成乘法的加強版。ℵ₀+ℵ₀、ℵ₀·ℵ₀ 確實塌回 ℵ₀（ch02/ch07，無限基數的加乘塌縮），但**取冪不塌縮**——2^ℵ₀ 嚴格大於 ℵ₀，這正是康托爾定理。加乘是把同一階的東西攤平，取冪是踏上新一階 | 你說不出「為什麼 ℵ₀·ℵ₀ 還是 ℵ₀、但 2^ℵ₀ 就跳上去了」——關鍵在對角線只在取冪時拉得開差距 |
| 「對角集 D 一定是 f 的某個像，只是你還沒找到」 | 你還沒接受反證法已經把這條路堵死。D 被造成跟**每一個** f(x) 都在第 x 格不同，不是「碰巧」不在像裡，是被設計成不可能在 | 你想去「第 d 列」找 D（像 ch04 想找「第 ∞ 列」），但 D = f(d) 這個假設本身就導致矛盾，沒有那個 d |
| 「冪集更大是因為 2ⁿ > n，無限版也是同個道理」 | 有限版可以靠數數（2ⁿ > n 是算術），但無限不能數，必須改用「抓出一個漏掉的元素」。把有限的算術理由搬到無限是搬錯了引擎——無限版的引擎是對角線，不是不等式 | 你被問「無限怎麼比大小」時只會說「2 的幾次方比較大」，講不出對角集 |
| 「𝔠 是最大的無限，再上去沒有了」 | 把 ch04/ch05 的「兩種無限」當成天花板。其實 𝔠 = ℶ₁ 只是塔的第一階，上面 2^𝔠、2^(2^𝔠) 無窮無盡 | 你以為集合論的無限就兩種（可數／不可數），不知道有 ℶ₂ 以上 |
| 「ℶ 記號是康托爾發明的」 | 把 aleph 和 beth 的歸屬混在一起。aleph 歸康托爾，beth 是皮爾士 1900 提議的 | 你把全書所有無限記號都掛在康托爾名下 |
| 「萬有集只是還沒人定義好，定義仔細點就有了」 | 沒聽懂「被證明不存在」和「還沒造出來」的差別。U 一存在就 |𝒫(U)| < |𝒫(U)|，是邏輯爆炸，不是定義不夠仔細 | 你還想替 `Set<Everything>` 找一個「正確的」定義 |

## 紙上推演

**1. 對 ℕ 親手跑一次康托爾定理** **[15 分鐘]** ★★
不看正文，對 A = ℕ 寫出康托爾定理的完整證明：假設 f: ℕ → 𝒫(ℕ) 滿射、造對角集 D = {n ∈ ℕ : n ∉ f(n)}、用兩種情況推出矛盾。然後回答：D 對應 ch04 對角線表裡的哪個翻轉序列？把 f(n) 都寫成特徵位元排成表，指出 D 是怎麼從對角線翻出來的。

**2. 找出破綻：「2^ℵ₀ 還是 ℵ₀」** **[10 分鐘]** ★
有人這樣論證：「ℵ₀ + ℵ₀ = ℵ₀，ℵ₀ · ℵ₀ = ℵ₀，無限基數做什麼運算都塌回 ℵ₀，所以 2^ℵ₀ 當然也是 ℵ₀。」指出這個論證錯在哪一步、為什麼加法乘法塌縮但取冪不塌縮。提示：康托爾定理直接給出 2^ℵ₀ > ℵ₀，所以結論一定錯，問題是找出那句話裡偷渡了什麼。

**3. 講給想寫 `Set<Everything>` 的工程師** **[15 分鐘]** ★★（口頭題）
一個同事說他要造一個「所有集合的集合」當作型別系統的頂層。不看書，用康托爾定理向他解釋為什麼這個東西不可能存在，講到他自己能重講「U 存在 ⇒ |𝒫(U)| ≤ |U| < |𝒫(U)| ⇒ 矛盾」這條鏈。然後加一句話點出這跟「列出所有不包含自己的清單」（ch10 預告）是同一個毛病。

**4. 有限版的對角集** **[10 分鐘]** ★
取 A = {1, 2, 3} 和 f(1) = {1, 3}、f(2) = ∅、f(3) = {2, 3}。手算對角集 D = {x : x ∉ f(x)}，驗證 D 不等於 f(1)、f(2)、f(3) 中任何一個（逐格檢查 D 和每個 f(x) 至少差一格）。然後回答：這個 f 漏掉了 𝒫({1,2,3}) 的 8 個元素裡的哪幾個，D 是其中之一嗎？

### 推演解答

**1.** 證明（與正文同構，這裡用 ℕ 具體化）：

```text
假設 f: ℕ → 𝒫(ℕ) 滿射。造 D = { n ∈ ℕ : n ∉ f(n) }。
D 是 ℕ 的子集，故 D ∈ 𝒫(ℕ)，由滿射假設 ∃ d 使 f(d) = D。
問 d ∈ D？
  若 d ∈ D ⇒ d ∉ f(d) = D，矛盾。
  若 d ∉ D ⇒ d ∈ f(d) = D，矛盾。
兩路皆死 ⇒ 無此 f ⇒ |ℕ| < |𝒫(ℕ)|。
```

對應 ch04 的對角線表：把每個 f(n) 寫成特徵位元，第 n 列就是 f(n) 的勾選表，對角線是「第 n 列第 n 位」（n 在不在 f(n) 裡）。D 的第 n 位 = 對角線第 n 位的**反**（n ∈ f(n) 就讓 n ∉ D，反之亦然）。所以 D 正是 ch04「沿對角線每位翻轉造出的那個新序列 s*」——當年那個「不在列表中的二元序列」，翻譯成集合語言就是「不是任何 f(n) 的那個子集」。同一個對象，兩種語言。常見錯路：想去「第 d 列」找 D，但證明的整個重點就是「假設有這個 d」會爆炸，所以根本沒有那一列（呼應 ch04「沒有第 ∞ 列」）。

**2.** 錯在「做什麼運算都塌回 ℵ₀」這句的「什麼運算」——它偷偷把取冪也算進「加法乘法那類」運算，但取冪不是同一類。正確的分界：

```text
ℵ₀ + ℵ₀ = ℵ₀     ← 塌縮（希爾伯特旅館，ch02）
ℵ₀ · ℵ₀ = ℵ₀     ← 塌縮（對角蛇行，ch03/ch07）
2^ℵ₀    > ℵ₀     ← 不塌縮！（康托爾定理）
```

加法、乘法是把「ℵ₀ 個 ℵ₀ 大小的東西」攤在同一階上重排，重排不改基數，所以塌回去。取冪 2^ℵ₀ = |𝒫(ℕ)| 不是重排，是「對 ℵ₀ 個元素各做一次獨立的二選一」，這恰恰是對角線能咬住的結構——對角集保證冪集裝不進原集合。換句話說，加乘塌縮和取冪長大的分水嶺，就是康托爾定理那條對角線：它只在「取所有子集」這個動作上拉得開差距。論證者沒有對「2^ℵ₀」這個具體運算驗算，憑「無限運算都塌縮」的模糊印象外推，外推到了一個反例上。（這個規律 ch07 會系統化：無限基數加乘塌縮、取冪才跳階。）

**3.** 模範說法要點（你的版本要命中這四點）：

1. **先講康托爾定理一句話**：任何集合 A，它的所有子集合起來（𝒫(A)）一定比 A 嚴格多——連有限版你都知道，3 個元素有 8 個子集。
2. **套到萬有集**：假設 U 裝下所有集合。U 的子集也是集合，所以也在 U 裡，於是 𝒫(U) ⊆ U，給出 |𝒫(U)| ≤ |U|。
3. **撞上定理**：但康托爾定理對 U 也成立，|U| < |𝒫(U)|。兩個合起來 |𝒫(U)| ≤ |U| < |𝒫(U)|，一個數嚴格小於自己，爆炸。
4. **結論＋點名同構**：所以「所有集合的集合」被證明不存在，不是還沒造出來。順帶一句：這跟「列一張清單，收所有不列自己的清單」是同一個自我參照的毛病（D 取恆等對應就變成 R = {x : x∉x}），ch10 會看到它的素顏。常見錯路：把「U 不存在」說成「U 太大裝不下」——不是容量問題，是邏輯矛盾，再大的容器都不行。

**4.** 逐格算對角集：

```text
        1   2   3
f(1)={1,3}  含1     不含2   含3
f(2)=∅      不含1   不含2   不含3
f(3)={2,3}  不含1   含2     含3

對角線（x 在不在 f(x)）：
  1 ∈ f(1)={1,3}？ 在  → 1 不收進 D
  2 ∈ f(2)=∅？     不在 → 2 收進 D
  3 ∈ f(3)={2,3}？ 在  → 3 不收進 D

D = {2}
```

驗證 D = {2} 和三個像都不同：f(1)={1,3} 在第 1 格含 1、D 不含，差第 1 格 ✓；f(2)=∅ 在第 2 格不含 2、D 含，差第 2 格 ✓；f(3)={2,3} 在第 3 格含 3、D 不含，差第 3 格 ✓。每一個都至少差「對角那格」，所以 D 不是任何 f(x)。

這個 f 的像是 {1,3}、∅、{2,3} 三個，漏掉了 𝒫({1,2,3}) 的另外五個：{1}、{2}、{3}、{1,2}、{1,2,3}。D = {2} 確實是漏掉的之一。注意對角線只「保證」抓到一個漏掉的（D），不保證抓到全部——三個元素本來就不可能滿射到八個子集，漏五個，對角線只負責當場點出其中一個現行犯，但這一個就足以證明 f 不滿。
