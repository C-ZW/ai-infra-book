# 不只一種無限：集合論與無限的直覺與驚嘆

> 一本寫給工程師、純為了欣賞而讀的集合論書。不是符號操練，不是嚴格公理化教科書，也不是為了轉職——是想讓你親身感受：人類怎麼學會**比較無限**，怎麼證明出「無限不只一種」，又怎麼一路走到連數學自己都管不到的邊界。

## 這本書在做什麼

「無限就是無限，哪有大小之分？」——這是幾乎所有人的直覺，而它錯了。19 世紀末，康托爾（Cantor）做了一件看似簡單卻翻天覆地的事：他不去回答「無限到底是什麼」，而是先把「**一樣多**」講清楚——兩個集合一樣多，意思是它們之間存在一個**一一對應**（把彼此的元素兩兩配對，不重不漏）。就這一個定義，讓無限第一次能被丈量、分級、做算術。然後對角線一劃，他證明了實數比自然數**嚴格地多**——無限有大小，而且不只一種。

這本書就是在追這件事，以及它一路長出來的所有驚嘆與危機。讀完，你應該能不看書、用自己的話對另一個工程師講清楚這六件事：

1. 「一樣多」的真正定義是什麼（**一一對應**），為什麼它比「數出來再比數字」更基本、能用在無限上。
2. 為什麼無限**不只一種**——可數（ℵ₀）與不可數（𝔠）的分界，對角論證怎麼證的。
3. 冪集定理：為什麼**沒有最大的無限**，無限有無窮多級、排成一座沒有頂的塔。
4. 為什麼天真集合論會自爆（羅素悖論），ZFC 與選擇公理怎麼重建地基、又付出什麼代價（Banach–Tarski 把一顆球變成兩顆）。
5. 連續統假設為什麼「**沒有答案**」——獨立性是什麼意思，哥德爾與科恩各證明了什麼（而且都**不是**「證出 CH 的真假」）。
6. 這一切怎麼活在計算裡——**對角論證就是停機問題**、可數就是可列舉、無限的分級就是可計算性與機率測度的邊界。

## 怎麼讀

- **架構圖先行**：每個 Part 的第一章都有全書地圖，先看清自己在哪、再鑽細節。
- **跟著同一條對角線走**：全書有一條脊椎——「排成一列，再沿對角線跟每一列作對」這一招。你會看著它出場四次：對角論證證明實數數不完（ch04）、冪集定理證明沒有最大的無限（ch06）、羅素悖論炸掉天真集合論（ch10）、哥德爾不完備困住數學自己（ch16）。同一條對角線，從第一道裂縫一路走到數學的自我極限——這是這本書最划算的對帳。
- **動手做是紙筆推演**：每章末的「紙上推演」用紙筆算、用嘴巴講。**講得出來才算懂**；講不清楚，就是還沒懂。本書純紙筆、沒有程式實驗——集合論的驚嘆在論證，不在數值。
- **「直覺的陷阱」是招牌**：集合論最容易誤解的地方（「無限＋1 還是無限，所以無限都一樣大」「獨立＝我們還沒證出來」「Banach–Tarski 是物理魔術」），每章都有一節專門拆解「這裡你的直覺正要騙你」。

## 書架上的鄰居

這本書會在需要時指向書架上的另外幾本，但本書自足、不必跨書也讀得懂：

- **《馴服無限：微積分的直覺與驚嘆》**——那本馴服「無限小／逼近」，這本馴服「無限大／多少」，是同一種勝利的兩個方向。ℝ 的構造（Dedekind 切割）、0.999…＝1、極限都住在那本。
- **《馴服隨機：機率與統計的直覺與驚嘆》**——為什麼機率非要 σ-代數不可、為什麼是「可數可加」而不是任意可加，根就在這本的 ch13（非可測集）。
- **《把系統寫成定理：TLA+ 與形式化方法》**——ZFC 的公理就是 spec、超限歸納就是 ranking function、不完備就是 model checking 的天花板；那本書的量詞攻防你會在這裡重逢。

那麼，從第一章開始：一個牧羊人不會數數，卻知道羊和石子「一樣多」——比較無限的故事，從這裡長出來。
