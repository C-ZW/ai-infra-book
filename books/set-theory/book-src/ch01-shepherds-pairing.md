這本書要帶你看的事，可以濃縮成六句話。我把它們放在最前面，當作全書的驗收清單——讀完最後一章，你應該能對著另一個工程師，把每一句都用自己的話講清楚（最後一章 ch19 會逐條口試）：

1. 「一樣多」的真正定義是什麼（**一一對應**），為什麼它比「數出來」更基本、能用在無限上。
2. 為什麼無限**不只一種**——可數與不可數的分界，對角論證怎麼證的。
3. 為什麼**沒有最大的無限**，無限有無窮多級。
4. 為什麼天真的集合論會自爆（羅素悖論），ZFC 與選擇公理怎麼重建地基、又付出什麼代價。
5. 連續統假設為什麼「**沒有答案**」——獨立性是什麼意思，哥德爾與科恩各證明了什麼。
6. 這一切怎麼活在計算裡——對角論證＝停機問題、可數＝可列舉。

這一章只負責第一句的前半截：把「一樣多」這個詞，從你以為的樣子，還原成它真正的樣子。

## 從你已知的出發

你每天都在判斷「兩個東西一樣多」，而且你判斷的方式，多半根本不是「數」。

你有兩張表，`users` 和 `user_profiles`。你想確認「每個 user 都恰好有一筆 profile、沒有孤兒、沒有重複」。你會怎麼做？如果你夠老練，你**不會**先 `SELECT COUNT(*)` 兩邊再比數字——數字相等只能證明「個數一樣」，證不出「對得上」：兩邊都是一百萬筆，也可能是「五十萬個 user 各有兩筆 profile、另五十萬個是孤兒 profile」。你真正會做的是 `JOIN ... ON user_id`，然後檢查這個 join 是不是**一對一又不漏**：左邊每筆恰好接到右邊一筆（不重），右邊每筆也都被接到（不漏）。

停在這裡看一眼，因為你剛剛做的，正是這本書的種子：**你用「能不能用唯一鍵兩兩接起來」判斷一樣多，而不是用「數出來的兩個數字相等」。** 你信任的是配對，不是計數。

這個直覺在你的工具箱裡到處都是：

- **完美 hash（perfect hash）**：一個把 n 個鍵打到 n 個槽、**無碰撞且填滿**的 hash。無碰撞＝沒有兩個鍵打到同一槽（這正是「不重」），填滿＝每個槽都被打到（這正是「不漏」）。一個完美 hash，就是鍵集合與槽集合之間一個明確的一一對應。
- **雙向 migration**：你寫一個 schema migration，再寫一個能完全還原的反向 migration，於是你確信新舊兩個資料模型「裝得下一樣多的東西、沒有資訊遺失」。你信的不是兩邊欄位數相等，是「來回都對得上、不丟不撞」。
- **序列化往返（round-trip）**：`decode(encode(x)) == x` 對所有 x 成立、且每個編碼字串都解得回某個 x——這是在說編碼與被編碼之間是一一對應。

這些你都做過，而且做得很穩。本書要做的，只是把這個你天天在用、卻沒給它名字的判準，**請到檯面上、給它名字、然後鬆開「有限」這個你一直默默假設的前提**——看看它能把你帶多遠。劇透：遠到你會起雞皮疙瘩。

## 牧羊人不會數，卻知道一樣多

把時間倒回有文字之前。一個牧羊人，早上把羊放出柵欄，傍晚要確認「一隻都沒少」。他不識數——沒有「三十七」這個詞，甚至沒有「數字」這個概念。他怎麼辦？

他在出柵欄時，每出一隻羊，就從地上撿一顆石子丟進袋子。一隻羊，一顆石子。傍晚羊回來，每進一隻，他就從袋子裡丟掉一顆石子。如果羊全進完了、石子也剛好丟光，他就知道：**一隻都沒少。** 如果還剩石子，少了羊；如果石子先丟光了還有羊在外面，那是別人的羊混進來了。

他從頭到尾**沒有數過**。他不知道有幾隻羊，也不需要知道。他知道的只有一件事：**羊和石子能不能兩兩配對、不剩。**

這個動作低到不能再低，低到我們會輕視它。但請你認真看它一眼，因為它藏著一個哲學上很深的東西：**「一樣多」不需要先知道「多少」。** 配對這個動作，比「數出一個數字」更原始。數數本身，其實就是拿一個標準的東西去配對——你數「一、二、三」，是把眼前的羊，一隻一隻配到「自然數」這把標準尺上的 1, 2, 3。**計數是配對的一個特例：拿 ℕ 當那袋永遠夠用的石子。**

我特意把這個畫面放在全書最前面，因為它就是脊椎的第一節骨頭。整本書，我會反覆問同一個問題：「這個集合，能不能跟另一個集合兩兩配對、不剩？」而那個被我拿來當標準石子袋的集合，會是自然數 ℕ。我把它叫做**「ℕ 這把尺」**——一切丈量無限的動作，都是拿某個集合去跟這把尺配對，看對不對得上、對到哪裡為止。這是全書的脊椎物件之一，請記住它的名字。

## 把判準寫成定義：一一對應

牧羊人的配對，數學上有個精確的名字。我先把工程版講清楚，再給正式定義——因為這兩個你其實都已經會了，只是名字陌生。

我們要的是兩個集合 A、B 之間一個對應 f: A → B，滿足兩個條件：

```text
不重（單射 injection）：  A 裡不同的元素，配到 B 裡不同的位置
                         a₁ ≠ a₂  ⇒  f(a₁) ≠ f(a₂)
                         ← 沒有兩個鍵打到同一個槽（hash 無碰撞）

不漏（滿射 surjection）： B 裡每個元素，都有人配到它
                         對每個 b∈B，存在 a∈A 使 f(a)=b
                         ← 每個槽都被打到（hash 填滿值域）
```

同時做到不重又不漏的 f，叫做**一一對應（bijection，亦譯雙射）**。我們寫 `A ↔ B` 表示「A 與 B 之間存在一個一一對應」。

不重保證「A 沒有元素被浪費地擠到同一格」，不漏保證「B 沒有元素落單沒人配」。兩個合起來，A 和 B 就被這個 f **完美地縫在一起**：每個 a 恰好對到一個 b，每個 b 恰好被一個 a 對到。你可以把 f 反過來讀，得到一個從 B 回到 A 的對應 f⁻¹，它一樣不重不漏——配對是對稱的，誰配誰都行。

於是我們把「一樣多」這個含糊的日常詞，**操作化**成一條乾淨的定義：

```text
|A| = |B|   ⟺   A ↔ B   （A 與 B 之間存在一一對應）
```

`|A|` 唸作「A 的基數（cardinality，亦稱勢）」，現在你只要把它讀成「A 的大小」。這條定義在說：**兩個集合一樣大，意思就是它們之間配得起來、不重不漏。** 沒有提到任何數字。

這一步看起來只是把日常話翻成符號，但它是康托爾（Georg Cantor）整套工作的閥門。在 1870 年代，他把「兩個集合有相同的『勢』（德文 Mächtigkeit，英文 power，即後來的 cardinality）當且僅當兩者間存在一一對應」立為定義（這個重心的轉移在他 1878 年的論文裡已經明確）。注意這個動作的方向：他不是先有「大小」這個概念、再發現一一對應剛好保持它；他是**反過來，用一一對應去定義大小**。大小不再是某種你要先看見、再去測量的東西；大小就是「配不配得起來」這件事本身。

為什麼這個方向的逆轉這麼重要？因為在有限世界，這兩個方向給的答案永遠一樣，你分不出差別，也不在乎用哪個。但到了無限，「先數出多少再比」這條路**根本走不通**——你數不完。配對卻照樣能做。康托爾的賭注是：**既然配對在有限和無限都能做，就讓它當定義；至於它在無限會推出什麼怪結論，我們認帳。** 這一章剩下的部分，就是去領第一張帳單。

（順帶補一條身世，一句話：康托爾不是為了研究無限才走進這裡的。他本來在啃一個分析學的硬問題——一個函數的三角級數（傅立葉級數）表示是不是唯一的；為了處理「容許無限多個例外點」的情形，他被迫去刻畫越來越複雜的「點的集合」，集合論就是從這裡長出來的。這條線屬於《馴服無限》的地盤，本書不展開，想看見「為什麼會有人需要這套東西」見《馴服無限》ch11/ch14。）

## Worked example：部分竟然和整體一樣多

定義立好了，現在領帳單。我們拿這把新尺去量一個你以為早就懂的東西，看看它吐出什麼。

考慮兩個集合：所有自然數 ℕ = {1, 2, 3, 4, …}，以及所有完全平方數 S = {1, 4, 9, 16, 25, …}。

直覺會立刻喊：S **明顯比** ℕ 少。為什麼？因為 S 是 ℕ 的一部分——平方數全是自然數，但絕大多數自然數（2, 3, 5, 6, 7, …）不是平方數。隨著數字變大，平方數還越來越稀疏（1 到 100 之間只有 10 個平方數，1 到 10000 之間也只有 100 個）。「整體大於部分」，這是歐幾里得就寫進《幾何原本》的公理，是你骨子裡的直覺。

現在用新定義量一次。我提出一個對應：

```text
f: ℕ → S,   f(n) = n²

n :  1   2   3   4   5   6   7   8  ...
f(n): 1   4   9  16  25  36  49  64  ...
```

我來逐項驗證它是不是一一對應——這個動作我會在全書每次造對應時都親手走一遍，不靠記憶，因為「我以為它對」和「我驗過它對」是兩回事。

**不重（單射）**：要證 n ≠ m ⇒ n² ≠ m²。設 n、m 都是自然數（≥ 1）且 n ≠ m，不妨 n < m。因為兩者都是正數，平方保持大小順序：n < m ⇒ n² < m²，所以 n² ≠ m²。沒有兩個不同的自然數平方到同一個數。✓ 不重。

**不漏（滿射）**：要證 S 裡每個元素都有人配到。任取一個平方數 k² ∈ S（這裡 k 是某個自然數）。問：有沒有 n 使 f(n) = k²？有，取 n = k，則 f(k) = k² = k²。每個平方數 k² 都有它的原像 k。✓ 不漏。

不重又不漏，所以 f 是一一對應。按定義，`|ℕ| = |S|`：**自然數和平方數一樣多。**

讓我把這句話的怪異程度說滿，不要滑過去。S 是 ℕ 的真子集（S 裡的東西 ℕ 全有，ℕ 裡有 2、3、5 等等 S 沒有的東西）。S 在 ℕ 裡越來越稀疏，密度趨近於零。直覺、歐幾里得、你的全部生活經驗，都說 S 比 ℕ 少。但按我們剛立的定義，它們一樣多——而且不是「差不多」、不是「都是無限所以隨便」，是**有一個明確的 f(n)=n²，我剛剛逐條驗證它不重不漏**。「整體大於部分」這條公理，在無限這裡，**破了**。

再造一個對照，把這件事釘死，省得你以為平方數有什麼特殊。取正偶數集 E = {2, 4, 6, 8, …}，造 g: ℕ → E，g(n) = 2n：

```text
n :  1   2   3   4   5  ...
g(n): 2   4   6   8  10  ...
```

不重：n ≠ m ⇒ 2n ≠ 2m（兩邊同除以 2 即知）。✓
不漏：任取偶數 2k ∈ E，取 n = k，則 g(k) = 2k。✓

所以 `|ℕ| = |E|`：偶數和全體自然數一樣多。可是偶數「明明只占自然數的一半」。一半，和全部，一樣多。

這不是文字遊戲，也不是「無限就是怪、別深究」的搪塞。這是新定義誠實運作的結果。你把「一樣多＝配得起來」當定義，它就**必然**推出「部分能和整體一樣多」——因為一個無限集合，真的可以和自己的一部分配得起來、不重不漏。這正是無限集合的**定義性特徵**（戴德金 Dedekind 後來就用這條當無限的定義：一個集合是無限的，當且僅當它能和自己的某個真子集一一對應）。有限集合永遠做不到這件事——把 {1,2,3} 配到它的真子集，一定有人落單。能跟自己的真子集配起來，是「無限」的指紋。

## 伽利略看見了，卻按兵不動

這個「平方數和自然數一樣多」的觀察，不是康托爾first發現的。早他兩個半世紀，伽利略（Galileo Galilei）就盯著它看過。

1638 年，伽利略出版了他生前最後一部科學著作《關於兩門新科學的對話》（Discorsi e dimostrazioni matematiche intorno a due nuove scienze，英譯 Two New Sciences）。在書裡，他借角色之口，把我們上面這條觀察攤了出來：每個自然數都對應到唯一一個平方數（n ↔ n²），所以平方數「不比」自然數少；可是平方數又顯然只是自然數的一小部分，所以「應該」更少。同一堆東西，一邊推出「不比較少」，一邊推出「明顯更少」——矛盾。這就是後世說的**伽利略悖論（Galileo's paradox）**。

伽利略看見了一一對應。這點要講清楚：他**真的看到了** n ↔ n² 這個配對，看到了「每個數恰好配一個平方數、反過來也是」。他離康托爾的結論，只差最後一步——只要他敢說「既然配得起來，就叫一樣多」，集合論的大門就在 1638 年打開了。

但他沒有跨那一步。他的結論是另一個方向的、極為謹慎的判斷。他斷定：「相等」「較大」「較小」這些詞，**只適用於有限的量，不能用在無限上**。用書裡的原話（英譯）：*"the attributes 'equal', 'greater', and 'less', are not applicable to infinite, but only to finite, quantities."*（「相等、較大、較小這些屬性，不適用於無限的量，只適用於有限的量。」）

請務必把這裡的歸屬分清楚，這是本章最容易被科普寫錯的一點，也是 landscape 鎖住的硬事實：

```text
常見錯誤寫法：「伽利略證明了平方數和自然數一樣多。」     ← 錯。
正確版：
  伽利略：看見了 n↔n² 的配對，結論是「大小這個詞不能用在無限上」
          （放棄比較，宣告無限量不可比）。
  康托爾：看見同一個配對，結論是「那就讓配對來定義一樣多」
          （重新定義，不放棄比較，放棄的是『整體必大於部分』）。
```

兩個人看著同一張圖，做了相反的選擇。伽利略選擇相信「整體大於部分」這條直覺，於是當配對和它打架時，他得出「無限量根本不能比大小」——把整個問題判出局。康托爾選擇相信「配對」這個動作，於是當它和「整體大於部分」打架時，他犧牲掉的是後者——他說，「整體大於部分」只是一條**對有限成立、對無限不成立**的直覺，不是什麼神聖公理。

我認為這是整個故事裡最被低估的一步，比後面任何花俏的定理都關鍵。難的從來不是看見那個配對——伽利略、甚至更早的人都看見過。難的是面對「配對」和「整體大於部分」的衝突時，**敢於把哪一邊當成可以丟掉的那一邊**。康托爾的賭注是：丟掉「整體大於部分」，留住「配對」。整本書接下來的所有驚奇，都是這個賭注兌現的紅利。

（不浪漫化一句：康托爾為這個賭注付了很重的個人代價，他晚年飽受克羅內克（Kronecker）等人的反對，也長年受疾病所苦。但「學術迫害把他逼瘋」是被嚴重渲染的傳說——他患的是雙相情感障礙，病因在生理，學術壓力至多是誘因。這段身世見 ch17、ch19，本章只取他這一步思想上的勇氣。）

## 為什麼非得是配對：在無限裡，「數出來」這條路是死的

你可能還在抵抗：好，配對是個漂亮的定義，但「數出來再比數字」哪裡錯了？在有限世界它明明也對啊。

對，在有限世界它對——而且它對，**恰恰是因為它偷偷地就是配對**。當你「數」一個有限集合數到 5，你在做的事是：把集合的元素一個一個配到 1, 2, 3, 4, 5 上，最後停在哪個數字，那個數字就是大小。「數出 5」這個動作的內核，是「和 {1,2,3,4,5} 建立了一一對應」。所以在有限世界，「數出來比數字」和「配對」給一樣的答案，不是巧合——前者是後者套了一層殼。

現在把集合換成無限的。「數出來」這條路立刻死掉，死得很徹底：

- 你**數不完**。沒有一個「最後停在哪」的數字，因為永遠沒有最後。「ℕ 有幾個元素」這個問句，期待一個自然數當答案，但沒有自然數能當這個答案——ℕ 比任何自然數都大。
- 退一步說「那就說它們都是『無限』、所以一樣大」？這更糟，這是把所有無限集合一鍋燴成同一個「∞」。而本書最核心、最震撼的發現（ch04）正是：**這鍋不能燴，無限不只一種大小。** 如果你一開始就用「都是無限所以一樣大」打發，你會親手把全書最美的風景鏟平。「∞」根本不是一個數，把它當數來算是一切混亂的源頭——這點 ch02 會正面處理。

配對這條路在無限裡卻**完全不受影響**。我要證明 ℕ 和平方數一樣多，我不需要數完任何一邊——我只要拿出 f(n)=n²，驗證它不重不漏，就結束了。我要證明它們**不**一樣多（這種事會在 ch04 發生），我也不需要數，我只要證明「任何想配的對應都必然漏掉某些元素」。配對和反配對，都不要求你數完。**這就是為什麼在無限的世界，配對是唯一還活著的工具，而計數已經死了。**

這也是這本書和書架上《馴服無限》的對偶之處。那本書馴服的是「無限小／逼近」——把「趨近」操作化成 ε-δ 與極限，於是 0.999…=1 這種事能被精確談論。這本書馴服的是「無限大／多少」——把「一樣多」操作化成一一對應，於是「不只一種無限」能被精確談論。兩本書是同一種勝利的兩個方向：**面對無限，人類的招數始終是同一招——不要妄想直接理解它的本質，而是找到一個在有限世界好用、又恰好能延伸到無限的『動作』，用那個動作把無限框住。** 在那本書是極限，在這本書是配對。

## 直覺的陷阱

這一章的概念簡單到危險：正因為「一樣多」你天天用，你會以為自己懂了，然後在後面幾章被自己的舊直覺反咬。以下是這章最常見的幾條錯路，每一條我都標出「會在哪一步把你帶溝裡」和「什麼徵兆代表你其實還沒切換過來」。

| 錯誤直覺 | 它哪裡錯 | 自我察覺的徵兆 |
|---|---|---|
| 「平方數明明是自然數的一部分，怎麼可能一樣多？整體當然大於部分。」 | 「整體大於部分」是**只對有限成立**的直覺，不是定義也不是公理。一樣多的定義是配對，不是包含關係。無限集合的指紋恰恰就是「能和自己的真子集配起來」。 | 你心裡用「誰包含誰」在判斷大小，而不是「配不配得起來」。一聽到「真子集」就反射性覺得「比較小」。 |
| 「平方數越來越稀疏，密度趨近於零，所以比自然數少。」 | 稀疏／密度是**排列上的幾何性質**，不是基數。多少由配對決定，和元素在數線上擠得密還是疏無關（ch03 會把這條釘死：ℚ 密到處都是，照樣和 ℕ 一樣多）。 | 你拿「分布、密度、間隔」當大小的證據。把「位置關係」和「個數」混為一談。 |
| 「無限就是無限，沒有大小之分，比大小本身就沒意義。」 | 這正是伽利略停下的地方——也正是本書要推翻的。配對這個定義恰恰**讓**無限可以比大小，而且 ch04 會證明真的有不同大小的無限。先用這句話打發，等於提前棄賽。 | 你覺得「比較無限的大小」這個問句本身是廢話、是文字遊戲。對「不只一種無限」這個預告無動於衷。 |
| 「`|A|=|B|` 只要造得出一個 f 把 A 打到 B 就行了吧？」 | 不夠。隨便一個 f 只證得了「A 不比 B 多」（若它不重）或「A 不比 B 少」（若它不漏）。要證一樣多，**同一個 f 必須同時不重又不漏**。少驗一個就是漏洞。 | 你造完對應就宣布勝利，沒有分別檢查不重和不漏。或者用「兩邊都是無限」當作不用驗證的藉口。 |
| 「先數出兩邊各有多少，再比數字——這在無限也能類推吧。」 | 無限集合**沒有**一個自然數能當「有多少」的答案，「數出來」這條路在無限直接斷掉。能延伸到無限的只有配對。 | 你還在找「ℕ 到底有幾個元素」的那個數字。期待無限的大小是某種你能寫下來的數。 |

這張表裡，第一條和第三條是這本書的死敵，會一路跟到 ch04、ch06。如果你發現自己在後面任何一章冒出「但整體不是應該大於部分嗎」的念頭，回到這一章——那代表你的舊直覺又把方向盤搶回去了。

## 紙上推演

純紙筆與口頭，不要寫程式、不要查。每題先自己做，再看解答。解答裡我會把常見錯路也標出來。

**推演 1 ★ [10 分鐘]** 造出 ℕ 與「3 的正倍數」集合 T = {3, 6, 9, 12, …} 之間一個明確的一一對應，寫出公式，並**逐條驗證**不重與不漏（不准只說「顯然」，每一條要能口頭說出理由）。

**推演 2 ★★ [15 分鐘]** 你的同事說：「一一對應的定義裡，不重和不漏只要有一個就夠了吧？反正都是把 A 接到 B。」請各舉一個具體的對應，證明他錯：(a) 一個 ℕ→ℕ 的對應，不重但漏（所以不是一一對應）；(b) 一個 ℕ→ℕ 的對應，不漏但重（所以不是一一對應）。並用一句話說出，少了其中一個各會讓「一樣多」的結論垮在哪。

**推演 3 ★★ [15 分鐘]** 「希爾伯特旅館」有可數無限個房間（編號 1, 2, 3, …），全部住滿了。現在門口來了一位新客。請你用**本章的配對語言**解釋：為什麼「客滿還能再住進一人」不荒謬？（提示：把「全部住滿」翻成一個一一對應，再說明為什麼可以重排出一個新的一一對應、空出一間房。這是 ch02 的預習，本題只要求你用配對的話講通，不要求公式之外的東西。）

**推演 4 ★★★ [20 分鐘・口頭]** 把「『一樣多』的定義是配對、不是數出來的數字」這件事，講給一個只懂國中數學、堅持「平方數當然比自然數少」的人聽，講到他**服氣**（不是被你唬住，是真的點頭）。寫下你會用的講法大綱：你會先說什麼、用哪個具體例子、怎麼處理他必然會丟回來的那句「可是平方數就是自然數的一部分啊」。

### 推演解答

**推演 1。** 取 f: ℕ → T，f(n) = 3n。

```text
n :  1   2   3   4   5  ...
f(n): 3   6   9  12  15  ...
```

不重：設 n ≠ m。則 3n 與 3m，兩邊同除以 3 得 n ≠ m，故 3n ≠ 3m。不同的 n 配到不同的位置。✓
不漏：任取 T 裡的元素，它形如 3k（k 是某個自然數，因為 T 就是 3 的正倍數）。取 n = k，則 f(k) = 3k。每個 3k 都有原像 k。✓
不重又不漏，所以 f 是一一對應，`|ℕ| = |T|`。

常見錯路：只寫「f(n)=3n 很明顯是對應」就收工。本書禁止「顯然」——你要能分別說出不重的理由（除以 3）和不漏的理由（任意 3k 的原像是 k）。漏掉「不漏」這半，你只證了 T 不比 ℕ 多，沒證一樣多。

**推演 2。** 
(a) 不重但漏：取 f: ℕ → ℕ，f(n) = n + 1，即 1→2, 2→3, 3→4, …。不重沒問題（n≠m ⇒ n+1≠m+1）。但它漏了——值域是 {2,3,4,…}，沒有任何 n 配到 1，所以不是滿射。不重這半隻撐起「ℕ 不比 ℕ 多」（|ℕ|≤|ℕ|），但因為它漏，**就拿不到另一半（不比少）**，所以單憑這個對應證不出兩邊一樣多——一樣多要不重**又**不漏，這裡只有不重。
(b) 不漏但重：取 g: ℕ → ℕ，g(1)=1, g(2)=1, g(3)=2, g(4)=3, …（把 1 和 2 都打到 1，之後 g(n)=n−1）。它不漏（每個自然數都被打到：k 由 g(k+1) 打到，1 還被 g(1) 多打了一次），不漏這半隻撐起「ℕ 不比 ℕ 少」（|ℕ|≥|ℕ|）。但它重——g(1)=g(2)=1，兩個不同輸入撞到同一個輸出，於是**拿不到「不比多」那一半**（你「用掉」了兩個元素才蓋住一個），照樣證不出一樣多。

一句話總結：不重撐起「A 不比 B 多」（沒有元素被擠在一起），不漏撐起「A 不比 B 少」（沒有元素落單）。要「一樣多」，兩根柱子缺一不可。（這兩個半邊，到 ch07 會升級成正式工具：兩個方向各一個單射，靠 CSB 定理拼出一一對應。）

**推演 3。** 「全部住滿」是什麼意思？意思是現在的住客和房號之間有一個一一對應——第 k 位住客住第 k 號房，房號集合 {1,2,3,…} 被完美蓋滿、沒有空房、沒有兩人擠一間。

來了新客，看似無處可放。但配對的定義不要求我們「找到一間現在空著的房」，它只要求「重排出一個新的一一對應，讓所有人（含新客）各有一間房、不重不漏」。做法：請每位現住客往後挪一間——住第 k 號的搬到第 k+1 號（room n → n+1）。這個搬法本身是一一對應（不重：不同住客挪到不同房；它在房號上的效果是把 {1,2,3,…} 一一對應到 {2,3,4,…}）。搬完之後，1 號房空了出來，新客住進去。現在所有人（無限位舊客＋1 位新客）和房號 {1,2,3,…} 之間，又是一個完整的一一對應。

不荒謬的關鍵在於：有限旅館客滿就真的住不下，因為有限集合**無法**和自己的真子集一一對應（總有人落單）。但 ℕ 是無限的，它能和自己的真子集 {2,3,4,…} 一一對應——這正是上一節說的「無限的指紋」。客滿旅館還能塞人，不是魔術，是「無限能和自己的真子集配起來」這件事的旅館版。常見錯路：用「無限＋1 還是無限」打發。這句話結論對，但它把「∞」當數在算，繞過了真正的機制（重排出新的一一對應）。ch02 會把「∞ 不是數、ℵ₀ 才是」講透。

**推演 4。** 一個能讓人服氣（而不是被唬住）的講法，大致這樣走：

先別碰平方數。問他一個他答得出來的問題：「這間教室裡，是學生多還是椅子多？」他會開始數。打斷他：「假設多到數不清——你還有別的辦法知道嗎？」引導他想到：請每個人都去坐一張椅子。如果剛好坐滿、沒人站著也沒有空椅，就一樣多；有人站著，人多；有空椅，椅子多。**他不用數，光靠『坐得下、不剩』就知道了。** 讓他自己說出這句話。

然後點破：你剛剛用的，根本不是「數」，是「配對」。「一樣多」真正的意思，從來就是「能不能兩兩配對、不剩」——數數只是配對的一種偷懶寫法（拿數字當椅子）。

現在才搬出平方數：每個數 n，給它配一張「椅子」n²。1 配 1，2 配 4，3 配 9……每個數都坐得到一張平方數椅子，每張平方數椅子也都剛好有一個數來坐——沒人站著，沒有空椅。按「坐得下、不剩就是一樣多」這條他剛剛親口同意的規則，平方數和自然數一樣多。

他一定會丟回來那句：「可是平方數明明就是自然數的一部分啊！」這時不要硬辯，要**承認**：對，在有限的東西上，「一部分」一定比「全部」少，這條規則你我從小都對。但這條規則是「有限」才有的脾氣。無限的東西就是有這個怪癖——它能和自己的一部分剛好配滿。你可以反問他：「我剛剛那個配對，你能指出哪個自然數沒被配到、或哪個平方數被配了兩次嗎？」他指不出來。指不出破綻，配對就成立，結論就成立——「一部分比較少」這條直覺，到無限這裡就是會破，這不是詭辯，是無限本來的樣子。

關鍵不是把他繞暈，是讓他**先同意「一樣多＝配得起來」這條規則**（用椅子的例子，他自己會同意），再讓平方數乖乖滿足這條規則。他服的是自己同意過的規則，不是你的口才。

## 自我檢核

講得出來才算過關，講不清楚就是還沒懂。對著空氣或一個假想的工程師同事，口頭回答：

1. 「一樣多」的真正定義是什麼？為什麼這個定義比「數出兩邊的數字再比」更基本？（要能說出「計數其實是配對的特例」。）
2. 一一對應的兩個條件（不重、不漏）各自保證了什麼？只有其中一個會怎樣？分別撐起「不比較多」還是「不比較少」？
3. 為什麼在無限集合上，「數出來再比數字」這條路是死的，而配對這條路還活著？
4. 伽利略看見了 n↔n² 這個配對，他的結論是什麼？康托爾看見同一個配對，結論又是什麼？兩人**到底差在哪一個選擇**上？（小心：別說成「伽利略證明了一樣多」。）
5. 「平方數和自然數一樣多」為什麼會跟「整體大於部分」打架？打架時，康托爾犧牲了哪一邊、留下了哪一邊？
6. 「能和自己的真子集一一對應」這件事，為什麼是「無限」的指紋？有限集合為什麼做不到？
7. 為什麼「平方數越來越稀疏、密度趨近零」不能拿來證明它比自然數少？（密度是什麼性質、多少是什麼性質，兩者為什麼是兩回事？）
8. 全書的脊椎物件「ℕ 這把尺」是什麼意思？接下來十八章，我們會拿這把尺反覆做哪一個動作？

## 延伸閱讀

- **Galileo, *Two New Sciences*（1638），「第一天」談無限與連續的段落。** 伽利略悖論的原始出處，看他怎麼在「平方數一一對應」和「整體大於部分」之間，選擇了放棄比較。讀那段對話本身（Salviati 與 Simplicio）比讀任何轉述都有力——你會看見一個人離真相只差一句話卻沒說出口。英譯多有公版（如 Dover 版）。
- **Wikipedia「Galileo's paradox」與「Two New Sciences」。** 快速核對年份（1638）、原文歸屬與那句「相等、較大、較小不適用於無限量」的英譯措辭，避免把功勞錯安到伽利略頭上。本書 landscape 即以此校過。
- **《馴服無限》ch11/ch14。** 想知道「康托爾為什麼會走進集合論」——三角級數唯一性與點集的故事都在那裡；本書刻意不展開的這條因果鏈，那本書是正主。
- **《馴服無限》ch03/ch14（ℝ 的構造、0.999…=1）。** 本書之後用到 ℝ 的性質時會一句話帶過、指向那裡；如果你對「實數到底是什麼」還心虛，先補那幾章，後面（ch04 起）會輕鬆很多。
- **Halmos, *Naive Set Theory*，前幾節（functions、cardinality 的開頭）。** 想看「一一對應定義一樣多」用最精簡的數學語言寫出來長什麼樣，這是經典教科書的標準寫法。薄、狠、無廢話；本書是它的「直覺與驚嘆」版，兩者對照讀很划算。
