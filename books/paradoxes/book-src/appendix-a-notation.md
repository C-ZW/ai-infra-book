# 附錄 A — 記號與機率速查

本附錄不是新的悖論，是一張速查表：把全書實際用到的數學記號、公式與基準數字集中放在一處，方便你複習某一章時回頭核對。每一條都附「這是什麼意思（白話）」與「第一次出現／主要使用的章節」。記號寫法照全書慣例——條件機率一律用直式 `|`（P(A|B)），行內優先用 Unicode 分數與符號（½、⅓、≈、≠、Σ、∞），多行推導放在 `text` 區塊裡。

## 一、機率的基本記號

### P(A) — 事件的機率

`P(A)` 讀作「事件 A 發生的機率」，數值介於 0 到 1 之間（0 代表不可能、1 代表必然）。這是全書預設讀者已具備的起點，第 1 章起就直接使用，不再另外定義。

### P(A|B) — 條件機率

`P(A|B)` 讀作「已知 B 發生的條件下，A 發生的機率」；直式 `|` 左邊是你想問的事件，右邊是你已經知道的資訊。這是全書「條件與資訊」主線（見 style-guide 驗收標準③）的核心記號——蒙提霍爾（ch02）、三囚犯（ch03）、貝特朗盒子（ch04）、男孩女孩（ch05）全部靠它把「多了一項資訊之後，機率怎麼重新分配」講清楚。

```text
P(A|B) = P(A∩B) / P(B)          ← 定義本身：在 B 這個縮小後的樣本空間裡，A 占的比例
```

- 首次出現／主要使用：ch02（蒙提霍爾）。
- 沿用：ch03、ch04、ch05、ch06、ch08、ch23。

### 獨立事件：P(A|B) = P(A)

兩個事件 A、B 獨立（independent），若且唯若已知 B 發生完全不改變你對 A 的判斷：

```text
P(A|B) = P(A)                    ← 知道 B 發生了，A 的機率原封不動
⟺ P(A∩B) = P(A)·P(B)             ← 等價寫法，兩者可互相推導
```

白話：獨立描述的是「這一次試驗本身沒有記憶」，不是「長期次數會被拉平」——賭徒謬誤（連紅後押黑）與熱手謬誤（連中後看好再中）都是把這個精確定義換成了「有一套機制在記帳」，只是記帳方向相反。

- 首次出現／主要使用：ch11（賭徒謬誤與熱手，精確定義獨立並用來拆穿兩種相反的直覺錯誤）。
- 隱含使用：ch10（賭徒輸光的每一步位移彼此獨立，是破產遞迴成立的前提）。

### 貝氏定理

貝氏定理（Bayes' theorem）把「證據反推假設」的機率，跟「假設推證據」的機率連起來：

```text
P(H|E) = P(E|H)·P(H) / P(E)

其中：
  H = hypothesis，你想知道機率的那個假設（例如「有病」）
  E = evidence，你觀察到的證據（例如「篩檢陽性」）
  P(H)   ← 先驗機率（prior）：看到證據之前，你對 H 的信念（基率／盛行率）
  P(E|H) ← 概似（likelihood）：已知 H 為真，觀察到 E 的機率（敏感度）
  P(H|E) ← 後驗機率（posterior）：看到 E 之後，更新過的 H 的信念

分母展開（把 H 為真、H 為假兩條路徑都算進去）：
  P(E) = P(E|H)·P(H) + P(E|非H)·P(非H)
```

白話：**檢察官謬誤（ch08）與偽陽性悖論（ch06）都是把 P(E|H) 誤當成 P(H|E)**——「這種巧合在清白者身上只有百萬分之一」不等於「被告清白的機率只有百萬分之一」，中間永遠隔著一個常被忽略的先驗 P(H)。等價的計算捷徑是自然頻率法（natural frequency，把機率換成「10 萬人裡有幾個」的人頭數去算，答案完全一樣，但多數人算對的比例大幅提高）。

- 首次出現／完整式：ch06（偽陽性悖論，第一次把「一直在暗地裡用」的工具正式攤開命名）。
- 沿用：ch03（三囚犯，貝氏算 P(A 存活|獄卒說 B)）、ch08（檢察官謬誤）、ch23（睡美人，對「被喚醒」這件事本身條件化）。

## 二、期望值與效用

### E[X] — 期望值

期望值（expected value）是隨機變數 X 所有可能結果，依各自機率加權平均後的值：

```text
E[X] = Σ pᵢ·xᵢ                   ← 對所有可能結果 i 加總：機率 × 結果值
```

白話：期望值是「長期玩很多次之後，平均每次拿到多少」，不是「你這一次該付的價」——聖彼得堡悖論（ch12）示範了兩者可以差到一個是無限大、一個是幾塊錢；兩個信封悖論（ch13）示範了「算出來的期望值比原本大 1.25 倍」這個論證，錯在把兩個不同條件下的金額，偷偷塞進同一個符號 X 裡。

- 首次出現／主要使用：background（style-guide 列為預設已知），ch12 起成為核心角色（聖彼得堡：Σ(n=1→∞) 2ⁿ·(1/2ⁿ) = ∞）。
- 沿用：ch13（兩個信封）。

### 對數效用：u(w) = ln(w)

白努利（Daniel Bernoulli）解聖彼得堡悖論的辦法，是把「錢的數量」換成「錢的效用（utility）」——具體用自然對數當尺度：

```text
u(w) = ln(w)                     ← w 為財富，ln 為自然對數
```

白話：對數會把「指數成長」壓成「線性成長」——賠付 2ⁿ 達克特的效用只有 n·ln2，而機率仍以 (1/2)ⁿ 指數縮小，兩者相乘後的和收斂成一個有限的天花板，不再發散。這解決了「聖彼得堡這一個」賭局，不代表能解決所有賠付成長更快的變體（本書誠實標示，效用理論完整深探交叉引用《在不確定中下注》）。

- 首次出現／主要使用：ch12（聖彼得堡悖論）。

## 三、計數與級數

### C(n,k) — 組合數

`C(n,k)` 讀作「從 n 個東西裡任選 k 個、不分順序」的組合數：

```text
C(n,2) = n×(n−1) / 2             ← 生日問題只會用到「任選 2 個」這個特例：配對數
```

白話：生日問題（ch09）那句沒說出口的假設，是把「有幾個人」當成問題的規模，實際上真正在互相比較生日的是「有幾對人」——配對數 C(n,2) 隨 n 平方成長，人數只隨 n 線性成長，直覺拿線性的尺去量一個平方成長的東西，才會嚴重低估機率（23 人只是線性小數，配對數 C(23,2)=253 已逼近 365 的七成）。

- 首次出現／主要使用：ch09（生日問題，C(23,2)=253）。

### Σ 與 ∫ 記號約定

全書的求和一律寫成「Σ(起點→終點) 通式」的形式，把求和的起訖點寫在 Σ 正下方或後面的括號裡，不留給讀者猜：

```text
Σ(n=1→∞) 2ⁿ·(1/2ⁿ)               ← 讀作「n 從 1 加到無窮大，每一項是 2ⁿ·(1/2ⁿ)」
```

積分（∫）在全書只在少數地方出現，用來檢驗「一個給定的密度函數，總機率加起來是不是等於 1」——例如兩個信封悖論（ch13）用它證明「每一塊錢都給相同機率密度」這種先驗，積分發散、湊不出 1，因此不是一個合法的機率分布（這種湊不出 1 的「分布」稱為不當先驗，improper prior）：

```text
∫₀^∞ f(x) dx = ∫₀^∞ c dx          ← 常數密度 f(x)=c 在無窮區間上的積分，恆發散（c>0 時）或恆為 0（c=0 時），兩者都不等於 1
```

- 首次出現／主要使用：Σ 為背景知識，ch09、ch12 起頻繁使用；∫ 首次具體出現於 ch13（兩個信封，不當先驗）。

### 幾何級數

```text
Σ(n≥0) rⁿ = 1/(1−r)              ← 公比 r 的幾何級數，僅當 |r| < 1 時收斂到這個值
```

白話：全書兩次真正用到這個公式，都是拿它當「收斂／發散」的分界尺——聖彼得堡悖論（ch12 練習）把賠付規則換成 rⁿ 達克特，公比變成 r/2，r<2 時收斂、r=2（本章原題）剛好卡在分界線上、r>2 時發散得更快；兩個信封悖論（ch13）的等比先驗（機率 (1/3)·(2/3)^k）本身就是一個公比 2/3 的幾何級數，用來驗證「這是一個合法、加總為 1 的機率分布」。

- 首次出現／主要使用：ch12（聖彼得堡，練習 1）、ch13（兩個信封，等比先驗）。

## 四、隨機過程：吸收邊界與破產機率

### 吸收馬可夫鏈與破產遞迴 q_i

把「手上的資本」看成一條會隨時間變化的馬可夫鏈（Markov chain）：狀態是目前資本 i（0 到 N 之間的整數），每一步以機率 p 往右走一格、機率 q=1−p 往左走一格；i=0 與 i=N 是兩道吸收態（absorbing state）——一旦走到就不再離開。設 q_i 為「起始資本 i 時，最終在 0 這道牆破產」的機率，公平版本（p=q=½）滿足：

```text
q_i = ½·q_{i-1} + ½·q_{i+1}       ← 「現在破產的機率」＝下一步兩種去向各自破產機率的加權平均
邊界條件：q_0 = 1（已經破產）、q_N = 0（已經贏光對手）

移項後（相鄰兩項的差恆定，收斂成一個等差數列）：
  q_{i+1} − q_i = q_i − q_{i-1}
解出：
  q_i = 1 − i/N                   ← 起始資本 i、總資本 N（對手資本 N−i），對照基準表 B8
```

不公平版本（p≠q）的封閉解（sketch，本書不重複展開代數）：

```text
q_i = [ (q/p)^i − (q/p)^N ] / [ 1 − (q/p)^N ]
```

白話：「公平」只保證單局期望值為 0，從來沒保證你碰不到 0 這道吸收牆——資本愈懸殊，破產機率愈接近確定值。這條記號也是 ch11（賭徒謬誤，借用「每一步無記憶」）與 ch14（帕隆多悖論，借用漫步語言講棘輪機制）的共同地基。

- 首次出現／主要使用：ch10（賭徒輸光問題）。
- 沿用：ch11、ch14。

## 五、共同知識與知識邏輯

### k 階知識運算子 Eᵏ 與共同知識 Cφ

設 φ 是某個事實（例如「島上至少有一個藍眼人」）。知識運算子（knowledge operator）一層一層往上疊：

```text
E¹φ  ：每個人都知道 φ                          ← 「互知」的第一層（也寫作 E(φ)）
E²φ  ：每個人都知道「每個人都知道 φ」            ← 第二層（也寫作 E²(φ)）
E^mφ ：疊了 m 層的「大家都知道大家都知道…φ」

Cφ   ：對所有 m，E^mφ 同時成立                  ← 共同知識(common knowledge)，也寫作 C(φ)
```

白話：「互知」（mutual knowledge，每個人都知道）跟「共同知識」（無限層都知道）不是同一回事，中間可能隔著一大段「大家都知道，但不確定大家知不知道大家知道」的灰色地帶。紅藍眼睛悖論（ch15）裡，島上恰有 k 個藍眼人時，宣告前已經疊出 k−1 層互知（E¹φ 到 E^(k−1)φ 皆成立），只差第 k 層（E^kφ）——遊客當眾、公開、同時的宣告，一次把這個斷點以上所有層級補齊，直接升級成 Cφ。兩位將軍問題（ch18）則反過來證明：只靠會遺失的私下訊息，不管交換幾層確認，永遠只能疊到有限層，永遠達不到 Cφ。

- 首次出現／完整定義：ch15（紅藍眼睛悖論）。
- 沿用：ch16（泥巴小孩，同一套歸納換一身乾淨外衣）、ch18（兩位將軍，證明私下訊息疊不出 Cφ）。

## 六、對數與尺度不變

### log₁₀ 與班佛定律

任何正數 x 都可以唯一寫成「一個 1 到 10 之間的數字，乘上 10 的某個整數次方」；取以 10 為底的對數（記作 log₁₀，班佛定律章之後簡寫成 log），跨數量級的資料在對數尺度下會呈現均勻分布，逼出首位數字的機率公式：

```text
P(首位數字 = d) = log₁₀(1 + 1/d)          ← d = 1, 2, …, 9
P(首位數字 = 1) = log₁₀2 ≈ 30.10%         ← 對照基準表 B12
```

白話：直覺以為「首位數字均勻分布（各 1/9）」，但這只有在資料**不**跨數量級時才成立；一旦資料橫跨多個數量級（人口、河流長度、股價……），尺度不變性會把均勻分布推到對數的世界裡去，首位是 1 的機率反而遠高於首位是 9。

- 首次出現／主要使用：ch25（班佛定律）。
