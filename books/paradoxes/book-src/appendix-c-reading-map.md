# 附錄 C — 悖論索引與閱讀地圖

本附錄是查閱工具，不是第二十八章——不重演任何悖論，只把全書已經講過的東西攤成一張可以隨手翻的地圖。三個部分：(a) 二十五個悖論對照它們「那句沒說出口的話」屬於 ch27 收出的哪一種假設類型；(b) 幾條可以跳著讀的路線；(c) 與姊妹書交界的地方，一句話說清楚哪裡該接去哪一本。

ch01（三步解剖框架）與 ch27（收官總表本身）不是具名的悖論，不計入下面的索引；索引涵蓋 ch02 到 ch26，共二十五章。

## (a) 悖論 ↔ 假設類型索引

假設類型沿用 ch27 收出的九種模式，這裡只列名稱，完整定義與逐條論證見 ch27：

①忽略條件化　②一個動作洩漏了資訊　③相互知識≠共同知識　④忽略吸收邊界　⑤期望值≠價格　⑥「隨機」或「勝過」未定義　⑦聚合改變方向　⑧傳遞性被預設　⑨不可測

「／」表示同一章橫跨兩種類型（ch27 已論證這是鏡頭疊加，不是分類沒做好）；「（九型之外）」表示該章明講無公認解，本來就不屬於這九種「補上假設就解決」的模式。招牌數字欄與 `_meta/running-examples.md`／ch27 總表一字不差。

## (b) 幾條閱讀路徑

不想按章節順序讀，也可以照題材跳著讀。每條路徑照書中原有的順序，只是抽掉了其他章節。

- **機率條件化線（ch02→ch03→ch04→ch05→ch06→ch08→ch23）**：條件機率永遠藏著「條件事件本身是怎麼被造出來的」這個問題——從主持人被迫的動作、獄卒的用詞、孩子性別怎麼問，一路問到「我現在醒著」這件事算不算新證據。
- **賭局與長期線（ch10→ch11→ch12→ch13→ch14）**：同一副骰子與銅板，從破產邊界、獨立性的精確定義、發散的期望值，到互換信封與混合輸局，一路拆穿「長期」「期望值」「該付的價」這三個詞常被偷渡的意思。
- **邏輯與共同知識線（ch15→ch16→ch17→ch18）**：共同知識為什麼不等於人人都知道，由藍眼小島、泥巴小孩、兩支軍隊三章交卷；中間插入的 ch17 意外絞刑刻意是同一個 Part 裡的對照組——同樣是認識論式的推理鏈，這次絆倒的是自我指涉，不是共同知識。
- **集體與選擇線（ch19→ch20→ch21→ch22）**：偏好、投票、路由、決策這四種「集體或對手涉入」的情境，分別拆穿「勝過可傳遞」「個體最優等於全體最優」「我的選擇與已經做好的預測彼此獨立」這幾句被悄悄預設的話。
- **隨機與測度線（ch24→ch25→ch26）**：貝特朗弦、班佛定律、巴拿赫–塔斯基三章一路把「隨機」「均勻」這些詞逼到定義的邊界，直到「體積」這個概念本身在不可測集上徹底失效。

## (c) 與姊妹書的交界

本書多處刻意不深挖，把更完整的理論留給同系列的其他悖論／欣賞書接手：

- **《馴服隨機》（prob）**：賭局與漫步三章（ch10、ch11、ch14）用到的大數法則、布朗運動、馬可夫鏈穩態，本書都只做到直覺層，完整的極限定理由《馴服隨機》接手。
- **《在不確定中下注》（decide）**：聖彼得堡（ch12）與紐康（ch22）在效用理論與理性選擇上的深探，兩章正文都明文交叉引用《在不確定中下注》；睡美人（ch23）的可信度之爭雖未在章內點名該書，但同屬「不確定下該相信什麼、該賭多少」的家族，讀完 ch23 想追下去，這是下一站。
- **《當理性互相碰撞》（game）**：紅藍眼睛（ch15）追溯共同知識的賽局論形式化（阿曼 1976 年〈Agreeing to Disagree〉與同意定理），這一面向更完整的賽局理論由《當理性互相碰撞》接手。
- **《不只一種無限》（sets）**：巴拿赫–塔斯基（ch26）需要的選擇公理強度（超出 DC、弱於完整 AC）、不可測集與勒貝格測度，是《不只一種無限》的主場，本書在此只做到 sketch 層。
- **《這句話無法被證明》（godel）**：意外絞刑（ch17）的自我指涉推理鏈，本書正文已明講它跟停機問題、跟哥德爾不完備定理裡那句「無法被證明」的自我指涉句是同一個家族的親戚，完整的形式化留給《這句話無法被證明》。
