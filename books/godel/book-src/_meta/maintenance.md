## 跨章基準數值表（改任一個必 grep 全書同步）

> 本書幾乎全是「數學恆真／史實已定案」，時效風險極低；最需要守的是**定理的精確條件**與**記號約定**的跨章一致。

| 基準 | 值 | 出現章 |

| 脊椎句 G（形式） | F ⊢ G ↔ ¬Prov_F(⌜G⌝)（對 ¬Prov_F 取不動點） | ch06/07/12/16 |
| 三條件（鐵律） | ① 一致（consistent）② 可遞迴公理化（公理集合可機械列舉）③ 夠強表達基本算術（含 Robinson Q，能定義 + 與 ×） | ch02/07/08/13/14/16 |

| HBL 三條件 | D1：F⊢φ⇒F⊢Prov(⌜φ⌝)；D2：F⊢Prov(⌜φ⌝)→Prov(⌜Prov(⌜φ⌝)⌝)；D3：F⊢Prov(⌜φ→ψ⌝)∧Prov(⌜φ⌝)→Prov(⌜ψ⌝)（第二定理用，Hilbert–Bernays–Löb） | ch08 |
| 哥德爾編碼約定 | 序列 (a₀,…,aₙ) → 2^(a₀+1)·3^(a₁+1)·5^(a₂+1)·…（指數＋1 避 0 冪）；解碼＝質因數分解唯一（FTA，《乘法的原子》） | ch05/06 |
| 對角化引理 | 對任意一元公式 ψ(x)，∃句 D，F ⊢ D ↔ ψ(⌜D⌝)（Carnap 1934 一般化；Gödel 1931 已用其特例） | ch06/09 |
| 對「真」的定義 | 「真」＝在標準模型 ℕ（自然數＋通常 +,×,<）中成立（⊨）；與「可證」（⊢）嚴格分開 | ch03/07/12 |
| 標準模型記號 | ℕ | ch03/12 |
| 說謊者 → 不可證 | 「這句話是假的」自爆（真假皆矛盾）；把「假」換成「在 F 內不可證」→「G 不可證」不自爆而得不完備 | ch04/06/16 |
| 塔斯基不可定義性 | 算術真理無法在算術內定義（Tarski 1933；Gödel 1930 已私下知道）——「可證」可表達、「真」不可定義 | ch03/12 |
| 圖靈停機問題 | 不存在程式 H 能對任意 (程式 P, 輸入 w) 正確判定 P(w) 會不會停（Turing 1936，對角化歸謬）；半可判定 | ch09/10 |
| 決定問題 | Entscheidungsproblem（Hilbert–Ackermann 1928）：一階邏輯有效性無一般判定程序——Church 與 Turing 1936 各自證否定 | ch02/09/10 |
| 完備性定理（勿混淆） | Gödel 1929：一階邏輯語意完備（每個語意有效的式子都可證）——與「不完備定理」是兩件事、不矛盾 | ch03/13/14 |
| 補洞長新洞 | F 加 G 當新公理 → 更強系統 F'，但 F' 仍滿足三條件 ⇒ 有自己的新 G'；無限不終止 | ch11/16 |
| 逃得掉的反例 | Presburger 算術（只有 +，無 ×）完備、可判定、可遞迴公理化；實閉體（Tarski）完備可判定——因為「夠強表達算術」這條被破壞 | ch13 |
| PA 證明論序數 | ε₀（epsilon-zero）；Gentzen 1936 用到 ε₀ 的超限歸納證 PA 一致 | ch15 |
| 爆炸原理 | 不一致系統能證明一切（ex falso quodlibet）——所以「一致」是底線，第二定理才這麼痛 | ch01/08 |
| Curry–Howard | 型別＝命題、程式＝證明（橋接，點到為止、不深推；Curry 1934 / Howard 1969，刊 1980） | ch01 |

> ω-一致 vs 一致：第一定理「¬G 不可證」原始需 ω-一致；**羅瑟 1936** 改進為只需一致。每次寫陳述必標清楚是哪個版本，不可混用。
> 第二定理比第一定理多要求 Prov_F 滿足 HBL D1–D3，所需算術略多；兩條定理條件不可互換。

## 脆弱／時效事實清單（以 landscape-2026-06.md 為準，引用前複查）

- ⚠️ **Lucas–Penrose 反機械論論證**（Lucas 1961/1959 宣讀、Penrose 1989）——哲學界對「漏掉的假設能否補上」**至今 2026-06 仍無共識**；本書 ch14 只指出論證缺的那個假設（人類心智一致且能識別自己的 G），不宣告終局，標 (2026-06)。
- ⚠️ **物理 TOE 是否受不完備限制**——開放且有爭議、非定理推論；ch14 以「有爭議的開放問題」框、不選邊。
- ⚠️ **ω-一致 vs 一致（T9）**：定理陳述最容易寫鬆的地方；每次出現必標。
- ⚠️ **圖靈論文年份 T11**：成文與核心結果 1936，卷標 1937，另有 1937 勘誤；本書用「1936」並一句註明刊期，見 landscape T11。
- ⚠️ **Lucas 論文年份 T12**：1959 牛津宣讀、1961 *Philosophy* 卷 36 刊出；本書引用年用 1961，一句註明 1959 宣讀，見 landscape T12。
- ⚠️ **Church–Turing 論題是論題非定理**：斷言直覺可計算性＝形式模型，無法被證明；別寫成「已被證明」。
- **跨書書名脆弱引用**：README／附錄 C 與各章指向姊妹書《乘法的原子》《思考的陷阱》《如何不騙自己》《把系統寫成定理》——primes/bias/scimethod 同屬「推理六書」，書名尚未最終定稿；若日後改名，需全書同步（grep 書名）。一律**主題式**引用、不寫姊妹書章號。
- 諾貝爾獎相關：本書不引用；若提及 Church–Turing 論題或圖靈獎，需用正確名稱（圖靈獎＝ACM A. M. Turing Award；諾貝爾獎無數學獎）。
- 歷史年份速查：希爾伯特 1900（23 問題）、Hilbert–Ackermann 1928（Entscheidungsproblem）、完備性定理 1929、Presburger 1929、哥德爾柯尼斯堡宣布 1930-09-07、希爾伯特電台演講 1930-09-08、哥德爾論文 1931（Monatshefte 卷 38：173–198）、Tarski 1933、Carnap 1934、Church 1936、Turing 1936、Rosser 1936、Gentzen 1936、Hilbert–Bernays 1939、Lucas 1961、Penrose 1989——以上均已查證，以 landscape 為準。
