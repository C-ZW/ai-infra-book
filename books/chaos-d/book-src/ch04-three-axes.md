# ch04 — 三條被混為一談的線：決定論、隨機、可預測

> **本章解決什麼問題**：前三章你看過了承諾（ch01 拉普拉斯）、第一道裂縫（ch02 三體）、與裂縫的機制（ch03 敏感依賴）。但要真正讀懂混沌，得先把三個一直被綁在一起講的詞拆開——決定論（determinism）、隨機（randomness）、可預測（predictability）。多數人（包括很多科普文章）把它們當同義詞輪流用，於是「混沌＝隨機」「決定論＝可預測」這兩個錯誤直覺就長出來了。這一章是全書最重要的概念清理：把三個詞釘成三條獨立的軸，畫出一張四象限表，然後指出混沌坐在哪一格。釘好了，後面 Part II 那條遞迴式的所有驚嘆才站得住。

## 從你已知的出發

你每天都在用一個「決定論造出來的假亂數」，而且你信任它信任到會拿它寫測試、拿它做負載分片、拿它當 retry 的 jitter——它就是 PRNG（pseudorandom number generator，偽隨機數產生器）。

想想你最熟的那段程式：

```text
rng = seedRandom(42)
a = rng()   // 0.3745...
b = rng()   // 0.9507...
c = rng()   // 0.7319...
```

`a`、`b`、`c` 看起來毫無規律——你盯著小數點後那串數字，看不出任何花樣，把它們畫成直方圖會是平的，做卡方檢定會通過。它「像隨機」。但你心裡很清楚一件事：**只要 seed 還是 42，這三個數永遠是這三個數。** 換一台機器、換一個禮拜、換一個資料中心，`seedRandom(42)` 跑出來的還是 `0.3745…`、`0.9507…`、`0.7319…`，一位不差。

這就是你早就會、卻可能沒停下來想清楚的那件怪事：**同一個東西，可以「完全被決定」和「看起來完全隨機」同時為真。**

你還會用更多這種東西：

- **雜湊（hash）**：`hash("user-1024")` 給你一串看似亂跳的十六進位，相鄰 key 的雜湊值天差地遠（這叫雪崩效應，avalanche）。但同一個輸入永遠給同一個輸出——所以你才敢拿它做一致性雜湊、做分片。「假亂」是它的賣點，「確定」是它能用的前提。
- **重放決定性（replay determinism）**：你做事件溯源（event sourcing），把同一串事件重放進同一個狀態機，要求得到 bit-identical 的結果。這是你親手強加的決定論——你要的就是「同輸入同輸出」這個性質。
- **retry 的 jitter**：backoff 加上一點「隨機」抖動，避免大家同時重試踩在一起。但那個抖動多半是 PRNG 給的——它是決定論的。

所以你早就活在這三個詞的交叉地帶，只是沒人逼你把它們拆開。這一章就來逼你拆。拆開以後你會看到：混沌跟你的 PRNG 是**同一個物種**——一條完全確定的規則，吐出一串看不出規律的東西。差別只在 PRNG 是人刻意設計來「假裝隨機」的工具，而混沌是自然界一條老老實實的物理定律，不小心也具備了同一種性質。這個對照是讀懂整本書的鑰匙。

## 第一條軸：決定論——關於「世界是怎麼演化的」

先給最乾淨的定義，這個定義來自 SEP（史丹佛哲學百科）的混沌條目，我認為它比任何物理課本都講得準：

> **一個模型是決定論的，若它具有「唯一演化（unique evolution）」：給定某個時刻的狀態，後續的狀態轉移歷史只有一條，且這條歷史完全由相關定律決定。**

請你停在「唯一演化」這四個字上。它說的不是「我們算得出未來」——它根本沒提到「我們」。它說的是：**現在這個狀態，只通往一個未來，不通往別的。** 這是關於世界本身的性質，跟有沒有人在觀察、算力夠不夠、量測準不準，統統無關。哲學上這叫「本體論（ontological）」性質——關於「是什麼」的，不是關於「我們知道什麼」的。

你的工程直覺裡早有一個完美對應：**純函數（pure function）**。

```text
f(x) = ...        // 同一個 x，永遠回同一個 f(x)
                  // 沒有讀時鐘、沒有讀全域變數、沒有 I/O
```

純函數就是「唯一演化」的離散工程版：同輸入只通往同輸出，不通往別的。你寫測試時依賴的就是這個——`expect(f(3)).toBe(9)` 之所以是個合理的斷言，正是因為 `f` 是決定論的。replay determinism、可重現建置（reproducible build）、bit-identical 重放，全都是你親手在系統裡安裝「唯一演化」這個性質。

決定論的反面是**非決定論（indeterminism）**：某個狀態可以通往不只一個未來，而且「選哪個」不由任何定律決定。注意這個「不由任何定律決定」很重要——它不是「我們不知道由什麼決定」，而是「真的沒有東西決定它」。量子力學的標準詮釋裡，一顆放射性原子何時衰變就是這種——不是我們算不出來，是宇宙自己還沒決定。這才是真正的本體論隨機。

**這條軸只有兩個值：決定論 / 非決定論。** 它問的問題是：「現在的狀態，通往一個未來，還是好幾個？」

## 第二條軸：隨機——關於「規則裡有沒有真的骰子」

第二條軸專門對付「隨機（randomness）」這個詞。它跟第一條軸看起來很像，但問的是一個更尖的問題：**這個系統的演化規則裡，有沒有一個「真正的隨機項」？**

所謂真正的隨機項，是指一個本體論上無法被任何前置條件決定的取值——不是「我們沒測到的變數」，而是「原則上就沒有那個變數」。擲一顆理想骰子（理想化成真隨機）、放射性衰變、量子量測，這些被認為有真隨機項。

關鍵的釐清在這裡：**「看起來亂」不等於「有真隨機項」。**

- 你的 PRNG 看起來很亂，但它的規則是 `Xₙ₊₁ = (a·Xₙ + c) mod m`——裡面**一個隨機項都沒有**。它在這條軸上站「決定論」這一側。
- π 的小數位 `3.14159265…` 看起來完全沒規律（統計上它通過所有隨機性檢定，數學家猜它是「正規數（normal number）」，每種數字串出現的頻率均等）。但 π 是一個固定的、可計算的常數——它的每一位都被它的數學定義唯一釘死。π 也站決定論這一側。
- 雜湊輸出同理：`hash(x)` 看起來亂，規則裡沒有骰子。

所以這條軸跟第一條軸其實是同一條軸的兩種問法——**有沒有真隨機項，就等於是不是唯一演化。** 我把它們分開講，是因為日常語言把「隨機」這個詞用得太鬆了：人們說「這數字好隨機」時，九成是在說「我看不出規律」（這是下一條軸的事），而不是在說「這裡有一顆真骰子」（這才是這條軸的事）。**把這兩種「隨機」分開，是本章要你拆掉的第一個混淆。**

（真隨機到底是什麼、機率分布怎麼描述它、為什麼擲骰子在統計上是可測的——這些是機率論的主場，見《馴服隨機》ch01。本書只需要「有沒有真隨機項」這一刀，不深入機率的內部。）

## 第三條軸：可預測——關於「我們能不能算出來」

第三條軸是最容易跟前兩條搞混、卻最該獨立出來的一條：**可預測性（predictability）。**

定義很簡單，但你要小心它的視角：**可預測性是「我們」能不能在實務上算出系統未來狀態的能力。** 注意這個「我們」——它出現了。可預測性是**認識論（epistemic）**的，它關於「我們能知道什麼」，不關於「世界是什麼」。它牽涉到：我們的初始量測有多準、我們的算力有多少、我們有多少時間。

SEP 那句話我認為值得你抄在便利貼上：

> **「對混沌與決定論的混淆，多半來自把決定論等同於可預測性。」**

這兩件事被等同了幾百年，因為在拉普拉斯的世界裡它們看起來真的是一回事：既然唯一演化（決定論）成立，那只要量得夠準、算得夠多，未來不就盡在掌握（可預測）了嗎？拉普拉斯惡魔（ch01）就是把這兩條軸焊死成一條的化身——「給我所有粒子的位置動量＋無限算力，過去未來盡在眼前」。

混沌做的事，一句話講完：**它把這兩條軸的焊接點敲斷了。** 它證明了存在這樣的系統——**唯一演化（決定論）完美成立，但可預測性實務上歸零。** 不是因為定律裡藏了骰子（沒有，它是純決定論的），而是因為對初始條件的指數敏感（ch03 的 SDIC）讓「量得夠準」這個前提變得永遠不夠準。你想多算遠一點，需要的初始精度就指數地往上爆——這就是 Part IV 要量化的「可預測地平線」（見 ch15）。

所以這三條軸是這樣分工的：

```text
軸                 它問什麼                          視角        值
────────────────────────────────────────────────────────────────────
決定論             現在的狀態通往一個未來還是多個？    本體論      唯一演化 / 非唯一
（determinism）     ← 關於「世界是怎樣」

隨機               規則裡有沒有真正的隨機項？          本體論      有骰子 / 沒骰子
（randomness）      ← 跟決定論其實同一刀的反面

可預測             我們實務上算得出未來嗎？            認識論      算得出 / 算不出
（predictability）  ← 關於「我們能知道什麼」
```

第一、二條軸是同一刀（有沒有骰子＝是不是唯一演化），講的是世界。第三條軸是完全獨立的另一刀，講的是我們。**混沌的全部反直覺，就濃縮在「這兩刀可以指向相反方向」這件事上**：世界這一刀說「完全決定、沒有骰子」，我們這一刀說「算不出來」。兩個都對，同時為真。

## 四象限：把三條軸壓成一張表

三條軸裡，第一條和第二條本質上是同一條（決定論 ⇔ 沒有真隨機項），所以真正獨立的是兩條：**「決定論 vs 隨機」** 和 **「可預測 vs 不可預測」**。兩條軸交叉，得到四個象限。這張表是本章的核心，請你慢慢看：

```text
                        │  可預測（算得出）        │  不可預測（算不出）
                        │  ＝統計上或實務上能掌握  │  ＝實務上無法掌握
════════════════════════╪══════════════════════════╪═════════════════════════
                        │                          │
  決定論                │  ① 行星運動              │  ② 混沌
  （唯一演化、無骰子）  │     鐘擺、彈道、純函數   │     邏輯斯諦映射、勞侖次、
                        │     replay 重放          │     三體、天氣、雙擺
                        │                          │
                        │  「拉普拉斯的甜蜜點」    │  「拉普拉斯惡魔死在這格」
                        │                          │  ★ 混沌就坐這裡 ★
                        │                          │
────────────────────────┼──────────────────────────┼─────────────────────────
                        │                          │
  隨機                  │  ③ 擲骰子、放射性衰變    │  ④ 沒有可用模型的
  （有真骰子）          │     大量分子的溫度壓力   │     雜訊／不可知過程
                        │     ← 單次測不準，       │     ← 連分布都抓不住、
                        │       但分布算得準       │       樣本也不夠
                        │                          │
                        │  「個別不知、整體可算」  │  「真的兩眼一抹黑」
                        │                          │
```

逐格走一遍，你會發現每一格你都認得：

**① 決定論 ＋ 可預測（行星運動）**——拉普拉斯的甜蜜點。兩體問題有橢圓軌道封閉解，明年今天火星在哪，算得出來。鐘擺、彈道、你的純函數、你的 replay 重放，全在這格。唯一演化成立，而且我們真的算得出。這是拉普拉斯惡魔以為的「全世界」。

**② 決定論 ＋ 不可預測（混沌）**——★ 這就是混沌的格子，本書的主角 ★。系統一條真隨機項都沒有，唯一演化完美成立（你給同一個初值，它永遠走同一條軌跡），但因為對初始條件指數敏感，你永遠量不夠準，於是實務上算不出長期未來。邏輯斯諦映射（ch05 起的脊椎）、勞侖次系統、三體、天氣、雙擺都在這格。**拉普拉斯惡魔就死在這一格**——不是因為世界不決定論了（它還是），而是因為「決定論 ⇒ 可預測」這個推論在這格是錯的。這是全書最該被你內化的一句話。

**③ 隨機 ＋ 可預測（擲骰子）**——有真骰子（理想化），所以單次結果你絕對算不出（這次擲幾點？不知道）。但**統計上可測**：擲很多次，1 到 6 各約六分之一，這個分布算得準到可以拿去開賭場賺錢。一大堆氣體分子的個別運動測不準，但溫度、壓力這些統計量穩定可算，也在這格。「個別不知、整體可算」是這格的標語。（為什麼真隨機在統計上反而很乖、可測，是《馴服隨機》的主題。）

**④ 隨機 ＋ 不可預測**——既有真隨機、連統計結構都抓不住（或樣本太短、太非平穩，分布都估不出來）。這格是「真的兩眼一抹黑」。實務上的某些不可知雜訊勉強算這格，但它在本書裡基本是個對照用的角落，不是研究對象。

我認為這張表是 ch01 到 ch03 鋪墊的那個「啊哈」的落點：**混沌不在第③格（它不是隨機），混沌在第②格（它是決定論的，但不可預測）。** 把混沌從第③格搬到第②格，就是這整本書要替你的直覺做的搬家工程。

## LCG：決定論造假亂的工程鐵證

要讓「決定但看不出規律」這件事從一句話變成你親手摸過的東西，最好的標本就是你天天在用的 PRNG。我們把它縮到能用紙筆走完的最小模型——一個小模數的線性同餘產生器（linear congruential generator, LCG）：

```text
Xₙ₊₁ = (5·Xₙ + 1) mod 16，    X₀ = 1
```

讀法：每一步把目前的數乘 5、加 1，再對 16 取餘數（只留 0 到 15）。`5` 是乘數、`1` 是增量、`16` 是模數、`1` 是種子（seed）。這四個數一旦定下，**整串輸出就被完全鎖死**——這正是第②格那個「唯一演化」的最小演示。我把每一步都算給你看（你可以拿筆自己驗，這是本章的 worked example）：

```text
n   │ 5·Xₙ + 1        │ mod 16            │ Xₙ₊₁
────┼─────────────────┼───────────────────┼──────
0   │ 5·1  + 1 = 6    │ 6                 │ 1   ← 種子
1   │ 5·6  + 1 = 31   │ 31 − 16 = 15      │ 6
2   │ 5·15 + 1 = 76   │ 76 − 64 = 12      │ 15
3   │ 5·12 + 1 = 61   │ 61 − 48 = 13      │ 12
4   │ 5·13 + 1 = 66   │ 66 − 64 = 2       │ 13
5   │ 5·2  + 1 = 11   │ 11                │ 2
6   │ 5·11 + 1 = 56   │ 56 − 48 = 8       │ 11
7   │ 5·8  + 1 = 41   │ 41 − 32 = 9       │ 8
8   │ 5·9  + 1 = 46   │ 46 − 32 = 14      │ 9
9   │ 5·14 + 1 = 71   │ 71 − 64 = 7       │ 14
10  │ 5·7  + 1 = 36   │ 36 − 32 = 4       │ 7
11  │ 5·4  + 1 = 21   │ 21 − 16 = 5       │ 4
12  │ 5·5  + 1 = 26   │ 26 − 16 = 10      │ 5
13  │ 5·10 + 1 = 51   │ 51 − 48 = 3       │ 10
14  │ 5·3  + 1 = 16   │ 16 − 16 = 0       │ 3
15  │ 5·0  + 1 = 1    │ 1                 │ 0
16  │ 5·1  + 1 = 6    │ 6                 │ 1   ← 回到種子，整串重來
```

把 X₀ 到 X₁₅ 那一列抽出來，這就是這台「亂數產生器」吐出的序列：

```text
1, 6, 15, 12, 13, 2, 11, 8, 9, 14, 7, 4, 5, 10, 3, 0,  然後 1, 6, 15, ... 無限重複
```

現在請你做兩件事，這兩件事是這個 worked example 的全部重點：

**第一，盯著這串數字找規律。** `1, 6, 15, 12, 13, 2, …`——它上上下下亂跳，沒有遞增、沒有週期性的小花樣、沒有明顯的等差等比。如果我把這 16 個數洗一洗給你，跟一串真擲 16 面骰子的結果擺在一起，你**肉眼分不出哪串是 LCG、哪串是骰子**。它徹底「看起來隨機」。這就是第②格的「不可預測」的玩具版——當然，這裡的「不可預測」是對不知道規則的人而言；知道規則就秒算，所以它沒有混沌那種「知道規則也測不準」的本事（那個本事來自指數敏感，是 ch14 的事）。但「看不出規律」這個表象，它演得有模有樣。

**第二，現在我告訴你它藏的鐵律。** 這串數字一點都不亂——它**剛好把 0 到 15 每個數字各走過一次，一個不漏，然後才回到起點**。這叫「滿週期（full period）」。0、1、2、…、15 全到齊，這不是巧合，是這四個參數（5, 1, 16, seed=1）的數學結構保證的（滿足赫爾–多貝爾條件，Hull–Dobell theorem）。所以這台機器有一個你肉眼看不見、卻鐵一般成立的性質：**它在「看起來最亂」的表象底下，藏著「每個值恰好出現一次」的完美秩序。**

我認為這個 worked example 是整章的縮影，因為它把全書的中央張力先預演了一遍：**表面的亂（看不出規律）和底層的鐵律（滿週期、完全確定）在同一個東西裡共存。** 第②格的混沌就是這件事的「升級版」——把 LCG 換成一條物理定律（或邏輯斯諦映射），把「看不出規律」升級成「即使知道規則、也因指數敏感而算不準長期」，秩序與亂的共存就從工程玩具變成了自然界的鐵律。

**對照真擲骰。** 拿一顆真的 16 面骰子擲 16 次，跟上面那串並排：

```text
LCG（決定論）   ：1, 6, 15, 12, 13, 2, 11, 8, 9, 14, 7, 4, 5, 10, 3, 0
真骰子（隨機）  ：11, 4, 4, 9, 16, 2, 11, 7, 1, 1, 13, 8, 8, 5, 16, 3   ← 一種可能結果
```

兩個關鍵差別，正好對應前面兩條軸：

1. **真骰子的下一個值，宇宙自己都還沒決定**（第②/③格的本體論差別）——你重置不了它，它沒有 seed。LCG 你只要 `seed=1` 重來，永遠是 `1, 6, 15, 12, …`，一位不差（這就是 replay determinism）。
2. **真骰子允許重複**（上面那串有兩個 4、兩個 1、兩個 8、兩個 16），因為每次擲都獨立、不記得前面擲過什麼；**LCG 在一個週期內不重複**（它記得——其實是它被規則逼著走完所有值才回頭）。「真隨機會撞重複、決定論的假亂在週期內反而不重複」這個反差，本身就是一條好用的鑑別線索。

這就是「決定論造假亂」的工程鐵證：你手上這台機器，規則裡一顆骰子都沒有，輸出卻能假亂到騙過肉眼、騙過統計檢定——而且它假亂的底下還藏著鐵律。混沌做的是同一件事，只是它不用人類設計，自然界自己就會。

## 直覺的陷阱

混沌是人類直覺最常崩壞的地方，而崩壞幾乎都從「把三條軸當一條」開始。這一章守的是兩個最致命的混淆——它們是 landscape 標記的 T3 與 T4，也是整本書反覆要替你校正的兩顆螺絲。

| 陷阱 | 錯誤直覺長怎樣 | 它在哪一步把你帶溝裡 | 正確版（怎麼自我察覺） |
|---|---|---|---|
| **T4：決定論 ＝ 可預測** | 「既然系統是決定論的，那原則上就算得出未來，只是工程上難一點。」 | 你會把混沌的不可預測誤判成「電腦不夠快／模型不夠好」的工程問題，以為加算力、加精度就能突破。於是你永遠不理解可預測地平線（ch15）為什麼是「買不過去」的牆。 | 決定論是本體論（唯一演化），可預測是認識論（我們算不算得出）——兩條獨立的軸。混沌證明了「決定論 ⇏ 可預測」。**徵兆**：當你說「理論上算得出」時，停下來問「是『世界有唯一答案』還是『我們能算到那個答案』？」——這兩件事差很遠的時候，你就站在第②格了。 |
| **T3：混沌 ＝ 隨機** | 「混沌系統那麼亂、那麼測不準，那它本質上就是隨機的吧。」 | 你會把混沌丟進機率／隨機過程的框架去處理，去找它的「機率分布」當成全部真相，而錯過它真正的身分——一條**完全確定**的規則。你會看不到它有形狀（相空間、奇異吸子，Part III）、有機制（拉伸摺疊，ch16）、有鐵律（Feigenbaum，ch08）。隨機沒有這些，混沌全都有。 | 混沌是**完全決定論**的：同初值給同軌跡、規則裡沒有真隨機項。表面的「亂」來自對初值的敏感依賴加上粗粒化觀察，是**認識論**的不可預測，不是**本體論**的隨機（SEP：chaotic behavior is always deterministic）。**徵兆**：你能不能重置它？混沌給它同一個 seed（初值）能完全重來，真隨機不能——這是區分第②格與第③格最快的試金石。 |

兩個陷阱其實是同一個病的兩種發作：**把「我們算不出來」（認識論）誤當成「世界裡有骰子／世界沒有定數」（本體論）。** 混沌偏偏就是「世界完全有定數、我們卻算不出來」這個夾縫——它逼你把這兩層分開。分不開，你就會在 T3 和 T4 之間來回跌。

（一個延伸提醒，留到 ch17 處理：肉眼看，混沌的時間序列和真雜訊長得幾乎一樣。要把它們**真正分辨開**，需要相空間重建那套技術——這裡先把框架立好：混沌在第②格、雜訊在第③/④格，原則上是兩種東西。怎麼動手分辨是 ch17 的事。）

## 紙上推演

下面三題都是純口頭／紙筆推演，不要碰鍵盤。重點不是算得多快，是能不能把每一步的「為什麼」講出來——講得出來才算這一章過關了。

---

**推演題 1 ★ [10 分鐘] — 把你系統裡的東西填進四象限**

回到你自己的後端系統，替四個象限各找**一個真實例子**（不能用本章舉過的）。然後回答這個關鍵子題：**retry 的 jitter（那個「隨機」抖動）落在哪一格？** 把你的判斷理由講清楚。

---

### 推演解答

四個象限的範例沒有標準答案，但好的填法應該長這樣（你的可以不同，但理由要站得住）：

- **① 決定論 ＋ 可預測**：純函數的單元測試（同輸入同輸出，斷言算得出）；事件溯源的重放（replay 給 bit-identical 結果）；一個固定種子的雪花 ID 生成器在受控環境下的序列。共通點：唯一演化成立，而且你真的算得出、敢拿去寫斷言。
- **② 決定論 ＋ 不可預測**：依賴執行緒排程時序的並行 bug（race condition）——規則全是確定的（沒有骰子），但實務上「哪個執行緒先到」對奈秒級的時序指數敏感，於是復現不出來、預測不了。這是你系統裡最像混沌的東西。（嚴格說它的不可預測來自難以控制的外部時序而非純 SDIC，但「決定卻測不準」的味道是對的。）
- **③ 隨機 ＋ 可預測**：負載均衡下單台機器的瞬時請求數（單次測不準，但長期分布／平均 QPS 穩定可估，能拿去做容量規劃）；A/B 測試的單次分流結果不知道、但整體比例算得準。
- **④ 隨機 ＋ 不可預測**：一個全新、樣本極少、又非平穩的外部 API 延遲——連分布都估不出來，這禮拜的統計到下禮拜就失效。

至於 **retry 的 jitter**：這是本題的陷阱題，答案取決於你怎麼問——

- 從**機制**看，jitter 多半是 PRNG 給的，PRNG 是**決定論**的（同 seed 同序列）。所以「jitter 的產生」嚴格屬於第①/②格那一側（決定論），不是真隨機。
- 但從**你使用它的視角**看，你要的效果是「讓各節點的重試時刻彼此獨立、像撒開的」，你把它**當成隨機在用**，而且在「整體上它有效打散了重試」這個意義上它是**統計可測**的（第③格的味道）。

所以正確的回答不是選一格，而是點破：**jitter 是「決定論的機制，被當成隨機來用」**——它正是本章開頭那句「決定論的假隨機」的活標本。如果你脫口說「jitter 是隨機的」，停一下：它有 seed 嗎？能重放嗎？能的話，它就不是第③格的真骰子。

---

**推演題 2 ★★ [15 分鐘] — 換一組 LCG 參數，手算驗證「決定但看不出規律」**

把本章的 LCG 改成 `Xₙ₊₁ = (3·Xₙ + 1) mod 8`，種子 `X₀ = 0`。手算迭代直到序列開始重複，寫出完整序列。然後回答：(a) 它的週期是多少？(b) 它有沒有走遍 0 到 7 所有值？(c) 這台機器跟本章那台（滿週期 16）相比，哪台「假亂」假得比較好？為什麼？

---

### 推演解答

逐步手算（乘 3、加 1、mod 8）：

```text
X₀ = 0
X₁ = (3·0 + 1) mod 8 = 1  mod 8 = 1
X₂ = (3·1 + 1) mod 8 = 4  mod 8 = 4
X₃ = (3·4 + 1) mod 8 = 13 mod 8 = 5
X₄ = (3·5 + 1) mod 8 = 16 mod 8 = 0   ← 回到 X₀，開始重複
```

序列：`0, 1, 4, 5, 0, 1, 4, 5, …`

(a) **週期是 4**（`0, 1, 4, 5` 然後重來）。

(b) **沒有**走遍 0 到 7——它只碰到 `{0, 1, 4, 5}` 這四個值，`2, 3, 6, 7` 從沒出現。這是因為這組參數**不滿足滿週期條件**（乘數 3 與這個模數的搭配不對），所以週期只有模數 8 的一半。

(c) **本章那台（滿週期 16）假得比較好。** 兩個理由：第一，週期 16 比週期 4 長得多，短週期一眼就會被看穿在循環（`0,1,4,5,0,1,4,5…` 你數第二輪就發現它在重複了）；第二，滿週期那台 16 個值全用上、分布均勻，這台只用了一半的值、`2,3,6,7` 永遠不出現，做卡方檢定會立刻露餡。**這題的教訓**：「決定論能不能假亂得逼真」極度依賴參數選擇——同樣是決定論的 LCG，好參數騙得過統計檢定，壞參數一眼穿幫。這也是為什麼真實 PRNG 的乘數／模數都是被精心挑過、甚至註冊在案的。決定論造亂是一門工程，不是隨便乘一乘取餘數就成。

**推演題 3 ★★ [12 分鐘] — 抓出論證的破綻**

有人對你說：「混沌系統測不準，所以它一定有某種我們還沒發現的隨機性藏在裡面——不然怎麼會算不出來？決定論的東西不可能算不出來。」請找出這段論證的破綻在哪，並用一句話講清楚他混淆了哪兩條軸。

---

### 推演解答

破綻在最後那句**隱藏前提**：「決定論的東西不可能算不出來」。這句話把「決定論（唯一演化，本體論）」直接等同於「可預測（我們算得出，認識論）」——這正是 T4 陷阱、也正是拉普拉斯把兩條軸焊死的那個假設。混沌的整個存在，就是這個前提的反例：**一個系統可以唯一演化（決定論成立、沒有任何隱藏的隨機性）、同時實務上算不出長期未來（不可預測）。**

他的推論「測不準 ⇒ 一定有隱藏的隨機性」是錯的，因為「測不準」有兩種完全不同的來源：(1) 本體論的——規則裡真有骰子（第③/④格）；(2) 認識論的——規則完全確定，但對初值指數敏感，使我們永遠量不夠準（第②格）。混沌是第 (2) 種。他預設只有第 (1) 種來源，於是反推出一個不存在的「隱藏隨機性」。

**一句話**：他把「不可預測」（認識論）誤當成「隨機」（本體論）的證據，也就是同時踩了 T3（混沌＝隨機）和 T4（決定論＝可預測）——把「我們算不出來」錯讀成「世界裡有骰子」。正確的回答是：混沌測不準，不是因為它藏了骰子，而是因為它對初值敏感到讓「決定論 ⇒ 可預測」這個推論在它身上失效。

2. SEP 說「對混沌與決定論的混淆，多半來自把決定論等同於可預測性」——這句話在罵的具體是哪個推論？混沌怎麼當它的反例？

4. 你的 PRNG 看起來很亂，為什麼它**不是**隨機的（第②格而非第③格）？給我一個一秒鐘就能區分它和真骰子的操作性測試。
5. 本章那台滿週期 LCG，「最亂的表象」底下藏的「鐵律」是什麼？這跟全書的中央張力（敏感與鐵律共存）有什麼關係？
