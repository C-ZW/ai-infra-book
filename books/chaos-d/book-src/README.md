# 蝴蝶的鐵律

### 混沌理論的直覺與驚嘆

> 一條完全確定的規則，可以製造出原則上算得出、實務上測不準的行為；而這份「測不準」本身，又服從鐵一般、跨系統不變的普適律。
>
> 這本書想讓你親身感受這一刀——並且能用自己的話，對另一個工程師講清楚：**拉普拉斯的惡魔，到底死在哪一刀。**

---

這是一本寫給資深後端工程師的混沌理論「欣賞」書。它不教你解微分方程、不要你跑任何程式、不發 lab 作業。它只想帶你看懂一件事：**決定論（同樣輸入給同樣輸出）和不可預測，可以同時為真**——而且當系統測不準時，它測不準的「方式」竟然藏著一個跨越所有系統、像物理常數一樣硬的數字。

你已經在工作裡碰過混沌的影子，只是沒這樣叫它：浮點不可重現的 bug、`abs(a-b) < eps`、autoscaler 在兩個副本數之間 flapping、級聯故障的指數放大、retry 迴圈、PRNG 的「假亂」。這本書把這些直覺接上一個更大的故事。

## 這本書怎麼讀

- **純紙上推演**：每章的「動手做」是口頭自答與思辨題，不是要你開終端機。數值示範一律手算、攤在你眼前。
- **直覺與圖像優先**：必要的式子一定先講「它在說什麼」再給形狀，不堆推導。書裡的圖（分岔圖、Lorenz 蝴蝶吸子、Mandelbrot 集……）是作者預先算好的插圖，給你看的，不是要你跑的。
- **一條脊椎貫穿全書**：整本書反覆回到同一條遞迴式 `xₙ₊₁ = r·xₙ·(1 − xₙ)`。轉動唯一的旋鈕 r，你會親眼看著它從一個穩定值，分裂成混沌，再從混沌裡長出秩序的孤島。ch19 會把它七層意義一次對帳。
- **故障視角是賣點**：混沌正是人類直覺最常崩壞的領域。每章的「直覺的陷阱」段，專門守住那些被講爛了的庸俗版（「蝴蝶拍翅一定引發龍捲風」之類），還你正確版。

## 書架上的鄰居

這本書只用自己的欣賞聲音講「為何美、為何反直覺」；需要更深的運算時，它會把你指向書架上相鄰的書，而不是重講一遍：

- 隨機、機率、白噪音、不變密度 →《馴服隨機》
- 微分方程、Lorenz 系統、相空間軌跡、導數 →《馴服無限》
- Jacobian 特徵值、線性化、對數伸縮率 →《矩陣是動詞》

準備好了，就從 **ch01 — 拉普拉斯的承諾** 開始。你的直覺正準備在接下來每一章騙你一次——這正是值得慢慢讀的地方。
