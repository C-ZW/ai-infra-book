# 附錄 A — 記號與術語速查

> 這是一張**速查表**，不是教學。每個記號、每條公式都假設你已在正文某章遇過它；這裡只負責讓你在忘記「δ 唸什麼、α 是正是負、r∞ 到底幾位」時，能在十秒內查回來，並標出它**第一次正式登場**的章。要從零理解某個概念，回該章；要核對一個數字或一個符號的確切意義，留在這裡。

閱讀器不渲染 LaTeX，全書一律純文字 Unicode 記號（內文行內就是 xₙ、x*、f′(x) 這種寫法）。所以這張表也是純文字的——你看到的形狀就是書裡的形狀。

```text
• 下標 ₙ ₙ₊₁ ₀ 是「序列的第幾項」；上標（如 2³⁰、e^λ、10⁻⁹）是「次方」。
  位置不同、意義天差地遠——下標數項數、上標數冪次。
• |·| 在一維（logistic）是絕對值、在多維（Lorenz 的 Jacobian、見《矩陣是動詞》）
  是矩陣特徵值的「模」。同一個符號，維度變了意思跟著變。
```

## 二、同形記號歧義專節（重要）

混沌理論借用的希臘字母不多，於是同一個字母在不同脈絡常指不同東西。下面五條是本書（及書架相鄰書）最容易撞名的記號。**這節不是吹毛求疵——查一個數字前先確認你查的是哪個 λ、哪個 α，否則會把對的數字填到錯的格子裡。**

```text
λ —— 本書：Lyapunov 指數（相鄰軌跡分離的指數率，λ>0 ⇔ 混沌）。     ch14
     易撞：在《矩陣是動詞》與一般線性代數，λ 是矩陣的「特徵值
     （eigenvalue）」。兩者其實有血緣——多維不動點的穩定性正是看
     Jacobian 特徵值的模是否 <1，而 Lyapunov 指數可視為「長期平均
     的對數伸縮率」（見《矩陣是動詞》談特徵值的章）——但在本書，
     λ 一律指 Lyapunov 指數，不是某個矩陣的特徵值。看到 λ 先問：
     這是「軌跡發散速率」還是「矩陣的伸縮因子」？

α —— 本書：Feigenbaum 尺度常數，印**正量值** α ≈ 2.5029。          ch08
     陷阱：α 的**嚴格符號其實是負的**（−2.5029），那個負號記錄
     「每放大一層、圖上下翻轉一次」。本書照常見慣例印正量值並全書
     一致；你只要知道符號是「翻轉的標記」、量值（2.5029）才是收縮率。
     別在別處看到 −2.5029 就以為和本書矛盾——是同一個常數的兩種寫法。
     （α 在物理別處也常指角度、係數等，與本書無關。）

r —— 本書：邏輯斯諦映射的控制參數（唯一旋鈕，0 ≤ r ≤ 4）。          ch05
     陷阱：Lorenz 文獻常把本書寫成 ρ 的那個參數寫成 r（甚至有的
     教材把 logistic 的 r 寫成 a 或 μ）。**本書 logistic 一律用 r、
     Lorenz 一律用 ρ**，兩者不混。看到 r 就是 logistic 的旋鈕；
     看到 Lorenz 的加熱參數，它叫 ρ=28，不叫 r。

σ ρ β —— 本書：Lorenz 系統的三個參數，固定取 σ=10、ρ=28、β=8/3。   ch11
     注意 ρ 在這裡是 Lorenz 參數，**不是**《馴服隨機》裡當「相關
     係數 / 密度」的 ρ，也不是 logistic 的 r。σ 在這裡是 Lorenz
     的 Prandtl 數，**不是**機率書裡的標準差 σ。三個字母在本書範圍
     內只有 Lorenz 這一個身分；跨到機率書要重新對表。

D —— 本書：碎維度（fractal dimension），可為分數（如 1.2619）。     ch13
     易混：D 在別處可指「定義域」「資料集」「微分算子」等。本書
     D 永遠是「維度」這個量——而且重點正是它可以不是整數。
```

一句總結：**λ、α、r、σ/ρ/β、D 在本書內部各自只有一個身分，跨到書架上別本書才需要重新對表。** landscape 的待考 #3（α 符號）、#6（同形記號）就是這節的來源。

## 三、關鍵公式速查

每條先一句「它在說什麼」，再給形狀，最後標首次出現章。脊椎遞迴式是全書貫穿的同一條式子（見 ch05），其餘公式都是繞著它或繞著「敏感 vs 鐵律」這條中央張力長出來的。

### 脊椎：邏輯斯諦映射

```text
它在說什麼：把「目前狀態」餵回去算「下一步」的離散回授迴圈。
            r 是成長率、(1−x) 是擁擠抑制（backpressure）。r 是唯一旋鈕。
公式：       xₙ₊₁ = r·xₙ·(1 − xₙ)，   0 ≤ x ≤ 1，   0 ≤ r ≤ 4
首次：       ch05（ch03 先借用、明寫「第 5 章正式介紹」）
```

### 不動點與穩定判據

```text
它在說什麼：不動點是「餵進去等於吐出來」的值；穩不穩定看斜率夠不夠平。
            |斜率|<1 → cobweb 彈回（吸引）；|斜率|>1 → 彈開（排斥）。
不動點：     x* = 1 − 1/r                          （解 x*=f(x*) 得到）
導數：       f′(x) = r(1 − 2x)
代入不動點： f′(x*) = 2 − r
穩定判據：   |f′(x*)| < 1   ⇔   1 < r < 3 穩定      （r=3 臨界，|f′|=1）
首次：       ch06（x* 在 ch05 登場，穩定性在 ch06 講透）
```

### Lyapunov 指數

```text
它在說什麼：把「對初始敏感」壓成一個數——相鄰軌跡每步分離的對數速率。
            離散版：每步誤差乘 |f′|，λ 是沿真實軌跡 ln|f′| 的長期平均。
公式：       λ = 長期平均的 ln|f′(xₙ)|
判據：       λ > 0  ⇔  混沌；  λ < 0  ⇔  穩定；  λ = 0  ⇔  臨界
脊椎特例：   r = 4 時 λ = ln2 ≈ 0.6931（每步誤差約翻倍）
首次：       ch14
```

### 可預測地平線

```text
它在說什麼：精度 ε 與發散率 λ 一起決定的「時間牆」——能預測多久。
            精度×10（ε 縮小 10 倍）只多買固定一段時間 (1/λ)·ln10。
公式：       T ≈ (1/λ)·ln(1/ε)
首次：       ch15（λ 與 1/λ 的尺度在 ch14 先備好）
```

### 碎維度（自相似維度）

```text
它在說什麼：把「有多碎」變成一個數——細節如何隨尺度增生。可為分數。
            把整體拆成 N 份自相似複製、每份縮成原來的 s（1/s 是放大倍率）。
公式：       D = ln N / ln(1/s)
代入示範：   Koch  N=4、s=1/3 → D = ln4/ln3 ≈ 1.2619（介於 1 與 2）
             Cantor N=2、s=1/3 → D = ln2/ln3 ≈ 0.6309（介於 0 與 1）
首次：       ch13
```

### Mandelbrot 迭代

```text
它在說什麼：碎形家族裡和脊椎同血緣的迭代——問「哪些 c 讓序列不發散」。
            和 logistic 同屬「拿規則套在自己輸出上」的 z→z²+c 家族。
公式：       zₙ₊₁ = zₙ² + c      （z、c 是複數）
首次：       ch12
```

### Lorenz 系統（三式）

```text
它在說什麼：一台有摩擦的對流機——三條 ODE 給連續時間的相空間運動。
            寫出來給直覺（負回授追趕、x·z 與 x·y 兩個非線性乘積是火種），
            本書不要求解。
公式：       dx/dt = σ·(y − x)
             dy/dt = x·(ρ − z) − y
             dz/dt = x·y − β·z
參數：       σ = 10、 ρ = 28、 β = 8/3 ≈ 2.6667
首次：       ch11（ch03 先點名 0.506 故事的來源系統）
```
