# 寫作風格指南（內部文件，非書籍內容）

本文件是全書的寫作契約。每一章都必須遵守，否則全書會失去一致性。
本書書名（工作名）：《蝴蝶的鐵律：混沌理論的直覺與驚嘆》。

## 這本書是什麼、不是什麼

- **是**：一本「欣賞與直覺」的書。目標是讓一位資深後端工程師親身感受混沌理論最反直覺的那一刀——**一條完全確定的規則，可以製造出原則上算得出、實務上測不準的行為；而這份「測不準」本身，又服從鐵一般、跨系統不變的普適律**。讀完要有「原來決定論和不可預測可以同時為真、原來亂裡面藏著一個跨宇宙不變的常數」的震撼，**而且能用自己的話對另一個工程師講清楚震撼在哪、Laplace 惡魔死在哪一刀**。
- **不是**：
  - 不是教科書——不推導 ODE、不證明 Sharkovskii、不做符號動力學的嚴格建構；定理給直覺、給條件、給結論，嚴格證明指向延伸閱讀。
  - 不是「動手做」實作書——**全書純紙上推演，零可執行 lab、零要讀者去跑的程式**（2026-06-20 建書設定，與《把系統寫成定理》tla-theory、《馴服隨機》probability 同調）。書中插圖是作者預先算好的程式生成 SVG，那是給讀者看的圖、不是要讀者跑的程式。
  - 不是數學操練——不刷迭代計算題、不堆公式表；必要的式子一定先講透「它在說什麼」再給形狀，直覺與圖像優先、硬推導一律省。
  - 不是轉職用書——純興趣，不服務 AI Infra 敘事；但工程橋接要讓它讀起來像母語，不是外語。
- **驗收標準（全書六條主線，ch19 收官口試）**：
  ① **決定論 ≠ 可預測**：混沌殺死的是「決定論 ⇒ 可預測」這個推論，不是決定論本身；說得清 Laplace 惡魔死在哪一刀。
  ② **敏感依賴（SDIC／蝴蝶效應）的正確意義**：測不準的來源是「對初始條件指數敏感」，不是「微小原因必然造成某個特定大果」。
  ③ **鐵律——普適性**：不同的方程式、同一條通往混沌的路、同一個 Feigenbaum δ≈4.6692（本書最大驚嘆）。
  ④ **混沌有形狀**：相空間、奇異吸子、碎形與碎維度——亂之中的幾何秩序。
  ⑤ **機制與極限**：拉伸＋摺疊製造混沌、Lyapunov 指數把敏感量化、可預測地平線（天氣約兩週）為何不是「電腦不夠快」能解。
  ⑥ **混沌 ≠ 雜訊，且可被駕馭**：低維確定性 vs 高維隨機；控制與應用；哲學收束（謹慎、不過度宣稱）。

## 全書脊椎：同一條遞迴式（對應微積分那本的「同一題三算」、機率那本的「同一枚硬幣」）

一條遞迴式是全書的貫穿範例，每深入一個 Part 就重新認識它一次。各章在用到時**必須明寫「這是 ch05 那條遞迴式」**，並在 ch19 總對帳：

```text
邏輯斯諦映射（logistic map）：xₙ₊₁ = r·xₙ·(1 − xₙ)，    0 ≤ x ≤ 1，  0 ≤ r ≤ 4
```

- ch05 登場：一條離散回授迴圈（把「目前狀態」餵回「下一步」），r 是唯一旋鈕，cobweb（蛛網圖）讀法。
- ch06 不動點與穩定：x* = 1 − 1/r；穩定看 |f′(x*)| < 1，而 f′(x*) = 2 − r，所以 1 < r < 3 穩定。
- ch07 倍週期分岔：r 過 3 → 2-cycle，過 1+√6 ≈ 3.4495 → 4-cycle，級聯加倍。
- ch08 費根堡普適：分岔間距比 → δ ≈ 4.6692，**換成別的單峰映射還是同一個 δ**（鐵律）。
- ch09 混沌與秩序孤島：越過 r∞ ≈ 3.56995 進混沌帶；帶內有 period-3 窗口（r = 1+√8 ≈ 3.8284）。
- ch14 Lyapunov：r = 4 時 λ = ln2 ≈ 0.6931，每步誤差約翻倍（把 SDIC 變成一個數）。
- ch16 拉伸與摺疊：r = 4 共軛於帳篷映射＝拉伸 2 倍＋對摺，就是 λ = ln2 的來源、也是混沌的製造機制。
- ch19 收官：「同一條遞迴式，現在你懂它七層」總對帳表。

中央張力（全書反覆叩問的那句話）：**敏感（蝴蝶）與鐵律（普適）如何在同一條式子裡共存。** 每章都該至少碰一次這條張力。

## 數學記號規範（全書一致，附錄 A 是速查表；閱讀器不渲染 LaTeX，一律純文字 Unicode）

- **內文行內**用 Unicode：xₙ、xₙ₊₁、x*（不動點）、r（控制參數）、f(x)、f′(x)、f(f(x))、δ（Feigenbaum 間距常數）、α（Feigenbaum 尺度常數）、λ（Lyapunov 指數）、ε（誤差／擾動）、σ ρ β（Lorenz 參數）、≈、≠、≤、≥、→、∞、√、ln、∈、|·|（絕對值／模）、×、·（乘）。
- 脊椎遞迴式一律寫 `xₙ₊₁ = r·xₙ·(1 − xₙ)`；不動點 `x* = 1 − 1/r`；穩定判據 `|f′(x*)| < 1`，其中 `f′(x) = r(1 − 2x)`、`f′(x*) = 2 − r`。
- Lyapunov：`λ = 長期平均的 ln|f′(xₙ)|`；`λ > 0 ⇔ 混沌`；可預測地平線 `T ≈ (1/λ)·ln(1/ε)`。
- Lorenz：`dx/dt = σ(y − x)`、`dy/dt = x(ρ − z) − y`、`dz/dt = xy − βz`，σ=10、ρ=28、β=8/3。寫出來給直覺，**不要求解**。
- 碎維度：`D = ln N / ln(1/s)`（N 份、每份縮為原來的 s）；Mandelbrot 迭代 `zₙ₊₁ = zₙ² + c`。

- 數字基準：e ≈ 2.71828、ln2 ≈ 0.6931、√6 ≈ 2.4495、√8 ≈ 2.8284。跨章基準值（r∞、δ、α、λ、Lorenz 三數、碎維度…）以 outline.md 基準表與 landscape 為準，全書統一、不得互相矛盾。所有迭代值寫章時自己手算複核、保留 4 位小數。
