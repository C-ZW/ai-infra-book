# 思考的陷阱

### 邏輯謬誤與認知偏誤的直覺與驚嘆

> 謬誤不是愚蠢，是「在錯的脈絡用對的捷思」——而且你察覺不到自己正在犯。
>
> 這本書想讓你親身感受這一刀——並且能用自己的話，對另一個工程師講清楚：**為什麼大多數認知偏誤不是腦子壞了，而是一個對的捷思被錯置到不該用的脈絡，以及為什麼「察覺不到」本身，才是整本書真正的標靶。**

---

這是一本寫給資深後端工程師的謬誤與偏誤「欣賞」書。純興趣、和工作無關、純紙上推演——它不教你證明命題邏輯的完備定理、不要你跑任何程式、不給你一份「讀完就免疫」的偏誤清單。它只想帶你看懂一件反直覺的事：**你那顆天天做決定的腦袋，跑著一堆「夠快、夠省、平均夠好」的捷思，而大多數判斷錯誤不是因為你不夠聰明，是因為你在毫無感覺的情況下，把一個對的工具用在了錯的場合。**

你其實已經在工作裡碰過這些陷阱的影子，只是沒人用這些名字跟你講過：確認偏誤式 debug——先有「一定是 DB」的假設、只翻支持的 log、繞遠路；alert fatigue 的數學——告警命中率高但真故障基率低，於是「收到告警時真有事」的機率低得反直覺；「改了 A 同時 B 好了」的因果歸因陷阱；估點被第一個人喊的數字錨住；回滾 vs 前進的損失感受不對稱；「這週沒出事所以該出事了」的維運迷信。這本書把這些工程直覺接上一個更大的故事。

## 這本書怎麼讀

- **純紙上推演**：每章的「紙上推演」是手填論證結構表、判斷有效性、口頭自答的思辨題，不是要你開終端機。所有數值示範一律攤在你眼前的 ` ```text ` 表裡。
- **機制與脈絡優先**：每個偏誤先回答「它在解決什麼問題（捷思原本對在哪、省了什麼）」「它在哪個脈絡翻車」「為什麼當事人察覺不到」，再給名字。你不必背清單，你要理解機制。
- **誠實是賣點**：本書有一章（ch17）專門誠實地說：這門研究人類偏誤的學問，自己也中了偏誤的招。ego depletion、power posing、社會促發、面部回饋、backfire effect——這些名研究在再現性危機裡的複製狀態，本書一條條標出來。
- **講得出來才算過**：每章末的「自我檢核」是口頭題——對齊「能不能把這個偏誤的對的捷思/錯的脈絡講給另一個工程師聽」的深度。講得出來，才算真的懂了。

## 一條脊椎貫穿全書

整本書反覆回到**同一顆會反覆上當的大腦**，用雙系統透鏡一層一層重看：

```text
雙系統透鏡：System 1（快思）  vs  System 2（慢想）

  S1：自動、平行、省力、永遠開著、給出「感覺對」的答案
     工程橋接：cache 命中、熱路徑
  S2：刻意、序列、費力、容易累、會偷懶把 S1 的答案放行
     工程橋接：冷路徑、慢查詢、人工覆核

  認知偏誤 = S1 跑了一個捷思 → S2 沒攔下來 → 錯置到不該用的脈絡
  形式謬誤 = 推論的「邏輯結構」本身無效（與內容、捷思無關）
            → S1 憑「聽起來合理」就放行 → S2 沒做結構檢查
```

中央張力（全書反覆叩問的那句話）：**謬誤不是愚蠢，是「在錯的脈絡用對的捷思」——而且你察覺不到自己正在犯。**

## 書架上的鄰居

這本書只用自己的欣賞聲音講「為何反直覺、捷思如何在錯的脈絡翻車」；遇到它的鄰書主場時，它點到為止、把你指過去，而不是重講一遍：

- 機率機制（基率、貝氏更新、合取律證明、大數法則的稀釋機制、回歸均值的統計推導）→《馴服隨機》
- 規範基準（理性上該怎麼選、期望效用、展望理論的數學、沉沒成本規範理由）→《在不確定中下注》（叢集姊妹書）
- 制度防線（科學如何防自欺、對照實驗、預先註冊、p-hacking 機制、複製率方法論）→《如何不騙自己：科學方法》（叢集姊妹書）
- 哥德爾定理的精確陳述與條件（本書只給哥德爾濫用的「為何是濫用」）→《這句話無法被證明》（叢集姊妹書）
- 策略情境下的非理性公平偏好（本書只點到為止）→《當理性互相碰撞》（叢集姊妹書）

這五本書加上本書，構成「理性三角」與「推理六書」叢集：本書管心理機制（人怎麼錯）、科學方法管制度防線（怎麼防自欺）、決策書管規範基準（正確答案是什麼）。三角的三個角，各管各的地盤。

## 給誰讀

唯一讀者：一位 6 年以上的資深後端工程師。你不需要任何邏輯學或心理學背景——你需要的直覺，確認偏誤式 debug、alert fatigue、postmortem 歸因、sunk cost、回滾的損失感早就教會你了。本書把這些工程經驗當錨點，從「這台機器為什麼這樣判斷」起步，一路帶你看到「這門學問自己也中了招」。

準備好了，就從 **ch01 — 腦裡的快與慢：你察覺不到的那個答主** 開始。你的直覺正準備在接下來每一章騙你一次——而你偏偏不會有感覺。這正是值得你停下來、慢慢讀的地方。
