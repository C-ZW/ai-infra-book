# 附錄 A — 記號與讀法速查表

這本書的驗收方式是「用自己的話講給另一個工程師聽」，所以每一個記號都必須**唸得出來**——不是視覺上認得，而是能在口頭講解時自然說出口。下表彙整全書實際出現過的記號（不收書中沒用到的），分六組，給出寫法、怎麼唸、白話意義、首次出現章。當你卡在某個符號講不順時，回來查「怎麼唸」那一欄，把它讀順了再往下講。

## 角與函數

| 記號（寫法） | 怎麼唸（口頭講解用） | 白話意義 | 首次出現章 |
|---|---|---|---|
| `θ` | 「西塔」（theta） | 角度的預設代號；圓上一點轉了多少（見 ch02） | ch02 |
| `φ` | 「fai」（phi） | 相位、相位差、合成波的相位角（見 ch10、ch12） | ch10 |
| `ω` | 「omega」 | 角頻率，轉多快，單位 rad/s（見 ch10） | ch10 |
| `δ` | 「delta」 | Lissajous 兩軸之間的相位差（見 ch12） | ch12 |
| `π` | 「pai」（圓周率，≈3.14159） | 半圈的弧度；180°=π rad（見 ch02） | ch01 |
| `30°`、`90°` | 「30 度」「90 度」 | 度數制角度；右上小圈圈就是「度」（見 ch02） | ch01 |
| `rad`、「弧度」 | 「弧度」（radian） | 用半徑當尺量角：角度＝弧長÷半徑，全書預設單位（見 ch02） | ch02 |
| `sin θ`、`cos θ`、`tan θ` | 「sine 西塔」「cosine 西塔」「tangent 西塔」 | 圓上一點 (cos θ, sin θ) 的座標與其比值；不是邊長比，是繞圓的影子（見 ch01、ch03） | ch01 |
| `csc θ`、`sec θ`、`cot θ` | 「cosecant」「secant」「cotangent」 | 三個倒數函數：1/sin、1/cos、cos/sin；單位圓上是割線/餘割線段（見 ch03） | ch03 |
| `sin²θ` | 「sine 西塔的平方」 | (sin θ)²，先取 sine 再平方；不是 sin(sin θ)（見 ch01、ch03） | ch01 |
| `sin(a+b)` | 「sine、括號、a 加 b」 | 和角：先轉 a 再轉 b 的影子，≠ sin a + sin b（見 ch04） | ch04 |
| `√2`、`√3` | 「根號 2」「根號 3」（≈1.41421、≈1.73205） | 特殊角座標裡的無理數長度（見 ch03） | ch01 |
| `≈` | 「約等於」 | 數值近似，後面通常接複核過的小數（見全書） | ch01 |
| `s = r·θ` | 「s 等於 r 乘西塔」 | 弧長＝半徑×弧度角（見 ch02） | ch02 |

## 複數與極式

| 記號（寫法） | 怎麼唸（口頭講解用） | 白話意義 | 首次出現章 |
|---|---|---|---|
| `i` | 「i」（虛數單位） | 規則是 `i²=−1`；幾何上「乘以 i＝逆時針轉 90°」（見 ch07） | ch07 |
| `i² = −1` | 「i 平方等於負 1」 | 虛數單位的定義規則（見 ch07） | ch07 |
| `a + b·i` | 「a 加 b i」 | 複數的直角座標形：實部 a、虛部 b，平面上一支箭頭（見 ch07） | ch07 |
| `Re`、`Im` | 「實部」（real）「虛部」（imaginary） | 複數平面的橫軸標籤與縱軸標籤（見 ch07） | ch04 |
| `\|z\|` | 「z 的模」（modulus） | 複數箭頭的長度，r=√(a²+b²)（見 ch07） | ch07 |
| `arg z` | 「z 的輻角」（argument） | 複數箭頭與正實軸的夾角，逆時針為正（見 ch07） | ch07 |
| `r(cos θ + i·sin θ)` | 「r、括號、cosine 加 i sine」 | 極式（polar form）：長度 r、方向 θ（見 ch07） | ch07 |
| `z̄` | 「z bar」「z 的共軛」（conjugate） | 虛部變號，幾何上對實軸鏡射（見 ch07） | ch07 |
| `e^{iθ}` | 「e 的 i 西塔次方」 | 單位圓上方向 θ 的點＝cos θ + i·sin θ（見 ch08） | ch08 |
| `e^{iθ} = cos θ + i·sin θ` | 「Euler 公式」 | 指數在虛數方向就是繞圈：旋轉與指數接上線（見 ch08） | ch08 |
| `r·e^{iθ}` | 「r 乘 e 的 i 西塔次方」 | 極式的指數寫法：長度 r、方向 θ（見 ch08、ch09） | ch08 |
| `e^{iπ}+1=0` | 「e 的 i pai 次方加 1 等於 0」 | Euler 恆等式；其實只是「沿單位圓走半圈到 −1」（見 ch08） | ch08 |
| `(cos θ + i·sin θ)ⁿ` | 「整串的 n 次方」 | de Moivre 定理 = cos(nθ)+i·sin(nθ)：轉一個角再放大 n 倍角（見 ch09） | ch09 |
| `zⁿ = 1` | 「z 的 n 次方等於 1」 | n 次單位根方程；解落在單位圓的正 n 邊形頂點（見 ch09） | ch09 |

## 向量與點積

| 記號（寫法） | 怎麼唸（口頭講解用） | 白話意義 | 首次出現章 |
|---|---|---|---|
| 向量 `a`、`b` | 「向量 a」「向量 b」 | 平面上從原點出發的箭頭（首次出現會註明是向量）（見 ch06） | ch05 |
| `\|a\|` | 「a 的長度」「a 的模」 | 向量箭頭的長度（norm）（見 ch06） | ch06 |
| `a · b` | 「a dot b」「a 點 b」 | 點積（dot product）：座標版＝對應分量相乘相加；幾何版＝\|a\|\|b\|cos θ（見 ch06） | ch06 |
| `a·b = \|a\|\|b\|cos θ` | 「a dot b 等於、a 的長度、乘 b 的長度、乘 cosine 西塔」 | 點積＝投影＝對齊度；cosine similarity 的本體（見 ch06） | ch06 |
| `\|b\|cos θ` | 「b 的長度乘 cosine 西塔」 | b 打在 a 方向上的影子（投影長）（見 ch06） | ch06 |
| `a ⊥ b` | 「a 垂直 b」 | 正交（orthogonal）⇔ cos θ=0 ⇔ a·b=0（見 ch06） | ch06 |
| `⇔` | 「若且唯若」「等價於」 | 雙向邏輯等價（見 ch06） | ch06 |
| `R(θ)` | 「R of 西塔」「角 θ 的旋轉矩陣」 | 2×2 旋轉矩陣 [[cos θ,−sin θ],[sin θ,cos θ]]，逆時針為正（見 ch05） | ch05 |
| `R(a)·R(b)=R(a+b)` | 「R(a) 乘 R(b) 等於 R(a 加 b)」 | 旋轉可組合＝矩陣相乘＝和角公式（脊椎第二證）（見 ch05） | ch05 |
| `det R` | 「R 的行列式」（determinant） | 旋轉的 det=1，代表不改變面積（見 ch05） | ch05 |
| `R(θ)ᵀ`、`R(θ)⁻¹` | 「R 西塔的轉置」「R 西塔的逆」 | 旋轉的轉置＝逆＝R(−θ)（反向轉回去）（見 ch05） | ch05 |
| `(1,0)`、`(cos θ, sin θ)` | 「點 1、0」「點 cos、sin」 | 平面上的座標點／基底向量轉到哪（見 ch01、ch05） | ch01 |

## 波與相量

| 記號（寫法） | 怎麼唸（口頭講解用） | 白話意義 | 首次出現章 |
|---|---|---|---|
| `A·sin(ω·t + φ)` | 「A 乘 sine、括號、omega t 加 fai」 | 一個正弦波的全解剖：振幅 A、角頻率 ω、相位 φ（見 ch10） | ch10 |
| `A` | 「振幅」（amplitude） | 波擺多高（見 ch10） | ch10 |
| `T = 2π/ω` | 「週期 T 等於 2 pai 除以 omega」 | 週期：轉滿一圈所需時間（見 ch10） | ch10 |
| `f = 1/T` | 「頻率 f 等於 1 除以 T」 | 頻率：一秒重複幾次，單位赫茲（Hz）（見 ch10） | ch10 |
| 相量（phasor） | 「相量」（phasor） | 一支以 ω 旋轉的箭頭，它的影子就是波（見 ch10） | ch10 |
| `√(A²+B²)·sin(x+φ)` | 「根號 A 平方加 B 平方、乘 sine、括號 x 加 fai」 | 同頻相量加法：A·sin x + B·cos x 合成回同頻波（見 ch12） | ch12 |
| `Σ` | 「sigma」「總和」 | 一堆項相加；傅立葉是把無限多個旋轉箭頭疊起來（見 ch13） | ch13 |
| `Σ cₙ·e^{inx}` | 「sigma、c n、乘 e 的 i n x 次方」 | 傅立葉級數：每支箭頭半徑 \|cₙ\|、轉速 n（本輪 epicycles）（見 ch13） | ch13 |
| `∫` | 「積分」（integral） | 連續版的「乘加」；當投影／探測器用（見 ch06、ch13） | ch06 |
| `∫ sin(mx)sin(nx) dx = 0` | 「sine mx 乘 sine nx 的積分等於 0（m≠n）」 | 不同頻率正交＝互不干擾，傅立葉「各自結帳」的地基（見 ch06、ch13） | ch06 |
| `∞` | 「無窮大」 | 級數項數無限、圓是 n→∞ 的正多邊形（見 ch09、ch13） | ch09 |
| `−∞ … +∞` | 「負無窮到正無窮」 | 傅立葉級數求和的指標範圍（見 ch13） | ch13 |

## 微分與極限

| 記號（寫法） | 怎麼唸（口頭講解用） | 白話意義 | 首次出現章 |
|---|---|---|---|
| `(sin)′`、`sin′` | 「sine prime」「sine 的導數」 | sin 的導數＝cos（旋轉的速度還是旋轉）（見 ch02、ch11） | ch02 |
| `(sin θ)′ = cos θ` | 「sine 西塔的導數等於 cosine 西塔」 | 速度向量＝位置轉 90°，只在弧度下乾淨（見 ch11） | ch11 |
| `(cos θ)′ = −sin θ` | 「cosine 西塔的導數等於負 sine 西塔」 | 同上，差一個負號（速度方向）（見 ch11） | ch11 |
| `sin″`、`x″` | 「sine 兩撇」「x 兩撇」（二階導數） | 二階導把位置轉 180°＝指向圓心＝向心（見 ch11） | ch11 |
| `x″ = −x` | 「x 兩撇等於負 x」 | 彈簧方程，解是 sin/cos——三角函數住在一切振動裡（見 ch11） | ch11 |
| `d/dθ` | 「對西塔微分」「d by d 西塔」 | 對角度取變化率；`d/dθ e^{iθ}=i·e^{iθ}`（見 ch08、ch11） | ch08 |
| `lim` | 「極限」「lim」 | 趨近某值時的結果（h→0 的縮寫附在底下）（見 ch02、ch11） | ch02 |
| `lim sin h/h = 1` | 「sin h 除以 h、當 h 趨近 0、的極限等於 1」 | 小角下 sin h≈h；導數機制的核心極限（弧度）（見 ch11） | ch11 |
| `lim (1−cos h)/h = 0` | 「1 減 cos h、除以 h、的極限等於 0」 | 另一個小角極限，配合上式推出 sin′=cos（見 ch11） | ch11 |
| `sin x ≈ x`、`cos x ≈ 1 − x²/2` | 「sine x 約等於 x」「cosine x 約等於 1 減 x 平方除以 2」 | 弧度下的小角近似（見 ch02） | ch02 |

## 反函數與雙曲

| `sinh x`、`cosh x` | 「sinch x」（或「hyperbolic sine」）「cosh x」 | 雙曲正弦/餘弦＝(eˣ∓e⁻ˣ)/2；cos/sin 的雙曲孿生（見 ch15） | ch15 |
| `tanh x` | 「tanch x」（hyperbolic tangent） | 雙曲正切＝sinh/cosh；神經網路的激活函數（見 ch15） | ch15 |

| `cos x = (e^{ix}+e^{−ix})/2` | 「cosine x 等於、e 的 ix 加 e 的負 ix、除以 2」 | 圓函數的指數寫法，與 cosh 照鏡子（見 ch15） | ch15 |
| `y = a·cosh(x/a)` | 「y 等於 a 乘 cosh、括號 x 除以 a」 | 懸鏈線方程——垂下的纜線是 cosh，不是拋物線（見 ch15） | ch15 |
| `arsinh`、`arcosh`、`artanh` | 「反雙曲正弦」等（唸全名） | 反雙曲函數；artanh 是相對論「快度 rapidity」（見 ch15） | ch15 |
