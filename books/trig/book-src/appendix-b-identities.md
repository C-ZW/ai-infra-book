# 附錄 B — 恆等式「來源表」（不是背誦表）

> 這張表不是要你背的。整本書的招牌姿態只有一句：**恆等式不背，現推**。你會發現本書真正需要你「記住」的東西少得驚人——記最少的幾條種子，其餘全部能在紙上幾分鐘長出來。下面每一條恆等式都標了**它從哪裡推來**（哪一章、用什麼工具）與**一句話直覺**。讀法不是「從上到下背下來」，而是反過來：先看最後一節「如果只記這幾條」的最小集合，把那幾顆種子焊死在腦裡，然後對表上其餘每一條問自己「我能不能從種子把它推回來」。推得回來，就不必記；推不回來，回去讀那一章。

本書全部的恆等式共 **23 條**（含脊椎和角公式算一條、但它有四種證法）。下面分六組列出。每組標題就是它們共同的來源。

---

## 第 0 組：唯一不靠別的東西、直接從「圓」掉出來的種子

| 恆等式 | 從哪推來（見章） | 一句話直覺 |
|---|---|---|
| `sin²θ + cos²θ = 1`（畢氏恆等式） | 單位圓方程 `x²+y²=1` 把 `x=cosθ`、`y=sinθ` 代入（見 ch03） | 不是公式，是「圓上的點離原點永遠是半徑 1」這件事穿了件三角的外衣。 |
| `tan²θ + 1 = sec²θ` | 畢氏恆等式兩邊同除 `cos²θ`（見 ch03） | 同一條畢氏，換個角度量；也是切線段直角三角形的畢氏定理。 |
| `1 + cot²θ = csc²θ` | 畢氏恆等式兩邊同除 `sin²θ`（見 ch03） | 上一條的餘版本，除以 `sin²` 而不是 `cos²`。 |
| `tan θ = sin θ / cos θ` | 單位圓主定義（終邊撞 x=1 切線的高度，見 ch03） | tan 是「縱比橫」的斜率；csc/sec/cot 同理是 sin/cos/tan 的倒數。 |
| `cos θ = sin(θ + 90°) = sin(θ + π/2)` | 同一支旋轉箭頭換一根軸看影子（見 ch10） | cos 就是「超前 90° 的 sin」；sin/cos 之爭只是約定，不是本質差異。 |

這一組是真正的根：`sin²+cos²=1` 只靠「點在單位圓上」，不需要和角、不需要複數。其餘四條都是它的近親或單位圓的直接讀數。

---

## 第 1 組：全書脊椎——和角公式（一條公式，四種證法）

這是全書骨幹。下面兩條和角公式，本書**用四個完全不同的工具各推一次，四次推的是同一件事**：先轉 a 再轉 b，等於一次轉 a+b——「旋轉可組合」。能把四種證法並排講出來，是本書最重要的驗收之一。

```text
sin(a + b) = sin a · cos b + cos a · sin b
cos(a + b) = cos a · cos b − sin a · sin b      ← 那個減號來自「垂直分量的影子往回拐」
```

| 證法 | 怎麼推（一句話） | 見章 |
|---|---|---|
| 幾何 | 單位圓上把角 a 的點再轉 b，投影／旋轉一個直角三角形，公式的每一項都是圖上一段長度 | ch04 |
| 旋轉矩陣 | `R(a)·R(b)=R(a+b)`；把兩個旋轉矩陣乘開，元素自動長出和角公式 | ch05 |
| 複數乘法 | 極式相乘「輻角相加、模相乘」，用 ch04 和角展開（**不靠 Euler 級數**，避免循環） | ch07 |
| Euler | `e^{i(a+b)}=e^{ia}·e^{ib}`，左右各展開、比對實虛部，一行得證 | ch08 |

> 四種證法是同一件事——這是全書最大的驚嘆點之一。差角公式 `sin(a−b)`、`cos(a−b)` 把 b 換成 −b（用 cos 偶、sin 奇）即得，不另列。

---

## 第 2 組：和角公式的直接後裔——倍角、半角

這一組全部從第 1 組「令 b=a」或反解得到，是二級產物，沒有一條需要獨立記。

| 恆等式 | 從哪推來（見章） | 一句話直覺 |
|---|---|---|
| `sin 2θ = 2 sin θ cos θ` | 和角令 b=a（見 ch04；ch08 用 Euler 再證一次） | 把同一個旋轉做兩次，角度翻倍。 |
| `cos 2θ = cos²θ − sin²θ` | 和角令 b=a（見 ch04） | 三種寫法是同一條，下面兩條只是換掉 sin²/cos²。 |
| `cos 2θ = 1 − 2 sin²θ` | 上式用畢氏 `cos²=1−sin²` 換（見 ch04） | 不是新公式，是 `cos²−sin²` 配畢氏換一下。 |
| `cos 2θ = 2 cos²θ − 1` | 上式用畢氏 `sin²=1−cos²` 換（見 ch04） | 同上，換另一邊。 |
| `sin²(θ/2) = (1 − cos θ)/2` | 把 `cos2θ=1−2sin²θ` 反解再令 θ→θ/2（見 ch04） | 半角＝把倍角倒過來解、改個名字。 |
| `cos²(θ/2) = (1 + cos θ)/2` | 把 `cos2θ=2cos²θ−1` 反解再令 θ→θ/2（見 ch04） | 同上，正號版本。 |

---

## 第 3 組：和角公式相加減——積化和差／和差化積

把和角與差角公式相加或相減，乘積項一個翻倍、一個抵消。這一組是調變與拍頻的數學（見 ch12）。

| 恆等式 | 從哪推來（見章） | 一句話直覺 |
|---|---|---|
| `sin(a+b) + sin(a−b) = 2 sin a cos b` | 和角＋差角相加（見 ch04 預告、ch12 用） | 把「兩個頻率的積」換成「和差」。 |
| `sin(a+b) − sin(a−b) = 2 cos a sin b` | 和角−差角相減（見 ch12） | 同上，相減版本。 |
| `sin p + sin q = 2 sin((p+q)/2) cos((p−q)/2)`（和差化積） | 上式令 p=a+b、q=a−b 反解（見 ch12） | 「加」變「乘」：平均頻率 ×（差頻的一半）——拍頻整個藏在這裡。 |

> 對 cos 的和角／差角做同樣的加減，得到另兩條（`cos a cos b`、`sin a sin b` 的版本）。本書用到的就是上面這三條，cos 版本同理可推，不另列。

---

## 第 4 組：相量（向量）加法——同頻疊加

| 恆等式 | 從哪推來（見章） | 一句話直覺 |
|---|---|---|
| `A sin x + B cos x = √(A²+B²) sin(x + φ)`，`φ = atan2(B, A)` | 兩支夾 90°、長 A 與 B 的旋轉箭頭相加（畢氏定理＋輻角），見 ch12 | 同頻兩波相加還是同頻波；振幅是合成箭頭長度，相位是它的方向角。 |

> `φ=atan2(B, A)`，順序別顛倒：B 是「90° 方向（cos 那支）」的縱座標、A 是「0° 方向（sin 那支）」的橫座標（見 ch14 為什麼用 atan2 而不是 atan）。

## 第 5 組：複數與 Euler——把旋轉寫成代數

| 恆等式 | 從哪推來（見章） | 一句話直覺 |
|---|---|---|
| `e^{iθ} = cos θ + i·sin θ`（Euler 公式） | 問「什麼東西的變化率是把自己轉 90°」⇒ `d/dθ e^{iθ}=i·e^{iθ}`＝等速繞單位圓（見 ch08；級數鏡頭見《馴服無限》ch09） | 指數在虛數方向不發散，它繞圈；e^{iθ} 是以單位速率在單位圓上旋轉的箭頭。 |
| `e^{iπ} + 1 = 0` | Euler 公式代 θ=π（轉半圈到 −1，見 ch08） | 轉 180° 把 1 送到 −1，五個常數的合照。 |
| `e^{iπ/2} = i`、`i² = −1` | Euler 公式代 θ=π/2（轉 90°）；轉兩次 90°＝轉 180°（見 ch08） | `i²=−1` 不是硬背的規定，是「轉兩次 90° 等於轉 180°」的指數寫法。 |
| `(cos θ + i·sin θ)ⁿ = cos(nθ) + i·sin(nθ)`（de Moivre） | 「輻角相加」用 n 次（歸納法，見 ch07／ch09）；或 Euler 的 `(e^{iθ})ⁿ=e^{inθ}`（見 ch08／ch09） | 自己乘自己 n 次＝把角度乘 n 倍；de Moivre＝指數律＋Euler。 |
| `cos 3θ = 4 cos³θ − 3 cos θ`（與所有多倍角） | de Moivre 展開 `(cosθ+i·sinθ)³` 取實部（見 ch09） | 多倍角公式不必背——你要 n 倍角就展 n 次方，公式自己掉出來。 |

## 第 6 組：導數、單位根、雙曲——脊椎的延伸

| 恆等式 | 從哪推來（見章） | 一句話直覺 |
|---|---|---|
| `(sin)′ = cos`、`(cos)′ = −sin` | 等速繞單位圓的點，速度＝把位置逆時針轉 90°＝`(−sinθ, cosθ)`（見 ch11；同 `d/dθ e^{iθ}=i·e^{iθ}`，ch08） | 旋轉的速度還是旋轉。微分對這支箭頭做的事，就是再轉 90°。 |
| `lim_{h→0} sin h / h = 1`、`lim_{h→0} (1−cos h)/h = 0` | 差商＋幾何面積夾擠（弧度下才成立，見 ch11、回收 ch02） | 小角時 sin h≈h；這兩個極限是 `sin′=cos` 在「極限路」上剩下的全部。 |
| `sin″ = −sin`（`x″=−x`） | 二階導把位置轉 180°＝指向圓心＝向心（見 ch11） | 三角函數住在一切振動裡——彈簧、單擺、LC 電路的解都是 sin/cos。 |
| `1 + ζ + ζ² + … + ζⁿ⁻¹ = 0`（n 次單位根和為 0，n≥2） | 幾何：均勻分布的箭頭相消；代數：等比級數 `(ζⁿ−1)/(ζ−1)`，分子為 0（見 ch09） | 開 n 次方＝把圓等分成 n 份；對稱逼著它們相消。**注意 n≥2**，n=1 時和是 1。 |
| `cos x = (e^{ix} + e^{−ix})/2`、`sin x = (e^{ix} − e^{−ix})/(2i)` | Euler 公式 `e^{±ix}=cos x ± i·sin x` 相加／相減（見 ch15） | 把 cos/sin 從 Euler 反解出來，是雙曲孿生的母版。 |
| `cosh x = (eˣ + e⁻ˣ)/2`、`sinh x = (eˣ − e⁻ˣ)/2` | 上式「拔掉 i」（見 ch15） | i 是旋轉的標記；拿掉它，繞圓變成沿雙曲線拉伸。 |
| `cosh²x − sinh²x = 1` | 把 e 的組合乘開，正負號自己掉出來（見 ch15） | 對照 `cos²+sin²=1`，只差中間一個減號；對應雙曲線 `x²−y²=1`。 |

> `(sinh)′=cosh`、`(cosh)′=sinh`（沒有負號，因為指數裡沒有 i，見 ch15）——和圓函數的差別就藏在那個減號裡。

## 如果只記這幾條、其餘現推（最小集合）

我認為這本書真正要你「焊死在腦裡」的，只有 **三顆種子**。其餘 20 條都是它們的下游，能在紙上幾分鐘推回來：

1. **`sin²θ + cos²θ = 1`**（畢氏恆等式，ch03）——它不靠任何別的東西，直接是「點在單位圓上」。有了它：`tan²+1=sec²`、`1+cot²=csc²` 兩邊一除就有；倍角的三種 `cos2θ` 寫法靠它互換；「從 sin 求 cos」也靠它。

2. **和角公式 `sin(a+b)`、`cos(a+b)`**（脊椎，ch04，四種證法任挑一種會推）——它是恆等式叢林的根。**倍角**（令 b=a）、**半角**（倍角反解）、**積化和差／和差化積**（和差相加減）、**相量加法 `A sin+B cos`**（合成兩支箭頭）全部從它長出來。

3. **Euler 公式 `e^{iθ} = cos θ + i·sin θ`**（ch08）——它把指數與旋轉接線，是複數那一整側的引擎。有了它：和角公式一行再得（脊椎第四證）、**de Moivre**＝指數律、**多倍角**＝展開取實部、**cos/sin 的 e-組合**＝相加減、**雙曲 cosh/sinh**＝拔掉 i、**單位根**＝`e^{i·2πk/n}`、**`sin′=cos`**＝`d/dθ e^{iθ}=i·e^{iθ}`。

為什麼是這三顆？因為它們各自鎮守一條獨立的來路：**畢氏**鎮守「圓本身」（不靠別的）、**和角**鎮守「旋轉可組合」（幾何那一側）、**Euler**鎮守「旋轉＝指數」（代數那一側）。三條來路一旦在手，全書 23 條恆等式沒有一條是孤兒——每一條都能順著某顆種子推回去。這就是本書的姿態：**記最少、其餘現推。** 哪天你在半夜寫旋轉動畫、忘了某條公式，別翻表背它——回到這三顆種子，把它推出來。推得出來，才算真的會了。
