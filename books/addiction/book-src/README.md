# 大腦被劫持

> 成癮的機制

## 這本書是什麼

一本給工程師讀的、關於成癮的書。它的核心圖像是：**成癮是大腦的獎賞學習系統被劫持**——一套原本用來幫你「學會追求對生存有益的東西」的迴路，被一個外部訊號挾持，把它的學習能力反過來綁在一件正在傷害你的事情上。

這本書要拆掉一個你大概也有的直覺：「成癮＝越來越爽、停不下來」。真相幾乎相反，而那個反差正是全書的樞紐。

## 脊椎：兩條曲線分岔

多巴胺（dopamine）不是「快感分子」。它編碼的是**想要（wanting＝預測誤差／誘因顯著性）**，不是**喜歡（liking＝快感本身）**。在成癮裡，這兩者**分離**了，而且往相反方向走：

- **想要**這條曲線往上（敏化 sensitization）：線索觸發的渴求越來越強。
- **喜歡**這條曲線往下（allostasis）：同一個東西帶來的快感越來越少、基線越壓越低。

於是你得到那個殘酷的局面：**越來越想要、越來越不快樂**。每一章你都會回到這張「兩條曲線分岔」圖，問兩個問題：**這裡講的是想要還是喜歡？哪一段迴路被劫持了？**

## 怎麼讀

- 章節骨架固定：從工程直覺出發 → 拆機制 → 拆迷思（直覺的陷阱）→ 紙上推演（思想實驗）→ 自我檢核 → 延伸閱讀。
- 這是**純紙上推演**的欣賞書，所有「動手做」都在腦子裡完成。
- 全書四個 Part：獎賞系統的正常運作（I）、劫持怎麼一步步發生（II）、邊界與爭論——行為成癮、疾病模型 vs 學習模型、脆弱性、Rat Park（III）、出路與機制——大腦能不能修回去、治療在修哪一塊（IV）。

## 一個誠實的前提

**這本書是機制說明，不是醫療或戒斷建議。** 它有兩條刻意守住的紀律：第一，**成癮是疾病還是學習/選擇**，學界至今兩派並陳——這本書呈現雙方最強的版本，**不替你裁決**。第二，環境（Rat Park、越戰退伍軍人）是強力的調節因素，但**不證明「成癮全是環境」**：藥理與個體脆弱性是真的。你會帶走的不是一個簡單答案，而是一張能看懂自己與他人被什麼劫持的機制地圖。
