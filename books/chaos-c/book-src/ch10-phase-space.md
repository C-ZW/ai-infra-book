# ch10 — 相空間：把運動畫成一張圖

> **本章解決什麼問題**：前面九章你都在盯著一條離散遞迴式看，它每一步吐一個數。但混沌真正的「肖像」是連續系統在空間裡跑出來的形狀——勞侖次那隻蝴蝶就是一個形狀。要看懂形狀，得先有一張畫得下「整個系統當下狀態」的圖。這就是相空間（phase space）：每個自由度一根座標軸，系統此刻的狀態縮成一個點，時間流逝就讓這個點走出一條軌跡（trajectory）。本章建立這個鏡頭，並用單擺的三種相圖（閉環、螺旋進點、極限環）教你讀它；最後守住一條全書最省力的鐵律——**軌跡不能自交**，它直接從決定論推出來，ch11 會靠它逼出「奇異吸子為什麼非得是碎形」。

前兩個 Part，混沌對你來說是一串跳動的數字——脊椎遞迴式吐出 0.5130、0.7995、0.5130……你用 cobweb 圖看它彈跳。那是離散時間的世界，一步一個值。但歷史上混沌真正的肖像（勞侖次的蝴蝶、龐加萊的纏結）都活在**連續時間**裡，是空間中一條不斷流動的曲線。Part III 要把混沌畫出來給你看，而要畫，得先有畫布。這一章就是發那塊畫布。

## 從你已知的出發

你其實天天在用相空間，只是沒這樣叫它。

想想 state machine。你寫過訂單狀態機：`pending → paid → shipped → delivered`，外加 `cancelled`、`refunded` 這些岔路。你心裡那張狀態轉移圖，本質上就是一張**離散版的相空間**——每個圓圈是一個「系統可能處在的狀態」，每條箭頭是「從這個狀態走到下一個狀態」的演化規則。整張圖就是「這個系統所有可能的狀態，加上它們之間怎麼跳」。相空間做的是同一件事，只是把「有限個離散圓圈」換成「連續的座標空間」。

再想得具體一點。假設你要描述一台服務「此刻整體在幹嘛」。光一個數字不夠——你得同時知道：CPU 使用率、佇列積壓（queue depth）、in-flight 請求數、connection pool 剩幾條。**這四個數字一湊齊，就唯一決定了系統此刻的「狀態」。** 你做監控時，下意識會說「系統正處在高 CPU、深佇列、connection 快滿的那個角落」——你已經在腦子裡把這四個量當成四根座標軸，把系統想成那個四維空間裡的一個點了。這個四維空間，就是這台服務的相空間；那個點，就是它此刻的狀態。

時間往前走，CPU 降、佇列消、connection 釋放——那個點就在四維空間裡移動，劃出一條曲線。這條曲線就是系統的**相軌跡（phase trajectory）**。你做 observability 時做的事，說穿了就是**把這條高維軌跡投影到你看得見的二維平面上**——畫一張「CPU vs 佇列深度」的散點隨時間連線，就是把四維相空間投影到兩根軸上。Grafana 上那些隨時間掃過去的曲線，是相軌跡的影子。

所以本章不是在教你一個陌生的新東西，而是把你早就在用的直覺——「系統此刻的整體狀態是一個點，它隨時間走出一條路」——升級成一個能看見混沌形狀的精確鏡頭。差別只在：state machine 的狀態是離散的、可數的；相空間的狀態是連續的、稠密的，而正是這個「連續」，才裝得下混沌那種「被困在有限區域裡卻永不重複」的軌跡。

## 相空間：一個點裝下整個當下

先把定義講乾淨。

**相空間（phase space）的規則只有一條：系統有幾個獨立的自由度（degree of freedom），就配幾根座標軸。** 把這些座標軸湊成一個空間，系統「此刻的完整狀態」就是這個空間裡的一個點。系統隨時間演化，這個點就移動，走出來的路徑叫**軌跡（trajectory，也叫 orbit、相軌跡）**。

什麼叫「自由度」？就是「你得知道幾個數字，才能唯一決定系統下一刻怎麼走」。這是關鍵——不是「系統有幾個你看得到的量」，而是「**要預測未來，你最少得交代幾個量**」。

拿單擺（pendulum）當例子，它會貫穿這一整章。一個吊著的擺，你直覺以為只要知道它「現在擺到哪個角度」就夠了。錯。光知道角度 θ，你說不出它下一刻往哪走——同一個角度，它可能正往右盪、可能正往左盪、可能盪得快、可能盪得慢。要真正鎖死它的未來，你還得知道它**此刻的角速度 ω**（轉多快、往哪轉）。

```text
        單擺的狀態 = (角度 θ, 角速度 ω)        ← 兩個數字才鎖得住未來
```

角度與角速度，這兩個量湊齊，單擺的下一刻就被牛頓運動定律唯一決定了。所以單擺的相空間是**二維**的：一根軸是角度 θ，一根軸是角速度 ω。單擺此刻的狀態，就是這張 (θ, ω) 平面上的一個點。

為什麼需要兩個而不是一個？因為支配運動的是二階的關係——加速度由角度決定（重力把擺往低處拉），而要從加速度推出未來位置，你得先知道現在的位置**和**現在的速度。這對工程師其實很熟：一個只看「當前值」的系統和一個還看「當前變化率」的系統，行為天差地別。PID 控制器裡那個 D 項（微分項，看變化率）就是這個道理——只有 P（看當前偏差）的控制器，和加了 D（看偏差變化多快）的控制器，是兩種不同維度的動力系統。**相空間的維度，數的是「鎖死未來所需的最小資訊量」**，不是表面上有幾個讀數。

（一個小提醒：相空間的軌跡是由微分方程給定的——單擺的 (θ, ω) 怎麼隨時間變，寫出來就是兩條一階微分方程。本書只用這張圖的幾何直覺看軌跡長什麼樣、往哪跑，不解方程；「給變化率、求軌跡」這件事本身的工具見《馴服無限》ch10 微分方程。）

這個鏡頭最值錢的地方在於：**它把「時間」這條軸藏起來了。** 你平常畫監控，橫軸是時間、縱軸是某個指標，曲線一路往右掃。相空間不畫時間——它把所有座標軸都讓給「狀態量」，時間只是隱含在「點沿著軌跡移動」這件事裡。為什麼要這樣？因為這樣一來，「系統長期會穩定在哪、會週期繞圈、還是會永遠亂跑」這些**定性的形狀**，會直接變成圖上看得見的幾何：穩定＝軌跡縮成一個點，週期＝軌跡繞成一個圈，混沌＝軌跡纏成一團永不重複又不自交的東西。龐加萊當年面對三體問題解不出來，正是這樣轉念的（見 ch02）：與其硬解方程求出「t 時刻在哪」的公式，不如直接研究軌跡在相空間裡的**幾何形狀**。動力系統這門學問，就是從這個轉念誕生的。

## 單擺的三種相圖：閉環、螺旋、極限環

理論講夠了，來畫。單擺在三種條件下會給三種典型的相圖，每一種對應一類「系統長期會落到哪」的命運。把這三張圖看懂，你就握住了相空間最核心的讀法。

下面這張三連圖，橫軸一律是角度 θ（擺到哪），縱軸一律是角速度 ω（轉多快、正為一個方向、負為另一方向）。中央那個點是擺垂在最低處、靜止不動的平衡狀態 (θ=0, ω=0)。

```text
  (A) 無阻尼擺動：閉曲線（中性穩定，能量守恆）

         ω
         |        . - - - - - .
         |     .'               '.        每一圈 = 一次完整來回

         |  :      '.       .'      :       軌跡是封閉的環
         |   '.      ' - - '      .'        繞完一圈精準回到起點
         |     '.               .'
         |        ' - - - - - '
         |

         |    :  '.   .'  ;              = 點吸子 (point attractor)
         |    ;    ' - '  ;

         |       ' - - - '

  (C) 被驅動的擺：極限環（孤立的週期軌）

         ω
         |        . - - - - - .            外面的軌跡 -> 旋進這個環
         |     .'   _ _ _ _ _   '.         裡面的軌跡 -> 旋出到這個環
         |   .'   /           \   '.       環本身 = 穩定的週期軌道
       --+--|----|-----o-----|----|--- θ   o = 平衡點（已不穩定）
         |   '.   \ _ _ _ _ _ /   .'        粗環 = 極限環 (limit cycle)
         |     '.             .'            驅動補進的能量
         |        ' - - - - - '            正好抵掉阻尼耗掉的能量
```

逐一說它在講什麼。

**(A) 無阻尼擺動——閉曲線。** 理想化的單擺，沒有摩擦、沒有空氣阻力。你把它拉到一邊放手，它盪過去、盪回來，永遠這樣盪下去，幅度一點都不減。在相空間裡，這對應一條**封閉曲線（closed orbit）**：擺從最右（θ 最大、ω=0）開始往左加速（ω 變負、絕對值變大），經過最低點時速度最快（θ=0、|ω| 最大），衝到最左（θ 最小、ω=0），再盪回來……繞一圈精準回到起點，然後一模一樣再繞一圈。**封閉＝週期（periodic）**：軌跡閉合，意味著系統的狀態每隔固定時間就完全重複一次。拉得越用力，環就越大（外圈），但仍是閉環。這種「軌跡是一圈圈閉曲線、各自獨立、靠初始能量決定繞哪一圈」的結構叫**中性穩定（neutrally stable）**：推一下會換到鄰近的另一個閉環，但不會自己回到原來那圈，也不會散掉。能量守恆的系統都長這樣。

**(B) 有阻尼擺動——螺旋進不動點。** 現在加回真實世界的摩擦與空氣阻力。擺每盪一次，就被摩擦吃掉一點能量，幅度一次比一次小，最後停在最低點不動。相空間裡這對應一條**向內螺旋（inward spiral）**的軌跡：它還是一圈圈繞著平衡點轉，但半徑每繞一圈就縮小一點，像水流進排水孔，最終旋進中央那個點 (0,0) 停住。那個點就是這個系統的**不動點（fixed point）**，也是它的**點吸子（point attractor）**——不管你從哪裡放手（不管哪條螺旋線起步），最後都被「吸」到同一個點。這對你太熟了：這就是 ch06 那個穩定的不動點，只是現在從離散迭代換成連續流動。autoscaler 經過幾輪震盪後穩定在某個副本數不動——那個「穩定下來」，在相空間裡就是一條螺旋進點的軌跡。

**(C) 被驅動的擺——極限環。** 第三種最有意思，也是 (A) 和 (B) 都給不出來的。光有阻尼，擺會停死（走向點吸子）。但如果我們持續從外面**驅動（drive）**它——想像給它一個週期性的小推力，像盪鞦韆時你一下一下地推——驅動補進的能量，可以剛好抵掉阻尼耗掉的能量。於是擺既不會停死、也不會越盪越大，而是穩定在一個**固定幅度**的週期擺動上。相空間裡，這對應一條**孤立的封閉曲線**，叫**極限環（limit cycle）**。

極限環和 (A) 的閉環長得像，但本質天差地別，這個差別值得你停十分鐘：

```text
  (A) 無阻尼的閉環          vs       (C) 極限環

  一整片同心閉環               只有「一條」特定的閉環
  繞哪一圈由初始能量決定        不管從哪起步,都被吸到這同一條環
  推一下 -> 換到鄰圈,不回來     推一下 -> 自己旋回原來的環
  中性穩定(能量守恆)           漸近穩定(有耗散又有驅動)
  鄰圈無窮多、稠密填滿          環是孤立的,旁邊沒有別的閉環
```

關鍵在「孤立」與「吸引」。(A) 的閉環是一**整片**連續排列的環，你正好在哪一圈，純看一開始給了多少能量，環之間沒有誰吸引誰。(C) 的極限環是**唯一一條**：從環內出發的軌跡會向外旋進它，從環外出發的軌跡會向內旋進它，四面八方的軌跡都被這條環收編。它是個**吸子**——和點吸子同一個身份，只是穩態行為的形狀從「一個點」變成「一個圈」。這就是為什麼系統會自己進入穩定的振盪：不管初始狀態怎麼擾動，它都回到同一個振幅、同一個週期。心跳、生理時鐘、霓虹燈管的電子振盪、某些會「自己穩定地週期抖動」的控制迴圈，背後都是極限環。一個在 N 和 2N 副本間穩定來回 flapping、推一下擾動完還是回到原本那個來回節奏的 autoscaler——那不是壞掉，那是落進了一個極限環。

把這三張圖並排，你會看到一個遞進：**點吸子（穩定在一處）→ 極限環（穩定地繞圈）→**……下一步是什麼？如果有一種「吸子」，軌跡被吸進去、被困在有限區域裡，卻**既不停在一點、也不繞成乾淨的圈、而是永遠繞、永遠不重複**——那會是什麼形狀？那就是 ch11 的奇異吸子（strange attractor）。本章先把前兩種、加上下一節的環面（torus）鋪好，把那個位置空出來。

## 吸子的種類：系統長期會落到哪

把上面的觀察收成一張分類表。**吸子（attractor）就是「系統長期穩態行為的形狀」**——不管你從相空間哪裡出發（在某個範圍內），軌跡最終都會被吸進去、停在這個集合上。它回答的問題是：「這個系統放著不管，最後會幹嘛？」

```text
  吸子種類      相空間形狀     維度    系統長期在幹嘛           例子
  ----------    -----------    ----    --------------------    ----------------
  不動點        一個點         0       靜止、停在一處          阻尼擺停死;
   point fixed                                                 收斂的 autoscaler
  ----------    -----------    ----    --------------------    ----------------
  極限環        一條閉曲線     1       單一頻率的週期擺動       心跳;被驅動的擺;
   limit cycle                                                 穩定 flapping
  ----------    -----------    ----    --------------------    ----------------
  環面          甜甜圈表面     2       兩個不可通約頻率的       兩個獨立週期疊加;
   torus                               準週期運動               準週期振盪
  ----------    -----------    ----    --------------------    ----------------
  奇異吸子      碎形集合       分數    有界、非週期、敏感       勞侖次蝴蝶
   strange                            (這就是混沌)             -> 留給 ch11
```

前三種叫**正規吸子（regular attractor）**，維度是整數（0、1、2），行為都「可預測」：點吸子最後不動，極限環照固定週期重複，環面照兩個頻率準週期地重複。它們是 Part II 那條遞迴式在「穩定」「2-cycle」「更高週期」時的連續時間版親戚。

第三種**環面（torus）**值得多說一句，因為它是「正規」與「奇異」之間最後一道正常的關卡。想像系統同時有**兩個獨立的週期擺動**疊在一起——比如一個東西一邊以頻率 f₁ 上下振、一邊以頻率 f₂ 左右晃。如果這兩個頻率的比值是**無理數**(數學上叫「不可通約 incommensurate」)，那麼這兩個振動永遠湊不出一個共同的大週期，合起來的軌跡就**永遠不會精準重複**——但它也不是混沌！它規規矩矩地在一個甜甜圈（torus、環面）的表面上繞，一圈圈地、稠密地把整個甜甜圈表面鋪滿，卻永遠不真正回到起點。這叫**準週期（quasiperiodic）**運動。

這裡藏著一個容易踩的坑，先點明：**「永不重複」本身不等於混沌。** 環面上的準週期軌跡就永不精準重複，但它一點也不混沌——它不敏感、不發散，你把起點挪一點，軌跡只是平行地挪一點，不會指數放開。混沌要的是「永不重複」**加上**「敏感依賴」。環面有前者沒後者。這個區分 ch11、ch14 還會回來算帳。(技術上，環面吸子有兩個方向是「中性」的——對應那兩個頻率；奇異吸子則至少有一個方向在指數拉開，這就是 ch14 的 Lyapunov 指數要量化的事。)

奇異吸子是這張表的第四行，我故意把它留白指向 ch11。原因在下一節：要理解它為什麼非得是「分數維度的碎形」、為什麼非得活在三維以上，你得先握住本章最後也最重要的一條鐵律。

## 鐵律：軌跡不能自交

這是本章我最想讓你帶走的一句話，而且它的證明只要一行，卻撐起後面整個 Part III。

**在相空間裡，一條軌跡不能和自己交叉，兩條不同的軌跡也不能相交。**

為什麼？因為**決定論（determinism）**。回想全書第一刀（ch01、ch04）：一個確定性系統，給定當下狀態，未來就被唯一決定——同一個輸入，只有一個輸出。把這句話翻譯到相空間：

```text
  相空間裡的「一個點」 = 系統的「一個完整當下狀態」
  決定論          = 一個當下狀態,只有「一個」確定的未來
  ----------------------------------------------------------
  所以:每個點,只能有「一條」軌跡穿過,而且方向唯一
```

現在假設軌跡**真的**自交了——兩條軌跡（或同一條軌跡的兩段）在某個點 P 相交。那麼站在 P 這個點上，系統「下一刻往哪走」就有了**兩個**答案（沿著第一條軌跡走、或沿著第二條軌跡走）。但 P 是一個完整的當下狀態，決定論說它只能有一個未來。矛盾。所以軌跡**不可能**自交。

```text
        禁止的圖像:軌跡在 P 自交

              \         /
               \       /
                \     /
                 \   /
                  \ /
                   X  <- P 點:從這裡有兩個方向可走?
                  / \
                 /   \      系統處在 P 這個狀態時,
                /     \     「下一刻」到底往哪?
               /       \    決定論說只能有一個答案
                            -> 所以這個交叉根本不可能存在
```

慢一點，把這個直覺嚼透，因為它太省力、太多後果。一個工程師的類比：把相空間想成一張巨大的**確定性查表**——「現在在 P 這個狀態」這個 key，映到「下一個瞬間的速度方向」這個唯一的 value。這就是一個純函數（pure function）：同一個 key，永遠回同一個 value。如果軌跡自交，等於同一個 key 對到了兩個不同的 value——那這張表就不是函數了，系統也就不是決定論的了。**軌跡不自交，和「演化規則是個確定的函數」，是同一件事的兩種說法。** 你寫過的每一個 `nextState(currentState)`，只要它是純函數，它畫出來的軌跡就自動不會自交。

但這條鐵律有一個你一定要記住的**前提**，不然會被上面 (C) 的圖打臉：它只對**自主系統（autonomous system）**成立——也就是演化規則「不顯含時間」，規則本身不隨時間改變(`nextState` 只吃 `currentState`，不吃當前時刻 `t`)。無阻尼擺 (A) 和有阻尼擺 (B) 都是自主的，所以它們的軌跡乾乾淨淨不自交。但被驅動的擺 (C)，那個週期性的外加推力**是時間的函數**(這一刻推、下一刻不推)——它顯含時間，是**非自主（non-autonomous）**的。在只畫 (θ, ω) 的二維平面上，(C) 的軌跡確實會看起來自己穿過自己。

這不是反例，而是提醒你「自由度數對了沒」。修法很漂亮：把「時間」也當成一根座標軸補進相空間。驅動力是時間的函數，那就讓相空間多一維 t(或者多一維「驅動相位」)，狀態變成 （θ, ω, t）。在這個**擴維後的相空間**裡，(C) 的軌跡就不再自交了——原本「看起來在同一個 (θ, ω) 點交叉」的兩段，其實發生在**不同的 t**，在三維裡根本沒碰到，只是投影到 (θ, ω) 平面上才疊在一起。

```text
  非自主系統(被驅動的擺)在二維投影裡「看似自交」

  只看 (θ,ω):                    補上時間軸 (θ,ω,t):
                                  軌跡像螺絲一樣往 t 方向前進
       看似交叉                   同一個 (θ,ω) 位置
          X            ==>        在不同的 t 通過 -> 三維裡不碰
        /   \                     投影到平面才疊成「X」
                                  (這正是 ch11 蝴蝶吸子
                                   「看似交叉、其實在不同高度」的同一招)
```

記牢這個修法，因為 ch11 的奇異吸子會用**一模一樣**的招式：勞侖次蝴蝶的軌跡在二維投影上看起來交叉得亂七八糟，但在完整的三維相空間裡，它從不真的交——每個「看似交點」其實是兩段軌跡在不同的高度（第三個座標）擦身而過。「不能自交」這條鐵律從沒被破壞，只是你得在**夠多的維度**裡看它。

而這正是「奇異吸子為什麼必須活在三維以上」的伏筆，把它說透，你就提前看懂了半個 ch11:

- 軌跡被**困在有限區域**裡（它是個吸子，跑不出去）。
- 軌跡**永不重複、永不閉合**（否則它就是個極限環、就週期了，就不是混沌了）。
- 軌跡**不能自交**（決定論，本節鐵律）。

這三條在**一維**裡湊不出來：一維上你只能單調地走，要嘛衝出去、要嘛停下，變不出花樣。**二維**裡也湊不出來——這是個漂亮又深刻的結論（數學上叫 Poincaré–Bendixson 定理，本書只給結論不證）：在二維平面上，一條被困在有限區域、又不准自交的軌跡，**只剩兩條路**：要嘛旋進一個不動點（像 B），要嘛旋進一個極限環（像 C）。平面太擠了，「永不重複又不准自交又跑不出去」這三個願望在二維裡是**不可能同時實現**的——你想永不重複地繞，但不准自交，平面很快就被你自己的軌跡圍死，逼得你只能收進點或環。

所以混沌**至少要三維連續空間**才裝得下。到了三維，軌跡多了一個「上下」的自由度，可以在「看似要交叉」時從另一段的上方或下方鑽過去，於是它能永遠不重複地繞、又永遠不真的自交——代價是，它繞出來的東西不可能是光滑的面或乾淨的線，只能是一個被無限次摺疊、有著分數維度的**碎形**集合。這就是「奇異」的幾何來源。本章只負責把這個邏輯鏈鋪到你眼前；它怎麼一步步成真，是 ch11 的事。

（連續系統至少三維才有混沌——這是上面那條推理的結論。離散映射不受此限：脊椎遞迴式一維就混沌了，因為離散迭代「下一步直接跳到別處」，不必畫出連續不自交的曲線，規避了平面太擠的限制。兩者別搞混。）

## 直覺的陷阱

| 誤解 | 為什麼錯／會在哪一步把你帶溝裡 | 正確版 |
|---|---|---|
| 「相空間的軸就是空間的 x、y、z 座標」 | 相空間的軸是**狀態量**(自由度)，不是物理位置。單擺的相空間是 (角度， 角速度)，沒有一根是「空間位置」。把它當成物理空間，你會數錯維度、也看不懂「為什麼軌跡繞圈＝週期」。 | 相空間每根軸對應一個自由度＝「鎖死未來所需的最小資訊」之一。一個量＋它的變化率，常是兩根軸。 |
| 「知道當前位置（角度）就知道系統狀態」 | 漏掉了速度／變化率。同一個角度，擺可能正往左、正往右、快、慢——未來完全不同。漏掉一個自由度，你的相空間維度少一維，軌跡會「看似自交」騙你。 | 狀態要湊齊**所有**自由度。位置型系統通常還要配上速度（動量）。維度不足，軌跡的不自交性就會被假象破壞。 |
| 「軌跡永不重複＝混沌」 | 環面上的準週期軌跡就永不精準重複，但它完全不混沌——不敏感、不發散。把「不重複」直接等於混沌，你會把規規矩矩的兩頻率振盪誤判成混沌。 | 混沌＝永不重複**且**對初始條件敏感（SDIC，指數發散）。準週期只有前者。敏感性是 ch14 用 Lyapunov 指數量化的那一刀。 |
| 「軌跡在勞侖次蝴蝶圖上明明交叉了」 | 你看到的是三維軌跡投影到二維平面的影子，影子會疊。真實軌跡在三維裡那兩段處在不同高度，從沒碰到。把投影的假交點當真，會以為決定論在這裡破功了。 | 投影會製造假交點。在**完整維度**裡，自主系統的軌跡永不自交——這是決定論的鐵律，蝴蝶吸子完全遵守它（見 ch11）。 |
| 「被驅動的擺軌跡會自交，所以鐵律有例外」 | 它顯含時間，是非自主系統。在只畫 (θ,ω) 的二維裡會看似自交，但那是因為你少畫了時間這根軸。 | 把時間（或驅動相位）補成一根軸，擴維後軌跡照樣不自交。鐵律沒例外，是你的相空間維度開少了。 |

## 紙上推演

### 推演題 1 ★ **[10 分鐘]**

把你的一個有狀態的後端系統，畫成一張相空間草圖。具體做：(a) 選一個系統（autoscaler、限流器、消費佇列的 worker 任選），列出「要預測它下一刻怎麼走、你最少得知道哪幾個量」——這就是它的自由度；(b) 挑其中兩個量當座標軸，畫一張二維相圖；(c) 在圖上標出「系統正常穩定時」這個點大概在哪、它是哪種吸子。

#### 推演解答

以 autoscaler 為例。

(a) 自由度：要預測下一輪開幾個副本，你至少得知道**當前副本數 N** 和**當前負載 L**（每副本的請求率或佇列深度）。只知道負載不夠——同樣負載下，已經開了 2 個副本和已經開了 20 個副本，下一步動作完全不同。所以這是個（至少）二維系統，(N, L) 是它的狀態。(嚴格講還有「負載的變化趨勢」等，但兩維足夠畫草圖。)

(b) 畫一張橫軸 N（副本數）、縱軸 L（每副本負載）的平面。一條典型軌跡：流量突增 → L 飆高（點往上跳）→ autoscaler 反應、N 增加（點往右移）→ 每副本分到的負載下降、L 回落（點往下）→ 系統繞了一圈。

(c) 「正常穩定」對應一個**點吸子**：某個 (N*, L*)，負載和副本數匹配，系統停在那不動（就是 ch06 那個穩定不動點的連續版，軌跡螺旋進這個點）。如果你的 autoscaler 設定太激進、在 N 和 2N 間穩定來回 flapping 又收不住，那它落進的是**極限環**——一條閉軌，系統穩定地週期性擴縮，推一下還是回到那個來回節奏。常見錯路：只用一個量（只畫負載）當相空間，於是「同一個負載下系統有時擴有時縮」看起來自相矛盾——那是因為你漏了 N 這個自由度，把二維軌跡壓成一維才看到假矛盾。這正好親身演示了上面陷阱表的第二條。

### 推演題 2 ★★ **[15 分鐘]**

口頭推演單擺的三種相圖。閉著書，只憑物理直覺，描述：(a) 無阻尼擺從「最右、靜止」放手後，那個代表狀態的點怎麼在 (θ, ω) 平面上移動，繞出什麼形狀；(b) 加了阻尼之後，形狀怎麼變；(c) 為什麼 (a) 是一整片同心閉環、而被驅動的 (c) 只有孤零零一條環？講清楚「孤立」與「吸引」的差別。

#### 推演解答

(a) 起點：θ 最大（最右）、ω=0(靜止)，點在 θ 軸正端。放手後重力把擺往左拉，它加速：ω 變負（往左轉）且絕對值變大，θ 開始減小。點往左下方走。經過最低點 θ=0 時速度最快，|ω| 最大，點在 ω 軸負端最深處。衝過去到最左 θ 最小、ω 又回到 0，點到 θ 軸負端。然後盪回來，對稱地走上半邊。繞一圈精準回到起點(θ 最大、ω=0)，再一模一樣繞下一圈——**封閉曲線，週期運動**。拉得越用力，環越大。

(b) 加阻尼：每盪一次被摩擦吃掉能量，幅度一次比一次小。點還是繞著中心轉，但半徑每圈縮小——**向內螺旋**，最後旋進中央 (0,0) 停死。那個點是**點吸子**。

(c) 關鍵在「孤立」和「吸引」。無阻尼的閉環是一**整片**：你正好繞哪一圈，純由放手時給了多少能量（初始條件）決定，環和環之間是平等的鄰居，沒有誰吸引誰——把擺稍微多推一點，它換到外面相鄰的一圈，然後就待在那新的一圈，不會自己回到原來那圈（中性穩定）。被驅動的極限環是**唯一一條**：驅動補的能量剛好抵掉阻尼耗的能量，只有在某個特定振幅上才平衡。比這振幅大，阻尼耗的多於驅動補的，軌跡向內旋（振幅縮回去）；比這振幅小，驅動補的多於阻尼耗的，軌跡向外旋（振幅長回去）。四面八方的軌跡都被吸到這條環上——它是**吸子**（漸近穩定）。常見錯路：把兩者都叫「週期軌道」就混為一談。差別不在「是不是閉環」(都是)，而在「孤立且吸引」vs「成片且中性」——這也是「能量守恆系統」與「有耗散又有驅動的系統」的分水嶺。

### 推演題 3 ★★★ **[20 分鐘]**

有人主張：「我寫一個二維的連續確定性系統(兩個狀態量 x、y，演化規則 `dx/dt=f(x,y)`、`dy/dt=g(x,y)`,f、g 不含時間)，讓它的軌跡被困在一個有限的方框裡、而且永遠不重複地亂繞——這樣我不就在二維裡造出混沌了嗎？」用本章的鐵律證明：這個目標**辦不到**。破綻在哪？

#### 推演解答

辦不到，而且理由正是本章三條鐵律的交鋒。把這個系統的三個要求列出來：

1. **有界**：軌跡被困在有限方框裡（假設成立）。
2. **永不重複**：軌跡不閉合、不收斂到點（他要的「亂繞」）。
3. **不自交**：系統是自主的（f、g 不顯含時間）+ 決定論 → 軌跡不能自交（本章鐵律）。

現在在二維平面上同時滿足這三條。想像那條軌跡在有限方框裡爬。它不能自交（條件 3），所以它每爬一段，就把平面切成「軌跡這邊」和「軌跡那邊」——軌跡像一道不能跨越的牆，把可用的空間越圍越小。它又不能重複、不能收進點或環（條件 2），所以它得**一直**找新的、沒走過的地方繞。但平面是二維的，被一條不自交的曲線反覆切割後，剩下能走的區域會越來越窄，最後窄到無路可走：軌跡要嘛被自己圍死（只能往唯一的出路走，而那條出路只能通向某個不動點或某條閉環），要嘛就得跨過自己已經畫好的線——但那違反條件 3。

換句話說，在二維裡，「有界 + 不自交」這兩條會**逼**著軌跡要嘛收進不動點、要嘛收進極限環——這正是 Poincaré–Bendixson 定理的內容（本書只用結論）。永不重複的混沌，在二維平面上沒有容身之處。

破綻就在：他默認「二維夠用」。要破局，只能加一維——三維裡軌跡多了「上下」這個自由度，可以在「快要撞到自己畫過的線」時從上方或下方鑽過去，於是「有界 + 永不重複 + 不自交」三願才同時成真。代價是繞出來的集合不再是光滑曲線，而是分數維度的碎形（奇異吸子，ch11）。常見錯路：拿脊椎遞迴式（一維就混沌）反駁說「一維都能混沌，二維當然能」。陷阱在於脊椎是**離散映射**——它「下一步直接跳到別處」，不必在平面上畫出一條連續不自交的曲線，所以躲開了「平面太擠」這個限制。連續流（continuous flow）和離散映射在這件事上規則不同，不能混用。
