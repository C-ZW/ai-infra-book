一個容易看漏的細節：

```text
• a | b 這條豎線在本書是「整除」符號，讀「a 整除 b」，
  方向是「小的 a 在前、被它整除的大的 b 在後」。
  3|12 為真（12=3·4）；12|3 為假（3 切不成 12 份）。

• 上標（如 2²、p^(p−1)）是次方；下標在本書極少出現，
  主要見於 Eₙ（第 n 個歐幾里得數）。

• π(x) 和 ln 在本書永遠連在一起用：「π(x)~x/ln x」
  是質數定理的標準寫法，~ 是漸近記號（見§三歧義節）。
```

## 三、同形記號歧義專節（重要）

數論借用的字母不多，於是同一個符號在不同脈絡常指不同東西。下面這幾條是本書（及書架相鄰書）最容易撞名的記號。**查一個數字前先確認你查的是哪個 π、哪個 e、哪個 p，否則會把對的符號套到錯的情境。**

```text
π ── 本書：絕大多數情況是「質數計數函數 π(x)」，                  ch06
     定義為「≤ x 的質數個數」（一個整數值的遞階函數）。
     π(10)=4, π(100)=25, π(1000)=168, π(10⁶)=78498。
     易撞：圓周率 π ≈ 3.14159…，與質數計數函數完全不同。
     本書凡指圓周率，明確寫「圓周率 π」加以區分；
     裸寫 π(x) 或 π 後接數值，永遠是質數計數函數。

p, q ── 本書：一般用法是「代表某個質數」的變數。                   ch01
     例：「設 p 是質數、gcd(a,p)=1」「質數 q 整除乘積」。
     RSA 脈絡（ch17）下：p 和 q 是選定的兩個特定質數（基準
     p=3, q=11），n=p·q=33 是 RSA 模數；此時 p,q 從「任意質數
     變數」變成「具體選定值」。判斷法：看旁邊有沒有 n=pq 或
     RSA 的情境字眼——有就是 RSA 裡的固定因數，沒有就是泛指
     的質數變數。

e ── 本書：RSA 脈絡（ch17）的「公鑰指數 e」（基準 e=3）。        ch17
     易撞：自然常數 e ≈ 2.71828。本書凡指自然常數，
     一律用 ln（自然對數的底，不裸寫 e 當底數）；
     非寫不可時加「自然常數 e」明確區分。
     RSA 脈絡下的裸寫 e，永遠是公鑰指數，不是自然常數。

d ── 本書：RSA 脈絡（ch17）的「私鑰指數 d」（基準 d=7）。        ch17
     d 滿足 e·d ≡ 1 (mod φ(n))。全書只在 RSA 那章用 d 當私鑰；
     其他章的 d 若出現，是普通整數變數，上下文清楚。

φ ── 本書：歐拉函數 φ(n)（Euler's totient），                    ch11
     計算 1..n 中與 n 互質的整數個數。
     φ(p) = p−1；φ(pq) = (p−1)(q−1)；φ(60)=16。
     RSA 基準：φ(33)=φ(3)φ(11)=2·10=20。
     易撞：其他書的 φ 可能代表相位角或其他函數；本書 φ 永遠是
     歐拉函數，不換義。

~ ── 本書：漸近符號，π(x) ~ x/ln x 讀作                        ch07
     「比值 π(x)/(x/ln x) 趨近 1（x→∞）」，
     不是「π(x) 等於 x/ln x」。x/ln x 在有限 x 系統性低估
     π(x)（見 landscape T5 與 ch07「直覺的陷阱」）。
     裸寫 = 是錯的；你只能說「漸近相等」「比值趨近 1」。
     易撞：別本書的 ~ 可能表示「約等於」或「同分布」；
     本書 ~ 嚴格只表示「漸近」。
```

一句總結：**π=質數計數函數（不是圓周率）、p/q=質數變數（RSA 脈絡是固定因數）、e=RSA 公鑰（非自然常數）、d=RSA 私鑰、φ=歐拉函數、~=漸近非相等——同一符號的身分由「它在哪一章、它的鄰居是什麼」共同決定。** 跨到書架上別本書（微積分、機率、線代）要全部重新對表。

## 四、脊椎分解與關鍵公式速查

每條先一句「它在說什麼」，再給形狀與基準值，最後標章。這一節是全書「不得互相矛盾」的數字濃縮——改任何一個都要 grep 全書同步（維護鐵律），所以查到的就是定案值。

### Part I — 乘法的原子（ch01–03）

```text
質數定義（ch01）：
  它在說什麼：「恰有兩個正因數（1 與自身）的大於 1 整數」；
              1 不是質數（定義排除），也不是合數（沒有比 1 大的真因數）。
  形狀：p 為質數 ⇔ 正因數只有 1 和 p，且 p > 1。
  試除剪枝：判斷 n 是否為質數，只需試除到 ⌊√n⌋（因數成對出現，小的 ≤ √n）。

算術基本定理 FTA（ch02）：
  它在說什麼：每個 > 1 整數唯一分解為質數乘積（不計次序）。唯一性不是顯然——
              靠歐幾里得引理 VII.30 撐住。
  引理 VII.30：p|(a·b)，p 是質數  ⇒  p|a 或 p|b
  基準例：60 = 2²·3·5（全書唯一的標準分解，不計因數的排列順序）

歐幾里得數（ch03）：
  Eₙ = p₁·p₂·…·pₙ + 1（前 n 個質數的乘積加 1）
  E₁={2}→3 質   E₂={2,3}→7 質   E₃={2,3,5}→31 質
  E₄={2,3,5,7}→211 質  E₅={2,3,5,7,11}→2311 質
  E₆={2,3,5,7,11,13}→30031 = 59×509（合數！59 不在原列表 → 新質數仍找到）
  結論：Eₙ 本身未必是質數；但它的質因數一定不在原列表——這才是證明要的事。
```

### Part II — 質數有多稀（ch04–07）

```text
質數計數函數（ch06）：
  π(x) = 「≤ x 的質數個數」（只增不減的階梯函數）
  基準值：π(10)=4  π(100)=25  π(1000)=168  π(10⁶)=78498
  平均間隔：在 x 附近，下一個質數平均約 ln x 遠
            x=10⁶ 時：x/π(x) ≈ 12.7，ln(10⁶) ≈ 13.8（接近但不等）

質數定理 PNT（ch07）：
  它在說什麼：質數個別位置無規律，整體計數卻服從光滑的統計律——漸近，非相等。
  形狀：π(x) ~ x/ln x    （~ 是漸近，不是 =）
  精確逼近：Li(x) = ∫₂^x dt/ln t（系統性比 x/ln x 更接近 π(x)）
  基準對照（x/ln x 系統低估）：
    x=100    x/ln x ≈ 21.7    π(100)=25      低估
    x=10⁶   x/ln x ≈ 72382   π(10⁶)=78498   低估
  ln 基準值：ln10≈2.303  ln100≈4.605  ln(10⁶)≈13.816

對數積分 Li(x)（ch07）：
  Li(x) = ∫₂^x dt/ln t   ← 比 x/ln x 更接近真實 π(x) 的連續逼近
  Li(x) 在可計算範圍系統性大於 π(x)，但 Littlewood 1914 證這個不等式無窮次翻轉
  （Skewes 數量級首次反超，ch14）
```

### Part III — 模算術（ch08–11）

```text
同餘（ch08）：
  定義：a ≡ b (mod m)  ⇔  m | (a−b)（m 整除 a 與 b 的差）
  時鐘直覺：mod 12 的時鐘，15≡3，27≡3，… 都在同一格
  運算性質：加減乘可保持同餘（× 可保持）；除法要小心（需互質）
  工程版：a mod m == b mod m

輾轉相除法與貝祖（ch09）：
  gcd(a,b)=gcd(b, a mod b)（遞迴到餘 0 停）
  基準例：gcd(48,18)：48=2·18+12，18=1·12+6，12=2·6+0 → gcd=6
  貝祖：反向代回得 6 = 48·(−1) + 18·3
  模逆元存在 ⇔ gcd(a,m)=1（互質）；反向輾轉相除算出逆元

費馬小定理（ch10）：
  形狀一：a^(p−1) ≡ 1 (mod p)，p 質、gcd(a,p)=1
  形狀二：a^p ≡ a (mod p)（對所有整數 a 成立）
  基準例：p=7, a=2：2^6=64=9·7+1 ⇒ 2^6≡1 (mod 7)  ✓
  費馬 1640 陳述（無證明）；歐拉 1736 首證

歐拉函數與歐拉定理（ch11）：
  φ(n)＝1..n 中與 n 互質的整數個數
  公式：φ(p)=p−1；φ(pq)=(p−1)(q−1)；φ(p^k)=p^(k−1)·(p−1)
  乘性：若 gcd(m,n)=1 則 φ(mn)=φ(m)φ(n)
  質因數分解公式：φ(n) = n · ∏(1 − 1/p)（對 n 的每個不同質因數 p）
  基準例：
    φ(60) = 60·(1−1/2)·(1−1/3)·(1−1/5) = 60·1/2·2/3·4/5 = 16
    φ(33) = φ(3)·φ(11) = 2·10 = 20（RSA 基準）
    φ(12) = 4（互質者：1,5,7,11）
  歐拉定理：a^φ(n) ≡ 1 (mod n)，gcd(a,n)=1
            （n 為質數 p 時退回費馬小定理，φ(p)=p−1）
            歐拉 1763 年證推廣版。
```

### Part V — 應用與收束（ch16–20）

```text
梅森質數與最大已知質數（ch16）：
  梅森數：Mₚ = 2^p − 1，p 必須先是質數（但不保證 2^p−1 也是質數）
  反例：p=11 → 2^11−1 = 2047 = 23 × 89（合數，p 質 ≠ Mₚ 質）
  已知梅森質數序（精確到 2026-06 前幾個）：
    M₁=3(p=2), M₂=7(p=3), M₃=31(p=5), M₄=127(p=7)…
    p=11 → 2047 合數，跳過
  最大已知質數：
    M52 = 2^136279841 − 1（41,024,320 位，2024-10，Luke Durant，
          史上第一個用 GPU 找到的梅森質數）⚠️(2026-06)
    精確度守則：「第 52 個**被發現的**梅森質數」（the 52nd known），
    非「確定排序第 52 個」（指數落在尚未雙重驗證的區間，理論上
    可能藏更小未知梅森質數）
  前一紀錄：M51 = 2^82589933−1（2018，24,862,048 位）

RSA 玩具金鑰（ch17）：
  選 p=3, q=11（兩個質數）
  計算：n = p·q = 33；φ(n) = (p−1)(q−1) = 2·10 = 20
  選公鑰指數：e=3，需 gcd(e,φ)=1，即 gcd(3,20)=1 ✓
  算私鑰：d=7，需 e·d ≡ 1 (mod φ)，即 3·7=21=20·1+1 ≡ 1 (mod 20) ✓
  加密：c = m^e mod n（例：m=2 → c=2³=8 mod 33 → c=8）
  解密：m = c^d mod n（例：c=8 → 8^7 mod 33 → 2 ← 正確還原）
  保證可逆：歐拉定理 m^φ(n)≡1 → m^(ed)=m^(kφ+1)≡m (mod n)
  安全性：靠「分解 n 很難」的**經驗難度**，非已被證明永遠難（見 T9）
  e 的合法條件：gcd(e, φ(n))=1（不是 gcd(e,n)=1——兩個不同的條件）
  注意：e=3 與 p=3 數值巧合，但合法性看的是 gcd(3,20)=1 而非 gcd(3,33)≠1
```
