# 乘法的原子

### 數論與質數的直覺與驚嘆

> 每個整數都由質數相乘搭成，就像每種物質都由原子構成——但這些「乘法的原子」長在哪裡？它們看似隨機散落在數線上，卻服從一條鐵一般的統計律。亂與律，如何在同一批數裡共存？
>
> 這本書想讓你親身感受這一刀——並且能用自己的話，對另一個工程師講清楚：**質數為什麼既像亂數又有鐵律，以及歐幾里得那個兩千年前的五行證明今天為何還是美得想哭。**

---

這是一本寫給資深後端工程師的數論「欣賞」書。純興趣，和工作無關，純紙上推演——它不要你證明質數定理、不要你跑任何程式、不發 lab 作業。它只想帶你看懂一件反直覺的事：**質數是每個整數唯一的分解方式的建材，它們分散在數線上卻沒有生成公式，卻又服從一條讓人嘆氣的統計律——質數定理（PNT）告訴你，即使個別質數沒有規律，整體的計數函數 π(x) 卻如光滑曲線一般可測。**

你其實已經在用質數的某些性質了，只是憑手感，沒看過內臟：雜湊表桶數選質數是為了打散週期性 key；環形緩衝的繞回是 `index % size` 的模算術；公鑰密碼 TLS 靠的是「兩個大質數相乘容易、分解回去極難」的不對稱。這本書把這些工程直覺接上一個更大的故事。

## 這本書怎麼讀

- **純紙上推演**：每章的「紙上推演」是手算因數分解、輾轉相除、費馬小定理驗證、口頭重講一個證明——全是紙筆，不需要開任何終端機。所有數值示範一律手算，攤在你眼前的 ` ```text ` 表裡。
- **直覺與精確意義優先**：每個概念先回答「它在問什麼問題」（同餘在問「除以 m 後餘數相不相同」、PNT 在量化「質數平均多稀」、貝祖在保證「gcd 一定能寫成線性組合」），再給形狀。
- **講得出來才算過**：每章末的「自我檢核」是口頭題——對齊「能不能把這個概念講給另一個工程師聽」的深度。
- **直覺的陷阱是賣點**：數論充滿「看起來顯然其實要證」與「被媒體誤傳」的點。每章的「直覺的陷阱」段，專門守住那些常見誤解的正確版（「歐幾里得不是反證法」「有界間隙 ≠ 孿生質數已證」「PNT 是漸近不是相等」「驗證 ≠ 證明」）。

## 一條脊椎貫穿全書

整本書反覆回到**同一個分解 60 = 2²·3·5**，在每個新 Part 重新認識這個熟悉的小數一次。

- ch01 它登場：60 的因數、哪些是質數建材。
- ch02 它揭示唯一性：分解是不是只有這一套？（是，要證。）
- ch09 從它讀出 gcd：gcd(60, 18)=6，輾轉相除與貝祖都用它示範。
- ch11 從它算出 φ：φ(60) = 60·(1−1/2)·(1−1/3)·(1−1/5) = 16，歐拉函數的計算引擎。
- ch19/20 收束：60 這個小數一路被看穿了幾層，RSA 玩具金鑰也跟著脊椎回收。

同時貫穿全書的還有**脊椎 B——「質數長在哪裡」這個謎**：無窮（歐幾里得）→稀疏（篩法與沙漠）→平均密度（PNT）→細節誤差（黎曼）→最近距離（孿生猜想未解）→最大已知（M52，2026-06）。謎的每一層，ch19 對帳一次。

## 書架上的鄰居

這本書只用自己的欣賞聲音講「為何美／為何反直覺」，遇到需要更深工具的地方，它會把你指向書架上相鄰的書：

- 質數定理（PNT）的 ln 與漸近、對數積分 Li(x)=∫₂^x dt/ln t、黎曼 ζ 函數的解析延拓、頻譜直覺——解析數論的工具全部指向《馴服無限：微積分》。本書只用結果與「平均密度 1/ln x」的直覺。
- 質數的「機率式啟發」（Cramér 隨機質數模型、把「n 是質數」當機率約 1/ln n 來估）、個體不可測總體可測的大數法則——指向《馴服隨機：機率與統計》。本書點到「這是啟發式、非證明」即可。
- 算術基本定理（唯一分解）那章埋下的伏筆——「這個唯一性，日後哥德爾會拿來把整串邏輯證明壓成一個獨一無二的數字」——正面展開在《這句話無法被證明》談哥德爾編碼那章。這是全套最招牌的跨書連結之一：數論 owner FTA，那本書回頭明引。
- 貝祖等式的線性組合若想延伸到格（lattice）或線性結構，交界很弱，可一瞥《矩陣是動詞》；本書一律一句帶過、不展開。
