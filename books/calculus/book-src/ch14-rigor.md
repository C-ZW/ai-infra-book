# ch14 — 嚴格化：ε-δ、實數的補完與無窮小的平反

> **本章解決什麼問題**：ch04 的 Berkeley 之譏、ch11 的收斂之亂、ch12 的發散詐騙、ch13 的怪獸——四張傳票同時到期，「靠直覺擔保」的微積分正式破產。本章是重建工程：把 ch03 的 ε 挑戰賽法典化成 ε-δ 定義、給「連續」一個一句話的精確身分、用戴德金切割把數線上的洞補滿；最後講一個漂亮的歷史反轉——被驅逐的無窮小，二十世紀被羅賓遜平反。讀完這章，ch01 芝諾開出的三張帳單（無限多步、瞬間、無限可分）全部結清。

## 從你已知的出發

PM 說：「這個 API 要夠快。」這句話沒辦法驗收——誰來測、測哪些請求、多快算夠？你被這種需求坑過，所以你會把它改寫成條款：「任一個 5 分鐘觀測窗內，99% 的請求延遲 < 200 ms。」這句話可以驗收，而且驗收方式是**對抗式**的：QA 任挑一個觀測窗來刁你，你的系統都得守住。把形容詞換成帶量詞的不等式——這是你做過很多次的 spec 工程。

「越來越近」就是數學裡的「夠快」：一個禁不起驗收的形容詞。ch02 的雙重歸謬、ch04 的「h 趨近 0」、ch07 的「切到無限細」，全靠它撐著；乖例子上沒事，ch11 到 ch13 的事故證明它撐不住。本章要做的，就是替「越來越近」簽 SLA。而且你已經看過草約了——ch03 的 ε 挑戰賽。那時是故事版：對手出誤差容忍 ε，你回門檻 N，之後每一項都得進帶。本章把它寫成正式條文，再把條文沒寫到的最後一個漏洞——「逼近的目的地可能根本不存在」——用實數的完備性堵上。

## 賬單到期：四張傳票

先把債務清單攤開，看清楚要修的是什麼。

**傳票一（ch04）**：1734 年 Berkeley 指出，牛頓的增量「既不是零又是零」——消逝之量的鬼魂（見 ch04）。微積分算出的答案全對，邏輯地基是空的。一百多年沒有人能正面回應。

**傳票二（ch11）**：傅立葉宣稱任何週期函數都是 sin 的無限和，但「等於那個無限和」是什麼意思，沒有人說得清；逐項積分這種對有限和天經地義的操作，搬到無限和上無人背書（見 ch11）。

**傳票三（ch12）**：無限和不是加法是極限——忘記這件事的人，被重排定理示範了「同一堆數想加成多少就加成多少」（見 ch12）。

**傳票四（ch13）**：「連續的曲線總可以畫切線」這個兩千年的圖像信仰，被魏爾斯特拉斯一條處處連續、處處不可微的三角級數拆掉（見 ch13）。

四張傳票，同一個病根：**當時所有的定義都長在圖像上**。「接近」是腦中的一段動畫，「連續」是一筆畫不離紙，「和」是把東西堆起來。圖像當直覺極好——本書前十三章都在用——但圖像當定義，遇到怪獸時每個人看到的東西不一樣，吵架沒有裁判。需要的是另一種定義：不引用任何圖、不依賴任何人的想像力、能被最不友善的對手逐字檢驗。十九世紀數學家交卷的答案，就是把 ch03 那場遊戲寫成法律。

## ε-δ：把挑戰賽寫成法典

正式條文一行（極限的 ε-δ 定義（epsilon-delta definition））：

```text
lim(x→a) f(x) = L  ⟺  ∀ε>0 ∃δ>0 ∀x (0 < |x−a| < δ ⇒ |f(x)−L| < ε)
```

大一的你看到這行符號大概只想翻頁。先別翻——逐字翻譯之後，它就是 ch03 挑戰賽的對局規則，一個字都沒有多：

```text
∀ε>0           對手出招：任挑一個誤差容忍 ε，越小越刁鑽
∃δ>0           你回招：交出一個窗寬 δ——可以依 ε 量身訂做
∀x(0<|x−a|<δ   裁判驗收：凡是進了你的窗的 x（a 本人除外）
  ⇒|f(x)−L|<ε) 它的函數值就必須進對手的帶。漏一個 x 就輸
```

**量詞的順序就是出招的順序。**∀ε 寫在 ∃δ 前面，意思是對手先出、你後回，所以你的 δ 可以依 ε 訂做——這正是你能贏的原因。把順序對調成 ∃δ ∀ε，字面只差一個位置，意思卻是「你先押死一個 δ，之後任何 ε 都得接」——一個強到幾乎沒有函數能滿足的宣稱（紙上推演第 2 題會讓你親手算出它到底在說什麼）。如果你讀過書架上那本 TLA+ 書，∀∃ 的攻防結構你已經演練過；沒讀過也無妨，ch03 的挑戰賽就是完整的訓練。數列版 lim(n→∞) aₙ = L 同款，只是「窗寬 δ」換成「門檻 N」。

三個細節值得停下來看，每一個都是精密的工程決策。

**第一，0 < |x−a| 那個 0。**它把 x = a 本人踢出驗收範圍：極限從頭到尾不過問 f(a) 是多少，甚至不要求 f(a) 有定義。這一個小於號，就是 ch04 整章的合法性來源——差商 (f(x+h)−f(x))/h 在 h = 0 處根本沒定義，沒關係，定義從來不看那一點（見 ch04）。

**第二，條文裡沒有任何東西在動。**「趨近」「越來越」「消逝中」這些動詞全部蒸發了，剩下的只有靜態的不等式：給定的 ε、存在的 δ、對每個 x 成立的蘊涵。芝諾拿來開刀的「運動」、Berkeley 譏笑的「消逝中的量」，在條文裡連藏身處都沒有——沒有量正在變小，自然沒有鬼魂。第一張傳票，結清。

**第三，條文比直覺更會說話。**ch11 留下一句「逐點收斂成立、一致貼合不成立」，當時只能用 Gibbs 耳朵比劃（見 ch11）；現在可以一字不差寫下來——「每個 x 各自有自己的 N」是 ∀x ∃N，「一張 N 全場通用」是 ∃N ∀x，差別只在 ∃N 站在 ∀x 的前面還是後面。直覺裡糊成一團的兩種「收斂」，在量詞語言裡是兩個肉眼可辨的句子。順帶一提，ch05 賒的帳——極限的和、積、商運算定律——在這套語言下全部成為可逐條證明的定理（本書不證，見延伸閱讀的 Spivak）。

我認為這是整本書最被低估的聰明：一行式子裡裝著一座無限大的測試矩陣——每個 ε 一列——但每一張測試單都是有限檢查。大一時它折磨你，不是因為它難，是因為沒有人告訴你這是一場對局的轉播逐字稿；你拿到的是法典，沒拿到立法理由書。

## 完整示範：lim(x→3) x² = 9

來打一場函數版的正式比賽。先畫好球場：

```text
      y
      │                           *    f(x) = x²
      │                        *
  9+ε ┤┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄*┄┄┄┄┄┄┄┄┄  ← 帶的上緣（ε 是對手出的）
      │                  *
    9 ┤···············*···············  ← L = 9
      │            *
  9−ε ┤┄┄┄┄┄┄┄┄┄*┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄  ← 帶的下緣
      │      *
      │   *
      └──────────┴────┴────┴──────────  x
                3−δ   3   3+δ
                 └─────────┘
                 你回的 δ 窗
```

對手畫一條水平帶（9 ± ε），你要回一個豎直窗（3 ± δ），使曲線在窗內的部分整段住在帶裡——曲線穿出帶的位置，必須在窗外。ε 越小、帶越扁，你的窗就得越窄。

現在誠實一次。大學課本只印「正式證明」那一欄，乾淨得像第一次就寫對；數學家真正的工作現場是另一欄——**倒著走**：從想要的結論出發，反推需要什麼。兩欄並排：

```text
┌─ 草稿：倒著找 δ（不外流） ───────────┬─ 正式證明：正著寫（出版品） ───┐
│ 目標　把 |x²−9| 壓到 ε 以下          │ 1. 給定任意 ε > 0。            │
│ 拆解　|x²−9| = |x−3|·|x+3|           │ 2. 取 δ = min(1, ε/7) > 0。    │
│ 盤點　|x−3| 歸我管（它 < δ）；       │ 3. 設 0 < |x−3| < δ。          │
│ 　　　|x+3| 是路人，得先銬住         │ 4. 因 δ ≤ 1：|x−3| < 1，       │
│ 押注一　要求 δ ≤ 1，於是             │    故 2 < x < 4，|x+3| < 7。   │
│ 　　　|x−3| < 1 ⇒ 2 < x < 4          │ 5. 因 δ ≤ ε/7：                │
│ 　　　⇒ |x+3| < 7                    │    |x²−9| = |x−3|·|x+3|        │
│ 代回　|x²−9| < 7·|x−3| < 7δ          │           < δ·7 ≤ (ε/7)·7      │
│ 押注二　要求 δ ≤ ε/7，               │           = ε。（證畢）        │
│ 　　　使 7δ ≤ ε——到手                │                                │
│ 合併　δ = min(1, ε/7)                │ （對手任出 ε，皆已接住。）     │
└──────────────────────────────────────┴────────────────────────────────┘
```

左欄的劇情：誤差 |x²−9| 拆開後是兩個因子的乘積，|x−3| 是你直接控制的量，|x+3| 卻跟著 x 跑——它是個**變動的放大倍率**（在 x = 3 附近大約是 6）。對付變動倍率的招數，是先把窗釘死在一個範圍內：押 δ ≤ 1，倍率就被銬在 7 以下；剩下的事只是除法。min(1, ε/7) 的兩個成分各管一件事：1 銬路人，ε/7 壓誤差。

實彈試射一發：對手出 ε = 0.7，你回 δ = min(1, 0.1) = 0.1。抽查 x = 3.05：|x²−9| = |9.3025−9| = 0.3025 < 0.7 ✓；逼到窗邊 x → 3.1：誤差 → 9.61−9 = 0.61，仍 < 0.7 ✓——還留有餘裕，因為那裡的實際倍率只有 6.1，我們用 7 當保守上界。

幾件事要說破。第一，**δ 不唯一，醜不扣分**：押 δ ≤ 2 會得到倍率上界 8、答案 δ = min(2, ε/8)，照樣全贏；比的是能不能接住，不是誰的 δ 最大最優雅。第二，**倒著找、正著寫，不是作弊也不是滅證**。草稿欄是探案，證明欄是呈堂證供——證明的職責不是重演發現過程，是讓每一行都能被逐行查驗。你也不會把 debug 時的二十次亂試寫進 commit message。大一的創傷多半來自把出版品當原稿讀：你以為別人一眼就看出 min(1, ε/7)，其實人人都是倒著爬回來的。第三，這場勝利跟 ch03 的終局同款——一條公式接管對手的所有出招，無限場驗收一次清空。

## 連續：搭一句話的便車

極限定義好了，連續（continuous）就是搭便車的一句話：

```text
f 在 a 點連續  ⟺  lim(x→a) f(x) = f(a)
```

展開就是 ∀ε>0 ∃δ>0 ∀x (|x−a|<δ ⇒ |f(x)−f(a)|<ε)——跟極限的條文幾乎同一張，差別有二：目的地 L 換成 f(a) 本人，以及不再需要「0 <」（x = a 自己代入，誤差是 0，永遠達標）。這一句話裡藏著三項驗收：f(a) 要存在、極限要存在、兩者要相等。ch08 那個跳躍函數敗在第三項：左右極限各自存在卻不會合（見 ch08）。「一筆畫不離紙」的圖像退役，換上可以逐點開庭的條款——而 ch06 的極值定理、ch08 的 FTC 第 5 步，這些「連續就保證……」的承諾書，全都要等下一節把地基灌好才能兌現。

## 實數的補完：數線上的洞

最後一個漏洞最深。挑戰賽驗收的是「逼近 L」，但誰保證那個 L **存在**？

在有理數（rational numbers）的世界裡做個實驗。定義 xₙ ＝「小數點後 n 位、平方仍小於 2 的最大小數」：1, 1.4, 1.41, 1.414, 1.4142, …（複核：1.41² = 1.9881 < 2 < 2.0164 = 1.42²）。這個數列每一項都是有理數、遞增、有上界 1.5、相鄰項差距不超過 10⁻ⁿ——以一切直覺標準它都在「收斂」。但它的目的地是 √2，而 √2 不是有理數（紙上推演第 3 題給完整論證）。**夾擠程序執行得完美無缺，夾住的位置卻是一個洞。**

注意這有多陰險：ℚ 是稠密（dense）的——任兩個有理數之間還有無限多個有理數——畫在紙上密不透風，看起來就是一條完整的線。**稠密不等於完備**；你畫的那條「數線」自帶連續性，是一張預載了結論的圖。這正是「圖像不能當定義」最深的一個案例：洞用眼睛看不到，只有像上面那樣的數列才探測得到。

1872 年，戴德金（Richard Dedekind）給出了補洞的辦法——據他自述，想法成形於 1858 年 11 月 24 日，在蘇黎世教微積分、自覺講不清楚「連續」的那段日子，擱了十四年才出版。做法叫切割（Dedekind cut）：拿一把刀把全部有理數劈成左右兩堆，左堆每個都小於右堆每個。有的刀口正落在某個有理數上；有的刀口落空——例如「左堆＝負數與平方小於 2 的正數，右堆＝其餘」這一刀，左堆沒有最大值、右堆沒有最小值，刀口處什麼都沒有。**落空的刀口就是洞。**戴德金的動作是：把每一刀本身宣告為一個數——實數（real number）就是有理數的切割全體。原本的有理數對應砍中的刀，無理數就是那些落空的刀。用洞的形狀替洞發身分證，荒謬又天才；而這套定義只用到有理數和集合，一張圖都沒引用。

補完之後得到的性質叫**完備性（completeness）**：對實數怎麼切，刀口都落在一個數上；等價的常用形態是「遞增且有上界的數列必收斂」（ch09 賒的那一筆）。另一條補洞路線一句話存檔：把實數定義成有理數柯西序列（Cauchy sequence）的等價類——梅雷（Méray）1869 年首發、康托爾 1872 年的版本流傳最廣——殊途同歸。

完備性為什麼是地基？因為**全書每一次「取極限」都假設了目的地存在**：ch02 雙重歸謬夾出的那個面積、ch03 幾何級數部分和的歸宿、ch07 上和與下和會合的那一點、ch06「連續函數在閉區間上必有最大值」的那個最大值（見 ch02、ch03、ch07、ch06）——沒有完備性，這些全是對著可能不存在的地址寄包裹。順手還能把 ch03 用過的阿基米德性質正式入帳：若自然數全體在 ℝ 中有上界，完備性給出最小上界 s，但 s−1 之上必有某個自然數 n，於是 n+1 > s，矛盾——所以不存在「小於一切 1/n 的正數」，實數裡藏不了無窮小。

把人物與年份排好隊，就是嚴格化的譜系：波爾查諾（Bolzano）1817 年已有嚴格定義但幾乎無人讀到；柯西 1821 年《Cours d'analyse》把分析學重建在極限之上，證明裡已是 ε 式論證，定義本身卻還用「無窮小增量」的語言寫——觀念奠基在柯西；今天課本上的符號版定稿於魏爾斯特拉斯——1861 年柏林講義起，由他的學派系統化；地基則是 1872 年灌的漿：戴德金的切割、康托爾的柯西序列，同一年，魏爾斯特拉斯還在這一年的 7 月向柏林科學院展示了怪獸函數（見 ch13）——驗收新地基的壓力測試與地基本身同年交付。注意工程順序：大樓十七世紀就蓋了（牛頓、萊布尼茲），樑柱 1821 到 1861 年補強，地基 1872 年才灌——**數學史是從屋頂往下蓋的**，跟教科書的呈現順序正好相反（ch16 會回收這條線）。

## 歷史反轉：無窮小的平反

故事到這裡，標準版的結局是：無窮小被驅逐出境，ε-δ 一統江湖。真實的結局多一個轉折，而且漂亮得多。

1960 年，邏輯學家羅賓遜（Abraham Robinson）在普林斯頓的研討會上提出非標準分析（non-standard analysis），1966 年出版同名專著：在嚴格的模型論基礎上造出超實數（hyperreal numbers）ℝ*——一個真的含有無窮小的數系，裡面有大於 0 卻小於一切 1/n 的數，加減乘除樣樣合法。在 ℝ* 裡，導數可以名正言順地定義成「兩個無窮小相除再取最近的標準數」，萊布尼茲的 dy/dx 直覺整套平反——遲到了快三百年（見 ch04、ch07 兩度埋的伏筆）。Berkeley 的傳票於是被結清了**兩次**：第一次是魏爾斯特拉斯把無窮小驅逐出境（鬼魂不存在），第二次是羅賓遜發給它合法證件（鬼魂領到身分證）。1734 年那聲譏笑，兩個答案都對得起它。

但平反有附帶條件。其一，ℝ* 的無窮小帶著嚴格的使用規則——轉移原理（transfer principle）：用標準語言寫的敘述，在 ℝ 為真若且唯若在 ℝ* 為真。這保證了兩邊證出的標準定理一模一樣——**兩套地基撐同一棟樓**，選哪套是工程偏好，不是對錯。其二，天下沒有白吃的地基：ℝ* 為了容納無窮小，放棄了完備性本身（無窮小全體有上界、卻沒有最小上界——若有，它的一半或兩倍立刻製造矛盾）。其三，造出 ℝ* 的模型論機器（超濾子、超結構）相當重，本書不開箱——知道機器存在、且通過驗收即可。也別把平反讀成追溯特赦：歐拉們十八世紀的形式操作是真的在沒有安全帶的情況下飛（見 ch12），羅賓遜的無窮小繫著安全帶；結果論的「反正最後都能嚴格化」不是跳步的授權。

那為什麼今天的課本還是 ε-δ？因為它不需要任何新機器——量詞和不等式，每個審稿人都讀得懂；它已是分析學的通用語。用無窮小教大一微積分可不可行？可行——Keisler 真的寫了一本，免費放在網路上（延伸閱讀）——但生態系沒有翻盤的理由。地基的選擇，最終是社群的工程決策；這句話你在技術選型會議上應該聽過。

最後一句話把視野拉開：地基翻新沒有完結篇。1902 年，勒貝格（Henri Lebesgue）的博士論文《Intégrale, longueur, aire》把積分整個重蓋了一次——黎曼積分對付不了的病態函數（ch13 那種等級），在勒貝格積分下服服貼貼——那是下一場重建的故事，本書送你到門口（延伸閱讀）。嚴格化不是數學家的潔癖，是事故之後補的回歸測試套件；而 ε-δ 這份 spec 寫得夠好，好到連對手陣營被平反之後，它仍然是大家開會用的語言。

## 直覺的陷阱

| 錯誤直覺 | 它把你帶去哪 | 自我察覺的徵兆 |
|---|---|---|
| 「ε-δ 是用來求極限的」 | 拿定義硬算 L 算不出來，挫折感爆表 | 你想用條文「解出」L——其實 L 要先用別的方法猜到（數值表、代數約簡），定義只負責驗收 |
| 「δ 要找到最好的那個」 | 卡在「我的 δ 是不是最大」，永遠寫不完證明 | 你在比較 min(1, ε/7) 和 min(2, ε/8) 誰「對」——都對，能接住就贏 |
| 「量詞順序無所謂」 | ∃δ ∀ε 與 ∀ε ∃δ 當同一句話讀，整個定義垮掉 | 你寫出一個不依賴 ε 的 δ 還覺得理所當然；說不出「對手先出招」是哪個量詞 |
| 「稠密就是沒有洞」 | 以為 ℚ 密不透風所以夾擠必有歸宿 | 你說不出 ℚ 缺哪條性質、也說不出「遞增有上界」在 ℚ 裡哪裡會失敗 |
| 「完備性看圖就知道，不用講」 | 把要證的東西畫進圖裡當前提 | 你的論證出現「從圖上看，它總會夾到一點」 |
| 「嚴格化是數學家的潔癖」 | 把 ε-δ 課當無意義的折磨，整個 Part V 白上 | 你講不出任何一件「不嚴格真的出過事」的事故——ch11 到 ch13 就是事故報告 |
| 「無窮小平反＝當年怎麼用都沒錯」 | 拿羅賓遜替自己的跳步辯護 | 你說「反正最後都嚴格化了」——平反的是概念，不是當年的論證 |

## 紙上推演

**第 1 題 [10 分鐘] ★　對 lim(x→2) 3x = 6 打一場正式 ε-δ**
先寫草稿欄（倒著找 δ），再寫證明欄（正著走）。然後回答：這題為什麼不需要 min？

**第 2 題 [15 分鐘] ★★　量詞對調之後，條文在說什麼？**
把定義改成 ∃δ>0 ∀ε>0 ∀x (0<|x−a|<δ ⇒ |f(x)−L|<ε)。推出：什麼樣的 f 能通過這個顛倒版驗收？f(x) = x² 在 a = 3 過得了嗎？

**第 3 題 [15 分鐘] ★★　有理數世界裡，夾擠夾到空氣**
沿用正文的 xₙ：1, 1.4, 1.41, 1.414, …（n 位小數、平方 < 2 的最大者）。(a) 證明 xₙ² 與 2 的差距小於 4·10⁻ⁿ；(b) 由此推出：若 xₙ 在 ℚ 中有極限 L，則 L² = 2；(c) 證明沒有有理數的平方是 2，結案。最後用一句話把這件事接回黎曼和。

**第 4 題 [10 分鐘] ★★　口頭題：講給被大一微積分傷害過的朋友聽**
不看書，把「ε-δ 是挑戰賽的法典化」講到對方願意再給它一次機會。講完對照解答的要點清單自評。

### 推演解答

**第 1 題**：草稿——目標 |3x−6| < ε；拆解 |3x−6| = 3|x−2|；|x−2| 歸我管，要 3δ ≤ ε，押 δ = ε/3，沒有路人需要銬。證明——給定任意 ε > 0，取 δ = ε/3 > 0；設 0 < |x−2| < δ，則 |3x−6| = 3|x−2| < 3·(ε/3) = ε。（證畢）試射一發：ε = 0.3 → δ = 0.1，x = 2.09 時 |3x−6| = 0.27 < 0.3 ✓。
不需要 min 的原因：x² 那題的誤差因子 |x+3| 是隨 x 變動的放大倍率，得先用 δ ≤ 1 銬住；3x 的「倍率」恆等於 3，天生就被銬好了。一般規律：直線的 δ 一步到位，彎的東西才需要先控制倍率——而「彎的東西局部看起來像直線」正是 ch05 的世界觀，所以 ε-δ 證明裡的 min 戲法，本質上是在「先把局部圈出來」。

**第 2 題**：固定窗內任一個 x（x ≠ a）。顛倒版要求 |f(x)−L| < ε 對**每一個** ε > 0 成立——一個非負的數小於一切正數，只能是 0，所以 f(x) = L。結論：顛倒版驗收的是「f 在 a 附近（除了 a 本人）恆等於 L」——不是「逼近」，是「釘死」。f(x) = x² 在 a = 3 過不了：任何窗內都有 x ≠ 3 使 x² ≠ 9。所以 ∀ε ∃δ 和 ∃δ ∀ε 之間隔著「所有會動的函數」——量詞順序不是文法潔癖，是定義的全部內容。

**第 3 題**：(a) 由 xₙ 的定義，xₙ² < 2 ≤ (xₙ + 10⁻ⁿ)²（再加一格就超標）。於是 2 − xₙ² ≤ (xₙ+10⁻ⁿ)² − xₙ² = 2xₙ·10⁻ⁿ + 10⁻²ⁿ < 3·10⁻ⁿ + 10⁻ⁿ = 4·10⁻ⁿ（用了 xₙ < 1.5 與 10⁻²ⁿ ≤ 10⁻ⁿ）。抽查 n = 2：2 − 1.9881 = 0.0119 < 0.04 ✓。(b) 若 xₙ → L，極限運算定律給 xₙ² → L²；但 (a) 說 xₙ² → 2，極限唯一，故 L² = 2。(c) 設 L = p/q 為最簡分數且 p² = 2q²：左邊 p² 是偶數 ⇒ p 偶，寫 p = 2k 得 4k² = 2q² ⇒ q² = 2k² ⇒ q 也偶——與最簡矛盾。所以 ℚ 裡沒有這個 L：數列乖乖夾擠，卻夾到空氣。
接回黎曼和：黎曼和的機器正是兩條單調數列（上和、下和）互相夾擠（見 ch07），機器本身不保證夾住的位置是個「存在的數」——保證書是完備性簽的。同一句話的另一個版本：π 的十進位截斷 3.1, 3.14, 3.141, … 全是有理數、夾擠得漂漂亮亮，但 π 不是有理數（蘭伯特（Lambert）在 1760 年代用連分數證明，本書不證）——在只有 ℚ 的宇宙裡，ch02 阿基米德夾了一輩子的那個目標根本不存在。

**第 4 題**：要點清單（你的版本必須命中至少前四點）：①先講遊戲再給符號——對手出誤差 ε、你回窗寬 δ、窗內全進帶就贏；②量詞順序＝出招順序，∀ε 在前所以 δ 可以依 ε 訂做；③定義只驗收、不搜尋——L 是先猜出來再驗收的，拿定義求 L 注定挫折；④課本只印「正著寫」的出版品，沒印「倒著找」的草稿——你當年覺得自己笨，其實是被滅了證的工作流程騙了；⑤一行條文＝無限張驗收單，但每張都是有限檢查——這是人類對「無限」最成功的一次工程化收編。加分項：現場用 lim(x→2) 3x = 6 打一局示範。
