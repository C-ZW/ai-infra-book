# ch13 — 怪獸函數：連續但處處尖刺

> **本章解決什麼問題**：ch12 拆穿了「無限和不是和」，這一章拆的是更貼身的信仰——ch05 以來全書賴以維生的「局部線性」世界觀：任何連續曲線放大到夠近就變直線。魏爾斯特拉斯用一條處處連續、卻處處畫不出切線的曲線證明：「連續」與「可微」之間隔著一個宇宙。這隻怪獸不是獵奇收藏品，它住在布朗運動與你的監控曲線裡；而它造成的恐慌，直接逼出 ch14 的嚴格化工程。

## 從你已知的出發

打開 Grafana，看一條 P99 延遲曲線的七天視圖：毛毛的。直覺說這是解析度問題——放大到一天、一小時、一分鐘，總會平滑下來吧？你放大了，它還是毛的；再放大，每個尺度都有新一層的尖刺等著你。彷彿這條曲線的「毛」不是雜訊，是體質。股價走勢圖也一樣：月線、日線、分線，永遠鋸齒，從來沒有哪個縮放級距會讓它看起來像一條可以畫切線的曲線。

對比 ch05 的場景：容量規劃時那條 CPU 對 QPS 的曲線，整體弧度明顯，但你敢在工作點附近做線性外插——因為放大之後它就是直線。所以你的螢幕上其實住著兩個物種：一種曲線 zoom 會直，一種曲線 zoom 永遠毛。前者是 Part II 的天下；後者呢？十九世紀的數學家會說：後者不存在——至少不可能是一條「連續的曲線」。本章要看著這個信念被一條公式殺死，而兇器你已經很熟：無限疊加（見 ch11）。

## 十九世紀的信仰：連續曲線總有切線

先把被害人請上台。ch05 的世界觀是**局部線性**：光滑曲線放大到夠近就是直線，導數就是那條直線的斜率。當時我們誠實附註過：|x| 在 0 點放大一萬倍還是尖角——「光滑」是前提。但十九世紀初的主流直覺認為這個前提便宜得近乎免費：**連續的曲線，頂多在幾個孤立的壞點上畫不出切線，其餘地方都乖**。畢竟「連續」就是一筆畫不抬筆，你拿筆畫畫看——手畫得出來的曲線，哪一條不是大段大段平滑的？

這個直覺強到有人想把它升格成定理。1806 年，安培（André-Marie Ampère，就是電流單位那位）發表了一個論證，宣稱連續函數除了一些例外點都可微；這個結論一度以「安培定理」的名義流通，十九世紀不少教科書把它當成事實陳述（2026-06 查證；安培的「函數」定義含糊，他到底主張了多強的命題，史家仍有爭論）。注意這不是懶惰——當時「函數」的具體樣本就是多項式、三角、指數這些解析品種，全是乖孩子。樣本偏差餵養直覺，直覺寫進教科書。

然後是 1872 年 7 月 18 日。魏爾斯特拉斯（Karl Weierstrass）——就是 ch14 將要登場的、把 ε-δ 系統化的那位嚴格化教父——在柏林的皇家普魯士科學院宣讀了一條函數：

```text
W(x) = Σ(n=0→∞) aⁿ cos(bⁿπx)

魏爾斯特拉斯的原始條件：0 < a < 1，b 為正奇數，且 ab > 1 + 3π/2 ≈ 5.71
（滿足條件的最小 b 是 7）
```

他證明：這條函數**處處連續、處處不可微**。不是壞在幾個點，是壞在每一個點——任何一點都畫不出切線，任何尺度放大都不會變直。「安培定理」不只是錯，是錯得不能再錯：連續性對可微性的保證，精確值為零。

魏爾斯特拉斯本人沒有出版這個結果；首次印成白紙黑字是 1875 年，du Bois-Reymond——你在 ch11 結尾見過他，構造連續函數讓傅立葉級數發散的就是他——把它刊在 Crelle 期刊第 79 卷。數學界的反應可以用一句流傳極廣的話作代表：埃爾米特（Charles Hermite）在 1893 年 5 月 20 日致 Stieltjes 的信裡寫道，他「懷著驚恐厭惡之情，別過頭去不看這個連續卻沒有導數的函數的可悲瘟疫」。這封信確實存在（兩人通信集 1905 年出版，第 374 號信），但引用要誠實加註：信的語境半帶玩笑，把它讀成埃爾米特反對嚴格分析是過度解讀（2026-06 查證：出處可考，語氣與解讀有爭議）。

最後一塊事實拼圖：原始條件後來被大幅放寬。哈代（G. H. Hardy）1916 年證明，只要 **0 < a < 1 且 ab ≥ 1**，函數就處處不可微——b 不必是奇數，甚至不必是整數。記住這條，等一下我們親手疊怪獸時要靠它背書。

## 從一個尖點到處處尖點：不平滑的譜系

「處處不可微」聽起來抽象，先從你能完全掌握的一個尖點出發，把譜系走一遍。

**一個尖點**：f(x) = |x| 在 x = 0。回到 ch04 的定義，導數是差商的極限，而極限不在乎你從哪邊靠近——左右兩邊必須給出同一個答案：

```text
右邊靠近（h > 0）：(|0+h| − |0|) / h = h/h = +1     ← 每一步都是 +1
左邊靠近（h < 0）：(|0+h| − |0|) / h = −h/h = −1    ← 每一步都是 −1

兩個單邊極限各自存在、但不相等 → 差商的極限不存在 → 0 點不可微。
而連續性毫髮無傷：x → 0 時 |x| → 0 = |0|。
```

這就是「連續但不可微」的最小可重現案例：曲線沒有斷，但在 0 點問「方向是哪邊」，左右兩派吵翻，沒有共識。

**可數多個尖點**：把 |x| 接成鋸齒波，每個轉折處都是一個尖點——無限多個，但它們是孤立的，相鄰尖點之間還是平滑直線段。再把越來越小、越來越密的鋸齒一層層疊上去，尖點開始增殖、加密。這條路線走到極限，就是「處處是尖點」的怪獸（用鋸齒疊的版本史上真的有人做過，延伸閱讀的圖鑑裡一隻一隻排好）。

**處處尖點**：魏爾斯特拉斯用的不是鋸齒而是 cos，這讓結果更驚悚一級——他的每一層、乃至**每一個部分和，都是無限次可微的光滑函數**，連一個尖點都沒有。尖刺不住在任何一層裡，只住在無限疊加的極限裡。你應該覺得這個句式眼熟：ch11 的方波跳躍不住在任何一項 sin 裡，住在整條尾巴的集體行為裡。同一種無限的脾氣。

還有一件事比尖點更糟，先預告：|x| 在 0 點好歹左右各有一個安定的斜率（+1 與 −1），只是不相等；怪獸連這個都不給——等一下你會親眼看到差商不是「分裂成兩派」，而是乾脆發散。

## 怪獸的藍圖：兩本帳，一場拉鋸

現在拆解構造。W(x) 是無限多層波的疊加：第 n 層是 aⁿ·cos(bⁿπx)，振幅 aⁿ 一層比一層小（a < 1），頻率 bⁿ 一層比一層快（b > 1）。畫面感：第 0 層是一個慢悠悠的大波；第 1 層在它身上貼一層小 b 倍快的抖動；第 2 層再貼更小、更快的……每一層都給曲線添一代新的、更細的毛。

關鍵洞察是：**連續性與可微性各記各的帳，而且記的是兩本不同的帳。**

```text
第 n 層的「振幅預算」：aⁿ
  全部加總：Σ(n=0→∞) aⁿ = 1/(1−a)        ← 幾何級數，收斂（ch01 的老朋友）

第 n 層的「斜率預算」：aⁿ·cos(bⁿπx) 最陡處的斜率 = aⁿ·bⁿ·π = (ab)ⁿ·π
  全部加總：Σ (ab)ⁿ·π                      ← ab > 1 時，公比大於 1，發散
```

**連續性只看振幅帳**。前 N 層的部分和是有限個連續函數相加，連續；剩下的尾巴被 Σ(n≥N) aⁿ = aᴺ/(1−a) 罩住——這個誤差上限對**所有** x 同時成立，而且 N 夠大就能壓到任意小。整條曲線躲在一條會收斂的誤差帶裡，極限自然連續。工程師的嚴謹版到此為止，每一步你都能口頭講出理由；把「一致的誤差帶保住連續」做成正式定理，是 ch14 那條路上的事，本書不證。

**可微性看的是斜率帳**。想在一點畫切線，差商得安定下來。但你每把觀察尺度縮小 b 倍，就有下一層醒過來，帶著 (ab)ⁿπ 的斜率預算進場——ab > 1 時，越深層的抖動反而**越陡**。振幅在縮、斜率在漲，這就是 a 與 b 的拉鋸：**a 負責把振幅壓下去保住連續，b 負責把頻率撐上來毀掉光滑**，而勝負手是乘積 ab。誠實標示：「斜率預算發散」是方向感、不是證明——證明還得排除各層斜率剛好互相抵銷的可能，這正是魏爾斯特拉斯需要奇數 b 和 ab > 1 + 3π/2 的地方（讓各層在關鍵點上同向用力），哈代 1916 年用更重的機具把門檻磨到 ab ≥ 1。

這本雙帳本值得停十分鐘，因為它一句話回答了「為什麼不矛盾」：**同一個構造，振幅帳收斂、斜率帳發散，兩件事毫不衝突——連續性是 0 階性質，只管「位置別跳」；可微性是 1 階性質，要管「方向要安定」。** 直覺以為它們差不多，是因為日常樣本裡兩本帳總是一起收斂。把 ab 轉到小於 1，同一個無限疊加就是一條處處光滑的乖函數（斜率帳也變成收斂的幾何級數）；轉到 ab ≥ 1，怪獸出閘。我認為「連續與可微原來分屬兩本帳」是本章最值得轉述的一句話。

放大鏡下的畫面也能寫成一條精確的式子。把 Σ aⁿ cos(bⁿπx) 的第 0 層剝掉，剩下的恰好是整隻怪獸的縮小版：

```text
W(x) = cos(πx) + a·W(bx)
      ← 驗算：a·W(bx) = Σ aⁿ⁺¹ cos(bⁿ⁺¹πx)，正是 W(x) 少掉 n=0 那層
```

讀法：把鏡頭橫向放大 b 倍、縱向壓成 a 倍，看到的還是同一隻怪獸，只差一條光滑的修正項。所以「放大會變直」在這裡注定失敗——**每個 zoom 級距裡都藏著一隻完整的、等比例的怪獸**。你監控曲線「怎麼放大都毛」的體感，第一次有了代數形式。

## 親手疊一隻迷你怪獸

紙筆時間。為了讓 cos 全部落在背得出來的角度上，取 **a = 1/2、b = 3**，算前三層。先把誠實聲明放在最前面：b = 3 **不滿足**魏爾斯特拉斯的原始條件（ab = 1.5，遠小於 1 + 3π/2 ≈ 5.71；想用 a = 1/2 滿足原始條件，奇數 b 至少要 13）——選它純粹是為了手算可行、看清「疊加」這個動作本身。但它滿足哈代 1916 年的條件 ab = 1.5 ≥ 1，所以這條迷你版**真的**處處不可微，只是 1872 年的工具還證不動它，要等 1916 年的背書。

在 x = k/6 這些點上，三層的角度分別是 kπ/6、kπ/2、3kπ/2，全是單位圓上的熟面孔（cos(π/6) = √3/2 ≈ 0.8660）。逐層累加：

| x | 第 0 層 cos(πx) | 加第 1 層 → S₁ | 加第 2 層 → S₂ |
|---|---|---|---|
| 0 | 1 | 1.5 | 1.75 |
| 1/6 | 0.8660 | 0.8660 | 0.8660 |
| 1/3 | 0.5 | 0 | −0.25 |
| 1/2 | 0 | 0 | 0 |
| 2/3 | −0.5 | 0 | 0.25 |
| 5/6 | −0.8660 | −0.8660 | −0.8660 |
| 1 | −1 | −1.5 | −1.75 |

抽兩個點看引擎怎麼轉：

```text
x = 1/6（角度 π/6、π/2、3π/2）：
  S₂ = cos(π/6) + (1/2)cos(π/2) + (1/4)cos(3π/2)
     = 0.8660 + 0 + 0 = 0.8660          ← 新層全停在零點，按兵不動

x = 1/3（角度 π/3、π、3π）：
  S₂ = cos(π/3) + (1/2)cos(π) + (1/4)cos(3π)
     = 0.5 − 0.5 − 0.25 = −0.25         ← 兩層新抖動全力往下拽
```

同一條曲線，有的點新層沉默、有的點新層全力使勁——形狀就這樣被揉皺。畫出來（ASCII 解析度有限，看形狀趨勢就好）。一層，平滑的波：

## 差商為什麼拒絕安定

構造看完了，現在補方向感：差商到底怎麼個不收斂法？挑最舒服的點 x = 0 開刀，因為每一層在 0 點都是 cos(0) = 1：

```text
W(0) = 1 + 1/2 + 1/4 + … = 2                       ← 振幅帳，幾何級數

取 h = 3⁻ᵐ（剛好是第 m 層的半個週期），奇數 b 的魔法上場：
  第 n ≥ m 層的角度 = 3ⁿπ·3⁻ᵐ = 3ⁿ⁻ᵐ·π = 奇數 × π → cos = −1（精確值！）
  整條尾巴塌縮成幾何級數：Σ(n=m→∞) (1/2)ⁿ·(−1) = −2¹⁻ᵐ

m = 1（h = 1/3）：
  W(1/3) = cos(π/3) + 尾巴(n≥1) = 0.5 − 1 = −0.5
  差商 Q₁ = (−0.5 − 2)/(1/3) = −7.5

m = 2（h = 1/9）：
  W(1/9) = cos(π/9) + (1/2)cos(π/3) + 尾巴(n≥2)
         = 0.93969 + 0.25 − 0.5 = 0.68969
  差商 Q₂ = (0.68969 − 2) × 9 = −11.793
```

m = 3 起只多用 cos(π/27) ≈ 0.99324 一個新值（m = 4 同法，計算器代勞），整列排開：

| m | h = 3⁻ᵐ | W(h) | 差商 Qₘ = (W(h) − 2)/h | 與前一項的比值 |
|---|---|---|---|---|
| 0 | 1 | −2 | −4 | — |
| 1 | 1/3 | −0.5 | −7.5 | 1.875 |
| 2 | 1/9 | 0.68969 | −11.793 | 1.572 |
| 3 | 1/27 | 1.33808 | −17.872 | 1.515 |
| 4 | 1/81 | 1.66829 | −26.868 | 1.503 |

差商不是在某個值附近徘徊拿不定主意——它**單調暴衝**，而且比值一路貼向 1.5，正是 ab。劇本完全照雙帳本走：h 每縮小到原來的 1/3，下一層的斜率預算（×3 的頻率、×1/2 的振幅）接管現場，差商就再漲約 ab = 1.5 倍。每縮一級，醒一層，漲一輪，永不安定。

這裡可以挺直腰桿說一句重話：在 x = 0 這一點，我們**真的證死了**不可微——頭段每一項 cos θ − 1 ≤ 0 只會把差商往下推，所以 Qₘ ≤ −4·(1.5)ᵐ → −∞，差商沿這串 h 發散，極限不存在，沒有任何含糊。定理的怪獸之處在於「**每一個** x 都這樣」——壞點不是住在特殊位置，而是住滿整條數線——那需要真功夫（控制所有 x、所有靠近方式），本書不證。但你已經拿到了證明的靈魂：奇數 b 讓尾巴在關鍵點上集體同向（全是 −1，不互相抵銷），ab > 1 讓新層的斜率永遠蓋過舊層的總和。

## 大自然先養了怪獸

到目前為止，你可以把怪獸歸檔成「數學家的對抗性樣本」。十九世紀很多人正是這麼安慰自己的：真實世界的曲線才不會這樣。然後顯微鏡說話了。

十九世紀初，植物學家布朗（Robert Brown）在顯微鏡下看到懸浮在水裡的微粒永不停歇地亂抖——後來以他為名的布朗運動（Brownian motion）。1905 年愛因斯坦（Einstein）給出解釋：微粒被無數看不見的水分子隨機撞擊。實驗物理學家佩蘭（Jean Perrin）接著用量測把分子的存在釘死，並在 1913 年的《Les Atomes》裡寫下一段堪稱本章註腳的觀察（2026-06 查證）：在軌跡的任何一點上，連近似的切線都定不出來——「數學家想像出來的那些沒有導數的連續函數，被當成單純的數學珍玩是不對的；大自然對它們的暗示，不亞於對可微函數的暗示。」一位實驗物理學家，隔著四十年對魏爾斯特拉斯遙遙舉杯。

數學這邊由維納（Norbert Wiener）接棒，把布朗運動做成嚴格的隨機模型（Wiener process）。模型裡有一條正牌定理：**樣本路徑以機率 1 處處連續、處處不可微**（首證是 Paley–Wiener–Zygmund 1933；隨機過程理論本書不開門，只取這一個事實）。措辭要小心一次：處處不可微的是**數學模型的路徑**——物理微粒在分子尺度以下另有物理，模型在那裡退場。但在模型適用的好幾個數量級裡，「放大永遠毛」就是塵埃的日常。股價的標準隨機走勢模型繼承同一個性質——你看盤軟體裡那條怎麼縮放都是鋸齒的曲線，跟 1872 年柏林科學院黑板上的那條，是同一族。怪獸不是病態想像，是你每天盯著看的現實。

## 病態反例：數學的 fuzz testing

最後談方法論，因為這隻怪獸真正的遺產不是嚇人，是**測繪**。

把定理想成 API 合約：假設是輸入驗證，結論是行為保證。1872 年之前，分析學的很多「定理」是在腦中樣本全是乖函數的情況下寫的合約——輸入驗證形同虛設，因為沒人送過惡意輸入。魏爾斯特拉斯做的事，用你的行話說就是 fuzz testing：精心構造一個病態輸入，打進「連續 ⇒ 頂多幾個壞點不可微」這條隱性假設，看它當場崩潰。W(x) 是數學史上最著名的 fuzz case——而且跟好的 fuzz case 一樣，它的價值不在搞破壞，在**精確標出合約真正的邊界**：「連續」這個前提，買得到積分（ch07 的黎曼和靠它）、買得到中間值，但買不到哪怕一丁點的「方向」。要可微，就得明碼寫可微。

此後數學家的工作習慣被永久改變：每條定理的每個假設都要付錢，每個「顯然」都要過安檢——因為反例獵人就在外面巡邏。反例與定理從此成對出貨：定理畫出安全區，反例證明牆外真的有狼，兩者合起來才是完整的地圖。這場由怪獸（加上 ch11 的收斂之亂、ch12 的發散詐騙）逼出來的全面重建，就是 ch14 的主題。順帶一句收尾：這隻怪獸毛到連「維度」都不再是整數——本章手算的 a = 1/2、b = 3 版本，圖形的 Hausdorff 維度約 1.37，介於曲線與平面之間，而這個值遲至 2018 年才被嚴格證明（Shen，延伸閱讀），分形理論的門就開在這裡，本書到門口為止。

## 直覺的陷阱

| 錯誤直覺 | 它會把你帶去哪裡 | 自我察覺徵兆 |
|---|---|---|
| 「連續曲線頂多壞在幾個點」 | 跟十九世紀的教科書一起陣亡——「畫得出來」不等於「摸得出方向」 | 你用「手畫曲線」想像「連續函數」——手只畫得出乖函數，樣本偏差正在餵養你 |
| 「連續跟可微差不多，都是『不跳』」 | 把 0 階性質（位置別跳）與 1 階性質（方向要安定）混記一本帳 | 你說不出「振幅帳 Σaⁿ 收斂、斜率帳 Σ(ab)ⁿ 發散」這兩本帳各自管什麼 |
| 「光滑函數無限疊加還是光滑」 | 跟拉格朗日否決方波同一個坑（見 ch11）——有限世界的定理進無限世界要重新簽證 | 你在某個部分和裡找「開始變毛的那一層」——每個部分和都無限光滑，毛只住在極限裡 |
| 「放大夠多倍總會變直」 | 局部線性當萬靈丹，碰上自相似結構就翻車 | 你沒注意到 W(x) = cos(πx) + a·W(bx)——每個尺度裡都有一隻完整的怪獸 |
| 「抖得夠快就會不可微」 | 條件記成「b 很大」，漏掉拉鋸的另一頭 | 你講不出同一構造在 ab < 1 時是光滑的——勝負手是乘積 ab，不是 b 單獨 |
| 「怪獸是數學家的惡趣味，自然界沒有」 | 把布朗運動、股價、P99 曲線的鋸齒當成「還沒放大夠」 | 你以為再提高取樣率，監控曲線就會平滑下來 |

## 紙上推演

**題 1 [10 分鐘] ★** — 不看書，用 ch04 的差商定義把「|x| 在 0 點連續但不可微」完整走一遍：左右差商各算一次、說明極限為什麼不存在、再說明連續性為什麼沒事。最後加答一題：怪獸在一點上的「壞」與 |x| 在 0 點的「壞」，差在哪裡？

**題 2 [15 分鐘] ★★** — 回答本章的核心質疑：「無限疊加既然保得住連續，憑什麼毀得掉可微——這不矛盾嗎？」要求講出兩本帳各自的級數、各自管什麼性質，並用 ab 這顆旋鈕說明同一構造什麼時候乖、什麼時候是怪獸。

**題 3 [10 分鐘] ★★** — 有人說：「自然界的曲線放大到最後都會平滑，數學怪獸只是紙上談兵。」舉一個反例，並誠實說明「測量尺度」在這個論證裡扮演什麼角色——哪一段尺度裡怪獸是對的模型、到哪裡模型該退場。

### 推演解答

**題 1**　右邊：h > 0 時差商 = (|h| − 0)/h = h/h = +1，每一步都恰好是 +1。左邊：h < 0 時差商 = |h|/h = −h/h = −1。差商的極限要求不論從哪邊、沿哪串 h 靠近都收斂到同一個值；現在右派全體說 +1、左派全體說 −1，極限不存在，0 點不可微。連續性：x → 0 時 |x| → 0 = |0|，位置沒有跳，連續成立——「位置」與「方向」是兩回事，這正是題 2 的開場。加答：|x| 的壞是「左右各有安定的斜率，但不相等」（一個尖角）；怪獸的壞更深一層——以 x = 0 為例，差商序列 −4, −7.5, −11.8, −17.9, …，根本沒有任何一邊安定下來，是發散，不是分裂。常見錯路：把「不可微」一律想成尖角——尖角只是不可微的最溫和款。

**題 2**　不矛盾，因為兩個性質結的是兩本帳。連續性只需要**振幅帳**收斂：Σ aⁿ = 1/(1−a)，幾何級數，a < 1 就收斂——部分和連續、尾巴被一條對所有 x 同時成立的誤差帶罩死，極限連續（工程師的嚴謹版；正式定理見 ch14 提及的一致收斂，本書不證）。可微性要的是**斜率帳**安定：第 n 層最陡斜率 (ab)ⁿπ，ab > 1 時這本帳發散——h 每縮小 1/b，下一層帶著更大的斜率預算進場，差商每輪再漲約 ab 倍，永不安定。旋鈕：ab < 1 時斜率帳也是收斂的幾何級數，同一個無限疊加就是光滑函數；ab ≥ 1（哈代的門檻）怪獸出閘。一句話版本：**連續花的是振幅的錢，可微花的是斜率的錢；a 縮振幅、b 漲斜率，預算表可以一本盈餘一本破產。**

**題 3**　反例現成：布朗運動的軌跡——顯微鏡下的塵埃微粒，放大好幾個數量級，軌跡始終是毛的；數學模型（Wiener process）裡這是定理：路徑以機率 1 處處不可微。股價曲線同款。測量尺度的角色：「自然界的曲線」其實永遠是「某段尺度範圍內的測量結果」，所以「平滑」或「毛」都必須附帶適用尺度——塵埃軌跡在光學可及的尺度範圍內怎麼放大都毛（怪獸是對的模型），但鑽到分子碰撞的尺度以下，連「軌跡」這個概念都得重新定義，模型退場。正確的結論不是「自然界沒有怪獸」也不是「自然界就是怪獸」，而是：**在一大段尺度裡，不可微的怪獸是某些自然現象最好的模型——而「可微」反而是需要特別理由的例外**。佩蘭在 1913 年就是這麼說的。

### Python 小實驗

換台機器疊一隻條件更兇的：a = 1/2、b = 7（哈代條件 ab = 3.5 ≥ 1；原始條件一樣不滿足，誠實如前）。看兩件事：部分和的**數值**收斂得多快（振幅帳），以及 zoom 三次之後「表觀斜率」怎麼爆炸（斜率帳）。

```python
# Weierstrass partial sums: W_N(x) = sum over n < N of a^n * cos(b^n * pi * x).
# a = 1/2, b = 7: ab = 3.5 >= 1 satisfies Hardy's 1916 condition
# (not Weierstrass's original ab > 1 + 3*pi/2 -- see chapter text).
# Note: layers n >= 19 lose cos() phase accuracy in float64, but their
# total amplitude is < 4e-6, invisible at 5 decimals.
import math

A, B = 0.5, 7.0

def w(x, layers):
    return sum(A ** n * math.cos(B ** n * math.pi * x) for n in range(layers))

print("x         W_3       W_10      W_30")
for i in range(7):
    x = i / 12
    print(f"{x:.4f} {w(x, 3):9.5f} {w(x, 10):9.5f} {w(x, 30):9.5f}")

print("\nzoom into x0 = 0.5 (30 layers): apparent slope = range / width")
for k in range(4):
    width = 0.1 / 10 ** k
    vals = [w(0.5 - width / 2 + width * i / 400, 30) for i in range(401)]
    rng = max(vals) - min(vals)
    print(f"width {width:>7g}: range {rng:8.5f}   range/width {rng / width:7.1f}")
```

預期輸出（CPython 3，寫章時實際執行）：

```text

0.1667   0.64952   0.57679   0.57735

0.3333   0.87500   0.99902   1.00000
0.4167  -0.15944  -0.29857  -0.29886

width  0.0001: range  0.09978   range/width   997.8
```

讀法。上表是振幅帳的勝利：W₁₀ 與 W₃₀ 處處只差不到 0.002（尾巴 Σ(n≥10) (1/2)ⁿ ≈ 0.00195 罩死），函數的**值**穩如磐石——還附三個可以手驗的錨點：x = 0 時每層都是 cos 0 = 1，W = 2 整；x = 1/3 時 7ⁿ ≡ 1 (mod 6)，每層都讀到 cos(π/3) = 1/2，W = 1 整；x = 1/4 時每層都讀到 ±π/4 的 cos = √2/2，W = √2 ≈ 1.41421。表格全部命中。下表是斜率帳的災難：如果 W 在 0.5 可微，「窗內最大落差 ÷ 窗寬」應該隨放大收斂到 |W′(0.5)|；實際上窗寬每縮 10 倍，落差只縮到一半多一點，表觀斜率 13 → 70 → 208 → 998 一路暴漲（平均每放大十倍漲約 4.4 倍），毛刺在每個尺度等你。想加碼，把 0.5 換成任何一點、再多 zoom 兩級（記得同步加密取樣點，否則網格會漏掉比網格更細的毛——數值實驗自己也有解析度陷阱，ch11 的老叮嚀）。
