# 馴服無限：微積分的直覺與驚嘆

> 一本寫給工程師、純為了欣賞而讀的微積分書。不是考試解題訓練，不是嚴格分析學教科書，也不是為了轉職——是想讓你親身感受：人類怎麼把「無限」這頭怪獸，馴服成一門描述「變化」與「連續」的語言。

## 這本書在做什麼

微積分最容易被誤會成「一堆積分技巧」。其實它是人類思想史上的巔峰級突破之一：面對無限、連續、瞬間變化這三個卡了兩千年的概念，數學家沒有去回答「無限到底是什麼」，而是把它**操作化**了——化成「要多近有多近」的可驗收條款。這本書就是在追這件事，以及它一路長出來的所有驚嘆與陷阱。

讀完，你應該能不看書、用自己的話對另一個工程師講清楚這六件事：

1. 微積分在「馴服」什麼——無限、連續、瞬間變化，為什麼這是人類思想史上的巔峰突破。
2. 三個核心動作的直覺：極限（逼近）、微分（瞬間變化率）、積分（無限累積），各自在做什麼。
3. 那顆中心寶石：微積分基本定理為什麼是「面積＝斜率反過來」——兩件看似無關的事其實是同一件事的正反面。
4. 橋接你已經會的東西：變化率 ↔ 你做的 profiling；積分 ↔ 累積量；極限逼近 ↔ 數值方法；最佳化 ↔ 梯度下降。
5. 直覺在哪裡崩壞：芝諾悖論、0.999…＝1、發散級數的詐騙、處處連續卻處處不可微的「怪獸函數」。
6. 它怎麼長成描述整個會變世界的語言：從一條切線，到微分方程能「預測世界」、傅立葉能拆解任何訊號。

## 怎麼讀

- **架構圖先行**：每個 Part 的第一章都有全書地圖，先看清自己在哪、再鑽細節。
- **跟著同一題走**：全書有一條脊椎——拋物線下的面積 ∫₀¹ x² dx ＝ 1/3。你會看著它被算三次：阿基米德的幾何級數（ch02）、黎曼和的硬算（ch07）、FTC 的一行解（ch08）。同一題、三種武器，是這本書最划算的對帳。
- **動手做是紙筆推演**：每章末的「紙上推演」用紙筆算、用嘴巴講。**講得出來才算懂**；講不清楚，就是還沒懂。指定幾章附可選的 Python 小實驗（本機可跑、免費），想開電腦親眼看數值收斂時用。
- **「直覺的陷阱」是招牌**：微積分的偉大最常藏在直覺崩壞的地方，每章都有一節專門拆解「這裡你的直覺正要騙你」。

## 書架上的鄰居

這本書會在需要時指向書架上的另外兩本，但本書自足、不必跨書也讀得懂：

- **《馴服隨機：機率與統計的直覺與驚嘆》**——「機率即面積」就是這裡的積分；中央極限定理裡的鐘形曲線，面積即機率。
- **《把系統寫成定理：TLA+ 與形式化方法》**——ch14 的 ε-δ 量詞攻防（∀ε∃δ），你在那本書的 safety/liveness 證明裡見過同一個對抗結構。

那麼，從第一章開始：人類為什麼怕無限，又為什麼最後算得動它。
