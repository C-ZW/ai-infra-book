# 附錄 A — 記號與讀法速查表

> 這本書的驗收標準是「能不能不看書、口頭講給另一個工程師聽」。所以每個記號除了意思，更要緊的是**怎麼把它唸出來**——一個你唸不出口的符號，等於你還沒真的擁有它。下面每一列的第二欄就是給你出聲練習用的：照著唸，唸到順。記號一律 Unicode 純文字，全書不依賴 LaTeX 渲染。首次出現章以實際內文為準。

## 極限與逼近

| 寫法 | 怎麼唸（出聲練習用） | 白話意義 | 首次出現章 |
|---|---|---|---|
| `lim(n→∞) aₙ = L` | 「當 n 趨近無限大，a n 的極限是 L」 | 數列愈跑愈靠近 L，要多近有多近 | ch01 |
| `lim(x→a) f(x) = L` | 「當 x 趨近 a，f x 的極限是 L」 | 函數在 a 附近的逼近目的地（不是 a 點的值） | ch04 |
| `lim(h→0)` | 「當 h 趨近零」 | 差距縮到無限小時的那個目的地 | ch04 |
| `→` | 「趨近」 | 逼近某值，但不必等於它 | ch01 |
| `∞` | 「無限大」 | 沒有上界；是一個方向、不是一個數 | ch01 |
| `≈` | 「約等於」 | 在容許誤差內相等 | ch01 |
| `ε`（epsilon） | 「epsilon」 | 對手指定的誤差上限，要多小有多小 | ch03 |
| `δ`（delta） | 「delta」 | 你回應的輸入容許範圍 | ch14 |
| `⌈x⌉` | 「x 的天花板」「不小於 x 的最小整數」 | 無條件進位取整（ε 挑戰賽算 N 用） | ch03 |

## 全書記號慣例（與各章一致）

- **行內**用 Unicode 純文字：`f′(x)`、`lim(x→a)`、`Σ(n=1→∞)`、`∫₀¹ x² dx`、`∫[a,b] f(x) dx`。閱讀器不渲染 LaTeX，所有公式都讀得出、複製得走。
- **多行推導**一律放進 ` ```text ` 圍欄區塊，等號對齊、右側可加「← 為什麼」短註，不硬塞行內。
- **導數雙軌記號並用**：拉格朗日 `f′(x)` 與萊布尼茲 `dy/dx`；牛頓的 `ẋ` 只在 ch05 一句帶過，全書工作記號不用。
- 函數圖形用 ASCII 描點（`*` 點、`─ │ └` 軸），追求形狀直覺而非精確比例；所有圖通過 CJK=2 欄寬對齊檢查。
- 數字精度全書統一：π ≈ 3.14159、e ≈ 2.71828、ln 2 ≈ 0.6931，各章按需截短但不互相矛盾。
