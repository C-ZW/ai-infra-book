# 附錄 A — 記號與讀法速查表

> **這份表是給嘴巴用的，不是給眼睛用的。** 本書的驗收方式是「你能不能把一個概念講給另一個工程師聽」，所以每個記號都附一欄「**怎麼唸出來**」——口頭講解時實際要說出口的中文。看得懂符號不算過關，唸得出來、且唸的同時知道它在幹嘛，才算。每一條都標了「首次出現章」，想回頭看它怎麼登場，照章號翻回去。
>
> 本表只收**書中實際用過的記號**。線代課本裡常見但本書沒用到的（例如 `↦` 這個映射箭頭、張量的 ⊗、外積 ∧），這裡不列——列了你也用不到，反而干擾。

## 最重要的一條：行（column）vs 列（row），台灣慣例

這是全書最容易把你帶進溝裡的一條，先釘死再往下看。**台灣的慣例和中國大陸完全相反**——不是小差異，是字面顛倒。本書從頭到尾用台灣慣例。

> **行 = column = 直的（↕），縱向一排。**
> **列 = row = 橫的（↔），橫向一排。**
>
> 記法：「行」這個字是直立的人形，**直行**；「列」橫著排，**橫列**。
> 中國大陸把這兩個字的用法對調（他們「行」指 row、「列」指 column），**所以讀陸版教材或對岸資料時務必當場翻譯**，否則每一句話的行列都是反的。

```text
        行（column，直行，↕）
              │
              ▼
            | 2  1 |  ◄─ 列（row，橫列，↔）
            | 1  2 |  ◄─ 列（row，橫列，↔）
              ▲   ▲
              │   └─ 第二行（第二個 column）= (1, 2)ᵀ
              └───── 第一行（第一個 column）= (2, 1)ᵀ
```

唸出來的時候：「S 的**第一行**」要唸成「S 的第一個 column」、指的是 `(2,1)ᵀ`（直的那一排，也是 ê₁ 的去向）；「S 的**第一列**」是 `(2,1)`（橫的那一排）。本書最常講「**矩陣的行就是基向量的去向**」——這裡的「行」永遠是 column。首次釘清楚：ch05；行空間（column space）/ 列空間（row space）的對照在 ch10。

| 寫法／說法 | 怎麼唸出來（口頭） | 白話意義 | 首次出現章 |
|---|---|---|---|
| 行 / column | 唸「行（直行，column）」 | 矩陣縱向（直的）一排；向量預設寫成直排（行向量） | ch05 |
| 列 / row | 唸「列（橫列，row）」 | 矩陣橫向（橫的）一排 | ch07 |
| 行向量 | 唸「行向量」 | 直排的向量（n×1），本書向量預設這樣 | ch05 |
| 行空間 column space C(A) | 唸「A 的行空間／column space」 | A 的各行張成的空間（＝值域） | ch10 |
| 列空間 row space | 唸「列空間／row space」 | A 的各列張成的空間 | ch10 |

## 一、向量與空間（ch01–04）

| 寫法 | 怎麼唸出來（口頭） | 白話意義 | 首次出現章 |
|---|---|---|---|
| v、x、b、u | 「向量 v」「向量 x」… | 小寫字母 = 向量（有方向與大小的量、空間中的箭頭或點） | ch01（v、u）；x、b 在 ch07 當「解 x」「目標 b」正式登場 |
| (2, 1)、(a, b) | 「括號二逗一」＝「向量 (2,1)」 | 用座標數列寫的向量；預設是直排（行向量），躺平印在行內 | ch01 |
| (2, 1)ᵀ | 「(2,1) 轉置」 | 把橫排的座標立成直排（行向量）；ᵀ 見矩陣那組 | ch01 |
| ê₁、ê₂、êₙ | 「e-hat-1」「e-hat-2」（或「第一個標準基向量」） | 標準基底向量：ê₁=(1,0)、ê₂=(0,1)，沿座標軸、長度 1 | ch01 |
| a·u + b·v | 「a 乘 u 加 b 乘 v」 | 線性組合——全書最常出現的動作（向量的相加與純量倍） | ch01 |
| ℝ | 「實數 R」 | 所有實數 | ch02（在 ℝ² 裡首次成形） |
| ℝ²、ℝ³、ℝⁿ | 「R 二」「R 三」「R n」（或「二維／n 維實空間」） | n 維實向量空間——n 個實數座標的所有點 | ℝ² ch02；ℝⁿ ch03 |
| span{v₁, v₂, …} | 「span，v1、v2 的生成空間」 | 這組向量所有線性組合的集合＝「你能走到的所有點」 | ch03 |
| dim | 「維度／dimension」 | 一個空間的維度＝它一組基底的向量個數 | ch03 |
| ∈ | 「屬於」 | 「在……裡面」，如 y ∈ ℝ 唸「y 屬於實數」 | ch10 |
| { x : 條件 } | 「所有滿足……的 x 的集合」 | 集合建構式，如 {(0,y) : y∈ℝ} 唸「所有 (0,y)、y 跑遍實數」 | ch10 |

> 註：x 與 b 這兩個字母在 ch01–ch04 的算式裡零星出現（如座標 (x,y)），但作為「未知向量 x」「目標向量 b」的固定角色是在 ch07 解 Ax=b 時定下來的。

## 二、矩陣與變換（ch05–08）

| 寫法 | 怎麼唸出來（口頭） | 白話意義 | 首次出現章 |
|---|---|---|---|
| A、S、P、R | 「矩陣 A」「矩陣 S」… | 大寫字母 = 矩陣 = 一個線性變換（一個對空間做事的動詞） | S ch01；A 當泛指矩陣 ch05；R（旋轉）ch05；P（投影/換基底）ch08 |
| S | 「脊椎矩陣 S」 | 全書貫穿範例 S=[[2,1],[1,2]]，每個 Part 認識它一層 | ch01 |
| Av、Ax | 「A 作用在 v 上」「A 乘 x」 | 矩陣作用在向量上＝把變換施加到向量；＝「x 的各分量挑出 A 對應的行、加權相加」 | ch05 |
| AB | 「A 乘 B」（記得是「**先做 B 再做 A**」） | 矩陣相乘＝變換合成；(AB)v = A(Bv)，右先左後 | ch06 |
| I | 「單位矩陣 I」 | 「什麼都不做」的變換，AI = IA = A | ch06 |
| A⁻¹ | 「A 反矩陣／A inverse」 | 把 A 還原的變換，A⁻¹A = I；只有沒壓扁空間（可逆）才存在 | ch04（提到 B⁻¹）；ch08 正式 |
| R(θ) | 「R-theta／旋轉 theta 角」 | 旋轉矩陣 [[cosθ,−sinθ],[sinθ,cosθ]]，逆時針為正 | ch05 |
| → | 「映到／送到」，如 ℝ²→ℝ² 唸「R 二 映到 R 二」 | 函數箭頭，T:ℝⁿ→ℝᵐ 表示「從 n 維送進、m 維出來」 | ch05 |
| m×n | 「m 乘 n」（m 列 n 行） | 矩陣的尺寸：m 列、n 行（m 個 row、n 個 column） | ch05 |
| ∞ | 「無限多」 | 解的個數：0、1、∞（無限多解） | ch07 |

## 三、行列式與秩（ch09–10）

| 寫法 | 怎麼唸出來（口頭） | 白話意義 | 首次出現章 |
|---|---|---|---|
| det S、det A | 「S 的行列式／determinant of S」 | 行列式：變換把面積（體積）放大的**有號**倍率；det=0＝壓扁降維＝不可逆 | ch01（先偷看公式）；ch09 正式 |
| det(AB) = det A · det B | 「det AB 等於 det A 乘 det B」 | 連續做兩個變換、面積倍率相乘 | ch09 |
| ad − bc | 「a-d 減 b-c」 | 2×2 行列式公式 [[a,b],[c,d]] 的有號面積 | ch09 |
| rank、秩 | 「秩／rank」 | 變換後像還撐起幾維＝行空間的維度（投影把面壓成線＝秩 1） | ch08（提及）；ch10 正式定義 |
| 零空間 / null space / ker | 「零空間／null space／kernel」 | 被變換送到 0 的所有向量（投影把整條 y 軸壓成 0） | ch10 |
| N(A)、C(A) | 「A 的零空間 N-of-A」「A 的行空間 C-of-A」 | N(A)＝零空間、C(A)＝行空間（column space） | ch10 |
| Σ（求和） | 「sigma／總和」 | 把一串東西加起來，如 Σ(xᵢyᵢ₊₁−xᵢ₊₁yᵢ)；讀者已熟的求和記號 | ch09（鞋帶公式）；ch15 再用於內積 |

> 註：「行列式（determinant）」字面含「行」「列」兩字，但它**是一個純量（一個數），不是矩陣**——別被名字騙了。

## 四、特徵值與對角化（ch11–14）

| 寫法 | 怎麼唸出來（口頭） | 白話意義 | 首次出現章 |
|---|---|---|---|
| λ | 「lambda／特徵值」 | 特徵值：沿某個特徵方向的伸縮倍率（λ>1 拉長、λ=1 不動、λ<0 翻向、λ=0 壓到 0） | ch09（提及 det＝λ 乘積）；ch11 正式 |
| Av = λv | 「A v 等於 lambda v」 | 特徵向量方程：v 被變換後方向不變、只被拉長 λ 倍 | ch11 |
| det(A − λI) = 0 | 「det of A 減 lambda I，等於 0」 | 特徵方程——解它就抓出所有特徵值 | ch11 |
| tr A、tr S | 「trace of A／A 的跡」 | 跡：對角線元素之和；＝所有特徵值之和（驗算工具） | ch11 |
| i | 「虛數單位 i」 | 滿足 i²=−1；旋轉的特徵值會是複數，逼出它 | ch12 |
| e^{±iθ} | 「e 的正負 i-theta 次方」 | 旋轉 R(θ) 的特徵值（複數！），|·|=1、輻角＝旋轉角 | ch12 |
| ±i、±45° | 「正負 i」「正負 45 度」 | 成對共軛的複特徵值 / 角度 | ch12 |
| \|λ\| | 「lambda 的絕對值／模長」 | 複特徵值的長度（模）；旋轉時 \|λ\|=1（不縮放） | ch12 |
| r·e^{iφ} | 「r 乘 e 的 i-phi 次方」 | 一般複數的極式：r＝縮放、φ＝旋轉角（此處 φ 是輻角） | ch12 |
| A = PDP⁻¹ | 「A 等於 P D P 反矩陣」 | 對角化：P 的行是特徵向量、D 是特徵值對角矩陣 | ch13 |
| D = diag(3, 1) | 「diag 三一／對角 3、1」 | 對角矩陣，對角線放 3 和 1、其餘 0 | ch13 |
| Aⁿ | 「A 的 n 次方」 | 反覆作用 n 次同一個變換；＝P·Dⁿ·P⁻¹ | ch14 |
| φ | 「phi／黃金比」 | 黃金比 φ=(1+√5)/2≈1.618＝Fibonacci 矩陣的最大特徵值與成長率 | ch12（先作輻角 φ）；ch14 作黃金比 |
| ψ | 「psi」 | Fibonacci 矩陣的第二特徵值 ψ=(1−√5)/2≈−0.618 | ch14 |

> 註：希臘字母 φ 在書裡身兼兩職——ch12 當複數極式 r·e^{iφ} 的**輻角**（旋轉角），ch14 當**黃金比**那個常數。唸的時候都唸「phi」，但意思看上下文：前者是一個角度、後者是一個固定的數 ≈1.618。

## 五、內積與正交（ch15–18）

| 寫法 | 怎麼唸出來（口頭） | 白話意義 | 首次出現章 |
|---|---|---|---|
| u · v | 「u dot v／u 點 v」 | 內積（點積）：吃兩個向量、吐一個數＝Σuᵢvᵢ＝‖u‖‖v‖cosθ | ch01（線性公理裡的 a·v 是純量倍，內積正式在 ch15） |
| ⟨u, v⟩ | 「u、v 的內積」（角括號寫法） | 內積的另一種寫法，與 u·v 同義（有些書這樣寫，本書主用 u·v） | ch15 |
| ‖v‖ | 「v 的範數／v 的長度」 | 向量長度（L2 範數）＝√(v·v)＝畢氏定理的 n 維版 | ch15 |
| √ | 「根號／square root」 | 平方根，如長度 √2、√3、√5 | ch02（√2 反例）；ch15 大量用於長度 |
| cosθ | 「cos theta」 | 兩向量夾角的餘弦＝(u·v)/(‖u‖‖v‖)＝cosine similarity | ch15 |
| θ | 「theta」 | 角度（夾角／旋轉角） | ch04（極座標 (r,θ)）；ch12 旋轉角；ch15 夾角 |
| ⊥ | 「垂直於／正交於」 | 正交：u⊥v ⟺ u·v=0（垂直的代數判準） | ch10（一瞥）；ch15 正式 |
| x̂ | 「x-hat」 | 最小平方的「最近解」（無解方程退而求最接近的 x） | ch16 |
| ŷ = a + bx | 「y-hat 等於 a 加 b x」 | 配適（迴歸）線；帽子 ^ 代表「估計值／近似值」 | ch16 |
| AᵀA x̂ = Aᵀb | 「A 轉置 A，x-hat，等於 A 轉置 b」 | 正規方程：解最小平方（把 b 投影到行空間） | ch16 |
| e₁、e₂ | 「e-1、e-2」（單位正交軸） | Gram–Schmidt 扶正後的單範正交基向量（與 ê 標準基底區分） | ch17 |
| Q | 「矩陣 Q／正交矩陣 Q」 | 正交矩陣：行互相單範正交，QᵀQ=I，＝純旋轉/反射、不改長度角度 | ch17 |
| QR、A = QR | 「Q R 分解」 | A 拆成正交矩陣 Q × 上三角矩陣 R＝Gram–Schmidt 的矩陣記法 | ch17 |

## 六、SVD 與應用（ch19–21）

| 寫法 | 怎麼唸出來（口頭） | 白話意義 | 首次出現章 |
|---|---|---|---|
| A = UΣVᵀ | 「A 等於 U Sigma V 轉置」 | 奇異值分解（SVD）：任何矩陣＝旋轉(Vᵀ)→沿軸伸縮(Σ)→旋轉(U) | ch19 |
| σ、σ₁、σ₂ | 「sigma／奇異值」「sigma-1、sigma-2」 | 奇異值：最一般的伸縮倍率，σ₁≥σ₂≥…≥0＝單位圓被送成橢圓的半軸長 | ch19 |

| σ₁ ≥ σ₂ ≥ … ≥ 0 | 「sigma-1 大於等於 sigma-2……大於等於 0」 | 奇異值由大到小排、全非負 | ch19 |
| ≥、≤ | 「大於等於」「小於等於」 | 不等號 | ch10（秩的不等式）；ch19 奇異值排序 |
| M、Mπ = π | 「轉移矩陣 M」「M pi 等於 pi」 | 隨機矩陣 M（每行和為 1）；穩態分布 π 是它 λ=1 的特徵向量 | ch21 |
| π | 「pi／穩態分布」 | 穩態分布向量（長期各狀態佔比）；＝PageRank 的重要度向量 | ch21 |

## 怎麼用這份表自我驗收

挑任何一行，把「寫法」遮住，只看「白話意義」，試著**唸出**對應的記號並講一句它在幹嘛；再反過來，遮住右邊、看著符號唸出聲、講出意義。兩個方向都能流暢講出來，這個記號才算真的是你的。本書的六條主線（ch22 口試）每一條都會用到這裡的記號——`det=0`、`Av=λv`、`A=PDP⁻¹`、`u·v=0`、`A=UΣVᵀ`、`Mπ=π`——能對著符號把故事講出來，就是讀懂了。
