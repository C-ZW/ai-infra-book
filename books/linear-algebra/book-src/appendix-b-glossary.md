# 附錄 B — 術語表

> 這張表收錄全書二十二章實際用到、值得記住的術語。每條給：英文、台灣繁中、一句話定義（用本書「幾何優先」的聲音講——先回答「它對空間做了什麼」，不是教科書定義），以及它**第一次被正式講清楚**的章（forward reference 不算，要看實際定義在哪一章）。
>
> 排列照全書 Part 順序（向量與空間 → 矩陣即變換 → 行列式與秩 → 特徵值與對角化 → 正交與近似 → SVD 與應用），讓你能照著走一遍全書的脊椎。
>
> 三條台灣慣例釘死、全表一致：**行＝column（直行）、列＝row（橫列）**（與中國大陸相反，惡名昭彰的陷阱）；用「**特徵值／特徵向量**」，不寫大陸的「本徵」；用「**單範正交**」（亦作標準正交）。記號的讀法與寫法見附錄 A。

## Part I — 向量與空間

| English | 台灣繁中 | 一句話定義（幾何優先） | 首次出現章 |
|---|---|---|---|
| linearity | 線性 | 一個變換把方格網拉成另一個方格網——直線還是直線、原點不動、平行線還是平行而且等距；唯一的要求就這一條。 | ch01 |
| superposition | 疊加原理 | 線性系統對「一堆刺激之和」的反應，等於它對每個刺激單獨反應之和——可分解、可重組，工程師最愛的性質。 | ch01 |
| scalar | 純量 | 一個普通的數，用來把向量伸長、壓短或翻向（乘負號）。 | ch01 |
| vector | 向量 | 同一個東西的三張臉：幾何上的有向箭頭、座標上的一串數字、抽象上「能相加與純量倍」的元素；它只有兩個基本操作，其餘一切都是後果。 | ch02 |
| linear combination | 線性組合 | 把幾個向量各自伸縮再加起來（a·u+b·v），是全書最常出現的動作。 | ch02 |
| span | 生成空間／span | 一組向量靠相加與純量倍能到達的所有點；一個向量的 span 是一條線、兩個獨立向量的 span 是一個面。 | ch03 |
| linear independence | 線性獨立（線性無關） | 一組向量裡沒有人是多餘的——沒有任何一個能被其他人組出來（幾何上不共線、不共面）。 | ch03 |
| linear dependence | 線性相依（線性相關） | 至少有一個向量躺在其他向量的 span 裡，是個多餘的方向——它一出現，det 就是 0。 | ch03 |
| basis | 基底（基） | 剛好夠用又不重複的一組方向：線性獨立又能生成整個空間，少一個就到不了、多一個就重複。 | ch03 |
| dimension | 維度（維數） | 一個空間真正需要幾個獨立方向（＝基底的向量個數），且這個數跟你選哪組基底無關。 | ch03 |
| coordinates | 座標 | 向量是幾何本體，座標只是它「在某組基底下的名字」；(3,3) 意思是「3 個第一基向量加 3 個第二基向量」。 | ch04 |
| change of basis | 換基底 | 換一組座標軸去描述同一個向量／變換——本書的主旋律：換對基底，難題就現出原形（對角化、Fourier、PCA 都是換基底）。 | ch04 |
| similar (matrices) | 相似 | 同一個變換在不同基底下長出的不同矩陣，互為相似——對角化就是找一個跟原矩陣相似、但長得最簡單（對角）的版本。 | ch04 |

## Part II — 矩陣即變換

| English | 台灣繁中 | 一句話定義（幾何優先） | 首次出現章 |
|---|---|---|---|
| matrix | 矩陣 | 不是一張數字表格，是一個動詞——它對整個空間做一件線性的事；每一行記下一個基向量被搬到哪。 | ch05 |
| linear transformation / linear map | 線性變換／線性映射 | 把方格網拉成另一張方格網的動作，由「基向量的去向」完全決定（因為任何向量都是基向量的線性組合）。 | ch05 |
| column (of a matrix) | 行（直行，column） | 矩陣縱向的一排；讀一個矩陣的第 i 行＝讀第 i 個基向量被搬到了哪裡（台灣慣例，與大陸相反）。 | ch05 |
| row (of a matrix) | 列（橫列，row） | 矩陣橫向的一排；解 Ax=b 的列視角裡，每一列是一條方程＝一條直線或平面（台灣慣例，與大陸相反）。 | ch05 |
| matrix multiplication / composition | 矩陣乘法／合成 | AB 不是表格相乘，是「先做 B 再做 A」的動作合成；機械規則是被「要對應合成」這個要求逼出來的，不是反過來背。 | ch06 |
| identity matrix | 單位矩陣 | 什麼都不做的變換（I），把每個向量原封不動留在原地。 | ch06 |
| non-commutativity (AB≠BA) | 不可交換 | 動作順序要緊：先旋轉後剪切 ≠ 先剪切後旋轉，所以矩陣乘法一般 AB≠BA——非交換是規則不是例外。 | ch06 |
| row picture | 列視角 | 解 Ax=b 時把每一列當一條方程（直線／平面），解＝它們的交點。 | ch07 |
| column picture | 行視角 | 解 Ax=b 時把 Ax 讀成 A 各行的線性組合，解 x 是「要多少份各行才湊出 b」。 | ch07 |
| inverse matrix | 逆矩陣 | 把一個變換原封不動搬回來的動作（A⁻¹A=I），只有在「沒壓扁任何維度」時才存在。 | ch08 |
| invertible / singular | 可逆／奇異 | 可逆＝沒把空間壓扁（雙射、資訊沒丟、回得去）；奇異＝壓扁降維（det=0、資訊永久損失、回不去）。 | ch08 |
| condition number | 條件數 | 「輸入微動、輸出暴走」的敏感度；det≠0 保證理論上可逆，條件數高卻讓你解出來的答案是垃圾——「可逆」不等於「解得準」。 | ch08 |

## Part III — 行列式與秩

| English | 台灣繁中 | 一句話定義（幾何優先） | 首次出現章 |
|---|---|---|---|
| determinant | 行列式 | 一個數字，量這個變換把面積（體積）放大的有號倍率；det=0 一句話同時解釋了不可逆、無解、線性相依。 | ch09 |
| orientation | 定向 | 變換有沒有把空間翻面（左右手對調）；det<0＝翻了面（反射），det>0＝沒翻。 | ch09 |
| rank | 秩 | 變換之後空間還撐得起幾維（＝行空間的維度）；投影把二維壓成一維就是秩 1，量的是「你的資料真正幾維」。 | ch10 |
| null space / kernel | 零空間（核） | 被變換送到 0 的所有方向——「怎麼改輸入都不影響輸出」的方向，攜帶的資訊被徹底吃掉、回不來。 | ch10 |
| nullity | 零化度 | 零空間的維度——這個變換「瞎掉」了幾個方向。 | ch10 |
| rank–nullity theorem | 秩—零化度定理 | 一條守恆律：保住的維（秩）＋ 壓掉的維（零化度）＝ 你帶進來的維（輸入維度）。 | ch10 |
| column space | 行空間 | 各行向量張成的空間，也就是變換到得了的所有地方（值域 range）。 | ch10 |
| transpose | 轉置 | 把矩陣的行與列互換（Aᵀ，原本第 i 行變第 i 列）；湊齊四個基本子空間、定義對稱矩陣都靠它。 | ch10 |
| row space | 列空間 | 各列向量張成的空間（＝Aᵀ 的行空間）。 | ch10 |
| four fundamental subspaces | 四個基本子空間 | 行空間、零空間、列空間、左零空間——Strang 的教學框架（非古典具名定理），把一個矩陣的輸入／輸出側一次看全。 | ch10 |

## Part IV — 特徵值與對角化

| English | 台灣繁中 | 一句話定義（幾何優先） | 首次出現章 |
|---|---|---|---|
| eigenvector | 特徵向量 | 變換作用後方向不變、只被拉長或壓短的那些特殊向量——變換的「骨架」，沿著它再亂的變換都退化成單純伸縮。 | ch11 |
| eigenvalue | 特徵值 | 沿某條特徵方向的伸縮倍率（λ>1 拉長、0<λ<1 壓縮、λ<0 翻向、λ=0 壓到 0＝落在零空間）。 | ch11 |
| characteristic equation | 特徵方程 | det(A−λI)=0；它在問「哪個 λ 會讓 A−λI 壓扁（det=0），讓 (A−λI)v=0 有非零解」，抓出來的就是特徵值。 | ch11 |
| trace | 跡 | 對角線元素之和（tr A），恰等於所有特徵值之和——跟 det（＝特徵值之積）一起當驗算特徵值的免費哨兵。 | ch11 |
| complex eigenvalue | 複特徵值 | 旋轉沒有任何實方向留得住，逼出成對共軛的複特徵值 e^{±iθ}（模長＝1 代表純旋轉不縮放、輻角＝旋轉角）——「這個變換裡藏著旋轉」的指紋。 | ch12 |
| diagonalization | 對角化 | 換到特徵基底（A=PDP⁻¹），讓變換在每根軸上只是乘一個倍率——「複雜只是座標沒選對」。 | ch13 |
| diagonalizable | 可對角化 | 一個矩陣有 n 個線性獨立的特徵向量、湊得齊一組完整特徵基底，就換得出對角形。 | ch13 |
| defective | 有缺陷的（不可對角化） | 重根的特徵值湊不出足夠多獨立特徵向量（幾何重數<代數重數），換不出對角形——剪切就是經典反例。 | ch13 |
| Jordan normal form | 若爾當標準形 | 湊不齊對角形時退而求其次的近對角形（對角擺特徵值、緊鄰上方擺幾個 1）；本書只點一句，指向延伸。 | ch13 |
| matrix power | 矩陣的冪 | 反覆作用同一個變換（Aⁿ）；對角化後＝特徵值各自 n 次方（Aⁿ=P·Dⁿ·P⁻¹），把「乘 n 次」變成「次方」。 | ch14 |
| dominant eigenvalue | 主特徵值（dominant） | 絕對值最大的那個特徵值；反覆作用時它的方向勝出、主宰長期行為，其餘方向被指數洗掉——系統「忘記起點」的引擎。 | ch14 |

## Part V — 正交、投影與最佳近似

| English | 台灣繁中 | 一句話定義（幾何優先） | 首次出現章 |
|---|---|---|---|
| inner product / dot product | 內積／點積 | 一個求和運算 Σuᵢvᵢ，幾何上＝‖u‖‖v‖cosθ，把「多長」「夾幾度」「垂不垂直」全變成可以算的數。 | ch15 |
| norm / length | 範數／長度 | 向量的「大小」‖v‖＝√(v·v)，畢氏定理的 n 維版（你在 loss 裡見過的 L2 範數）。 | ch15 |
| cosine similarity | 餘弦相似度 | 正規化的內積 (u·v)/(‖u‖‖v‖)＝兩個方向有多像，與長度無關——embedding 比對用的就是它。 | ch15 |
| orthogonal | 正交 | 兩個方向互不洩漏、互相垂直；代數判準簡單到不可思議——內積為 0（u·v=0）。 | ch15 |
| projection | 投影 | 把一個夠不到的目標，垂直地投到你夠得到的子空間上，取那個最近的點。 | ch16 |
| least squares | 最小平方 | 方程組無解時（b 不在行空間裡）誠實地給最近答案：把 b 投影到行空間——解一個無解的方程，理直氣壯。 | ch16 |
| residual | 殘差 | 沒被模型解釋掉的偏差（b−Ax̂）；它垂直於你能表達的所有方向，就是「我盡力了、再也擠不出資訊」的代數證據。 | ch16 |
| normal equations | 正規方程 | AᵀAx̂=Aᵀb；名字裡的 normal 是「法向／垂直」，整條式子就是「讓殘差垂直於行空間」這個幾何條件的代數寫法。 | ch16 |
| orthonormal basis | 單範正交基底 | 一組互相垂直、長度都是 1 的座標軸；在這種座標下，座標＝直接內積、投影＝逐項內積相加，不必再解方程。 | ch17 |
| Gram–Schmidt | Gram–Schmidt | 把一組獨立向量逐個「減掉在前面方向上的投影、再正規化」，扶正成單範正交——一個一個剃掉「已有的成分」。 | ch17 |
| QR decomposition | QR 分解 | A=QR（Q 正交、R 上三角）＝Gram–Schmidt 的矩陣記法；解最小平方比正規方程數值穩，是「不要算逆、要分解」的正解。 | ch17 |
| symmetric matrix | 對稱矩陣 | A=Aᵀ（沿對角線翻過去長得一樣），所有矩陣裡最乖的一類——共變異、Hessian、任何「能量／相似度」的表都是它。 | ch18 |
| spectral theorem | 譜定理 | 對稱矩陣必有實特徵值、且有一組正交特徵向量基底，於是 A=QΛQᵀ（Q 正交、純旋轉換基底、無剪切）。 | ch18 |
| quadratic form | 二次型 | 對稱矩陣定義的「能量」函數 xᵀAx，等值線是橢圓，橢圓的軸正是特徵向量（特徵值大＝那方向陡＝短軸）。 | ch18 |
| positive definite | 正定 | 特徵值全大於 0＝二次型恆正＝開口向上的碗（有唯一最小值）——凸最佳化之所以乖的底層原因。 | ch18 |
| covariance matrix | 共變異矩陣 | 一排 metric 兩兩「一起變動的程度」排成的表；天生對稱（cov(x,y)=cov(y,x)），所以一定能正交對角化，PCA 才找得到正交主軸。 | ch18 |

## Part VI — SVD、應用與收官

| English | 台灣繁中 | 一句話定義（幾何優先） | 首次出現章 |
|---|---|---|---|
| singular value decomposition (SVD) | 奇異值分解 | 任何矩陣（不論方不方、滿不滿秩、能不能對角化）都是 A=UΣVᵀ——旋轉 → 沿垂直軸各自伸縮 → 旋轉；線代最一般的形狀定理。 | ch19 |
| singular value | 奇異值 | 最一般的伸縮倍率（σ₁≥σ₂≥…≥0）＝AᵀA 特徵值的平方根；單位圓被送成橢圓，半軸長就是奇異值。 | ch19 |
| low-rank approximation | 低秩近似 | 把 SVD 的小奇異值砍掉、只留前 k 大——用一個秩 k 矩陣逼近原矩陣（壓縮、去雜訊）。 | ch20 |
| Eckart–Young theorem | Eckart–Young 定理 | 截斷 SVD 留前 k 個奇異值，得到的就是所有秩 k 矩陣裡最接近原矩陣的那一個——丟掉小奇異值是可證的最佳壓縮，不是隨手砍。 | ch20 |
| principal component analysis (PCA) | 主成分分析 | 找資料「變異最大的正交方向」（主成分）＝共變異矩陣的特徵向量；降維＝只留前幾個主成分、丟掉低變異的冗餘。 | ch20 |
| principal component / principal axis | 主成分／主軸 | 資料變異最大的那幾個正交方向，正是共變異矩陣由大到小排序的特徵向量。 | ch20 |
| stochastic matrix | 隨機矩陣 | 每行（或每列）和為 1 的轉移矩陣；它必有特徵值 1，因為全 1 列向量是它的左特徵向量。 | ch21 |
| stationary distribution / steady state | 穩態分布／穩態 | 反覆轉移後系統的長期佔比 π，滿足 Mπ=π——也就是隨機矩陣 λ=1 的（右）特徵向量。 | ch21 |
| power iteration | power iteration（冪迭代） | 反覆乘同一個矩陣就收斂到主特徵向量（因為次大特徵值那塊成分以絕對值小於 1 的速率指數衰減）——算穩態與 PageRank 的引擎。 | ch21 |
| Markov chain | 馬可夫鏈 | 狀態機配上固定轉移機率（每條箭頭標一個機率）；演化一步＝拿目前的機率分布去乘轉移矩陣。 | ch21 |
| Perron–Frobenius theorem | Perron–Frobenius 定理 | 正（非負不可約）矩陣有唯一的、絕對值最大的實特徵值，對應特徵向量各分量為正——PageRank 那個「唯一正穩態」的存在性保證。 | ch21 |
| PageRank | PageRank | 把網頁連結圖當隨機衝浪者的轉移矩陣，重要度＝穩態分布＝dominant 特徵向量；Google 的十億帝國建在「解一個特徵向量」上。 | ch21 |
| damping factor | 阻尼因子 | PageRank 裡每頁 0.85 機率沿連結走、0.15 機率隨機跳轉的係數；隨機跳轉讓圖強連通，保證唯一收斂。 | ch21 |
