# 在不確定中下注

### 決策與理性的直覺與驚嘆

> 人類花了三百年把「在不確定下該怎麼選」寫成一套漂亮到近乎強制的數學，正常、聰明、受過訓練的人卻在最簡單的賭局前系統性地違反它——而且違反得很有規律。這本書要讓你親身感受這道裂縫，並且能用自己的話對另一個工程師講清楚：**聖彼得堡、Allais、Ellsberg 各自從哪一條公理把理性掀翻，而展望理論又如何用另一套數學把「人怎麼選」寫成方程式。**

---

## 這本書怎麼讀

**純紙上推演。** 每章的「紙上推演」是手算期望效用、反解確定等值、算貝氏後驗、倒推決策樹——不是要你開終端機。所有數值示範一律手算，攤在 ` ```text ` 表裡。你帶一支鉛筆就夠。

**直覺與精確意義優先。** 每個概念先回答「它在解決什麼決策問題」——期望效用在問「凹效用下該不該接這個賭」，資訊的價值在問「這次再驗一次值不值它的成本」，展望理論在問「人實際上把哪些東西算進去了」——再給形狀與公式。定理給直覺、給公理、給結論，嚴格證明指向延伸閱讀。

**講得出來才算過。** 每章末的「自我檢核」是口頭題，對齊「能不能把這個概念講給另一個工程師聽」的深度。優先問「為什麼」而不是「是什麼」。如果你能對另一個工程師講清楚「Allais 悖論為何讓人在兩道題之間必然不一致」，你才真正懂了那章。

**故障視角是賣點。** 決策理論正是人類直覺與規範理論最常打架的領域。每章的「直覺的陷阱」段，專門守住那些被講爛了的庸俗版（「期望值無限所以該無限下注」、「展望理論證明人不理性」、「損失趨避就是膽小」），還你正確版。

## 一條脊椎貫穿全書

整本書反覆回到**同一個帶不確定性的決策**，每深入一個 Part 就用一套新工具把它重算一次。這條脊椎有**兩條交織的線**：

**賭注線（the bet）**：擲一枚公正硬幣，正面贏 NT$200、反面輸 NT$100。期望值 +50，多數人拒絕。這個懸念從 ch01 開始，七層重看直到 ch18 對帳。

**檢查線（the test）**：某病盛行率 1%、檢查敏感度 99%、偽陽率 5%。驗出陽性後到底有多可能真的有病？貝氏告訴你：後驗約 16.7%——不是 99%，不是 95%，是 16.7%。這條線在 ch05 登場，ch07 問「要不要再驗一次」，ch18 收官。

```text
脊椎七層：同一個決策，七種眼光

  第一層  ch02  期望值          E[L]=+50，期望值正 ⇒ 該接（第一種演算法）
  第二層  ch03  期望效用        u=√，EU=8.66<10 ⇒ 拒絕（風險趨避）
  第三層  ch04  風險vs不確定    同一個賭，硬幣已知vs未知，模糊趨避額外發動
  第四層  ch05  貝氏決策        檢查線：後驗16.7% × 效用矩陣 → 最適行動
  第五層  ch07  資訊的價值      再驗一次的 EVSI：若行動不翻轉，精確等於零
  第六層  ch08–09 展望理論      v(−100)被λ放大 ⇒ 拒絕（機制是損失更重）
  第七層  ch18  五種演算法對帳    規範說X、人做Y、兩者都對——兩種問題的兩個答案
```

中央張力始終不變：**規範理性（你該怎麼選）與描述現實（人實際怎麼選）之間那道穩定的裂縫——它不是 bug，是兩套都對的數學在描述兩件不同的事。**

## 書架上的鄰居

這本書只做一件事：把「單人在不確定下該怎麼選」以及「人實際怎麼選」這兩件事說清楚。需要更深的東西時，它會指向書架上相鄰的書，而不是重講一遍：

- **機率的機制**——貝氏定理怎麼算、條件機率的推導、大數法則為何成立 →《馴服隨機》（本書 ch05–07 只用結果，機制在那裡；注意本書醫療檢查用盛行率 1%，那本書用 0.1%，是刻意差異）

- **偏離的心理機制**——展望理論的數學描述本書收了，人**為什麼**會這樣偏、雙系統與捷思怎麼運作、為什麼察覺不到 →《思考的陷阱》（理性三角的心理機制角，ch12 是明確交棒點）

- **多人策略互動**——當對手會對你的選擇反應，單人期望效用就不夠了 →《賽局：當理性互相碰撞》（本書 ch16 立界碑，細節在那裡；決策↔賽局是同一枚硬幣兩面）

- **制度防線**——信念更新作為科學確證、如何設計流程防自欺 →《如何不騙自己》（理性三角的制度角，ch17 是總交棒點）

- **凹凸函數與邊際遞減**——凹效用的幾何直覺、Jensen 不等式的圖像版 →《馴服無限》（ch03、ch08 觸及時點到指向）

這五本構成**理性三角**（決策＋偏誤＋科學方法）及其週邊，合起來才是完整的圖——本書承擔的是規範基準那一角。

## 給誰讀

唯一讀者：一位六年以上的資深後端工程師。你不需要任何決策理論背景——你需要的直覺，凌晨三點的 rollback 賭注、alert 偽陽性的折扣、漸進式放量的資訊成本、「多算幾步也不值」的滿意就停，早就教會你了。
