## 從你已知的出發

先別管伯努利、馮紐曼、康納曼這些名字。我們從你昨天可能就做過的一件事開始。

凌晨三點，pager 響了。某個服務的錯誤率從基線爬上來，但還沒到 SLA 違約線，也還沒有客訴。你面前有兩個動作：**rollback** 到上一版（確定能止血，但要中斷一個正在跑的資料遷移，代價是延後一週、團隊白做幾天），或是**再觀察十分鐘**（可能它只是一次抖動，自己會好；也可能它正在惡化，等十分鐘等出一場 P0）。

你會怎麼選？你不會去翻教科書。你心裡跑的是一個你早就跑熟、熟到沒意識到自己在跑的算式：

```text
「再觀察」這個動作，在我腦子裡大概長這樣：

   它只是抖動（七成？）── 那我省下一週的遷移工，很好
   它正在惡化（三成？）── 那我賠一場 P0，凌晨被叫醒第二次，外加事後檢討

   ⇒ 我把「每種結果有多可能」乘上「那個結果有多痛」，
      在心裡加起來，跟「rollback」那一欄比一比，挑比較不痛的。
```

這個「把每種結果有多可能、乘上它有多痛，加起來，挑比較不痛的」動作——**機率乘代價、加總、取較優**——就是這整本書要解剖的東西。決策理論不是什麼高深的新發明，它是把你這個凌晨三點的直覺，慢動作拆開、寫成數學、然後問一句殺人的話：**你心裡那個算式，真的是你以為的那個算式嗎？**

你已經有的直覺，比你想的多。容量規劃時你算「這個 region 掛掉的機率乘上掛掉的損失」，決定要不要多開一組備援——那是**期望成本最小化**，是這本書 ch02 的主角。你寧可把容量稍微過配一點、也不要冒「尖峰打爆」的險，明明過配天天在燒錢——那是**風險趨避**，是 ch03 的主角。你收到一個 alert，知道它命中率高但真故障的基率很低，所以心裡先打個折「八成又是偽陽」——那是**貝氏**，是 ch05 的主角。你猶豫「要不要再多跑一輪壓測才敢上線」——那是**資訊的價值**，是 ch07 的主角。

這本書不會從零教你這些直覺。它做的是把它們擦亮、命名、然後指給你看：哪幾個你以為很穩的直覺，其實在某些賭局前會穩定地、可預測地把你帶下溝。

## 一個決策最少需要幾個零件

在拆你的凌晨三點之前，先把它的零件數清楚。決策理論有一個乾淨到近乎冷酷的觀察：**任何一個「在不確定下選一個動作」的決策，最少只要三個零件就描述完了。**

```text
決策的三要素

  ① 行動（act）       你能控制的、要從中挑一個的東西。
                      例：{ rollback, 再觀察 }、{ 治療, 不治療 }、{ 接這個賭, 不接 }

  ② 狀態（state）     你控制不了、而且不確定的那個「世界到底是哪一種」。
                      例：{ 只是抖動, 正在惡化 }、{ 有病, 健康 }、{ 硬幣正面, 反面 }

  ③ 報酬（outcome）   每一個「行動 × 狀態」的格子裡，你拿到什麼。
                      例：rollback × 抖動 = 白白延後一週遷移
                          再觀察 × 惡化 = 一場 P0
```

這個三分法不是我隨手切的。它是 Savage 1954 年那套主觀期望效用理論的地基（見 ch06），而且切得很有道理：**狀態是你「信念」的所在，報酬是你「慾望」的所在。** 世界到底是哪一種（狀態），你只能去**相信**——它由不得你；每個結果（報酬）有多好或多壞，是你**想要**或不想要它——這是你的價值。把這兩件事分開，是整套理論能往下走的關鍵。一個動作，正式講，就是「一個從狀態到報酬的對應」：你選了動作，世界選了狀態，兩者一交叉，報酬就定了。

工程師看到這裡應該很眼熟：這是一張二維表。行是你的動作，列是世界可能的樣子，每一格填一個報酬。你維運生涯裡畫過的每一張「if 這樣 then 那樣」的權衡表，骨子裡都是這張表。決策理論接下來要回答的，全部是同一個問題的變形：**面對這張表，到底該挑哪一行？**

而「該挑哪一行」之所以變成一門需要三百年、好幾個諾貝爾獎才講清楚的學問，是因為列上有「不確定」這三個字。如果你**知道**世界是哪一種，這就不是決策、是查表。難就難在你不知道——你只有信念，而且信念可能是錯的，可能還能更新，甚至可能根本測不出來。

## 把賭注擺上桌：脊椎的第一條線

現在把這本書要陪你走完十八章的那個決策正式請上台。它刻意簡單——簡單到你一眼覺得沒什麼好算的，而那正是它的殺傷力。

> **脊椎・賭注線（the bet）**：擲一枚公正硬幣。正面，你贏 NT$200；反面，你輸 NT$100。要不要接這個賭？

用決策論的記法，把這個賭注（在決策論裡叫**彩券**（lottery）——別誤會，它只是「結果＋機率」的組合，跟刮刮樂無關）寫成一行：

```text
L =（+200, 0.5; −100, 0.5）

  讀法：有 0.5 的機率拿到 +200，有 0.5 的機率拿到 −100。
  分號隔開不同的「結果 ; 機率」對；單位 NT$。

  攤成三要素表：

                  狀態：正面 (p=0.5)   狀態：反面 (p=0.5)
  行動：接這個賭          +200               −100
  行動：不接               0                   0
```

這一行記法 `L =（+200, 0.5; −100, 0.5）`，全書會出現幾十次。每次你看到它，就是「ch01 那個賭注又回來了」。請現在就讓它住進你的腦子。

關於這個賭注，本章只給你**一個**數字、留**一個**懸念。

數字是：如果你把「機率乘報酬加總」這個動作硬套上去——0.5 乘 200，加上 0.5 乘負 100——平均下來，每接一次這個賭，你**長期**會賺 50 塊。這個 +50 叫期望值，正式怎麼算、它到底是什麼意思、為什麼說「長期」三個字是關鍵，全留給 ch02。本章你只要接受一件事：**這個賭的期望值是正的。** 數學上，它是個划算的賭。

懸念是：**多數人不接。**

把這兩句話並排看一分鐘：

```text
   一個期望值為正的賭   ── 數學說：划算，接。
   多數正常人 ── 拒絕。

   ⇒ 誰錯了？是人不理性，還是「划算」這個詞被你誤會了？
```

這就是全書第一道裂縫的種子，也是脊椎賭注線要承載的整條劇情。它會在 ch02 用期望值看一次（划算，該接）、ch03 用效用看一次（拒絕原來是理性的！）、ch08–09 用展望理論看一次（拒絕背後是「賠的比賺的更重」）、ch18 做總對帳。同一個賭，五種眼睛，五種看法。本章不揭曉任何一種——我只把賭注擺好、把「期望值為正卻被拒絕」這個怪事釘在你眼前。

如果你現在心裡冒出「那當然不接啊，200 沒多到讓我願意冒掉 100 的險」——很好，記住這個感覺。你剛剛沒有算錯任何數學，你只是動用了一個**還沒被命名**的東西。這本書接下來幾章，就是去把那個東西揪出來、給它名字、再問它合不合規矩。

## 把檢查擺上桌：脊椎的第二條線

脊椎不只一條線。賭注線講的是「報酬」那一側——同樣的機率，不同的價值。但決策的另一半難題在「狀態」那一側：**世界到底是哪一種，你根本不知道，只能靠證據去猜。** 為了把這一半講透，本書還有第二條脊椎線。

> **脊椎・檢查線（the test）**：有一種病，在人群裡的盛行率是 1%（一百人裡一個有）。有一種檢查，敏感度 99%（真有病的人，99% 會驗出陽性），偽陽率 5%（其實健康的人，也有 5% 會被驗成陽性）。現在，**你驗出了陽性。你到底有多可能真的有病？**

本章一樣，只給設定、留懸念，**一個數都不算**。

先讓你的直覺自己跳一下。檢查敏感度 99%，聽起來很準。你驗出陽性。你心裡那個數字是多少？九成？九成五？

```text
   盛行率 1%、敏感度 99%、偽陽率 5%
   你驗出陽性
   ⇒ P(真的有病 | 驗出陽性) = ？？

   多數人脫口而出：「九成多吧，檢查那麼準。」
   正確答案（ch05 揭曉）：差得遠。
```

我先不告訴你正確的數，但給你一個方向感：它比你直覺低**非常多**。低到「驗出陽性」這件聽起來很嚇人的事，其實沒你以為的那麼確定。為什麼？因為你的直覺偷偷把「真有病的人有多大機率驗出陽性」（這個的確是 99%），跟「驗出陽性的人有多大機率真有病」（這個完全是另一回事）混成了同一個數。這兩個條件機率方向相反、數值天差地別——把它們搞混，是 ch05 要解剖的**基率忽視**。

你會問：這跟賭注有什麼關係，怎麼算同一個脊椎？關係在這裡——**驗出陽性之後，你還得做一個決策：治療，還是不治療。** 而「該不該治療」不只看「多可能有病」（那是狀態的信念），還要看「猜錯各自有多慘」（那是報酬，又回到效用矩陣）。檢查線最後會和賭注線在 ch18 會合：兩條線都是同一個動作——**機率乘代價、加總、取較優**——只是一條的難點在報酬怎麼估，一條的難點在機率怎麼更新。

這條線也是和姊妹書的接點。「驗出陽性後到底多可能有病」這個後驗機率**怎麼算**，本書不重推——機率機制請看《馴服隨機》ch05。先說個你後面會用到的細節：那本書用的盛行率是 0.1%（後驗約 1.94%），本書**刻意改成 1%**（後驗約 16.7%），讓「驗出陽性」這件事剛好落在「有點可能、但遠不確定、所以值得認真做決策」的區間——這個差異是設計，不是手滑，後面每次引用都會提醒你。

## 規範還是描述：這本書真正在追的東西

到這裡，賭注和檢查都擺好了。但這本書的主角其實不是它們，而是夾在它們和「該怎麼選」之間的那道裂縫。要看見裂縫，得先把一個詞拆成兩半。

「決策理論」這個詞，其實同時指兩門很不一樣的學問。它們長得像、用同一套符號、卻在回答**兩個不同的問題**：

```text
            規範（normative）                 描述（descriptive）
            ─────────────────                 ─────────────────
  問的是    你「應該」怎麼選？                 人「實際」怎麼選？
            理性的選擇長什麼樣                 真實的腦袋怎麼運作

  產出      一套演算法（期望效用最大化）         一套模型（展望理論）
            從幾條公理逼出來                   從一堆實驗歸納出來

  本書位置  Part I–II                         Part III
            ch02–07 講「該怎麼選」            ch08–12 講「實際怎麼選」

  招牌人物  馮紐曼、Savage                     康納曼、特沃斯基

  代表句    「違反這條公理，你會被穩賺套利」   「人對變化估值，不對狀態估值」
```

這個區分是全書的中軸線，請花一分鐘把它焊牢。**規範**理論回答「該怎麼選」——它不太管人實際上怎麼做，它問的是「一個理性的決策者，面對這張表，邏輯上**必須**怎麼挑才不自相矛盾」。它的美，在於你只要接受幾條看起來無害到不行的公理（後面會講：完備、遞移、連續、獨立），這套理論就能**逼**出唯一的演算法。你幾乎沒得選——要嘛接受期望效用最大化，要嘛承認自己的偏好能被人牽著鼻子無限抽水。這是 ch03 的高潮，也是我認為這本書最該讓你「啊」一聲的地方之一。

**描述**理論回答「人實際怎麼選」——它不管你該不該，它就是去看人在賭局前**真的**做了什麼，然後找出規律。而它找到的規律，驚人地穩定：人不是隨機亂選、不是「有時對有時錯」，而是**系統性地、可預測方向地**偏離規範理論。偏離本身能被另一套漂亮的數學（展望理論）描述到能預測你會往哪邊歪。

於是這本書真正的那句話，那句它會反覆叩問你的話，是這個：

> **規範理性（你該怎麼選）與描述現實（人實際怎麼選）之間，有一道穩定的裂縫。它不是 bug，是兩套都對的數學，在描述兩件不同的事。**

請特別擋掉一個太容易上鉤的結論：「所以人是不理性的、所以那套漂亮的數學是錯的。」**不是這樣。** 規範理論沒有因為人不照做就失效，就像「最短路徑演算法」不會因為通勤族不走最短路就被推翻——它從來不是在預測人,它是在定義「最佳」。而展望理論也不是在罵人笨，它是在精確地描述「人實際把哪些東西算進去了」。兩套數學各自都對，只是回答的問題不同。能不能扛住這個張力、不滑進「所以人很笨」那個廉價結論，是讀懂這本書的門檻。ch12 和 ch17 會專門回來和解這道張力，但從本章起，你每讀一章都該問一次：這一章碰到的，是「該怎麼選」還是「實際怎麼選」？

## worked example：把賭注翻成一個只有一格的決策

本章不算數，但要示範一件事：怎麼把一句白話的「要不要賭」，翻成決策理論能處理的形狀。這個翻譯動作本身就是後面每一章的起手式，現在先練一次。

```text
白話：「擲公正硬幣，正面贏 200、反面輸 100，要不要接？」

第 1 步：列出行動（你能控制的）
   行動 = { 接這個賭, 不接 }                       ← 兩個選項，挑一個

第 2 步：列出狀態（你控制不了、不確定的）
   狀態 = { 正面, 反面 }
   機率 = { 0.5,  0.5 }                            ← 公正硬幣，已知 50/50

第 3 步：填報酬表（每個「行動 × 狀態」一格）
                       正面 (0.5)      反面 (0.5)
       接這個賭          +200            −100
       不接                0               0

第 4 步：寫成彩券記法
       接 → L =（+200, 0.5; −100, 0.5）
       不接 → L₀ =（0, 1）              ← 確定拿 0，一個退化的彩券

第 5 步：問題定型
       在 { L, L₀ } 之間挑一個。
       這就是「一格決策」——單人、單步、機率已知。
       後面整本書，都是在回答「這一格該挑哪個」，外加把
       「機率已知」「只有一步」「沒有對手」這些假設一條條鬆開。
```

自我複核：這張表對嗎？行動兩個、狀態兩個、四個報酬格——對。機率加起來 0.5 + 0.5 = 1——對，沒有漏掉哪種世界。「不接」這一行兩格都填 0——對，因為不接就沒有任何金錢變動。記法 `L =（+200, 0.5; −100, 0.5）` 和上面基準表一致——對。這張表簡單到讓人想跳過，但請注意它已經藏了三個假設：① 機率是**已知**的（0.5/0.5，你信得過這枚硬幣）；② 報酬**等於金額**（效用就是你實際拿到的數字，沒有風險趨避或參考點）；③ 沒有**對手**（硬幣不會看你下注就偷偷變偏）。這三個假設，後面三個 Part 會分別拆掉：

```text
  鬆開「機率已知」    → 機率要靠證據去更新  → Part II（檢查線、貝氏）
  鬆開「報酬＝金額」  → 報酬要換成效用、甚至看參考點 → ch03、Part III
  鬆開「沒有對手」    → 對手會反應你的選擇  → ch16 交棒給《賽局》
```

預告一件會讓你停一下的事：**同一張表，ch02 用期望值算會說「接」，ch03 用期望效用算會說「拒絕」——而且兩個答案都對。** 不是其中一個算錯了。它們在回答「該不該接」時，偷偷用了不同的「該」。這就是為什麼一個四格的表，值得一本書。

## 直覺的陷阱

本章還沒進入任何具體機制，但有一個陷阱現在就該擋掉，因為它會毀掉你讀後面十七章的姿勢。

**陷阱：把「人實際怎麼選」直接當成「人該怎麼選」的反證——一看到人偏離理論，就宣布「所以人不理性」或「所以那套理論沒用」。**

```text
  錯誤直覺在哪一步把你帶下溝：

  你讀到「多數人拒絕一個期望值為正的賭」，
  → 直覺第一反應：「那這些人不理性」或「期望值理論錯了」
  → 你已經滑進溝裡了。

  你預設了「規範理論就該預測人的行為」——但它從來不是幹這個的。
  規範理論定義「最佳」，不預測「實際」；
  說人偏離它，就像說「通勤族不走最短路、所以最短路演算法錯了」。
```

| | 大眾版（錯誤直覺） | 正確版（本書守住的） |
|---|---|---|
| 人拒絕正期望值賭 | 「人太膽小、不理性」 | 拒絕可能正是理性（凹效用下，見 ch03）；也可能是穩定的描述性偏離（展望理論，見 ch08–09）——兩種解讀都不是「笨」 |
| 規範 vs 描述 | 「兩個衝突，必有一個錯」 | 兩套數學回答兩個不同問題，可並存、不互相否證（見 ch12、ch17） |
| 展望理論 | 「證明了人不理性、期望效用沒用」 | 它是更好的**描述**，期望效用仍是合理的**規範**（見 ch12） |

**怎麼自我察覺你掉進去了：** 當你心裡冒出「所以人很笨／所以這理論沒用」這種話，就是警報。停下來，問自己一句：**我現在說的是「該怎麼選」還是「實際怎麼選」？** 這兩個問題的答案不一樣，是正常的、是這本書的全部重點，不是矛盾。能穩住這個區分，你才讀得動 Part III；穩不住，展望理論在你眼裡就只剩「一堆人類很蠢的實驗」，那就把這本書最精彩的部分讀廢了。

## 紙上推演

本章是框架章，推演也是框架性的——練的是「把白話決策翻成三要素表」這個全書起手式，不是算數。全書純紙上推演，紙筆即可，不必碰任何程式。

### 推演題

**第 1 題（★，10 分鐘）** 用你自己的話，不照抄本章，定義決策的三要素：行動、狀態、報酬。然後說清楚一件事：為什麼「狀態」是你信念的所在、「報酬」是你價值的所在？如果有人把這兩者搞混（例如「我很想要正面，所以正面比較可能」），他犯了什麼錯？

**第 2 題（★★，15 分鐘）** 挑一個你**真的**維運過的「要不要動作」決策（rollback、要不要擴容、要不要發 hotfix、要不要 page 某人……）。把它畫成一張完整的三要素表：列出至少兩個行動、至少兩個你當時不確定的狀態、把每個「行動 × 狀態」格子的報酬填上（用文字描述後果即可，不必算錢）。畫完後檢查三件事：機率有沒有漏掉某種世界？有沒有哪一格你其實填不出來、因為你根本沒想過那種狀態？哪一格的報酬最痛？

**第 3 題（★★，15 分鐘）** 本章說脊椎賭注「期望值為正，多數人卻拒絕」。**先不要用效用、不要用任何後面章節的工具**，只用你現在的直覺，寫下三個可能的理由，解釋「為什麼一個期望值為正的賭，拒絕它也可能不是不理性的」。寫完後，把每個理由標上：它在質疑這張表的哪個部分——是質疑「報酬」（200 跟 100 的價值不對等？）、質疑「機率」（我不信這硬幣真的公正？），還是質疑「期望值」這個演算法本身（長期平均跟我這一次有什麼關係？）？

### 推演解答

**第 1 題：** 三要素中，**行動**是你能直接控制、要從中挑一個的選項集合；**狀態**是你控制不了、而且不確定的「世界到底是哪一種」；**報酬**是每個「行動 × 狀態」交叉格裡你實際拿到的後果。

狀態是信念的所在，因為世界是哪一種**由不得你**——你只能去相信、去估機率、去蒐證據更新；你的願望改變不了它。報酬是價值的所在，因為每個結果有多好多壞，**完全是你的事**——同一筆錢、同一次當機，對不同人、不同處境，價值可以天差地別。把這兩者分開，是整套理論能往下走的關鍵：機率管狀態、效用管報酬，兩者最後相乘加總才得到決策（這正是 ch05 貝氏決策的形狀）。

把兩者搞混的典型錯誤是「**我很想要它，所以它比較可能**」——把報酬（我多想要正面）偷渡進機率（正面多可能）。願望不改變機率。這個錯誤在工程上很常見：你**希望**這次只是抖動、不是真故障，於是不知不覺把「正在惡化」的機率在心裡調低了——這不是估計，是自我安慰。決策理論逼你把「我想要哪種」和「我相信哪種」寫在不同欄位，正是為了擋掉這種偷渡。

**第 2 題：** 沒有標準答案，這題練的是「翻譯」這個動作本身。一個合格的答案長這樣（以 rollback 為例）：

```text
            狀態：只是抖動           狀態：正在惡化
  rollback   白做幾天遷移工           止血成功，但仍白做遷移工
             （早收手了，虧）         （幸好收手，但代價是遷移延後）
  再觀察十分鐘  賺到，自己好了         一場 P0，凌晨被叫第二次 + 事後檢討
```

自我檢查的三個重點：(1) 你列的狀態有沒有蓋住所有可能的世界？很多人只列了「會好」跟「會壞」，漏了「會壞但慢到我來得及反應」這種中間狀態——漏掉的狀態，就是你決策時的盲區。(2) 有沒有哪格填不出報酬？填不出來，通常代表你當時根本沒沙盤推演那種情況。(3) 最痛的那格（這裡是「再觀察 × 正在惡化」），就是你真正在迴避的尾部風險——你後面的所有保守傾向，多半是衝它來的。

**第 3 題：** 這題的目的是讓你**親手**指出裂縫在哪一塊，再由後面章節接手。三個合理的理由，分別質疑不同部分：

```text
  理由（直覺版）                          它在質疑表的哪部分    後面哪章接手
  ──────────────────────────────         ──────────────       ──────────
  「賠掉 100 的痛，比賺到 200 的爽更重」    報酬（價值不對稱）    ch03 效用 / ch09 損失趨避
  「我才不信這硬幣真的剛好公正」            機率（50/50 可疑）    ch04 風險 vs 不確定
  「期望值是長期平均，我只賭這一次」        演算法（EV 適不適用）   ch02 期望值的陷阱
```

常見錯路：很多人三個理由全擠在「報酬不對稱」那一格，沒意識到另外兩塊也能各自把賭注掀翻。這題若你三塊都點到了，恭喜——你剛剛獨立地畫出了 Part I 的地圖：ch02 質疑演算法、ch03 質疑報酬、ch04 質疑機率本身可不可信。另一個常見錯路是直接跳去用「效用」「展望理論」這些詞——但你還沒學它們，這題刻意要你用**裸的直覺**講，因為後面每一章的價值，正是把你這裡的裸直覺替換成精確的機制。

## 自我檢核

口頭自答，講得出來才算過關。優先回答「為什麼」。

1. **為什麼**一個決策最少只要三個零件（行動、狀態、報酬）就描述完了？少掉任何一個會怎樣——譬如一個沒有「不確定狀態」的問題，它還算不算決策？
2. 「狀態是信念的所在、報酬是價值的所在」——用你自己的話講一遍，並舉一個把兩者搞混的例子。
3. 脊椎賭注 `L =（+200, 0.5; −100, 0.5）`，本章只給了你一個數字和一個懸念。那個數字是什麼意思（不必算，講概念）？那個懸念又是什麼？
4. 脊椎檢查（盛行率 1%、敏感度 99%、偽陽 5%），「真有病的人 99% 驗出陽性」跟「驗出陽性的人多可能真有病」，**為什麼**是兩個不同的數？哪一個是你真正想知道的？
5. **規範**和**描述**的決策理論，各自回答哪個問題？為什麼說「人偏離規範理論」不等於「規範理論錯了」？舉一個非決策的類比。
6. 為什麼說「人很笨」是讀這本書最該擋掉的結論？如果不是「笨」，那「人系統性地拒絕一個期望值為正的賭」這件事，該怎麼擺進你的世界觀？
7. 把「要不要 rollback」翻成三要素表——你能在不看本章的情況下，當場畫出來嗎？
