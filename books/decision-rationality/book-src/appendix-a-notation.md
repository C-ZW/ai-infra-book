# 附錄 A — 記號與公式速查

> 這是一張**速查表**，不是教學。每個記號都假設你已在正文某章遇過它；這裡只負責讓你在忘記「λ 是損失趨避還是別的、α 指展望曲率還是其他、π(p) 是機率本身還是被扭曲過的決策權重」時，能在十秒內查回來，並標出它**第一次正式登場**的章。要從零理解某個概念，回該章；要核對一個符號或一個基準數字，留在這裡。

閱讀器不渲染 LaTeX，全書一律純文字 Unicode 記號——你看到的形狀就是書裡的形狀。

## 一、全書脊椎一眼定位

全書十八章重看的是**同一個帶不確定性的決策**，有賭注線與檢查線兩條，先把關鍵數字釘在這裡，後面所有記號都繞著它轉。

```text
脊椎賭注（the bet）：ch01 登場，被七層重看直到 ch18
  L = (+200, 0.5; −100, 0.5)    單位 NT$
  E[L] = 0.5·(+200) + 0.5·(−100) = +50    ← 正期望值，但多數人拒絕
  u(x) = √(x+100)，財富基準 W=100          ← 全書統一凹效用
  u(不賭) = √100 = 10
  EU(L) = 0.5·√300 + 0.5·√0 = 8.6603 < 10 ← 拒絕，風險趨避
  CE：u(CE)=8.6603 ⇒ CE = 8.6603²−100 = −25.0
  風險溢酬 = E[L]−CE = 50−(−25) = 75

脊椎檢查（the test）：ch05 登場，ch07 問資訊價值，ch18 收官
  盛行率 P(病) = 1%，敏感度 = 99%，偽陽率 = 5%
  後驗 P(病|陽性) = (0.99·0.01)/(0.99·0.01+0.05·0.99) = 0.0099/0.0594 ≈ 16.7%
  自然頻率交叉驗算：1 萬人→99 真陽+495 偽陽=594 陽性，99/594=16.7% ✓
  效用矩陣：u(治療,病)=−10，u(不治療,病)=−100，
            u(治療,健康)=−20，u(不治療,健康)=0
```

**⚠️ 注意**：本書盛行率 1%、後驗 16.7%；《馴服隨機》ch05 同題但用 0.1%、後驗約 1.94%，是刻意差異。引用本書檢查數字時用 1%/16.7%。

## 三、同形記號歧義專節（重要）

決策理論的字母不多，同一個符號在不同脈絡常指不同東西。以下幾條是本書（及書架相鄰書）最容易撞名的記號。**查一個數字前先確認你查的是哪個 λ、哪個 α、哪個 δ，否則會把對的數字填到錯的格子裡。**

```text
λ —— 本書：損失趨避係數（loss aversion coefficient），TK1992 中位估計 2.25。    ch09
     意義：失端陡度相對得端的放大倍數。λ=2.25 表示「失 1 元的痛≈得 2.25 元
     的爽」。注意：(a) λ 不是圓周率；(b) 書架上《蝴蝶的鐵律》混沌書的
     λ 是邏輯方程的生長參數，和損失趨避毫無關係；(c) 學術論文裡 λ
     有時也出現在 Poisson 率或正則化項——看到 λ 先問它的脈絡是
     「損失趨避」還是「別的數學」。
     ⚠️ TK1992 的 2.25 是一組估計、不是常數。meta-analysis（Brown 等
     2024）給 λ≈1.31（CI 1.10–1.53），TK 範圍 1.8–2.7，引用必 hedge
     並標 (2026-06)。

α, β —— 本書：展望理論價值函數的曲率參數（TK1992）。                           ch09
      v(x)=x^α (x≥0)；v(x)=−λ(−x)^β (x<0)；TK1992 中位 α=β=0.88。
      意義：α 控制得端凹的程度（α<1 ⇒ 凹），β 控制失端凸的程度。
      易撞：書架上其他書的 α β 可能是學習率、迴歸係數、型一型二誤差
      等。本書 α β 只出現在展望理論的 v(x) 式子裡，別和 ch03 那個
      凹效用 u(x)=√(x+W) 的冪次混淆——那是 1/2 次方，不叫 α。

γ, δ —— 本書：機率權重函數的形狀參數（TK1992）。                               ch09
      w_得(p)=p^γ/(p^γ+(1−p)^γ)^(1/γ)，γ=0.61（得）；
      δ=0.69（失）（w_失 用 δ 取代 γ）。
      意義：γ, δ<1 ⇒ 小機率被高估、大機率被低估（倒 S 型）。
      易撞：書架上《賽局》的 δ 是折扣因子（下一回合的貼現率 0≤δ<1）；
      看到 δ 先問它的鄰居是「機率 p」還是「報酬/未來」。

π(p) —— 本書：偶爾用 π(p) 指機率權重（決策權重），和 w(p) 同義。              ch09
       ⚠️ 圓周率 π≈3.14159 在本書完全不出現；看到 π(p) 就是決策權重、
       不是圓周率乘以 p。附錄 A 統一用 w(p) 避免混淆，正文偶爾出現的
       π(p) 都指決策權重。

u(x) vs v(x) —— 兩個截然不同的函數，不能互換。                                ch03/ch08
      u(x)：規範效用函數，吃「最終財富 x」，全書基準 u(x)=√(x+100)，
             用於期望效用 EU=Σ pᵢu(xᵢ)（ch03）；這是「該怎麼選」的工具。
      v(x)：展望理論價值函數，吃「相對參考點的得失 x」，S 型分段冪；
             用於展望價值=Σ w(pᵢ)v(xᵢ)（ch08–09）；這是「人實際怎麼算」的描述。
      同一個賭注 L=(+200,0.5;−100,0.5)，u 看最終財富（100+200=300 或 0），
      v 看參考點偏離（+200 是得、−100 是失）——記帳系統完全不同。

L（彩券）vs L（賭注） —— 本書用同一字母有兩層意思。                           ch01/ch02
      廣義：任何一個「結果+機率」組合都可寫成 L=(x₁,p₁;x₂,p₂;…)。
      脊椎賭注：L=(+200,0.5;−100,0.5) 是全書那個具體例子。
      脈絡決定是哪個；多數時候「L」出現在等式旁就是脊椎賭注。

W（財富基準）—— 本書：u(x+W) 裡的 W=100，是全書統一的財富基準。            ch03
      ⚠️ 若用 W=1000：EU(L)=0.5·√1200+0.5·√900=32.32>√1000=31.62，
      賭注「竟被接受」，裂縫消失。故全書定案 W=100。這是教學設計選擇、
      不是經驗常數，各章不得另用 W（維護鐵律）。
```

一句總結：**λ＝損失趨避係數（ch09）、α β＝展望曲率（ch09）、γ δ＝機率權重參數（ch09）、π(p)＝決策權重（≠圓周率）、u vs v＝規範效用 vs 描述價值函數（ch03 vs ch08）——同一符號的身分由「它在哪一章、相鄰的式子是什麼」共同決定。** 跨到書架上別本書（混沌、機率、線代、賽局）要全部重新對表。

## 四、脊椎賭注與關鍵公式速查

每條先一句「它在說什麼」，再給形狀與數值，最後標章。這一節是「不得互相矛盾」的基準數字濃縮。

### 期望值（ch02）

```text
它在說什麼：可重複情境下的長期單次平均，不是「這一次會發生的」。
E[L] = Σ pᵢ·xᵢ
脊椎賭注：E[L] = 0.5·(+200) + 0.5·(−100) = +50
聖彼得堡：E = Σ(n=1→∞) (1/2ⁿ)·2ⁿ = ∞（每項貢獻恰為 1，發散）
截斷版（最多第 N 次）：期望值 = N（無限來自尾部）
```

### 期望效用與確定等值（ch03）

```text
它在說什麼：最大化效用的期望，不是金額的期望；凹效用 ⇒ 風險趨避。
EU(L) = Σ pᵢ·u(xᵢ)
脊椎基準：u(x)=√(x+100)，W=100
  u(不賭) = √(0+100) = 10
  EU(L) = 0.5·√300 + 0.5·√0 = 0.5·17.3205 + 0 = 8.6603
  EU(L) = 8.66 < u(不賭) = 10  ⇒  拒絕（理性的風險趨避）
確定等值 CE：u(CE) = EU(L)
  8.6603² − 100 = 75.0 − 100 = −25.0
  CE = −25（你願意接受確定失去 25 元來換掉這個賭）
風險溢酬 = E[L] − CE = 50 − (−25) = 75

Jensen 不等式（凹效用下成立）：u(E[L]) ≥ E[u(L)]
  u(50) = √150 = 12.25 > EU(L) = 8.66  ✓
```

### 貝氏決策（ch05）

```text
它在說什麼：後驗信念 × 效用矩陣 → 最適行動；兩個缺一不可。
後驗：P(θ|D) ∝ P(D|θ)·P(θ)   ← 概似 × 先驗
最適行動：a* = argmax_a Σ_θ P(θ|D)·u(a,θ)

脊椎檢查驗算（陽性後的最適行動）：
  P(病|陽性) ≈ 16.7%，P(健康|陽性) ≈ 83.3%
  EU(治療)  = 0.167·(−10) + 0.833·(−20) = −1.67 − 16.67 = −18.33
  EU(不治療) = 0.167·(−100) + 0.833·0  = −16.67
  −16.67 > −18.33  ⇒  最適行動是「不治療」（在此矩陣下）
  若把 u(不治療,病) 改成 −500，最適行動翻轉為「治療」
```

### 資訊的價值（ch07）

```text
它在說什麼：資訊值錢，若且唯若它可能讓你換一個更好的行動。
EVPI = E[max_a u(a,θ)] − max_a E[u(a,θ)]    ← 神諭情境下的增益
EVSI = E_D[max_a Σ_θ P(θ|D)·u(a,θ)] − max_a Σ_θ P(θ)·u(a,θ)

核心性質：
  EVSI ≥ 0    （多知道不會更糟，在期望意義上）
  若資訊不改變最適行動 a*：EVSI = 0（精確等於零，不是「約等於」）
```

### 展望理論（ch08–09；TK1992，引用必 hedge，標 2026-06）

```text
它在說什麼：人估的是相對參考點的變化，損失被放大 λ 倍，機率被倒 S 扭曲。
價值函數：
  v(x) = x^α              (x ≥ 0，得)
  v(x) = −λ·(−x)^β        (x < 0，失)
  TK1992 中位估計：α=β=0.88，λ=2.25
  ⚠️ meta-analysis（Brown 等 2024）給 λ≈1.31（CI 1.10–1.53）；
     引用必 hedge：「一組常被引用的估計、有再現性討論、非常數、(2026-06)」

機率權重：
  w(p) = p^γ / (p^γ + (1−p)^γ)^(1/γ)
  γ=0.61（得）、δ=0.69（失）；小機率被高估、中大機率被低估（倒 S 型）

脊椎賭注展望重算（參考點=0，先用 w(p)=p 簡化）：
  v(+200) = 200^0.88 ≈ 105.9
  v(−100) = −2.25·100^0.88 ≈ −129.5
  展望價值 ≈ 0.5·105.9 + 0.5·(−129.5) = −11.8 < 0  ⇒  拒絕
  ⚠️ 若改用 λ=1.31：v(−100)≈−1.31·57.5=−75.3
     0.5·105.9 + 0.5·(−75.3) = +15.3 > 0  ⇒  接受
     ——足以翻轉結論，參數不確定的影響不可忽略。

校驗常數（手算複核用）：
  200^0.88 = exp(0.88·ln200) = exp(0.88·5.2983) = exp(4.6625) ≈ 105.9
  100^0.88 = exp(0.88·4.6052) = exp(4.0526) ≈ 57.5
  √300 ≈ 17.3205，√1000 ≈ 31.6228，ln2 ≈ 0.6931
```

### vNM 四公理（ch03）

```text
它在說什麼：滿足這四條公理 ⇒ 你的偏好等價於最大化某個 EU。
① 完備性（completeness）：任兩個彩券之間你有偏好或無差異。
② 遞移性（transitivity）：A≻B、B≻C ⇒ A≻C；違反 ⇒ money pump。
③ 連續性（continuity）：偏好可被機率連續插值；極端結果不「無限可怕」。
④ 獨立性（independence axiom）：X≻Y，把兩者都按機率 p 混入同一個 Z
   後，你仍該 (p·X+(1−p)·Z) ≻ (p·Y+(1−p)·Z)；共同項可約掉。
表現定理（不證明，給直覺）：滿足①②③④ ⇔ 存在效用函數 u 使得
   L₁≻L₂ ⟺ EU(L₁)>EU(L₂)，且 u 在正線性轉換下唯一。
```

### Allais 對賭（ch11）

```text
它在說什麼：客觀機率完全透明，受試者還是違反獨立公理（確定性效應）。
題 1：A=（確定 100 萬）vs B=（10% 得 500 萬, 89% 得 100 萬, 1% 得 0）
題 2：C=（11% 得 100 萬, 89% 得 0）vs D=（10% 得 500 萬, 90% 得 0）
多數人選 A 又選 D：把兩題共同的「89% 得 100 萬」消去後，
   A 對剩下 11%：確定 100 萬
   D 對剩下 10%：得 500 萬（另 1% 得 0）
   A 與 D 在同樣的「89%」背景下，對剩下 11% 偏好相反  ⇒  違反獨立公理。
```
