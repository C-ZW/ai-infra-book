# ch09 — 失眠的機制與 CBT-I：當排程器被條件化反應綁架

> **本章解決什麼問題**：前幾章建立了睡眠的正常樣貌——脊椎的維護視窗、雙歷程排程器（Process S × Process C）、以及三件維護工作。ch08 檢視了排程器被強制跳過時（剝奪）的代價。本章換個角度：當排程器沒被強制跳過，而是被一套學習來的條件化反應「鎖住」，進不了維護視窗時，會發生什麼事？這是慢性失眠的機制圖像，也是理解為什麼 CBT-I（認知行為治療失眠）比安眠藥更治本的關鍵。本章只講機制與證據，不提供任何治療建議——有睡眠障礙請就醫，本書是機制說明，非醫療指引。與 ch08（剝奪代價）、ch10（排程器硬體壞掉的猝睡症）、ch12（咖啡因/安眠藥）構成 Part III「故障與後果」的完整故障地圖。

## 從你已知的出發

想像一個快取系統出了這樣的問題：原本「key=床」應該對應「value=睡眠驅動、副交感放鬆」，但在幾次壞的經驗之後，這個 key 被寫入了「value=清醒焦慮」。之後每次你「請求」這個 key，傳回的都是錯誤的 value——不是睡眠，而是警覺。更糟的是，這個錯誤快取關聯愈命中愈鞏固，形成正回饋迴圈：你愈睡不著愈焦慮，焦慮又讓你更睡不著，「床」這個 key 的「清醒焦慮」 value 被一次次強化寫入。

這不是比喻，這是慢性失眠的核心機制——條件化（classical conditioning）讓睡眠環境本身變成過度激發的觸發器。

你的系統工程直覺在這裡是準的：
- **錯誤的快取關聯**＝古典制約把「床/就寢時間」連結到「清醒/焦慮」
- **正回饋迴圈**＝「睡不著→焦慮→更睡不著」這個循環不會自我修正
- **壓症狀 vs 改根因**＝安眠藥讓你暫時入睡，但錯誤映射還在；CBT-I 直接處理那個錯誤的條件化連結
- **提高命中率（睡眠效率）**＝睡眠限制療法壓縮在床時間，讓你躺床時幾乎必定是在睡覺，藉此重建「床＝睡」的關聯

本章的核心問題是：失眠是一種什麼樣的「系統狀態」？它為什麼用某種介入能治本而另一種只是熱修補？

## 慢性失眠是什麼：不只是「睡不著」

在進入機制之前，先確認我們在講同一件事。

按照美國睡眠醫學會（AASM）的 ICSD-3 標準，**慢性失眠（chronic insomnia disorder）**的診斷需要同時滿足：
- **入睡困難、維持睡眠困難、或過早清醒**，且對睡眠品質不滿意
- 每週至少三夜出現
- 持續**三個月以上**
- 在有足夠睡眠機會與環境的前提下仍發生
- 造成**日間功能損害**（疲勞、注意力下降、情緒擾動、工作表現下滑）

第三條（持續三個月以上）是區分急性與慢性失眠的關鍵：如果只是某段時間的偶發性睡不著（急性失眠，通常有壓力觸發），那是不同的狀況；而最後兩條（足夠睡眠機會、日間功能損害）則是把失眠跟單純的睡眠剝奪、以及無困擾的短睡者區分開來。慢性失眠在成人中估計盛行率約 **6–10%**（同時要求夜間症狀與日間功能損害）；若只計「失眠症狀」（不要求功能損害），比例更高、達 20–30%。（⚠️ 估計值因研究定義略有不同）

三個月以上的慢性化，是這個疾病最有趣的地方：**原來的觸發事件（工作壓力、感情問題、生病）可能早就消失了，但失眠繼續存在**。這暗示「維持失眠的因素」跟「一開始引發失眠的因素」不是同一件事。

## 失眠的三 P 模型：為什麼一開始的觸發消失後，失眠還持續

Spielman 等人 1987 年提出的三 P 模型（3P model），至今仍是失眠機制最廣泛引用的概念框架，也是 CBT-I 的理論基礎。三個 P 是：

**第一 P：素因（Predisposing factors）**

讓你比別人「容易失眠」的底層特質。包括基因傾向（有家族史）、性格特點（高度神經質、焦慮傾向、完美主義）、生理特徵（靜止時心率較高、過度激發的神經系統基調）。素因本身不足以讓你失眠，它只是讓你的「失眠門檻」較低。

工程類比：某台機器的 CPU 基礎負載本來就比別台高；在同樣的請求量下，它更容易先跑滿。

**第二 P：誘因（Precipitating factors）**

實際觸發第一次失眠的急性事件——工作截止日、喪親、搬家、換工作、手術後疼痛、時差。誘因通常很清楚，「從那週開始就睡不好」。誘因本身往往是暫時的，大多數人在誘因消失後會自然恢復睡眠。但對素因較高的人，這個短暫壞眠可能進入第三個階段。

**第三 P：維持因素（Perpetuating factors）**

這是失眠慢性化的關鍵。為了應對睡不著的焦慮，人們發展出一系列「補償行為」，這些行為反而把失眠鎖住了：

- **增加在床時間**（早點躺、晚點起、睡不著也躺著），期望靠這個增加睡眠機會——結果是床上清醒的時間增加，「床」和「清醒」的連結被強化
- **白天補眠**（降低夜晚的睡眠驅力 Process S）
- **白天不敢活動**（擔心消耗體力會更累，但反而讓 S 的累積速度降低）
- **睡前反覆「努力」入睡**（反而啟動認知監測與焦慮，升高激發程度）
- **建立對睡眠的焦慮性想法**（「我明天有重要會議，必須睡著」「我只剩 5 小時可以睡了」）

這些行為短期內感覺合理，長期卻把失眠維持住了。等到誘因消失，維持因素已經自我鞏固，失眠不需要誘因也能繼續運作。

## 過度激發：失眠的生理與認知面

三 P 模型是行為框架。它的生理層面稱為**過度激發（hyperarousal）**，這是慢性失眠研究裡最被廣泛接受的神經生物學圖像（⚠️ 標注：這是主流圖像，機制細節仍在活躍研究中）。

過度激發有三個相互強化的面向：

**生理激發**：研究顯示慢性失眠者與正常睡眠者相比，不只在夜晚，**白天也表現出更高的基礎激發水準**——心率較高、皮質醇（壓力激素）濃度較高、體溫調節略有不同、全身代謝率偏高。這不只是「晚上緊張」，而是系統性的喚醒閾值偏高。

**認知激發**：「睡前腦子轉個不停」是失眠患者最常描述的症狀。認知層面的激發包括：
- 反覆出現的擔憂性思維（rumination）——不只是睡眠相關，常波及生活各層面
- 對睡眠本身的「睡眠監控（sleep monitoring）」——「我現在睡著了嗎？」這個問題本身讓你清醒
- 對隔日後果的災難化想法（catastrophizing）——「睡不著明天絕對毀了」

**條件化激發**：這是最機制化的那一塊。透過古典制約（classical conditioning），**「床」「臥室」「就寢時間」等原本中性的刺激，因為一次次在其中經歷清醒與焦慮，被學習成觸發激發的線索**。就像巴夫洛夫的狗對鈴聲分泌唾液，慢性失眠患者對「躺進床」這個動作分泌焦慮。

你可以用一個具體的對比來感受這個機制：同一個人，在自己的床上清醒地翻來覆去兩個小時，卻在沙發上看電視時不小心睡著。這不是體力或睡眠驅力的問題——是「沙發」還沒有被條件化成焦慮線索，而「床」已經是了。

## 脊椎歸位：失眠破壞了排程器的哪個部分？

在把失眠放回全書脊椎之前，先問一個問題：失眠到底是 Process S 出了問題，還是 Process C 出了問題？

答案是**以 Process S 為主，但過度激發疊在上面干擾整個系統**。

一般失眠者的 Process C（生理時鐘）本身通常沒有損壞——他們不是時型錯位或相位延遲（那是 ch11 的主題）。問題是：**本來 S 累積夠高、C 也到了促眠相的「雙重允許」時刻，排程器發出了「現在可以進入維護視窗」的信號，但條件化的過度激發攔截了這個信號**。

用更精確的方式說：失眠是排程器的兩個歷程（S 和 C）都還在正常工作，但一個「來自學習的覆蓋機制」讓維護視窗的啟動被阻斷。排程器本身沒壞，但每次它試圖啟動維護視窗，條件化的高激發狀態就把門關上。

這正是失眠與 ch08 剝奪、ch10 猝睡症的根本差異：
- **剝奪**：外力強制縮短維護視窗（比如熬夜或鬧鐘），排程器沒壞
- **猝睡症**：促醒看門狗壞掉，清醒/睡眠狀態機非法轉移（ch10）
- **慢性失眠**：排程器正常，但條件化的過度激發每次都擋住維護視窗的入口

這個圖像解釋了一個失眠患者常有的痛苦體驗：「我明明很累，但就是睡不著。」S 很高、C 也在促眠相——排程器說「進去」——但門被激發的條件化反應頂住，進不去。

## Worked Example：失眠的正回饋迴圈，走完一圈

這是本章的機制核心。讓我們把失眠的慢性化過程逐步走完，每一環都標出「正回饋在哪裡」：

```text
第 0 步：誘因出現
    工作截止日、關係問題、搬家——
    某週因為壓力睡得很差。
    此時：急性失眠，正常反應。
    │
    ↓
第 1 步：第一個維持因素出現——延長在床時間
    「我昨晚沒睡好，今晚要早點躺多補一點。」
    → 在床時間增加（比如從 7h 增到 9h）
    → 但睡眠需求沒增加，多出的 2h 是清醒時間
    → 床上清醒比例上升（睡眠效率下降）
    → 「床」開始關聯到清醒。
    │
    ↓
第 2 步：睡眠監控與焦慮啟動
    「我怎麼還沒睡著？」
    → 認知監控啟動（prefrontal arousal）← 正向回饋開始
    → 對明日後果災難化：「睡不著我明天會完蛋」
    → 焦慮升高 → 腎上腺素/皮質醇升高 → 生理激發升高
    → 更難入睡
    │
    ↓
第 3 步：條件化強化
    重複幾週後：「躺進床」這個動作本身
    觸發焦慮（古典制約完成）。
    → 不需要外部壓力，光是就寢時間到來
      就足以啟動整套激發反應
    → 誘因（工作截止日）早已消失，
      但失眠已自主運行。
    │
    ↓
第 4 步：正回饋自我鎖定
    睡不著 → 更焦慮 → 條件化更強
    → 更睡不著（迴圈閉合）
    
    補償行為持續加劇：
    - 白天補眠（降低夜晚 S）→ 夜晚更難入睡
    - 更多清醒時間在床上 → 條件化更強
    - 咖啡因用量增加（ch12）→ 夜晚腺苷被進一步壓制
    │
    ↓
第 5 步：慢性失眠
    三個月以上，觸發事件早不在了。
    排程器（S×C）正常工作，
    但過度激發的條件化擋住維護視窗入口。
    「我明明累，但躺下就清醒。」
```

整個迴圈的工程直覺：**這是一個沒有自我修正機制的正回饋系統**。在沒有外部介入的情況下，每一輪迴圈都讓條件化更強、激發門檻更低，系統傾向於「鎖住在失眠狀態」。

## CBT-I 的機制：逐一打斷正回饋

慢性失眠是一個正回饋系統——那打破它的方法，就是逐一打斷它的幾個關鍵環。CBT-I（認知行為治療失眠，cognitive behavioral therapy for insomnia）正是這樣設計的。

這裡值得先強調：**CBT-I 是成人慢性失眠的一線療法，這是穩固共識**。美國內科醫師學會（American College of Physicians, ACP）2016 年臨床指引在《Annals of Internal Medicine》給出「強推薦（strong recommendation）」，優先於藥物。（指引年份：2016；指引可能更新，讀者可查 AASM 最新版確認。）

CBT-I 不是一個技術，而是幾個技術的組合，每一個打斷失眠迴圈的不同環節：

### 刺激控制（Stimulus control）：清掉錯誤快取關聯

**原理**：切斷「床」與「清醒/焦慮」的條件化連結，重建「床」與「睡眠」的關聯。

**核心規則**（以下為機制說明，非治療建議）：
- 床只用於睡眠（與性，兩者之外都在床外進行）
- 如果躺床超過一定時間仍未入睡，離床去另一個房間做平靜活動，直到睡意重新升起才回床
- 無論夜晚睡多少，固定起床時間（錨定 Process C）

機制邏輯：每次你在床上清醒，就是一次「床→清醒」的條件化強化。刺激控制的目的是**打破這個配對**——讓你在床上的時間幾乎都是在睡覺，讓「床」重新成為睡眠的獨有線索。

工程翻譯：你想讓 `key=床` 的快取值變回 `value=睡眠`，方法是讓未來每一次命中這個 key 都傳回正確的 value，一次次覆蓋掉錯誤的條件化記憶。

### 睡眠限制（Sleep restriction）：提高 Process S、重建睡眠效率

**原理**：讓 S（睡眠壓力）高到足以真正蓋過條件化激發。

維持因素裡有一條：「擔心睡不夠而增加在床時間」。但在床時間增加不等於睡眠增加，只是增加了「清醒躺床」的比例（睡眠效率下降）。睡眠限制反其道而行：**先壓縮在床時間，讓睡眠效率上升，再逐步調整**。

機制邏輯：壓縮在床時間 → S 的累積在有限窗口內無法被稀釋 → 更快入睡、睡得更實 → 睡眠效率提高 → 睡眠與床的連結重建 → 逐步放寬在床時間至適當長度。

這裡的機制與 Process S 直接掛鉤：睡眠限制的早期介入等於人工製造一個「S 更高、蓋過激發閾值」的狀態。研究顯示睡眠限制增加慢波睡眠（N3）的比例，表示它確實在讓 Process S 驅動的深睡功能正常化。

⚠️ 重要邊界：睡眠限制在某些族群（如雙相情緒障礙症、癲癇）有禁忌，且執行細節因人而異——這裡是機制說明，執行請在醫師或臨床睡眠專家指導下進行，不是本書提供的自助步驟。

### 認知治療（Cognitive restructuring）：打斷災難化思維

**原理**：挑戰並重構對睡眠的災難化信念，降低認知激發。

常見的失眠維持性想法包括：
- 「我必須睡滿 8 小時，否則明天完全沒用」（⚠️ 事實：美國國家睡眠基金會（National Sleep Foundation, NSF）建議是範圍 7–9 小時，且一夜稍差不等於功能全毀，見 ch08 的剝奪證據）
- 「我已經失眠這麼久，我的大腦一定壞掉了」（錯誤歸因）
- 「我再也不可能睡得好了」（過度類化）

認知治療不是叫你「正向思考」，而是幫你辨認這些想法的邏輯問題，用更準確的評估取代。**焦慮降低 → 認知激發降低 → 維持因素減弱 → 條件化更容易消退**。

### 睡眠衛教（Sleep hygiene education）：移除外部干擾

這是 CBT-I 裡「最淺」的一層，但也是常被過度強調的一層。睡眠衛教針對的是一些外部因素（咖啡因時機、睡前光照、環境溫度、固定作息），目的是「不要讓外部因素替排程器添麻煩」。

**重要的是**：光靠睡眠衛教無法治療慢性失眠——如果條件化的過度激發還在，規律作息只是減少雜訊，根本問題沒有被碰。CBT-I 之所以有效，是前三個技術（刺激控制、睡眠限制、認知治療）在處理根因；睡眠衛教是輔助。

（睡眠衛教的詳細證據分級在 ch15。）

### 放鬆技術（Relaxation techniques）：降低生理激發

漸進式肌肉放鬆、控制呼吸、正念等，直接針對過度激發的生理面——降低在就寢時的皮質醇、心率、肌肉張力。這不是在壓制睡眠困難，而是把「身體的靜止程度」調回一個讓 Process S 和 C 的合成信號能夠生效的狀態。

## CBT-I vs 安眠藥：改根因 vs 熱修補

安眠藥（苯二氮平類 benzodiazepines 與 Z 藥如 zolpidem）的機制是透過增強 GABA（γ-胺基丁酸，一種抑制性神經傳導物質）的活性，抑制神經系統的整體激發，讓你進入一個「看起來像睡著」的狀態。

幾個機制層面的問題：

**睡眠結構可能未正常化**：苯二氮平類已知會壓低 N3 慢波睡眠（深睡），這意味鞏固（ch05）與校準（ch07）這兩件維護工作可能沒有充分進行。Zolpidem（Z 藥）對 N3 的影響在研究中有不一致的結果，但長期慢性使用者的 N3 確實受損。用藥理方式壓制激發以入睡，不等於讓大腦進入自然的維護視窗工作模式。

**根因未碰**：「床＝清醒焦慮」的條件化連結沒有被處理。藥效消失的夜晚，條件化還在。

**依賴、耐受、反彈性失眠**：長期使用安眠藥後，身體適應了那個高 GABA 活性的基線，停藥時反而出現「反彈性失眠（rebound insomnia）」——比服藥前更難入睡。這是工程裡的「熱修補債」：補丁在，系統勉強運行；補丁拿掉，原問題回來，而且往往更嚴重。

**長期療效比較**：一篇 2024 年的系統性回顧（涵蓋 37 項研究的網絡後設分析）顯示，CBT-I 的長期（中位數 24 週）緩解率約 **41%**，高於以藥物治療為起點的 **28%**。在大多數頭對頭比較試驗中，短期效果相當，但 12 個月追蹤時 CBT-I 顯著優於安眠藥。（⚠️ 數字來自 2024 年後設分析，方法學可能差異，標記此處為單一來源。）

用工程語言總結這個對比：

```text
安眠藥                         CBT-I
━━━━━━━━━━━━━━━━━━━━           ━━━━━━━━━━━━━━━━━━━━
打補丁：壓住激發讓你入睡       改根因：打斷條件化迴圈
入睡時間較快（短期）           入睡改善較慢（需幾週）
停藥反彈失眠                   效果停療後持續
睡眠結構可能受影響             靠 S 自然驅動深睡
條件化連結未處理               重建「床=睡」連結
適合急性失眠短期過渡           慢性失眠一線
```

（⚠️ 醫療邊界：安眠藥在某些情況下有其適應症，本書只講機制，用藥決策請由醫師評估。本書不提供任何藥物建議。）

## 直覺的陷阱

失眠是睡眠科普裡迷思密度極高的區域——部分因為每個人都有「睡不好」的親身體驗，感覺直覺很可靠，實際上卻常被錯誤引導。

| 常見錯誤直覺 | 錯在哪裡 | 正確版本 |
|---|---|---|
| **「睡不著就多躺一會兒，總會睡著」** | 在床上清醒的時間是「床＝清醒」條件化的强化，不是補眠。增加在床時間且睡不著，只讓失眠更穩固。 | 睡不著且無法入睡時，離床比留在床上更有利於重建「床=睡」的連結（刺激控制原理）。 |
| **「安眠藥讓我睡著，所以它治好了我的失眠」** | 安眠藥壓住了激發讓你入睡，但條件化的「床=焦慮」連結沒有消除。停藥後大多數人失眠回來。這是壓症狀，不是解決根因。 | CBT-I 指引優先於藥物，因為它打斷條件化迴圈，改善在停療後持續。藥物有其短期角色，但不是慢性失眠的長期解法。 |
| **「失眠是因為壓力太大，解決壓力就好了」** | 誘因確實常是壓力，但慢性失眠的維持靠的是條件化與維持行為，不是原始誘因。誘因消失後，條件化還在。 | 三 P 模型把誘因（第 2 P）與維持因素（第 3 P）分開：你需要處理的是第 3 P，光靠等待誘因消失無效。 |
| **「努力睡著比較有效——我需要更努力入睡」** | 入睡是一個必須「放手」才發生的過程，不是可以主動努力達成的目標。「試圖睡著」本身啟動認知監控與焦慮，升高激發，讓睡眠更難發生。 | CBT-I 的悖論之一：讓自己不那麼在乎睡著，反而更容易睡著（降低認知激發）。 |
| **「我的大腦一定有什麼結構問題讓我失眠」** | 大多數慢性失眠沒有可測量的結構損傷，而是功能性的條件化問題。過度激發確實有生理測量基礎，但這不是「硬體壞掉」（那是猝睡症的故事，見 ch10）。 | 慢性失眠通常是學習來的——這意味它也可以被「反學習」。CBT-I 的效果正是利用了這個可塑性。 |
| **「失眠一定要靠藥物，CBT-I 只是心理安慰」** | ACP 2016 指引的強推薦有隨機對照試驗基礎，CBT-I 的緩解率在多項後設分析中顯著優於安眠藥（長期）。它針對的是真實的條件化與過度激發機制，不是安慰劑效應。 | CBT-I 是有機制基礎的行為介入，不是「想開點就好」。但它需要時間與一致性，不如藥物即刻生效，這是真實的取捨。 |
| **「我晚上補眠白天睡一下，總比一直失眠好」** | 白天補眠降低夜晚的睡眠驅力（Process S），讓夜晚入睡更難，進一步鞏固失眠迴圈。這是最常見的維持行為之一。 | 治療慢性失眠期間，通常建議限制日間補眠（特別是長時間補眠），以維持夜間 S 的強度。（⚠️ 這是機制說明，執行細節請遵醫囑。） |

## 紙上推演

### 題目一：把白話翻成機制鏈  ★★ [20 分鐘]

**情境**：一個朋友說：「奇怪，我一躺到床上就清醒腦子開始轉，但在客廳沙發上看電視卻很容易打瞌睡。去臥室反而睡不著。」

**任務**：把這個白話現象翻成機制語言，說清楚：
1. 這是什麼學習機制在作用？
2. 「床」和「沙發」在這個人身上代表什麼不同的條件化狀態？
3. 如果你用 CBT-I 的「刺激控制」邏輯，第一步你會建議（從機制角度）做什麼？

（⚠️ 提醒：這是紙上推演，不是給你的治療建議；也不是在叫讀者自行試驗治療失眠——有失眠請就醫。）

### 推演解答

**機制**：這是古典制約（classical conditioning）的典型展示。床這個中性刺激（CS，條件刺激）因為多次與清醒/焦慮體驗配對（US，非條件刺激），學習出「清醒激發」的條件化反應（CR）。沙發則因為不常與「嘗試入睡/焦慮」掛鉤，保持中性——在放鬆的脈絡（電視、低要求）下，高漲的 S（睡眠壓力）能夠自然發揮，所以睡意出現。

**不對稱的條件化**：
- `key=床` → 因為多次失眠經驗，value 已被寫成 `清醒/警覺/焦慮`
- `key=沙發` → 從未進入「嘗試睡著/監控睡眠」的情境，value 保持 `中性放鬆`

**刺激控制的第一步**（從機制角度）：讓床重新成為睡眠的獨有線索，辦法是「當你在床上且無法入睡超過一段時間，起床到沙發上，直到睡意出現再回床」。這樣做的效果是：讓未來「在床」的時刻，幾乎都對應睡眠——逐漸把 `key=床` 的 value 覆寫回 `睡眠`。同時，不讓「床上清醒焦慮」的配對繼續強化。

常見錯路：「在沙發上睡著就直接睡沙發好了」——這修補了當晚，但沒有打斷「床＝清醒」的條件化，失眠繼續。你需要的不是找別的地方睡，而是重建床的正確連結。

### 題目二：找壞論證的破綻  ★ [10 分鐘]

**論證**：「安眠藥讓我睡著了，所以它治好了我的失眠——我只要在睡不著的時候吃一顆就好了，不需要做什麼複雜的 CBT-I。」

**任務**：指出這個論證的問題：「症狀消失」等於「根因被解決」嗎？從機制層面說明為什麼不是。

### 推演解答

**問題點一：混淆「症狀」與「病因」**

安眠藥解決的是「今晚入睡」這個現象（症狀），沒有碰觸「床＝焦慮」的條件化連結（根因）。藥物的 GABA 增強效果壓住了神經激發，讓你進入睡眠狀態——但等藥效退去，條件化還在，明天不吃藥的夜晚，問題回來。

這是工程裡最經典的「先讓 production 稳住」vs「把根因修掉」的對比：熱修補快，但下次觸發一樣的條件，問題再現。

**問題點二：停藥的反彈性失眠**

長期使用安眠藥後，大腦對 GABA 的抑制性效果產生了習慣性適應（tolerance）。停藥時，GABA 抑制解除，神經系統在沒有補償之下呈現「反彈性激發」——反彈性失眠往往比服藥前更嚴重。「只要睡不著就吃一顆」的策略可能通往一條愈來愈難退出的路徑。

**問題點三：睡眠結構的問題**

入睡不等於正常睡眠。苯二氮平類壓低 N3 慢波睡眠，Z 藥的長期使用者也出現 N3 降低。藥物誘導的睡眠不一定讓大腦完成它在深睡期應該完成的維護工作（鞏固與校準）。你「睡著了」，但維護視窗的內容可能不完整。

**不是說安眠藥沒有角色**：在急性失眠的短期過渡期（例如親屬去世、手術恢復），藥物有其位置。問題是把短期工具用於慢性問題、且認為它治本了。

### 題目三：口頭重講脊椎  ★★ [15 分鐘]

**任務**：不看書，用以下幾個關鍵詞把「失眠的機制與 CBT-I 如何對症它」講給另一個後端工程師，控制在 3 分鐘內：
- 排程器（S × C）
- 條件化 / 錯誤快取關聯
- 正回饋迴圈
- 改根因 vs 熱修補

### 推演解答（參考骨架）

「大腦有個睡眠排程器，靠兩個信號合成決策：Process S（睡眠壓力，睡不夠就累積）和 Process C（生理時鐘，到點就促眠）。正常情況下兩者合力打開維護視窗，人就睡著了。

慢性失眠的問題是：排程器本身沒壞，但有一個學習來的覆蓋機制擋住了維護視窗的入口——古典制約讓『床』這個原本中性的刺激，被一次次的清醒焦慮體驗寫成了『清醒激發』的觸發器。現在只要躺床，條件化就啟動焦慮，把排程器的促眠信號蓋掉。

這形成了一個沒有自我修正能力的正回饋迴圈：睡不著→焦慮→條件化更強→更睡不著。

安眠藥是熱修補——靠壓住神經激發讓你今晚入睡，但不碰那個錯誤的快取關聯。藥退了，問題回來，長期還有反彈性失眠的風險。

CBT-I 是改根因——刺激控制直接清掉『床=焦慮』的錯誤連結，睡眠限制提高 S 讓它能蓋過殘留的激發，認知治療打斷災難化思維降低認知激發。ACP 2016 的指引說它是慢性失眠的一線，優先於藥物，原因正在這裡：CBT-I 改了根因，效果停療後持續。」
