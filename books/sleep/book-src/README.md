# 大腦每晚的維護工作

> 睡眠的科學

## 這本書是什麼

一本給工程師讀的、關於睡眠的書。它的視角很簡單：**睡眠是大腦每晚的維護視窗**——就像你的服務在低峰時段跑 GC、壓實日誌、清碎片、重整索引。白天你一直在累積技術債（代謝廢物、待鞏固的記憶、被學習推高的突觸），晚上這個視窗打開，系統做三件維護工作：**清除、鞏固、校準**。

這不是一本「睡好覺的 N 個技巧」書。它要給你的是**機制**：為什麼身體知道現在該睡、深睡和 REM 各在修什麼、以及——很重要地——**哪些被科普講得斬釘截鐵的事，其實還在激烈爭論中**。

## 脊椎：三件維護工作 ＋ 一個排程器

- **三件維護工作**：清除（代謝廢物）／鞏固（記憶）／校準（突觸與情緒）。
- **排程器**：雙歷程模型——睡眠壓力（Process S）× 生理時鐘（Process C）——決定維護窗什麼時候開、哪一段做什麼。

每一章你都會把當章現象歸位：**這是哪一件維護工作？由排程器的哪個歷程驅動？** 連反例（猝睡症＝排程器的促醒看門狗壞掉）也要說清楚它破壞了脊椎的哪一塊。

其中「清除」這隻腳（膠淋巴系統）目前**證據最不確定**：一個被一千篇文章引用為「已證實」的機制，2024 年被另一組有完整方法論的研究者用相反數據正面挑戰，至今無共識。這一章（ch04）是全書零信任主題的招牌示範。

## 怎麼讀

- 章節骨架固定：從工程直覺出發 → 拆機制 → 拆迷思（直覺的陷阱）→ 紙上推演（思想實驗，**不需要你追蹤自己的睡眠**）→ 自我檢核 → 延伸閱讀。
- 這是**純紙上推演**的欣賞書，所有「動手做」都在腦子裡完成。
- 全書四個 Part：為什麼要睡（I）、三件維護工作的機制（II）、故障與後果——剝奪、失眠、猝睡症、輪班（III）、邊界與實踐——咖啡因、短睡、夢、睡眠衛生的證據分級（IV）。

## 一個誠實的前提

**這本書是機制說明，不是醫療建議。** 涉及失眠、CBT-I、安眠藥、睡眠障礙的章節，講的是「這套機制怎麼運作、證據到哪裡」，不是診斷或處方。真有睡眠困擾，請找專業醫療。這本書能給你的，是看懂那些建議背後在修哪一塊維護工作——以及分辨哪些流行說法被誇大了。
