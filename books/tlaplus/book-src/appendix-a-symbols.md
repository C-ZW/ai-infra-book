# 附錄 A — 符號速查表

收錄原則：只收本書各章**實際用過**的符號與關鍵字，依首次出現章排序，方便當複習索引——從頭往下掃一遍，等於把全書的記號重走一次。「首次出現章」指該符號以其本義第一次實際使用的章；若更早的章只是預告（「見 chNN」式的一句話），以括號註記。本書慣例（ch06 立約）：**內文與紙上推演用 Unicode，引用 spec 原文時保留 ASCII**，所以每個符號都有兩種寫法，兩欄都列。

## 主表：數學與 TLA+ 符號

| Unicode 寫法 | TLA+ ASCII | 怎麼唸 | 意思 | 首次出現 |

| ∈、∉ | `\in`、`\notin` | 屬於、不屬於 | 元素在不在集合裡；action guard 的常客（m ∈ queue） | ch02 |
| ⟨…⟩ | `<<...>>` | 角括號（tuple） | tuple 與狀態的寫法：⟨queue = {m1, m2}, …⟩ | ch02 |
| ↦ | `\|->` | 映到、對應到 | 函數逐格給值：[m1 ↦ 0, m2 ↦ 0] | ch02 |
| ≠ | `#` 或 `/=` | 不等於 | 兩值不相等 | ch02 |
| ∅ | `{}` | 空集合 | 沒有任何元素的集合 | ch02 |
| ∧ | `/\` | 且（and） | 兩邊都成立 | ch03 |
| ∨ | `\/` | 或（or） | 至少一邊成立 | ch03 |
| ¬ | `~` | 非（not） | 否定 | ch03 |
| ⇒ | `=>` | 蘊涵、若…則 | 前件真則後件必真；前件假則整句為真（vacuous truth） | ch03 |
| ≡ | `<=>` | 等價 | 兩邊真值永遠相同 | ch03 |
| ∀ | `\A` | 對所有 | 全稱量詞 | ch03 |
| ∃ | `\E` | 存在 | 存在量詞 | ch03 |
| ⊆ | `\subseteq` | 包含於 | S ⊆ T：S 的每個元素都在 T 裡 | ch04 |
| ∪ | `\cup`／`\union` | 聯集 | 合併兩個集合 | ch04 |
| ∩ | `\cap` | 交集 | 兩個集合的共同部分 | ch04 |
| ∖ | `\` | 差集 | 扣掉：queue ∖ {m1}（ch04 內文以 `\` 引入，ch06 起內文寫 ∖） | ch04 |

| ≜ | `==` | 定義為 | 給左邊的名字下定義；是定義、不是賦值 | ch04 |
| [S → T] | `[S -> T]` | 從 S 到 T 的函數集 | 定義域是 S、值都落在 T 的所有函數 | ch04 |
| \|S\| | `Cardinality(S)` | S 的基數 | 元素個數（ASCII 寫法見 ch12 的 Consensus spec） | ch04 |
| a..b | `..` | a 到 b | 整數區間 {a, a+1, …, b}；sequence 的定義域是 1..n | ch04 |

## TLA+ 結構關鍵字

| 關鍵字 | 意思 | 首次出現 |
|---|---|---|
| `TRUE`、`FALSE`、`BOOLEAN` | 布林常值，與收容它們的集合 BOOLEAN = {TRUE, FALSE} | ch04 |
| `VARIABLES` | 宣告狀態變數（Lamport 原文亦見單數 `VARIABLE`，ch11） | ch06 |
| `UNCHANGED` | UNCHANGED ⟨x, y⟩ ≜ x′ = x ∧ y′ = y：這些變數這一步不動 | ch06 |
| `ENABLED` | ENABLED A：站在這個狀態，A 這種步走得出去（≠ 會發生） | ch06 |
| `EXCEPT`（與 `!`） | [f EXCEPT ![c] = m]：抄表改一格的函數更新句型 | ch06 |
| `THEOREM` | 作者主張 spec 滿足某性質；寫下不等於成立，只是立字據 | ch07 |
| `WF_v(A)` | weak fairness：A 持續 enabled 就終須發生 | ch07 |
| `SF_v(A)` | strong fairness：A 反覆 enabled 就終須發生（比 WF 強） | ch07 |
| `MODULE` | 模組名，必須與檔名一致 | ch08 |
| `EXTENDS` | 引入標準模組（Naturals、FiniteSets、Sequences…），連算術都要 import | ch08 |
| `CONSTANTS` | 宣告可調參數，不寫死（單數 `CONSTANT` 見 ch11 原文；ch06 曾預告） | ch08 |
| `ASSUME` | 常數必須滿足的前提，開工前先驗（ch04 曾預告；ch16 另有 TLAPS 用法） | ch08 |
| `IF`／`THEN`／`ELSE` | 條件運算式 | ch08 |
| `LET … IN` | 區域定義：先取名再用，如 ch12 Paxos 的 Q1b／Q1bv | ch12 |
| `Len`、`Append` | Sequences 模組：序列長度、尾端追加（raft.tla 精讀） | ch13 |
| `INSTANCE … WITH x <- e`、`M!Op` | 模組實例化與參數代換；`!` 前綴避免撞名（ch06 曾預告） | ch15 |

## TLAPS 證明關鍵字

除註明者外，皆於 ch16 精讀真實 TLAPS 證明時首次出現。

| 關鍵字 | 意思 | 首次出現 |
|---|---|---|
| `QED` | 本層目標證畢、由列出的步湊齊（手寫作 Q.E.D. 自 ch05；作關鍵字寫 QED 自 ch12 的層級證明） | ch05 |
| `<1>1.` | ⟨1⟩1 的 ASCII 寫法：層級證明的步驟編號 | ch16 |
| `BY e1, e2 DEF d1` | 本步的理由欄：用事實 e1、e2，展開定義 d1；TLAPS 不自動展開定義 | ch16 |
| `OBVIOUS` | 不引額外事實，光靠當前上下文就該成立 | ch16 |
| `SUFFICES ASSUME … PROVE …` | 把本層目標改寫成「假設 P、證 Q」 | ch16 |
| `NEW` | 任取一個元素（∀ 引入）：ASSUME NEW m \in Nat | ch16 |
| `USE DEF` | 此後所有義務自動展開某定義的速記宣告 | ch16 |
| `LEMMA` | 抽出來重用的小定理 | ch16 |
| `PTL` | 時序葉步驟的後端呼叫（LS4）：如「由不變量規則」的機器版 | ch16 |

## 本書自有記號

- **⟨1⟩1 層級證明編號**（ch05 起，全書手寫證明的標準格式）：讀作「第 1 層第 1 步」。⟨2⟩ 步隸屬它上方最近的 ⟨1⟩ 步，依此類推；每層以 Q.E.D. 步收攏。讀法是可調深度的：先只讀 ⟨1⟩ 層看骨架，哪步可疑就鑽進它的下一層。到 ch16 它原樣變成 TLAPS 的證明語言（`<1>1.`）。
- **狀態 tuple 寫法**（ch02 起）：一個狀態＝所有變數的一組賦值，寫成 ⟨queue = {m1, m2}, working = idle, ledger = [m1 ↦ 0, m2 ↦ 0]⟩。手追 behavior 時逐狀態列出。
- **手追驗證記號 ✓／✗**（ch02 起）：逐 guard、逐合取打勾或打叉，「每條都驗、驗完蓋章」的紙上紀律。
- **behavior 記號**（ch07）：σ = s₀ → s₁ → s₂ → … 表示一條（可無限的）behavior；σ⁽ⁱ⁾ 是它從位置 i 起的後綴。
- **「← 你在這裡」路線圖**（ch01 起）：每個 Part 的第一章開頭放一張全書 ASCII 地圖，箭頭標出當前位置——先全局、再細節。
