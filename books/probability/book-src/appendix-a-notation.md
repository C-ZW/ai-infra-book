# 附錄 A — 記號與讀法速查表

> 本書的驗收靠口述：你要能把每個概念講給另一個工程師聽。講解時你會碰到記號——而**一個你唸不出口的記號，就是一個你還沒真正內化的概念**。這張表的用途就是讓每個出現過的數學符號都有一句「能直接唸出來的中文」，旁邊配一句白話意義，最後標出它在書裡第一次正經出場的章。
>
> 表只收錄**書中實際寫過**的記號（依各章 grep 萃取），不收沒用到的。同一個寫法在書裡有兩種意思的（最惡名昭彰的是 π 和 α），分兩列標清楚——這正是口述時最容易講錯的地方。記號一律純文字 Unicode，跟內文一致；閱讀器不渲染 LaTeX。
>
> 「首次出現章」標的是該記號**正經承擔意義**的第一章；少數記號更早有一閃而過的預告（如 λ、z），會在該列備註。

## 一、基礎機率（集合、事件、條件）

| 記號寫法 | 怎麼唸（口語講解用） | 白話意義 | 首次出現章 |
|---|---|---|---|
| `Ω` | 「大歐米伽」／「樣本空間 Omega」 | 一次隨機實驗所有可能結果的集合 | ch02 |
| `∅` | 「空集合」 | 什麼都沒發生的事件，機率為 0 的不可能事件 | ch02 |
| `Aᶜ` | 「A 的餘事件」／「A 補」 | A 沒發生；P(Aᶜ)=1−P(A)，「至少一個」幾乎都靠它算 | ch02 |
| `P(A)` | 「A 的機率」 | 事件 A 發生的機率，是落在 0 到 1 之間的一個數 | ch02 |
| `P(A∩B)` | 「A 交 B 的機率」／「A 且 B 都發生的機率」 | A 和 B 同時發生 | ch03 |
| `P(A∪B)` | 「A 聯 B 的機率」／「A 或 B 至少一個發生的機率」 | A、B 至少一個發生；容斥：P(A)+P(B)−P(A∩B) | ch03 |
| `P(A|B)` | 「在 B 發生的條件下、A 的機率」 | 把世界縮到 B 裡再看 A；=P(A∩B)/P(B) | ch04 |
| `P(A∩B)=P(A)P(B)` | 「A 交 B 等於 P(A) 乘 P(B)」 | 獨立的正式定義：知道 B 不改變 A 的機率 | ch04 |
| `≠` | 「不等於」 | 例如 P(A|B)≠P(A) 就是「不獨立」 | ch04 |
| `⇏` | 「推不出」／「不蘊涵」 | 例如「不相關 ⇏ 獨立」——左邊成立不保證右邊 | ch04 |
| `P(H|D)` | 「在資料 D 之下、假設 H 的機率」 | 貝氏要的後驗：看到證據後對假設的信念 | ch05 |
| `P(D|H)` | 「在假設 H 之下、資料 D 的機率」 | 概似：假設為真時這份資料有多容易出現 | ch05 |
| `P(H|D)=P(D|H)·P(H)/P(D)` | 「P(H given D) 等於 P(D given H) 乘 P(H) 除以 P(D)」 | 貝氏定理：把概似翻成後驗 | ch05 |

> 口述地雷：`P(A|B)` 和 `P(B|A)` 是兩回事，唸的時候 given 後面那個是「已知的條件」。把這兩個唸反，就是檢察官謬誤（ch05）。

## 二、隨機變數、期望與動差

| 記號寫法 | 怎麼唸（口語講解用） | 白話意義 | 首次出現章 |
|---|---|---|---|
| `X ~ ...` | 「X 服從……分布」 | `~` 唸「服從」，右邊是這個隨機數字的行為規格 | ch11 |
| `F(x)=P(X≤x)` | 「F of x 等於 X 小於等於 x 的機率」 | 累積分布函數 CDF；你查 P99 反查的就是它 | ch07 |
| `Σ` | 「sigma，總和符號」／「把後面的東西全加起來」 | 求和；離散期望 E[X]=Σ x·P(x) 用它 | ch07 |
| `Σ(i=1→n) xᵢ` | 「i 從 1 加到 n 的 x sub i」 | 把 x₁ 到 xₙ 全部加起來 | ch07 |
| `≤` / `≥` | 「小於等於」／「大於等於」 | 區間與尾端機率常用 | ch02 |
| `E[X]` | 「X 的期望值」／「E of X」 | 長期平均、機率加權的重心，不是「最可能值」 | ch08 |
| `E[X+Y]=E[X]+E[Y]` | 「X 加 Y 的期望等於各自期望相加」 | 期望線性性——**不需要獨立**，本書最划算的工具 | ch08 |
| `μ` | 「mu，平均／期望」 | 母體平均值 E[X]；分布中心在哪 | ch09 |
| `μ_X` | 「X 的 mu」／「X 的平均」 | 指明是哪個變數的平均（兩變數登場時用） | ch10 |
| `Var(X)` | 「X 的變異數」 | 離散程度＝偏差平方的期望 E[(X−μ)²] | ch09 |
| `σ` | 「sigma，標準差」 | 跟資料同單位的離散尺度；√Var(X)，SRE 的「幾個 sigma」 | ch09 |
| `σ²` | 「sigma 平方，變異數」 | 變異數的另一寫法（σ 的平方就是 Var） | ch09 |
| `SD(X)` | 「X 的標準差」 | 同 σ；本書 σ 與 SD(X) 互通（見下方一致性說明） | ch09 |
| `E[X²]` | 「X 平方的期望」 | 算變異數的捷徑：Var=E[X²]−μ² | ch09 |
| `X̄` | 「X bar」／「樣本平均」 | 把 n 個值相加除以 n，Sₙ/n | ch09 |
| `Sₙ` | 「S sub n」／「前 n 項的和」 | n 個隨機變數相加的總和 | ch09 |
| `σ/√n` | 「sigma 除以根號 n」 | 標準誤：平均比個體穩 √n 倍（√n 法則的招牌） | ch09 |
| `Cov(X,Y)` | 「X 與 Y 的共變異」 | 兩變數一起變動的方向與強度（帶單位） | ch10 |
| `ρ` | 「rho，相關係數」 | 洗掉單位的線性關聯強度，落在 −1 到 1 之間 | ch10 |
| `z=(x−μ)/σ` | 「z 等於 x 減 mu 再除以 sigma」 | 標準化／z 分數：離自己平均幾個標準差 | ch10 |

> 口述地雷：`E[X]` 唸「期望」不是「平均最可能出現的值」；`ρ=0` 只代表「沒有**線性**關聯」，不代表獨立（ch10）。`z` 分數在 ch10 先以公式現身回指 ch09，到 ch14–15 才成為你天天用的工具。

## 三、分布（具名分布與其參數）

| 記號寫法 | 怎麼唸（口語講解用） | 白話意義 | 首次出現章 |
|---|---|---|---|
| `X ~ Bernoulli(p)` | 「X 服從成功機率 p 的伯努利分布」 | 單次成敗（脊椎硬幣的單擲就是它） | ch11 |
| `X ~ Binomial(n, p)` | 「X 服從 n 次、單次成功機率 p 的二項分布」 | n 次獨立試驗的成功數；硬幣 n 擲 ~ Binomial(n,1/2) | ch11 |
| `C(n,k)` | 「n 取 k」／「C n k」 | 從 n 個裡選 k 個的組合數，二項 PMF 的係數 | ch03 |
| `λ` | 「lambda」 | 卜瓦松的平均發生次數；λ 同時是期望也是變異數 | ch11 |
| `e^(−λ)·λ^k/k!` | 「e 的負 lambda 次方，乘 lambda 的 k 次方，除以 k 階乘」 | 卜瓦松 PMF；稀有事件計數的普適形狀 | ch11 |
| `e` | 「歐拉數 e」（≈2.71828） | 自然常數；e^(−1)≈0.3679 全書反覆出現 | ch11 |
| `X ~ N(μ, σ²)` | 「X 服從平均 μ、變異數 σ² 的常態分布」 | 鐘形，兩個旋鈕：μ 平移、σ 縮放 | ch12 |
| `~` | 「服從」 | 連接隨機變數與它的分布規格 | ch11 |
| `α`（冪律） | 「alpha，尾巴指數」 | 冪律尾巴 P(X>x)≈x^(−α) 的陡峭度（長尾世界） | ch12 |
| `∫` | 「積分符號」／「把曲線下的面積加起來」 | 連續分布的機率＝密度的積分（離散用 Σ、連續用 ∫） | ch07 |
| `∫[a,b] f(x) dx` | 「f of x 從 a 到 b 的積分」 | a 到 b 之間、密度曲線下的面積＝該區間機率 | ch12 |
| `N(0,1)` | 「標準常態」／「平均 0、變異數 1 的常態」 | 標準化後的鐘形，所有常態的共同尺 | ch14 |
| `z` | 「z 分數」 | 同一把標準常態尺上的座標，配 1.96、68-95-99.7 | ch14 |
| `1.96` | 「一點九六」 | 95% 雙尾的臨界值（±1.96σ 蓋住 95%） | ch15 |
| `√(2π)` | 「根號 2 pi」 | 常態 PDF 的歸一化常數（π 是圓周率，見下方歧義） | ch14 |

> 口述地雷：連續分布**單點機率為 0**，密度 f(x) 不是機率、是「機率的濃度」（ch12）；要積分才變機率。`1.96` 在 ch05 曾以「1.96%」這個無關的算術結果一閃而過，作為**臨界值**的意義是 ch15 才上場。

## 四、統計推論（母體、樣本、估計、檢定）

| 記號寫法 | 怎麼唸（口語講解用） | 白話意義 | 首次出現章 |
|---|---|---|---|
| `p̂` | 「p-hat」／「p 帽」 | 樣本比例（看得到、隨機）；用來反猜母體 p | ch09 |
| `p`（母體比例） | 「母體比例 p」 | 母體裡真正的比例（固定常數、未知、看不到） | ch16 |
| `Var(p̂)` / `SD(p̂)` | 「p-hat 的變異數／標準差」 | 樣本比例的抖動大小，SD(p̂)=1/(2√n)（公正硬幣） | ch09 |
| `x̄` | 「x bar」／「樣本平均」 | 從一筆樣本算出的平均，點估計 | ch17 |
| `估計 ± 1.96·標準誤` | 「估計值正負 1.96 乘標準誤」 | 95% 信賴區間的構造（民調 ±3% 由它來） | ch17 |
| `H₀` | 「H zero」／「H naught」／「虛無假設」 | 「沒有效果／硬幣公正」的對照假設，被拿來反證 | ch18 |
| `α`（顯著水準） | 「顯著水準 alpha」 | 事先畫的拒絕門檻（慣例 0.05），不是冪律那個 α | ch18 |
| `p`（p 值） | 「p 值」 | P(資料至少這麼極端 \| H₀ 為真)；**不是** P(H₀ 為真) | ch18 |
| `z=(60−50)/5=2.0` | 「z 等於 60 減 50 再除以 5，等於 2」 | 硬幣檢定的標準化統計量（60/100 正面） | ch18 |

> 口述地雷：信賴區間的正確讀法是「這個**方法**長期有 95% 的區間蓋住真值」，不是「真值有 95% 機率落在這個區間」（真值不是隨機的，ch17）。p 值唸的時候務必把 given H₀ 唸出來——漏掉它就是把 p 值誤讀成「結果出於偶然的機率」（ch18）。

## 五、過程、極限與比較

| 記號寫法 | 怎麼唸（口語講解用） | 白話意義 | 首次出現章 |
|---|---|---|---|
| `lim(n→∞)` | 「n 趨近無窮大時的極限」 | 擲無窮多次的理想化，頻率詮釋的承諾 | ch01 |
| `→` | 「趨近」／「收斂到」 | 比例 → 0.5、標準化和 → N(0,1)（也用作「映到」） | ch01 |
| `∞` | 「無窮大」 | 聖彼得堡期望＝∞、漫步期望回返時間＝∞ | ch02 |
| `≈` | 「約等於」 | 近似值，如 0.5073≈、e^(−1)≈0.3679 | ch03 |
| `√` | 「根號」 | 平方根；√n 從 ch09 一路長到 ch22 的 √Δt | ch03 |
| `P(|X−μ|≥kσ) ≤ 1/k²` | 「X 離 mu 至少 k 個 sigma 的機率，小於等於 1/k 平方」 | 車比雪夫不等式，大數法則的引擎（只靠變異數） | ch13 |
| `|正−反| ~ √n` | 「正面減反面的絕對差，量級約根號 n」 | 比例收斂、但絕對差距發散（全書最反直覺的對帳） | ch13 |
| `X̄ₙ` | 「X bar sub n」／「n 個樣本的平均」 | 樣本平均；標準化後 → N(0,1)（CLT） | ch14 |
| `(N−i)/N` | 「N 減 i 再除以 N」 | 賭徒破產機率（本金 i、目標 N、公正賭局） | ch20 |
| `π`（穩態向量） | 「穩態分布 pi」／「pi 向量」 | 馬可夫鏈的長期狀態佔比，解 π·P=π | ch21 |
| `π·P=π` | 「pi 乘 P 等於 pi」 | 穩態方程式：乘完轉移矩陣還是自己（不動點） | ch21 |
| `π₀`、`π₁`、`πₙ` | 「pi zero、pi one、pi sub n」 | 第 0、1、n 步的狀態分布（迭代收斂到 π） | ch21 |
| `Δx ~ √Δt` | 「delta x 約等於根號 delta t」 | 隨機漫步取極限成布朗運動的尺度律 | ch22 |
| `~N(0, t)` | 「服從平均 0、變異數 t 的常態」 | 布朗運動在 t 時刻的位置分布 | ch22 |

## 同形記號歧義（口述前務必分清）

書裡有幾個寫法**長一樣、意思不同**，唸的時候要靠上下文選對，否則整段講解會走鐘：

| 寫法 | 意思 A | 意思 B | 怎麼分 |
|---|---|---|---|
| `π` | 圓周率（≈3.14159，常態歸一化常數 √(2π)，ch14） | 穩態分布向量（馬可夫鏈，π·P=π，ch21） | A 是純數字常數；B 後面接得上「向量／轉移矩陣／穩態」 |
| `α` | 冪律尾巴指數（P(X>x)≈x^(−α)，ch12） | 顯著水準（拒絕門檻 0.05，ch18） | A 在分布／長尾脈絡；B 在假設檢定脈絡 |
| `p` | 母體比例／單次成功機率（固定常數，ch11、ch16） | p 值（檢定的證據強度，ch18） | 看它前面是「比例／機率」還是「值」 |
| `σ` | 標準差（單一隨機變數的離散尺度，ch09） | 條件機率裡偶見的下標／無關用法 | 本書 σ 一律指標準差，無歧義 |

另外三組「同物不同寫」——刻意保留、不是錯，但講解時要知道指的是同一件事：

- **`σ` 與 `SD(X)`**：同指標準差。本書 ch09 起就明說兩者互通（「標準差 SD(X) 或 σ」），σ 在公式裡較常用、SD(X) 在強調「這是某個變數的標準差」時用。唸法都唸「標準差」。
- **`σ²` 與 `Var(X)`**：同指變異數。σ² 是緊湊寫法、Var(X) 是函數寫法。
- **`x̄`（小寫）與 `X̄`（大寫）**：`X̄` 是隨機變數「樣本平均」這個量本身（會抖、有分布，ch09、ch14）；`x̄` 是某一筆具體樣本算出來的那個數（一個固定值，ch17 點估計）。大小寫的區別＝「隨機變數 vs 它的一次實現值」，這是統計推論的核心語感。

> 唸法總原則：**每個希臘字母先唸它的英文名（mu、sigma、rho、lambda、alpha、pi），再接它在本書的角色**——「sigma，也就是標準差」。一個你能完整唸出口、又能補一句白話角色的記號，才算真的是你的。
