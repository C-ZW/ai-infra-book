## 從你已知的出發

你對大數法則的信任，深到你根本沒意識到它是個需要被證明的東西。

回想你怎麼判斷一支 API 的可靠度。你不會跑一次請求、成功了就宣布「這支 API 100% 可用」，也不會失敗一次就宣布它壞了。你會看一段時間的監控——一萬次請求裡 23 次失敗，你說「錯誤率大約 0.23%」。然後你做了一個你從沒質疑過的跳躍：你把這個**樣本的錯誤率**，當成這支 API**真實的、長期的**錯誤率來用，拿去算 SLA、拿去設告警閾值。

停在這裡。你憑什麼相信「我量到的 0.23% 會接近那個看不見的真值」？

你憑的就是大數法則。你心裡有個信念：樣本夠大，量到的頻率就會穩定下來、靠近真實機率。這個信念你用在每一個地方——

- **A/B test 要跑夠久**。你不會看一百個 user 就拍板說 B 版轉換率比 A 版高。你知道樣本太小、數字會亂跳，要等到累積夠多樣本，兩邊的轉換率才「穩」下來、差異才可信。你在等的，就是樣本平均收斂。
- **抽樣監控**。你不記錄每一筆請求的延遲，你抽 1% 算 P50/P99，然後相信這個抽樣的分位數會接近全量的真實分位數。你賭的是「樣本夠大、抽樣統計量會收斂到母體真值」。
- **負載測試找容量上限**。你跑幾輪壓測、取平均吞吐量，當成這台機器的「真實容量」。一輪可能受 GC、受鄰居噪音干擾，但多跑幾輪平均起來，雜訊會被抹掉。

這三件事的數學骨架是同一條：**樣本平均會收斂到期望值。** 這就是大數法則。你的職業生涯有一大半建立在它上面，只是你從沒把它寫成定理。

但這裡有個陷阱，會讓人賠錢、會讓人在 on-call 時做錯判斷。同樣這條「長期會穩定」的直覺，換個場景就反過來咬你：「**最近故障特別多，照理說接下來該平靜一陣子了吧？**」——這句聽起來像「跑久了會穩定」的延伸，其實是大數法則的**冒牌貨**，叫賭徒謬誤，它偷渡了一個大數法則從來沒給過的承諾。本章後半段會把這兩者一刀切開，那一頁我會請你停下來十分鐘。

先把「樣本平均會收斂」這件事，從信念變成定理。

## 弱大數法則：到底在保證什麼

先把話講精確，因為這個定理最容易被人記成一個它沒說的版本。

設你有一串**獨立同分布（independent and identically distributed, iid）**的隨機變數 X₁, X₂, X₃, …——白話就是「同一個實驗一直重複、每次互不影響」。脊椎硬幣就是最乾淨的例子：每一擲是一個伯努利隨機變數，正面記 1、反面記 0，每擲獨立、分布都一樣。設每個 Xᵢ 的期望值是 μ（對公正硬幣，μ = 1/2，這是 ch08 算過的）。

定義**樣本平均**：

```text
        X₁ + X₂ + … + Xₙ
X̄ₙ  =  ──────────────────      ← 前 n 次的平均
                n
```

對硬幣來說，X̄ₙ 就是「前 n 擲的正面比例」（因為正面記 1、反面記 0，加起來就是正面數，除以 n 就是比例）。

**弱大數法則（weak law of large numbers, WLLN）** 說的是：

> 對任意你指定的小誤差 ε（不管多小），當 n 變大，「樣本平均離期望值超過 ε」這件事的機率會趨近 0。寫成式子：
>
> ```text
> lim(n→∞) P( |X̄ₙ − μ| ≥ ε ) = 0
> ```

讀出來：**你愛多嚴格就多嚴格地畫一個誤差帶（μ ± ε），擲得夠多，樣本平均落在帶子外面的機率就被壓到要多小有多小。**

這裡有兩個字要你特別盯著看，因為它們是後面拆穿賭徒謬誤的關鍵。

第一個字是**機率**。弱大數法則沒有保證「樣本平均一定會落在帶子裡」。它保證的是「落在帶子外的**機率**趨近 0」。永遠有一絲可能你超級衰、擲了一百萬次還是偏離得很遠——只是那個機率小到可以忽略。它是個機率陳述，不是鐵口直斷。

第二個字是**平均**（或者說**比例**）。收斂的是 X̄ₙ，是除過 n 的那個東西，不是正面的**個數**、不是「正面減反面」的差。這個區別微妙到 99% 的人會滑過去，而它正是整個賭徒謬誤的命門。先把它記在心裡，後面算給你看。

歷史上第一個證明這件事的是雅各布·伯努利（Jacob Bernoulli），收在他 1713 年出版的《推測術》（Ars Conjectandi）裡——這本書他本人已於 1705 年過世，由侄子整理付印，出版時人都走了八年（照 landscape；2026-06 已查證）。這是機率論史上第一個極限定理，被視為整門學科的奠基石。伯努利本人對這個定理看得極重（坊間流傳他稱之為「黃金定理」，但 landscape 提醒這個命名可能是後人把拉丁文 autem 誤讀成 aureum 的訛傳，我們就保守地說「他自認分量極重的核心結果」）。他要回答的正是一個你天天在問的問題：**從有限次觀察，能不能可靠地估出那個看不見的真實比例？** 他證明了：能，只要你觀察得夠多。

伯努利的證明很硬。我們不走他的路，走一條一個半世紀後才出現、但簡單到優雅得驚人的捷徑——車比雪夫的那條不等式。

## 車比雪夫不等式：只靠變異數，就把「離期望遠」壓下去

這一節是本章的數學心臟。它要回答一個看起來不可能簡單回答的問題：

> 我只知道一個隨機變數的期望值 μ 和變異數 σ²（連它長什麼分布都不知道），能不能對「它離期望值很遠」這件事的機率，給一個保證上界？

直覺會說：不知道分布怎麼可能保證？分布可以長得很怪啊。但答案是能，而且那個保證乾淨到你會懷疑是不是哪裡偷吃步。這就是**車比雪夫不等式（Chebyshev's inequality）**：

```text
                    1
P( |X − μ| ≥ kσ ) ≤ ───      ← 離期望超過 k 個標準差的機率，上界是 1/k²
                    k²
```

讀出來：**任何**有期望、有有限變異數的隨機變數，它離期望值超過 k 個標準差的機率，不會超過 1/k²。不管它是硬幣、是骰子、是延遲、是收入、是什麼妖魔鬼怪分布，這個上界一律成立。

代幾個 k 進去感受一下：

```text
k = 2  →  P(|X − μ| ≥ 2σ) ≤ 1/4  = 25%     ← 離 2 個標準差以外，頂多 25%
k = 3  →  P(|X − μ| ≥ 3σ) ≤ 1/9  ≈ 11.1%
k = 5  →  P(|X − μ| ≥ 5σ) ≤ 1/25 = 4%
```

這個界很鬆——對常態分布，離 2σ 以外的真實機率只有 4.55%（ch15 的 95.45% 法則），車比雪夫卻只敢保證「頂多 25%」。但**鬆不是缺點，是賣點**：它鬆是因為它對**所有**分布都成立，它買的是普適性。一個你完全不認識的分布，也能對它下這個保證。這種「幾乎什麼都不知道、卻能保證點什麼」的力量你其實很熟——就像你不知道某服務內部怎麼實作，但能對它的 P99 給一個契約上界。車比雪夫不等式是機率世界的 SLA。

### 為什麼成立：一個你能口頭講出來的論證

我不會給你測度論的嚴格證明，但我會給你一個**每一步都能口頭說出理由**的版本，這在本書的嚴謹度標準裡叫「工程師的嚴謹」。

核心觀念只有一句話：**變異數是「偏差平方的平均」，所以遠處的偏差因為被平方，會在變異數裡佔掉一大塊預算；而變異數的總預算是固定的（就是 σ²），於是遠處不可能擠進太多機率。**

把它拆細。回憶 ch09 變異數的定義：

```text
σ² = Var(X) = E[ (X − μ)² ]      ← 偏差平方的期望（平均）
```

這是一個「平均」，它把每個可能值的「偏差平方」按機率加權加起來。現在做一個關鍵的觀察：假設有一坨機率躲在「離 μ 至少 kσ」的遠方。對這坨機率裡的每一個值，它的「偏差平方」(X−μ)² 至少是 (kσ)² = k²σ²。

那麼光是這坨遠方的機率，貢獻給變異數的量，至少是：

```text
（遠方那坨的機率） × k²σ²   ≤   整個變異數 σ²
                                  ↑
              因為變異數是所有偏差平方的加權總和，
              遠方這坨只是其中一部分，不可能超過全部
```

把不等式整理一下，兩邊除以 k²σ²：

```text
P(|X − μ| ≥ kσ)  ≤  σ² / (k²σ²)  =  1/k²
```

得證。整個論證的靈魂就一句白話：**遠處的偏差被平方放大，吃掉的變異數預算太貴，所以遠處塞不下太多機率。** 變異數是一筆固定的預算，遠方的機率想多待一點，就得付 k² 倍的平方稅，付不起，所以只能稀稀落落。

這就是為什麼我說它優雅得驚人。你只給它兩個數字（μ 和 σ），它連分布長相都不問，就能反手給你一個對所有遠方偏差的鐵保證。一個半世紀的數學家用各種精巧手法逼近大數法則，最後發現引擎可以這麼簡單。

關於它的歸屬，landscape 特別交代了一個常被寫錯的點，我照正確版講：這條不等式**不是車比雪夫一個人的**。是 Bienaymé 在 1853 年先陳述了它（沒給證明），車比雪夫（Chebyshev）在 1867 年給出更一般的證明（2026-06 已查證，兩個來源確認）。所以更公平的名字是 **Bienaymé–Chebyshev 不等式**。當年 Liouville 編期刊時，還特意把 Bienaymé 的文章排在車比雪夫前面刊出。別只寫「Chebyshev 1867」——那會抹掉先陳述它的人。順帶一提，車比雪夫的學生馬可夫（就是 ch21 那個馬可夫）後來又給了一個證明。

### 用它一槍打穿弱大數法則

現在來看這條不等式怎麼直接生出大數法則——這是它最漂亮的應用，而且只要一個你 ch09 學過的事實當燃料。

回憶 ch09 的 **√n 法則**：n 個 iid 的隨機變數，每個變異數 σ²，它們的**平均** X̄ₙ 的變異數是 σ²/n（因為獨立時變異數相加，n 個 σ² 加起來是 nσ²，再除以 n² 就是 σ²/n）。換句話說：

```text
E[X̄ₙ]   = μ           ← 樣本平均的期望，就是真值（它瞄得準）
Var(X̄ₙ) = σ²/n        ← 樣本平均的變異數，隨 n 變小（它越來越穩）
```

這兩行就是全部的關鍵。樣本平均**瞄得準**（期望就是 μ），而且**隨 n 越來越穩**（變異數 σ²/n 在縮小）。現在把車比雪夫不等式直接套在 X̄ₙ 身上——把車比雪夫裡的「X」換成「X̄ₙ」、「σ²」換成「σ²/n」、把「kσ」那個距離直接寫成你指定的誤差 ε：

```text
                      Var(X̄ₙ)      σ²/n        σ²
P( |X̄ₙ − μ| ≥ ε )  ≤  ─────────  =  ──────  =  ──────
                        ε²            ε²         n·ε²
```

盯著最右邊那個 σ²/(n·ε²)。ε 是你固定選的（不管多小都行），σ² 是個固定常數，分母裡有個 n。**當 n → ∞，這整個東西 → 0。**

```text
P( |X̄ₙ − μ| ≥ ε )  ≤  σ²/(n·ε²)  ──n→∞──►  0
```

弱大數法則，證完了。

停下來體會一下剛剛發生了什麼。我們沒對分布做任何假設（只要有限變異數），只用了兩件事：樣本平均瞄得準（期望是 μ）、隨 n 變穩（變異數是 σ²/n）。然後車比雪夫不等式把「變穩」翻譯成「跑到誤差帶外的機率被壓到 0」。**「混沌如何生出秩序」這個從 ch01 就懸著的大哉問，核心機制就藏在那個 σ²/n 的 n 裡——你每多擲一次，就替平均多買一分穩定，買到無窮多次，平均就被釘死在 μ 上了。**

這是本書第一個「秩序從混沌浮現」的完整論證，每一步你都能口頭講給另一個工程師聽。我認為這是整本書最划算的一頁——三行代數，換一個定理。

## 弱與強：依機率 vs 幾乎必然（直覺帶過）

剛剛證的叫「弱」大數法則，那一定有個「強」的。它們差在哪？這個差別很微妙，但值得你知道有這回事，免得哪天看到「強大數法則」一臉茫然。

弱版說的是：對每一個固定的 n，「此刻的樣本平均落在誤差帶外」的機率很小，而且隨 n 趨於 0。這叫**依機率收斂（convergence in probability）**。它管的是「某一個時間點的快照」——在第 n 擲這個時刻，偏離的機率多大。

強版（**強大數法則，strong law of large numbers, SLLN**）說的更狠：樣本平均這條**序列本身**，幾乎必然會收斂到 μ。這叫**幾乎必然收斂（almost sure convergence）**。它管的不是某個時間點的快照，而是**整條軌跡的命運**——它保證的是：你擲下去這條無窮的序列，它的樣本平均「從某一刻之後就再也不會大幅偏離」，而且這件事發生的機率是 1。

用工程的比喻抓這個差別：

```text
弱（依機率）：     對任意固定時刻 t，「此刻在誤差帶外」的機率 → 0
                   （像在問：第 t 秒這個快照，超標的機率多大？）

強（幾乎必然）：   整條時間序列「最終穩定下來、不再超標」的機率 = 1
                   （像在問：這條曲線會不會收斂？答：100% 會。）
```

差別在於：弱版允許「偶爾有一擲突然彈出去」零星發生（只要機率趨於 0），不保證軌跡會「永遠安定」；強版直接保證軌跡會安定。強版更強，蘊含弱版。

本書**不證強版**（它需要的工具超出我們的範圍，嚴格證明見延伸閱讀的測度論教材）。記住一句話就夠：**弱版說「每個時刻偏離的機率趨於 0」，強版說「整條軌跡幾乎必然會穩定」；日常你信的「擲久了一定會穩」技術上靠的是強版，但弱版已夠你建立所有工程直覺。** 兩者都指向同一結論：擲得夠多，樣本平均會回到期望值。

好。地基打好了。現在來拆全書最危險的一個誤解——而且它正是用上面這個定理偽裝出來的。

### 賭徒謬誤：硬幣連開五次正面，第六次呢？

場景。**這就是 ch01 那枚公正硬幣**，你已經很熟它了：P(正) = 1/2。現在你連續擲了五次，全是正面。正正正正正。

問：第六次，正面的機率是多少？

很多人——而且是很聰明的人——心裡會冒出一個聲音：「都連五個正面了，反面該來了吧？反面的機率應該大一點。」這個聲音就是**賭徒謬誤（gambler's fallacy）**。

正解是：**第六次，正面機率還是 1/2，反面也是 1/2。一絲一毫都沒變。**

為什麼？因為硬幣**沒有記憶**。這是 ch04 的獨立性，現在以最赤裸的形式回來找你。每一擲都是一個獨立事件，硬幣不知道、也不在乎前五次發生了什麼。它沒有一個「欠了五個反面」的帳本，沒有一個「該平衡一下了」的內建衝動。前五次正面這個資訊，對第六次的機率**完全沒有作用**，因為獨立的定義就是「條件機率等於無條件機率」——P(第六次正 | 前五次全正) = P(第六次正) = 1/2。

這個謬誤有個著名的真實案例，貴到值得記住。1913 年 8 月 18 日，蒙地卡羅賭場的一張輪盤桌，球**連續開出 26 次黑色**（2026-06 已查證）。從大概第十幾次開始，賭客瘋狂往紅色下注——他們堅信「黑這麼多次了，紅鐵定要來補了」。結果黑一直開、一直開，賭客一輪輪加碼想等紅來「修正」，輸掉了數百萬法郎。每一次他們都在重犯同一個錯：以為輪盤記得前面開過什麼、以為它欠他們一個紅。輪盤什麼都不欠。每一轉都是嶄新的、跟過去毫無瓜葛的 18/37 紅、18/37 黑、1/37 綠。

### 那大數法則不是說會「平衡回來」嗎？——不，它說的是稀釋

這裡是全章的命門，請你務必跟上，因為賭徒謬誤的狡猾之處，就在於它盜用了大數法則的名義。

賭徒的心理是這樣運作的：「大數法則說正面比例長期會回到 50%。我現在正面太多了，那大數法則總得想辦法把它拉回來吧？所以接下來反面該變多，來補償。」

這個推理**錯在哪**？它把大數法則理解成一個**會主動修正偏差的力量**——好像宇宙有個校正機制，看到你正面太多，就調高反面的機率來補。**沒有這種東西。** 硬幣的機率永遠是 1/2，雷打不動，大數法則沒有、也不可能去動每一次擲幣的機率（那會違反獨立性）。

那大數法則到底怎麼把比例拉回 50% 的？答案是兩個字，請刻進腦子裡：**稀釋（dilution），不是修正（correction）。**

差別是這樣的。假設你前 5 擲拿了 5 個正面、0 個反面，比例是 100%，偏離 50% 偏得離譜。大數法則**不會**讓接下來的反面變多去「還債」。它做的事情是：讓你再擲一千次、一萬次，這一千次、一萬次裡正反各半（因為機率始終是 1/2），於是最前面那 5 個多出來的正面，被淹沒在後面這一大池子大致各半的結果裡，它對**比例**的影響被稀釋到趨近於 0。

打個你熟的比方。一桶水，一開始倒了 5 克鹽進去（這 5 克鹽就是那 5 個「多餘」的正面）。怎麼讓它的含鹽濃度降下來？賭徒謬誤的想法是「把鹽撈出來」（修正——讓未來反面多到把正面抵銷掉）。但大數法則的做法是**一直加清水**（稀釋——讓未來大量各半的擲幣不斷加進來）。那 5 克鹽從頭到尾都在桶裡、一克沒少，但加了一萬公升水之後，濃度趨近於 0。**偏差沒有被消除，它被淹沒了。**

這就是為什麼「比例收斂回 0.5」和「硬幣沒有記憶」可以同時成立、毫不矛盾。前者講的是分母 n 變超大之後，固定的那點偏差被攤薄；後者講的是分子的每一項永遠獨立、永遠 50/50。大數法則作用在分母上（靠 n 變大稀釋），不作用在分子上（不去碰每一擲的機率）。賭徒謬誤的全部錯誤，就是以為它作用在分子上。

### 致命一擊：「正面減反面」的絕對差，期望其實在變大

如果上面還沒讓你脊背發涼，這一段會。它是同一組數據的另一種讀法，而且它跟比例收斂**同時為真**——這是本書最重要的一次「同一組數據兩種讀法」對帳。

我們追蹤兩個量。一個是**正面比例**（正面數 / n），大數法則說它收斂到 0.5。另一個是**正面數減反面數的絕對差** |正面 − 反面|，這量的是「偏離完美 50/50 的絕對距離」。

賭徒的直覺會猜：既然比例收斂到 0.5，那「正面減反面」應該也越來越接近 0 吧？**錯得徹底。** 真相是：

> 隨著 n 變大，「正面比例」收斂到 0.5，但「正面減反面」的典型絕對差**反而越來越大，大約以 √n 的速度成長**。

兩個量往**相反方向**跑。比例收斂、差距發散。同一組擲幣，同時發生。

怎麼來的？ch11 算過，n 擲硬幣的正面數服從 Binomial(n, 1/2)，它的標準差是 √n / 2（ch09、ch11 的基準）。而「正面 − 反面」= 正面數 − (n − 正面數) = 2×正面數 − n，它是正面數的兩倍再平移，所以它的標準差是正面數標準差的兩倍：2 × (√n/2) = √n。所以「正面減反面」的典型波動量級就是 **√n**——n 越大，這個絕對差的典型大小**越大**，不是越小。

```text
n        正面比例會靠近      |正面−反面| 典型量級 ~√n
──────────────────────────────────────────────────
100      0.5（±0.05 左右）   ~10
10,000   0.5（±0.005 左右）  ~100
1,000,000 0.5（±0.0005）     ~1000
```

看懂這張表，你就懂了整個謬誤。**比例的偏差在縮小（因為分母 n 衝得比分子的偏差快），但絕對的偏差在擴大（√n 是會長大的）。** 賭徒謬誤的人腦補了一個「絕對差會被拉回 0」的世界，但真實世界裡絕對差是發散的——硬幣連開五次正面之後，那「多出來的 5 個正面」不但不會被未來的反面補掉，從絕對數量看，未來典型的不平衡量還會比 5 更大。比例之所以還能收斂，純粹是因為分母漲得更快，把它稀釋掉了。

我把這句話單獨拎出來，因為它是本章你最該能轉述的一句：

> **驚嘆點，請務必能轉述**：大數法則保證的是**比例**回到 0.5，靠的是「分母 n 暴漲、把偏差稀釋掉」，**不是**「偏差被反向修正、被補回來」。事實上「正面減反面」的絕對差期望是隨 n 變**大**的（約 √n）。所以「連開五個正面、反面該來補了」這句話，把稀釋誤認成修正，把一個發散的量誤認成會回零的量——它在兩個層次上同時錯了。獨立事件沒有記憶，系統不欠你任何東西。

**熱手謬誤（hot-hand fallacy）** 是 landscape 特別用 ⚠️ 標起來、要我們別寫錯的。1985 年 Gilovich、Vallone、Tversky 的經典研究主張：籃球員「手感發燙、連續命中」是錯覺，人類只是在隨機資料裡看見不存在的型態，**結論是「熱手是謬誤」**——這被當定論講了三十多年。但 2018 年 Miller 與 Sanjurjo 在《Econometrica》指出（2026-06 已查證），1985 年的分析藏了一個**系統性選擇偏差**：在有限序列裡，「連續命中之後再投中」這個經驗頻率，**即使在純隨機（獨立）假設下也會系統性偏低**。去掉這個偏差校正後，**1985 年「熱手是假的」結論被推翻，反而冒出支持「連續命中」確實存在的顯著證據。** 所以正確的當代理解是：**熱手可能是真的，至少 1985 年的否證站不住腳。** 別把「熱手是謬誤」當鐵案——很多教科書（甚至作者本人）還停在 1985 年的版本。這給你一個美麗的對照：**賭徒謬誤是真的謬誤**（獨立事件確實沒記憶），但**「熱手是謬誤」這句話本身可能才是謬誤**（運動員的投籃可能根本不獨立）。差別全在「事件到底獨不獨立」。

## 紙上推演

純紙筆，數字手算。三題，做完你對大數法則的理解會從「聽過」變成「能講」。

### 推演題 1 — 用車比雪夫估「比例偏離 0.5 超過 ε」的上界 **[15 分鐘]** ★★

**這就是 ch01 那枚硬幣。** 擲 n = 1000 次。用車比雪夫不等式，估「正面比例偏離 0.5 超過 0.05」的機率上界——也就是 P(|X̄ₙ − 0.5| ≥ 0.05) 的上界。需要的料：單擲的變異數 σ² = 1/4（ch09 基準）。一步步算，最後說一句這個界鬆不鬆、為什麼還是有用。

#### 推演解答

用上面推導好的形式 P(|X̄ₙ − μ| ≥ ε) ≤ σ²/(n·ε²)，把數字代進去。μ = 0.5、σ² = 1/4、n = 1000、ε = 0.05：

```text
                            σ²            1/4              0.25
P(|X̄ₙ − 0.5| ≥ 0.05)  ≤  ────────  =  ────────────  =  ──────────
                          n · ε²        1000 × 0.05²      1000 × 0.0025

                                          0.25
                                     =  ────────  =  0.1
                                          2.5
```

上界是 **0.1，也就是 10%**。讀出來：擲 1000 次公正硬幣，正面比例偏離 0.5 超過 0.05（也就是落到 0.45 以下或 0.55 以上）的機率，**頂多 10%**。

交叉驗算（換個方式確認沒算錯）：分母 n·ε² = 1000 × (0.05)² = 1000 × 0.0025 = 2.5；分子 σ² = 0.25；0.25 / 2.5 = 0.1。✓ 兩次都是 0.1。

這個界**很鬆**。如果你拿 ch14、ch15 的常態近似去算真實機率，會發現它遠小於 10%（正面數的標準差是 √1000/2 ≈ 15.8，偏離 50 達 0.05×1000 = 50 個，相當於 50/15.8 ≈ 3.16 個標準差，常態下這機率約 0.16%）。車比雪夫的 10% 比真實的 0.16% 大了六十幾倍。但它鬆得有道理——它沒用到「硬幣的正面數近似常態」這個額外知識，它只用了變異數，所以它對**任何**有限變異數的分布都成立。它給你的是一個**保證下限的安心**：就算你對分布一無所知，你也能拍胸脯說「偏離這麼多的機率不會超過 10%」。在工程上這種「最壞情況的保證」常常比「精確值」更值錢。

### 推演題 2 — 賭徒謬誤錯在哪？指出它偷渡了什麼 **[15 分鐘]** ★★

一個人說：「這枚公正硬幣連開了五次正面。根據大數法則，比例長期要回到 50%，所以第六次反面的機率比正面高。」請精確指出這段推理的破綻：(a) 第六次反面機率到底多少、為什麼；(b) 他把大數法則的「稀釋」誤解成了什麼；(c) 他若改追蹤「正面減反面」的絕對差，會發現什麼跟他的直覺相反的事。

#### 推演解答

**(a) 第六次反面機率就是 1/2，正面也是 1/2，沒有變。** 因為每一擲獨立，硬幣沒有記憶——前五次的結果對第六次的機率毫無作用。獨立的定義就是「條件機率＝無條件機率」：P(第六次反 | 前五次全正) = P(第六次反) = 1/2。他假設了硬幣「記得」前面開過正面、並會「想辦法補反面」，這個記憶不存在。

**(b) 他把「稀釋」誤解成了「修正」。** 大數法則讓比例回到 0.5，靠的是「再擲一大堆、各半的結果把前面那 5 個多餘正面**淹沒/稀釋**掉」（分母 n 暴漲），**不是**「未來反面變多、把正面**抵銷/修正**掉」（調高反面的機率）。他以為大數法則會去動「下一擲」的機率——但大數法則作用在分母 n 上，從不碰每一擲的機率（碰了就違反獨立性了）。那 5 個多餘的正面從頭到尾都在，只是被稀釋到對比例沒影響。鹽沒被撈出去，是水加多了。

**(c) 他會發現絕對差不但不回 0，反而變大。** 「正面 − 反面」的典型絕對差約 √n，隨 n 增長而**擴大**（n=100 時 ~10、n=10⁴ 時 ~100）。他的直覺以為「比例回 0.5」意味著「正反個數會拉平」，但真相是比例收斂、絕對差發散，兩個量往相反方向跑。所以連他「該補回來」的那個量——絕對差——本身都不是會回零的東西。他在兩個層次上同時錯了：機率不會變（分子），絕對差會變大（不會回零）。

常見錯路：有人會退一步說「好啦機率沒變，但長期『大概』會平衡吧」。要小心這個「大概」——它對**比例**成立（比例會穩到 0.5），對**絕對差**不成立（絕對差發散）。把這兩個分清楚，才算真懂。

### 推演題 3 — 口頭題：向喊「該開大了」的人講清楚 LLN 真正保證什麼 **[15 分鐘]** ★★★

賭場裡，輪盤連開好幾把小，旁邊有人扯著嗓子喊「憋這麼久了，該開大了，快壓大！」請你用不超過兩分鐘、不寫一條公式，把大數法則**真正**保證的東西講給他聽，並讓他明白他喊的那句話錯在哪。講完請自評：你有沒有不小心也用到「平衡」「補回來」這類會把對方帶溝裡的詞。

#### 推演解答

一個能過關的口頭版本（你用自己的話講，抓住骨架就好）：

「輪盤沒有記憶。它不知道剛剛開了幾把小，也不在乎——下一把開大開小的機率，跟第一把一模一樣，跟前面那串小**完全無關**。你說『憋久了該開大』，等於假設輪盤心裡有個帳本、欠了你幾個大、會想辦法還——它沒有這個帳本。

那大數法則不是說長期大小會各半嗎？是。但它兌現這個承諾的方式，不是『讓接下來的大變多來補』，而是『再轉超級多把、各半的結果把前面這串小**淹沒**掉』。前面這串小一把都不會被抵銷，它們會一直在那；只是你轉了上萬把之後，它們佔的比例被攤薄到看不見。**是稀釋，不是補償。**

而且更顛覆的是：如果你去數『大的次數減小的次數』那個絕對差，它隨著轉的把數**越來越大**，不是越來越小。所以連你想等的那個『拉平』都不會發生。獨立的東西沒有『該不該』，輪盤不欠你。你壓大壓小，期望值都一樣是賠的——這才是賭場真正穩賺的地方。」

**自評重點**：回頭檢查你有沒有講出「補回來」「平衡一下」「該輪到」這種詞。只要冒出來，你就在用對方的錯誤框架，反而強化了他的謬誤。正確的講法核心永遠是兩個詞——**獨立（沒記憶）** 和 **稀釋（不是修正）**。把這兩個講清楚，這題就過了。

## 自我檢核

口頭自答，講得出來才算過關，講不清楚就是還沒懂：

1. 弱大數法則精確在保證什麼？請特別講清楚「收斂的是**比例/平均**、而且收斂的是**機率**」——這兩個字眼為什麼非盯緊不可？
2. 車比雪夫不等式憑什麼能在「不知道分布長相」的情況下給出機率上界？用「變異數是固定預算、遠處偏差被平方放大、所以塞不下太多機率」這個骨架講一遍。
3. 車比雪夫不等式怎麼一步生出大數法則？關鍵是哪個量隨 n 變小、為什麼（提示：σ²/n）？
4. 賭徒謬誤錯在哪？「稀釋」和「修正」差在哪裡——大數法則是哪一個、為什麼這個區別是整個謬誤的命門？
5. 為什麼「正面比例收斂到 0.5」和「正面減反面的絕對差約 √n 在變大」可以**同時**為真、毫不矛盾？
6. 弱大數法則和強大數法則的差別，用「某時刻的快照」vs「整條軌跡的命運」講一遍（依機率 vs 幾乎必然）。
7. 「熱手是謬誤」這句話現在還能不能照講？2018 年發生了什麼、它跟賭徒謬誤的關鍵差別在哪（提示：事件獨不獨立）？
8. 回到 ch02：當時說「機率＝長期頻率」只是三種詮釋之一、還欠一個正當化。這一章怎麼把那張支票兌現了？

## 延伸閱讀

- **回看 ch02、ch09**（本書）——ch02 的頻率詮釋當時只能當「應該對」的直覺用，這一章給了它定理當地基，讀那一節「三張臉」現在會有完全不同的踏實感；ch09 的 √n 法則和 σ²/n 標準誤是本章車比雪夫證明的唯一燃料，沒吃透那個先回去補。
- **預告 ch14（中央極限定理）**（本書）——大數法則說樣本平均「去哪」（收斂到 μ），ch14 接著回答「在 μ 附近**怎麼抖**」，答案永遠是鐘形。本章的車比雪夫給的是「鬆」的尾巴界，ch14 會給你「緊」的精確形狀。帶著本章那張「比例收斂、差距 √n 發散」的表去讀 ch14，你會看到那個 √n 正是常態標準誤的來源。
- **預告 ch20（隨機漫步）**（本書）——把「正面減反面」這個 √n 量級的東西畫成一條隨時間走的軌跡，就是隨機漫步。本章說它的典型大小約 √n，ch20 會說它「必定回到原點、但期望回返時間無限」——同一個量的時間軸版本，更反直覺。
- Grinstead & Snell, *Introduction to Probability*, 第 8 章「Law of Large Numbers」（免費 PDF，2026-06 照 landscape 之 Statistics LibreTexts 鏡像）——用車比雪夫證弱大數法則的標準教材版，跟本章同一條路但更形式化；想看「工程師的嚴謹」升級成「數學家的嚴謹」長什麼樣，讀這一章。強大數法則的證明也在進階教材裡，本書略過的那塊在這裡補。
- Miller & Sanjurjo (2018), *Surprised by the Hot Hand Fallacy? A Truth in the Law of Small Numbers*, Econometrica（2026-06 已查證）——熱手 2018 翻案的原始論文。讀它的核心論證「在有限序列裡，連續命中後再命中的經驗頻率即使在獨立假設下也系統性偏低」——這個選擇偏差本身就是一個獨立的驚嘆點，而且它直接打臉了三十年的「定論」。
- 蒙地卡羅 1913 輪盤連開 26 次黑的軼事（Wikipedia「Gambler's fallacy」，2026-06 已查證）——史上最貴的賭徒謬誤現場教學。讀它順便看看「連 26 次黑」的機率有多小（約六千六百萬分之一）——但這不代表第 27 把紅的機率有任何改變，這正是謬誤的核心。
