# 馴服隨機：機率與統計的直覺與驚嘆

> 一本寫給工程師、純為了欣賞而讀的機率統計書。不是考試解題訓練，不是測度論教科書，也不是統計工具手冊——是想讓你親身感受：人類怎麼在最不可預測的東西裡，找到鐵一般的秩序。

## 這本書在做什麼

機率統計最迷人的地方，是它把一件聽起來矛盾的事變成了真的：**單一事件完全不可測，總體卻精確可測。** 一枚硬幣下一次是正是反，沒人說得準；但擲一萬次，正面比例會以驚人的精度逼近一半。這本書就是在追這件事——以及它一路長出來的所有驚嘆與陷阱。

讀完，你應該能用自己的話對另一個工程師講清楚這六件事：

1. 機率到底在量化什麼（無知？長期頻率？信念？三種詮釋撐著同一個數）。
2. 秩序怎麼從混沌浮現（大數法則的精確意義——不是「偏差被修正」，是「被稀釋」）。
3. 中央極限定理為什麼讓鐘形無所不在。
4. 貝氏：證據怎麼一步步改變信念。
5. 統計推論的大膽（從一千人推斷兩億人）與它的崩壞（p 值、複現危機）。
6. 人類對隨機的直覺，在哪裡、為什麼系統性地崩壞。

## 怎麼讀

- **架構圖先行**：每個 Part 的第一章都有全書地圖，先看清自己在哪，再鑽細節。
- **跟著同一枚硬幣走**：全書有一條脊椎——一枚公正硬幣，每個 Part 重新認識它一次（古典機率 → 期望與變異 → 二項 → 大數法則 → 中央極限 → 假設檢定 → 隨機漫步）。看著它越長越深，是這本書最划算的讀法。
- **動手做是紙筆推演**：每章末的「紙上推演」用紙筆算、用嘴巴講。**講得出來才算懂**；講不清楚，就是還沒懂，別急著翻下一章。沒有任何程式作業。
- **「直覺的陷阱」是招牌**：機率是人類直覺最常出錯的領域，每章都有一節專門拆解「這裡你的直覺正要騙你」。

## 書架上的鄰居

這本書會在需要時指向書架上的另外兩本，但本書自足、不必跨書也讀得懂：

- **《馴服無限：微積分的直覺與驚嘆》**——「機率即面積」「處處連續卻處處不可微的怪獸」在那裡有微積分的視角。
- **《把系統寫成定理：TLA+ 與形式化方法》**——信賴區間與 p 值背後的 ∀∃ 量詞攻防，你在那本書見過。

那麼，從第一章開始：人類為什麼怕隨機，又為什麼最後算得動它。
