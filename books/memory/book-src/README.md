# 大腦怎麼編造過去

> 記憶作為重建的科學

## 這本書是什麼

一本給工程師讀的、關於記憶的書。它的核心主張會讓你不太舒服：**你的記憶不是一段錄影回放，而是每次「想起」都當場重新建構的東西**——而重建，可以被污染、被植入、被你自己改寫，而且你完全感覺不到。

你習慣把記憶想成一塊硬碟：寫進去、放著、需要時讀出來，內容不變。這本書要說的是：記憶比較像一個**每次讀取都會順手改寫自己**的系統。鮮明不等於正確；自信不等於準確。

## 脊椎：每次 read 都觸發一次 write

整本書只繞著一句工程隱喻轉：**提取記憶（read）這個動作本身，會把記憶帶回不穩定狀態並重新寫入（write）**。每一章你都會至少停下來一次，問那句招牌問題：

> 這裡是 read，還是 read 觸發了 write？

從誤導訊息效應（一個動詞改寫了你「看到」的東西）、商場走失（整段沒發生的童年被植入）、到再鞏固（提取打開了一個分子層級的改寫窗口）——它們都是同一件事的不同化身。

**但要同時守住一條防線**：重建論說的是「可被污染、鮮明≠正確」，**不是「記憶全不可信、一切都是編造」**。你昨天吃了什麼、上週開會的大致內容——多數日常記憶大致準確。這本書要的是準確的不信任，不是虛無。

## 怎麼讀

- 章節骨架固定：從你已知的工程直覺出發 → 拆機制 → 拆迷思（直覺的陷阱）→ 紙上推演（思想實驗，**不需要你做記憶測驗**）→ 自我檢核 → 延伸閱讀。
- 這是**純紙上推演**的欣賞書，所有「動手做」都在腦子裡完成。
- 全書四個 Part：先打破「錄影帶」的幻覺（I），看重建怎麼被污染（II），拆開海馬迴與再鞏固的系統機制（III），最後談邊界與應用——目擊證詞、冤案、怎麼記得更牢（IV）。

## 一個誠實的前提

讀完這本書，你不會更不相信自己的記憶，而是會**用對的方式**不相信它：知道哪些情境（情緒、暗示、反覆敘說、他人的版本）最容易讓記憶悄悄被改寫，並在那些時刻多停十秒。
