# 附錄 A — 名詞對照速查表

> **本附錄的用途**：口頭驗收用。每一條的「白話定義」是「你能不能對另一個工程師三句話講清楚這個詞」的標準答案。表中術語均萃取自 ch01–ch16 實際出現的內容，未出現的詞不收錄。首次出現章以「最早定義或完整展開」的章為準（預告性提及不算）。

## A1　核心機制術語

| 中文 | English | 白話定義（口頭驗收用） | 首次出現章 |
|---|---|---|---|
| 編碼 | encoding | 事件發生當下大腦把它「寫進去」的過程；是有損壓縮，細節在這一步就丟了 | ch02 |
| 儲存 | storage | 編碼後記憶在腦中靜置的階段；會衰減、會被干擾，但不是唯讀 | ch02 |
| 提取 | retrieval | 回憶一段記憶的動作；不是唯讀——提取本身會讓記憶回到可改寫的狀態 | ch02 |
| 重建（性）記憶 | reconstructive / constructive memory | 記憶不是讀取原檔，而是每次提取都根據現有線索＋基模重新拼湊出來；細節常是當場現編的 | ch01 |
| 基模 | schema | 大腦對「這類情境通常長什麼樣」的模板；編碼和提取時都會用它來補洞，補完之後你分不出哪些是真記得、哪些是填空 | ch04 |
| 要旨 | gist | 編碼時保留下來的「事情大意」；細節丟了，提取時用要旨重建細節，所以細節常是現編的 | ch02 |
| 逐字痕跡 | verbatim trace | 與要旨相對——精確到字面的細節記憶；比要旨衰減更快，老了之後多半只剩要旨 | ch02 |
| 模糊痕跡理論 | Fuzzy-Trace Theory | 布雷納德（Brainerd）與蕾娜（Reyna）提出：記憶同時編碼兩條痕跡（gist＋verbatim）；verbatim 先衰減，年長後反而更靠要旨補洞，假記憶率更高 | ch02 |
| 鞏固（系統鞏固） | (systems) consolidation | 記憶從剛編碼的「海馬迴依賴」狀態，逐步搬遷為「皮質依賴」的長期儲存；需要數天到數年 | ch08 |
| 再鞏固 | reconsolidation | 已鞏固的記憶被提取後，會回到不穩定狀態，需重新合成蛋白質才能再次穩定；這段窗口記憶可被改寫——「read 觸發 write」的分子機制 | ch09 |
| 不穩定（可塑）狀態 | labile state | 提取後記憶暫時回到的可修改狀態；窗口期過後才重新穩定（re-stabilization） | ch09 |
| 邊界條件 | boundary conditions | 再鞏固不是每次提取都觸發的條件限制；例如記憶太舊、太強、或提取方式未帶來預測誤差，就可能不進入不穩定狀態 | ch09 |
| 預測誤差 | prediction error | 「你以為會發生 A，結果是 B」的落差訊號；可能是觸發再鞏固的關鍵條件之一 | ch09 |
| 消退 | extinction | 透過反覆讓制約線索出現而不給予強化，讓制約反應漸漸消失；不等於記憶被抹除——原始記憶可能還在，消退是新學了「不要反應」 | ch09 |
| 系統鞏固（標準模型） | standard model of consolidation | 記憶最終完全從海馬迴搬至皮質，遠期記憶不再需要海馬迴參與提取 | ch08 |
| 多重痕跡理論 | multiple trace theory | Nadel & Moscovitch 的挑戰：生動的情節記憶可能永遠需要海馬迴參與提取；每次提取都生成新痕跡 | ch08 |

## A2　記憶分類術語

| 中文 | English | 白話定義（口頭驗收用） | 首次出現章 |
|---|---|---|---|
| 陳述記憶 | declarative memory | 可以用語言說出來的記憶，包含事件記憶與語意記憶；HM 案例顯示它依賴海馬迴 | ch08 |
| 程序記憶 | procedural memory | 技能、習慣等「怎麼做」的記憶；不依賴海馬迴，HM 能學新技能卻不記得學過 | ch08 |
| 情節記憶 | episodic memory | 有時間地點脈絡的個人事件記憶（「那天下午我在哪」）；生動的情節記憶可能永遠需要海馬迴（多重痕跡理論） | ch08 |
| 語意記憶 | semantic memory | 去脈絡化的事實與概念知識（「台北是台灣首都」）；比情節記憶更早脫離海馬迴 | ch08 |
| 自傳式記憶 | autobiographical memory | 關於自己生命經歷的長期記憶；HSAM 的超群能力只在這個領域 | ch14 |
| 工作記憶 | working memory | 此刻意識裡短暫握著的少量資訊；秒級就掉，不是本書主題，但 HM 保留了它 | ch02 |
| 前瞻記憶 | prospective memory | 「記得未來要做某件事」的記憶（提醒自己明天帶傘）；HSAM 者在此不優於常人 | ch14 |
| 閃光燈記憶 | flashbulb memory | 重大驚奇事件（爆炸、死訊）發生時的鮮明個人情境記憶；主觀很生動但客觀不更準 | ch10 |

## A3　遺忘與干擾術語

| 中文 | English | 白話定義（口頭驗收用） | 首次出現章 |
|---|---|---|---|
| 遺忘曲線 | forgetting curve | 艾賓浩斯發現的先快後慢衰減曲線；用節省法量化，Murre & Dros 2015 單人複製成功 | ch03 |
| 節省法 | method of savings | 艾賓浩斯的測量法：「重學省下多少時間」作為記憶保留量，而非問「記得幾個」 | ch03 |
| 無意義音節 | nonsense syllables | 艾賓浩斯自創的子音-母音-子音材料（如 DAX、ZOL），用來排除意義對記憶的影響 | ch03 |
| 痕跡衰減 | trace decay | 記憶痕跡隨時間自然減弱；可以解釋遺忘，但難以解釋「取不到 ≠ 不存在」的現象 | ch03 |
| 順向干擾 | proactive interference | 舊記憶干擾新記憶的學習（學了日文之後學中文，日文詞一直跳出來） | ch02 |
| 逆向干擾 | retroactive interference | 新記憶干擾舊記憶的提取（學了新密碼，舊密碼想不起來） | ch02 |
| 提取引發遺忘 | retrieval-induced forgetting (RIF) | 提取某些記憶的同時，會壓制其他相關記憶的可及性；遺忘不只是衰減，提取本身也會造成 | ch03 |
| 適應性遺忘 | adaptive forgetting | 遺忘有功能：清除過時、衝突的舊資訊，降低干擾；遺忘不是純粹的壞事 | ch03 |
| 可得性／可取性 | available / accessible | 記憶「還在儲存裡」（available）≠「此刻能取出」（accessible）；提取失敗不代表記憶消失 | ch02 |
| 編碼特定性原則 | encoding specificity principle | 提取情境越接近編碼情境，提取越成功（Tulving & Thomson 1973）；這是「線索不對就取不到」的正式表述 | ch02 |
| 話到嘴邊 | tip-of-the-tongue (TOT) | 記得詞的語意與詞彙層（lemma）但就是說不出完整發音；illustrates available ≠ accessible | ch02 |

## A4　記憶污染與假記憶術語

| 中文 | English | 白話定義（口頭驗收用） | 首次出現章 |
|---|---|---|---|
| 誤導訊息效應 | misinformation effect | 事後接觸到含錯誤資訊的問句或說法，會污染對原始事件的記憶；Loftus & Palmer 1974 的經典示範 | ch05 |
| 假記憶 | false memory | 對從未發生的事件形成的記憶，主觀上與真記憶同樣鮮明且充滿細節 | ch06 |
| 想像膨脹 | imagination inflation | 只是在腦中想像一個未發生的事件，就會顯著提高「它發生過」的主觀自信（Garry et al. 1996） | ch06 |
| 關鍵誘餌詞 | critical lure | DRM 典範中，從未出現在詞表裡、但被語意聯想網路自動激活而被誤記成「看過」的詞（如 sleep） | ch07 |
| 回想否決 | recollection rejection | DRM 中阻止假記憶的機制：若你還記得「我記得在聽詞表時沒看到 sleep 這個字」的逐字細節，就能否決誘餌——但逐字痕跡衰減後這個否決就失效了 | ch07 |
| 激活—監控帳 | activation–monitoring framework | 解釋 DRM 假記憶的框架：語意網路自動激活誘餌詞（不可避免），來源監控失敗時就誤記為「看過」 | ch07 |
| 發展逆轉 | developmental reversal | DRM 研究的反直覺發現：成人的假記憶率反而高於兒童，因為成人的語意基模更發達，要旨主導更強 | ch07 |
| 記得／知道判斷 | remember / know judgment | 實驗測量主觀記憶品質的二分判斷：「記得」（remember）＝有清晰的情節感；「知道」（know）＝確定見過但無細節 | ch07 |
| 來源監控 | source monitoring | 判斷一段記憶「從哪裡來」的後設推斷歷程（親眼看到／聽說的／自己想像的）；記憶本身不附來源標籤，是事後推斷的 | ch13 |
| 來源歸因錯誤 | source misattribution | 來源監控失敗：把「聽說的」誤記成「親眼看到的」，把「想像的」誤記成「真實發生的」 | ch13 |
| 隱記症（無意識抄襲） | cryptomnesia | 記得一個念頭的內容、但忘了它來自別人，於是誤以為是自己的原創；來源監控失敗的一種 | ch13 |
| 既視感 | déjà vu | 把當下新經驗誤判為「曾經歷過」的熟悉感；機制有多種競爭假說（顳葉小放電、雙路徑、格式塔熟悉），未有定論 | ch13 |
| 熟悉感 | familiarity | 「感覺見過」的快速自動信號，不帶情節細節；與回想（recollection）相對，被認為是雙路徑記憶識別的其中一條 | ch13 |
| 回想 | recollection | 帶有時間地點情節細節的主動提取；與熟悉感相對，既視感時有熟悉感卻缺乏回想 | ch13 |
| 記憶從眾 | memory conformity | 共同回憶時，他人報告的錯誤會被自己的記憶併入；Roediger、Meade & Bergman 2001 | ch12 |
| 需求特徵 | demand characteristics | 受試者猜測實驗者期待什麼、並依此調整行為的現象；Bartlett 原始研究的方法學批評之一 | ch04 |

## A5　記憶污染的機制爭議

| 中文 | English | 白話定義（口頭驗收用） | 首次出現章 |
|---|---|---|---|
| 覆寫 | overwriting / trace alteration | Loftus 的原始假說：誤導訊息直接取代原始記憶痕跡 | ch05 |
| 共存（加來源混淆） | coexistence + source confusion | McCloskey & Zaragoza 1985 的反駁：原始痕跡可能還在，只是來源監控失敗讓你優先提取了錯誤版本 | ch05 |
| 反應偏誤 | response bias | 替代解釋：受試者沒有真正改變記憶，只是猜測問者的期待而改變回答——Loftus 誤導訊息研究需排除這個解釋 | ch05 |
| 恢復記憶 | recovered memory | 創傷記憶多年後在治療中被「找回」的說法；記憶戰爭的核心爭議，目前是 unsettled debate | ch06 |
| 壓抑 | repression | Freud 式的主動把創傷推出意識的機制；「壓抑作為特殊記憶機制是否存在」本身有爭議，部分研究者用「尋常遺忘」解釋 | ch12 |

## A6　神經機制術語

| 中文 | English | 白話定義（口頭驗收用） | 首次出現章 |
|---|---|---|---|
| 海馬迴 | hippocampus | 內側顳葉的關鍵腦區；新陳述記憶的形成必需它（HM 案例）；也負責空間導航，是情節記憶的系統鞏固起點 | ch08 |
| 杏仁核 | amygdala | 調節記憶鞏固強度的腦區，而非「情緒記憶的硬碟」；情緒事件時釋放壓力荷爾蒙，強化海馬迴的鞏固 | ch11 |
| 記憶調節假說 | memory modulation hypothesis | McGaugh 等人提出：杏仁核的角色是調節其他腦區的鞏固強度，不是直接儲存記憶 | ch11 |
| 模式分離 | pattern separation | 海馬迴把相似的輸入編碼成不同的表徵，避免混淆（類似資料庫不同 primary key） | ch08 |
| 模式補全 | pattern completion | 海馬迴用部分線索補全完整記憶（類似根據 partial key 查到整筆記錄） | ch08 |
| 海馬迴索引理論 | hippocampal indexing theory | Teyler & DiScenna 提出：海馬迴不存記憶本身，而是存一張指向皮質各分片的索引；鞏固後皮質直連、索引可退場（但多重痕跡理論說情節記憶的索引可能永久保留） | ch08 |
| 茴香黴素 | anisomycin | 蛋白質合成抑制劑；Nader et al. 2000 大鼠再鞏固實驗使用的藥物（注入杏仁核），可破壞再鞏固——不是 propranolol | ch09 |
| 順行性失憶 | anterograde amnesia | 無法形成新的陳述記憶；HM 手術後的主要症狀 | ch08 |
| 逆行性失憶 | retrograde amnesia | 手術前數年的記憶受損；HM 的次要症狀，童年記憶相對保留 | ch08 |
| 解離 | dissociation | 一種記憶壞了、另一種好好的；HM 的程序記憶保留而陳述記憶全毀，這個解離證明記憶是多個可分離系統 | ch08 |
| 鏡像描繪 | mirror drawing | 米爾納（Milner）設計的任務：看著鏡子裡的星形描邊；HM 每次都更好卻從不記得練過，是程序記憶保留的鐵證 | ch08 |
| 消退 | extinction | 反覆暴露制約線索但不給強化，恐懼反應逐漸消失；不等於再鞏固（消退是「學了新的抑制」而非覆寫原恐懼） | ch09 |
| 自發恢復 | spontaneous recovery | 消退後過一段時間，恐懼反應再度出現；說明原始記憶並未消失，消退只是暫時壓制 | ch09 |
| 去抑制 | renewal | 換了情境後消退的反應重新出現；進一步說明消退是情境特定的，原始恐懼還在 | ch09 |
| 武器聚焦效應 | weapon focus effect | 目擊者注意力被武器吸引，導致對武器的記憶增強但對周邊（如犯罪者臉孔）的記憶更差；Fawcett 2013 meta-analysis 確認但指出現實外推有限 | ch11 |
| 倒 U 形曲線 | inverted-U curve | 情緒強度對記憶的影響呈倒 U：適度情緒增強編碼，過度壓力（皮質醇過高）反而損害提取 | ch11 |

## A7　Bartlett 的重建扭曲型態

| 中文 | English | 白話定義（口頭驗收用） | 首次出現章 |
|---|---|---|---|
| 同化 | assimilation | 把陌生細節改寫成符合自己文化基模的版本（《鬼魂的戰爭》裡獨木舟變成船）；write 的一種 | ch04 |
| 削平 | leveling | 刪去難以理解或跟基模不符的細節，讓故事更流暢簡潔 | ch04 |
| 銳化 | sharpening | 放大並保留那些讓故事「更合理」的特定細節，讓它更突出 | ch04 |
| 連續再生 | serial reproduction | Bartlett 的一種方法：一人傳一人（像傳話遊戲），研究錯誤如何在社會傳播中累積 | ch04 |
| 重複再生 | repeated reproduction | Bartlett 的另一種方法：同一人在不同時間點多次回憶同一故事；War of the Ghosts 用的是這種 | ch04 |

## A8　閃光燈記憶與情緒記憶術語

| 中文 | English | 白話定義（口頭驗收用） | 首次出現章 |
|---|---|---|---|
| 一致性 | consistency | 閃光燈記憶研究的量測指標：現在的說法與過去記錄的說法相符程度；Hirst 9/11 追蹤發現自信高但一致性快速下降 | ch10 |
| 主觀生動度 | vividness | 記憶感覺清晰、有畫面的程度；與客觀準確度脫鉤——感覺很生動並不代表更準確 | ch10 |
| 校準 | calibration | 信心與準確率的對應關係；「說 90% 確定的案例，實際有幾成是對的」——閃光燈記憶的自信校準很差 | ch10 |

## A9　法庭與應用術語

| 中文 | English | 白話定義（口頭驗收用） | 首次出現章 |
|---|---|---|---|
| 目擊者誤認 | eyewitness misidentification | 目擊者在指認時錯認無辜者為犯罪者；美國清白專案統計約 69% 的冤案涉及此問題 | ch12 |
| 跨種族辨識劣勢 | other-race effect | 辨認與自己種族不同的臉孔時，準確率顯著下降的穩固現象 | ch12 |
| 盲化主持 | double-blind lineup | 主持指認的警察也不知道哪個是嫌疑人；防止無意識提示污染目擊者的前置程序 | ch12 |
| 暗樁 | confederate | 實驗中假扮成另一個受試者（或目擊者）的實驗同謀；Wells & Bradfield 1998 用暗樁對目擊者說「答對了」以研究事後確認對自信的影響 | ch12 |
| 記憶戰爭 | Memory Wars | 1980–90 年代「壓抑／恢復記憶」vs「假記憶可被植入」之間的學術與法庭論戰；目前仍是 unsettled debate | ch12 |
| 來源監控框架 | source monitoring framework | Johnson, Hashtroudi & Lindsay 1993 提出：記憶不附帶來源標籤，來源是事後根據記憶的特徵（知覺細節、時空脈絡、認知運作痕跡）推斷的，因此可能推錯 | ch13 |

## A10　記憶技術與穩固正面結論

| 中文 | English | 白話定義（口頭驗收用） | 首次出現章 |
|---|---|---|---|
| 間隔效應 | spacing effect | 分散練習勝過集中練習（臨時抱佛腳）；Ebbinghaus 已觀察到，是複製度最高的記憶研究發現之一 | ch15 |
| 測試效應（提取練習） | testing effect / retrieval practice | 主動提取（考自己）比重讀更能鞏固長期記憶；因為每次提取就是一次重建練習，鍛鍊的是重建路徑本身 | ch15 |
| 記憶宮殿（位置記憶法） | memory palace / method of loci | 把要記的東西轉換成生動影像，掛在熟悉空間路線的各個位置；之所以有效，是因為它順著大腦「重建」的特性而非抵抗它 | ch14 |
| 流暢度錯覺 | fluency illusion | 重複閱讀感覺很順暢，誤以為自己已經記住了；但流暢感是「路徑熟悉」而非「記憶穩固」的副產品 | ch14 |
| 合意的困難 | desirable difficulties | Bjork 提出：學習時製造一些提取困難（間隔、測試）反而有助於長期保留；困難感本身是鍛鍊的證明，不是失敗的信號 | ch15 |
| 提取強度／儲存強度 | retrieval strength / storage strength | Bjork 的區分：臨時抱佛腳提高提取強度（短期可及），但沒有提高儲存強度（長期保留）；間隔＋測試兩者都提高 | ch15 |
| 超憶症（高度超群自傳式記憶） | hyperthymesia / HSAM (Highly Superior Autobiographical Memory) | 能對個人生命中每一天做近乎完整回憶的極罕見現象；只在自傳式領域超群，一樣會產生 DRM 假記憶——脊椎的實驗化身 | ch14 |
